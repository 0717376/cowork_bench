# Методология секторного анализа

Финансовые показатели:
- Динамика и тренды цен
- Рыночная капитализация
- Концентрация по секторам
- Измерение волатильности

Интеграция академических исследований:
- Сопоставление выводов исследований с наблюдаемыми рыночными данными
- Выявление мест, где теория совпадает или расходится с практикой
- Использование академических моделей для прогнозирования ротации секторов (sector rotation)

Стандарты отчётности:
- Подкрепляйте все утверждения данными
- Ссылайтесь на конкретные статьи
- Формулируйте практический инвестиционный тезис
