Вы — аналитик отдела продаж. Ваша задача — проанализировать показатели продаж по регионам из корпоративного хранилища данных и подготовить презентацию для руководства.

Сначала выполните запрос к хранилищу данных ClickHouse, чтобы получить суммарную выручку и количество заказов по каждому региону продаж. Данные находятся в базе данных SALES_DW, схема PUBLIC. Соедините таблицу ORDERS с таблицей CUSTOMERS, чтобы получить выручку в разрезе регионов (колонка REGION). В хранилище представлены следующие регионы: Европа, Азиатско-Тихоокеанский регион, Северная Америка, Ближний Восток и Латинская Америка.

Затем создайте файл презентации PowerPoint с именем Regional_Sales_Review.pptx в рабочем каталоге. Презентация должна содержать не менее 5 слайдов: Слайд 1 — титульный слайд с заголовком «Regional Sales Performance Review». Слайд 2 — «Выручка по регионам» с таблицей или сводкой по выручке и количеству заказов для каждого региона. Слайд 3 — «Детальный разбор лидирующего региона», посвящённый региону с наибольшей выручкой (Европа), с описанием его показателей. Слайд 4 — «Анализ отстающего региона», посвящённый региону с наименьшей выручкой (Латинская Америка), с описанием его показателей. Слайд 5 — «Рекомендуемые действия» с не менее чем двумя стратегическими рекомендациями.

Важно: инструмент создания PowerPoint не сохраняет текущий рабочий каталог, поэтому при создании и сохранении файла .pptx укажите полный (абсолютный) путь к файлу Regional_Sales_Review.pptx внутри рабочего каталога.

После этого создайте событие в Google Calendar с названием «Regional Sales Review Meeting» через 14 дней от сегодняшней даты, длительностью один час. В описании укажите, что это ежеквартальный обзор региональных продаж.

В завершение отправьте презентацию по электронной почте на адрес sales-leadership@company.example.com с адреса reporting@company.example.com. Тема письма должна быть «Regional Sales Performance Review». В теле письма подчеркните, что Европа является лидирующим регионом, а Латинская Америка требует особого внимания.

Файл Sales_Target_2024.pdf в вашем рабочем каталоге содержит плановые показатели по регионам для справки.
