# groundtruth_workspace

Эталонные значения вычисляются проверяющим скриптом ЧЕСТНО на лету из схемы
`moex.*` (последние цены закрытия по 6 отслеживаемым тикерам) и количества
позиций из `initial_workspace/holdings.csv`. Заранее заготовленных файлов-ответов
здесь нет — это исключает «подсматривание» эталона агентом.
