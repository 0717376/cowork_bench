Ты — портфельный аналитик управляющей компании «Капитал-Инвест» (Москва). Инвестиционный комитет поручил подготовить отчёт о диверсификации портфеля российских акций, торгующихся на Московской бирже (MOEX).

В рабочей директории лежит файл `holdings.csv` с текущими позициями портфеля: колонки `symbol` (тикер MOEX с суффиксом `.ME`) и `quantity` (количество акций). Отслеживаемые тикеры (валюта — рубли):
`SBER.ME`, `GAZP.ME`, `LKOH.ME`, `TCSG.ME`, `MGNT.ME`, `MTSS.ME`.

Используй MCP `moex-finance` (тот же интерфейс, что и yahoo-finance: `get_stock_info`, `get_historical_stock_prices` и т.д.). Для каждого тикера из `holdings.csv` получи текущую (последнюю) цену закрытия и сектор (`sector`).

Выполни шаги ровно в таком порядке.

Шаг 1. Для каждой позиции вычисли `Current_Value = последняя цена закрытия × количество` (в рублях). Просуммируй стоимости всех позиций — это `Total_Portfolio_Value`. Сгруппируй стоимости по секторам и посчитай вес каждого сектора в процентах от общей стоимости портфеля.

Шаг 2. Создай документ Word `Portfolio_Diversification_Report.docx` в рабочей директории. Документ должен содержать:
- заголовок «Отчёт о диверсификации портфеля»;
- раздел «Состав портфеля» (Holdings) с таблицей: по одной строке на каждый из 6 тикеров с колонками тикер, сектор, количество, текущая цена, текущая стоимость;
- раздел «Анализ диверсификации» (Diversification) с указанием общей стоимости портфеля `Total_Portfolio_Value` и весов по секторам в процентах;
- раздел «Рекомендации» (Recommendations) — минимум два предложения с выводами о концентрации и диверсификации портфеля на основе данных.

Шаг 3. Создай презентацию PowerPoint `Portfolio_Diversification_Deck.pptx` в рабочей директории: титульный слайд с заголовком, слайд со составом портфеля (6 тикеров) и слайд с распределением по секторам и общей стоимостью портфеля.

Шаг 4. Отправь письмо с адреса `analyst@capital-invest.ru` на адрес `investment-committee@capital-invest.ru` с темой ровно `Portfolio Diversification Report`. В теле письма укажи общую стоимость портфеля и краткое резюме по диверсификации (на русском языке).
