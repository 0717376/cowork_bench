Вы готовите итоговую сводку по учебному курсу для методического семинара кафедры. Найдите в системе управления обучением (LMS) курс под названием "Applied Analytics & Algorithms" за осенний семестр 2013 года (Fall 2013). Соберите сведения о составе участников: сколько было зачислено студентов, преподавателей и ассистентов преподавателя (TA). Также соберите оценки по всем сданным заданиям этого курса.

Создайте книгу Excel с именем Course_Summary_AAA_F13.xlsx в рабочей директории с тремя листами. Первый лист должен называться "Enrollment Stats" со столбцами Student_Count, Teacher_Count и TA_Count, содержащими одну строку данных с соответствующими количествами. Второй лист должен называться "Assignment Performance" со столбцами Assignment_Name, Avg_Score, Max_Score, Min_Score и Submission_Count. Для каждого задания курса вычислите средний балл, округлённый до двух знаков после запятой, максимальный балл, минимальный балл и общее число оценённых сдач. Учитывайте только сдачи, у которых есть оценка (исключайте неоценённые). Отсортируйте задания по названию в алфавитном порядке. Третий лист должен называться "Grade Distribution" со столбцами Grade_Range, Student_Count и Percentage. Вычислите для каждого студента его средний балл по всем его оценённым сдачам в этом курсе, затем распределите студентов по диапазонам оценок: "A (90-100)" для среднего 90 и выше, "B (80-89)" для среднего от 80 до 89, "C (70-79)" для среднего от 70 до 79, "D (60-69)" для среднего от 60 до 69 и "F (<60)" для среднего ниже 60. Столбец Percentage должен показывать долю каждого диапазона оценок от общего числа студентов, округлённую до одного знака после запятой. Отсортируйте диапазоны оценок в алфавитном порядке.

Затем создайте таблицу с названием "AAA F13 Course Dashboard" на онлайн-платформе для работы с таблицами. На первом листе разместите сводку с названием курса, общим количеством студентов, общим числом заданий и средним баллом по курсу в целом (среднее значение средних баллов всех заданий, округлённое до двух знаков после запятой).

Наконец, создайте презентацию PowerPoint с именем Course_Summary_AAA_F13.pptx в рабочей директории. Первый слайд должен быть титульным и отображать полное название курса "Applied Analytics & Algorithms (Fall 2013)". Второй слайд должен представлять обзор состава участников, включая количество студентов, преподавателей и ассистентов (TA). Третий слайд должен освещать результаты по заданиям, показывая название и средний балл каждого задания. Четвёртый слайд должен представлять распределение оценок, показывая каждый диапазон оценок с количеством студентов и процентом. Пятый слайд должен содержать ключевые выводы и рекомендации, включая общее число студентов, средний балл по курсу в целом и название задания с наивысшим средним баллом.

Примечание: при работе с инструментами Excel и PowerPoint указывайте АБСОЛЮТНЫЙ путь к файлам .xlsx и .pptx внутри рабочей директории (рабочая директория указана в системном сообщении), поскольку у этих инструментов может не сохраняться текущая рабочая директория.
