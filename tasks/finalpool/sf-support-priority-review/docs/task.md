Мне нужен анализ обращений в службу поддержки с разбивкой по уровню приоритета. Пожалуйста, получите данные из базы данных нашего центра поддержки (хранилище данных ClickHouse, логическая база SUPPORT_CENTER) и проанализируйте, насколько хорошо мы укладываемся в целевые показатели SLA. Целевое время ответа по SLA составляет 4 часа для приоритета High, 8 часов для Medium и 24 часа для Low.

Создайте Excel-файл с именем Support_Priority_Report.xlsx с двумя листами. При создании файла используйте абсолютный путь к Support_Priority_Report.xlsx внутри рабочей директории. Первый лист "Priority Analysis" должен содержать столбцы Priority, Ticket_Count, Avg_Response_Hrs (округлённое до 2 знаков после запятой), SLA_Target_Hrs (целевые значения из политики выше), SLA_Met — значение "Yes", если среднее время ответа укладывается в целевой показатель SLA, и "No" в противном случае, и Avg_CSAT (средний балл удовлетворённости клиентов, округлённый до 2 знаков после запятой). Отсортируйте по столбцу Priority в алфавитном порядке.

Второй лист "Summary" со столбцами Metric и Value должен включать строки: Total_Tickets, Overall_Avg_CSAT (как средневзвешенное по количеству обращений, округлённое до 2 знаков после запятой), Priorities_Meeting_SLA (количество приоритетов, укладывающихся в SLA) и Priorities_Missing_SLA (количество приоритетов, не укладывающихся в SLA).

После создания отчёта отправьте письмо на адрес support-manager@company.com с темой "Support Priority Analysis Report", в котором кратко изложите результаты анализа и отметьте все приоритеты, которые не укладываются в целевые показатели SLA.
