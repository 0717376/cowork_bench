Вы — менеджер по складским запасам нашего интернет-магазина электроники на платформе InSales. Настало время еженедельного обзора запасов. Пожалуйста, выполните следующий рабочий процесс, чтобы подготовить отчёт о запасах, обновить общую панель мониторинга и уведомить складскую команду о товарах, которые нужно дозаказать.

Сначала выгрузите текущий каталог товаров из нашего магазина. Для каждой категории товаров соберите количество товаров, суммарный остаток на складе и среднюю розничную цену по категории. В магазине восемь категорий.

Затем обратитесь к финансовому сервису за последними ценами закрытия акций ПАО «ЛУКОЙЛ» (тикер LKOH.ME). Котировки сырьевой компании служат ориентиром для динамики стоимости сырья и комплектующих в производстве электроники. Получите данные о ценах закрытия за пять последних торговых дней, чтобы отслеживать тренд себестоимости.

Теперь создайте книгу Excel с именем Inventory_Restock_Report.xlsx в рабочей директории с тремя листами. Первый лист должен называться "Category Inventory" и содержать столбцы Category, Product_Count, Total_Stock и Avg_Price. Перечислите все восемь категорий товаров, отсортированных по названию категории в алфавитном порядке. Округлите Avg_Price до двух знаков после запятой. Второй лист должен называться "Restock Alerts" и перечислять каждый товар, текущий остаток которого менее десяти единиц. Включите столбцы Product_Name, SKU, Current_Stock, Category и Regular_Price. Для товаров, относящихся к нескольким категориям, используйте первую указанную категорию. Отсортируйте этот лист по текущему остатку по возрастанию, затем по названию товара по возрастанию. Третий лист должен называться "Cost Trend" со столбцами Date и Close_Price, перечисляющий пять последних торговых дней с ценами закрытия LKOH.ME, отсортированными от новых к старым.

После того как файл Excel готов, синхронизируйте данные листа Category Inventory с общей таблицей. Создайте новую таблицу с названием "Inventory Dashboard" с листом "Category Inventory", содержащим те же столбцы и данные, что и первый лист файла Excel.

Наконец, отправьте уведомление по электронной почте на адрес warehouse@company.com с темой "Low Stock Alert". В теле письма перечислите каждый товар, у которого менее десяти единиц на складе, включая название товара, его SKU и текущий остаток. Это даст складской команде понятный список позиций для дозаказа.
