Наша команда e-commerce-маркетинга хочет привязать продуктовые промоакции к актуальным технологическим темам из популярных видео на YouTube, чтобы запускать маркетинговые кампании вокруг контента, который наши клиенты уже смотрят.

Из базы видео получи все видео канала Fireship, опубликованные начиная с 1 января 2024 года. Из этого набора выбери 10 видео с наибольшим числом просмотров. Для каждого из 10 видео определи основную технологическую тему по заголовку видео, используя следующие правила: если заголовок содержит "DeepSeek", "AI", "OpenAI", "GPT", "Grok", "Claude" или "vibe", тема — "AI"; если заголовок содержит "Linux", тема — "Linux"; если заголовок содержит "Windows", тема — "Windows"; если заголовок содержит "JavaScript", "CSS", "TypeScript", "React", "Node" или "Deno", тема — "JavaScript/Web"; если заголовок содержит "Python", тема — "Python"; если заголовок содержит "security", "hack", "Hackers" или "encrypted", тема — "Security"; для всех остальных — "Tech/General".

Затем получи все товары из нашего интернет-магазина InSales. Для каждого из 10 видео найди товары, у которых название или описание содержит (как подстроку, без учёта регистра) любое из следующих ключевых слов: "laptop", "USB", "hub", "adapter", "TV", "monitor", "tablet", "watch", "headphone", "camera". Запиши каждое совпадение «видео — товар» с указанием ключевого слова, по которому оно сработало: если для товара подходит несколько ключевых слов, бери первое в порядке перечисления выше.

Создай в рабочей директории Excel-файл с именем Marketing_Opportunity_Report.xlsx с тремя листами.

Первый лист должен называться Video_Topics и содержать следующие столбцы в указанном порядке: Rank (1-10), Title, View_Count, Publish_Date (в формате YYYY-MM-DD), Main_Topic. Отсортируй по Rank по возрастанию.

Второй лист должен называться Product_Matches и содержать столбцы: Video_Title, Product_ID, Product_Name, Product_Price, Match_Keyword. Включи все найденные совпадения; Product_Name указывай полностью, как он возвращается магазином. Отсортируй строки по Rank видео по возрастанию, а внутри одного видео — по Product_ID по возрастанию. Если для видео не найдено ни одного совпадения с товаром, не включай это видео в данный лист.

Третий лист должен называться Summary и содержать ровно три строки: в первой строке метка "Total_Videos_Analyzed" и количество проанализированных видео (10); во второй строке метка "Total_Product_Matches" и общее число строк на листе Product_Matches; в третьей строке метка "Most_Common_Topic" и тема, которая встречается чаще всего среди топ-10 видео.

Отправь письмо на адрес marketing@company.com с темой "Tech Video Marketing Opportunities". В теле письма опиши топ-3 найденных совпадения «видео — товар», указав для каждого заголовок видео и название подобранного товара.
