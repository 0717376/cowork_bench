Вы — координатор корпоративной программы велнеса. Ваша задача — составить план здорового питания на 5 рабочих дней, запланировать ежедневные велнес-перерывы на обед в корпоративном календаре и сообщить о программе всем сотрудникам.

Задача состоит из трёх частей.

Часть 1. Используя базу рецептов kulinar (русская база из 50 блюд), составьте план питания на 5 дней — с понедельника по пятницу. На каждый день выберите три блюда: одно на завтрак, одно на обед и одно на ужин. Подбирайте разнообразные, питательные блюда, подходящие для корпоративной программы велнеса. Для получения рецептов используйте инструменты kulinar (например, getAllRecipes, getRecipesByCategory, recommendMeals). Для каждого блюда отметьте название рецепта, категорию и краткую заметку о пользе для здоровья.

Часть 2. Создайте файл Excel с именем Wellness_Meal_Plan.xlsx в рабочей директории с двумя листами. Первый лист должен называться "Meal Plan" и содержать следующие столбцы строго в этом порядке: Day, Meal_Type, Recipe_Name, Category, Notes. Добавьте по одной строке на каждый приём пищи (завтрак, обед, ужин) для каждого из 5 дней — минимум 15 строк. Значения в столбце Day должны быть Monday, Tuesday, Wednesday, Thursday, Friday. Значения в столбце Meal_Type должны быть Breakfast, Lunch или Dinner. В столбце Recipe_Name указывайте реальные названия блюд из базы kulinar (на русском). Столбец Notes должен содержать краткую пользу для здоровья или велнес-заметку для каждого блюда. Второй лист должен называться "Summary" и содержать два столбца: Metric и Value. Включите такие строки: Total_Recipes (минимум 15), Unique_Categories (число различных категорий блюд из листа Meal Plan), Days_Covered (должно быть 5).

Часть 3. Запланируйте 5 ежедневных событий "Healthy Lunch Break" в Google Calendar — по одному на каждый будний день (с понедельника по пятницу) текущей недели задачи. Каждое событие должно длиться 1 час и начинаться в 12:00 (полдень). Чтобы определить нужную неделю, используйте время запуска задачи (launch time). Название события должно быть в точности "Healthy Lunch Break".

После выполнения вышеперечисленного отправьте письмо на all-staff@company.example.com от wellness@company.example.com с темой "5-Day Corporate Wellness Meal Plan". Тело письма должно быть на русском языке и включать: краткое описание целей программы велнеса (снижение стресса и улучшение концентрации/фокуса), краткое резюме плана питания (с упоминанием 5 дней и 15 приёмов пищи), а также призыв к сотрудникам участвовать в ежедневных событиях календаря Healthy Lunch Break.

В исходной рабочей директории находится файл Wellness_Program_Guide.md с описанием контекста и целей корпоративной программы велнеса.
