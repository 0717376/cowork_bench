# Методология расчёта ROI обучения

## Обзор
Этот документ описывает методологию расчёта отдачи от инвестиций (ROI) во внутренние программы обучения.

## Формула ROI
ROI рассчитывается как отношение улучшения эффективности к пороговому значению ожидаемого улучшения, выраженное в процентах:

ROI (%) = (Среднее улучшение эффективности / 0.25) * 100

Где:
- Среднее улучшение эффективности = среднее значение (current_avg_rating - baseline_rating) по всем подразделениям
- 0.25 — стандартный порог улучшения, представляющий ощутимый прирост эффективности

## Расчёт улучшения эффективности
Для каждого подразделения:
1. Получите текущую среднюю оценку эффективности из HR-хранилища данных
2. Сравните её с базовым значением до обучения (из department_benchmarks.csv)
3. Вычислите дельту = current_avg - baseline

## Интерпретация
- ROI > 100%: обучение превосходит ожидания
- ROI 50-100%: обучение соответствует ожиданиям
- ROI < 50%: обучение неэффективно, рассмотрите реструктуризацию

## Правила принятия решений
- Если среднее улучшение < 0.15 балла: рекомендуйте реструктуризацию программы
- Если среднее улучшение >= 0.15 балла: рекомендуйте расширение на другие подразделения
