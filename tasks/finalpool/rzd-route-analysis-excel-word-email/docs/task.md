Мне нужно проанализировать варианты скоростных поездов «Сапсан» из Москвы (станция Москва-Ленинградская) в Санкт-Петербург на 10 марта 2026 года, чтобы помочь нашей команде выбрать оптимальный маршрут для предстоящей деловой поездки. Запроси в системе РЖД все доступные поезда, отправляющиеся из Москвы в Санкт-Петербург в эту дату, и зафиксируй ключевые детали по каждому маршруту. Приоритет команды — минимальная длительность поездки.

Создай в рабочей директории книгу Excel под названием Route_Analysis.xlsx. Добавь лист Routes — по одной строке на каждый доступный поезд — со следующими колонками: Train_Code, Destination, Depart_Time, Arrive_Time, Duration_Min (длительность поездки в целых минутах как целое число), First_Class_Price, Second_Class_Price, Available_Second_Class_Seats. Отсортируй строки по времени отправления от самого раннего к самому позднему. У «Сапсана» три класса обслуживания: Эконом, Эконом+ и Бизнес. Заполни Second_Class_Price ценой класса «Эконом» (эконом-класс), а First_Class_Price — ценой класса «Бизнес» (все цены в рублях, ₽). Колонка Available_Second_Class_Seats должна содержать фактический показатель доступности мест из системы РЖД для эконом-класса (это может быть число или текстовый индикатор, например «много»).

Добавь второй лист Summary с колонками Label и Value. Включи следующие строки в указанном порядке: Total_Routes (общее число найденных поездов), Earliest_Departure (время отправления первого поезда), Latest_Departure (время отправления последнего поезда), Cheapest_Second_Class (код поезда с наименьшей ценой эконом-класса), Fastest_Route (код поезда с минимальной длительностью в минутах), Most_Available_Seats (код поезда с наибольшей доступностью мест эконом-класса). Если по какому-то показателю несколько поездов равнозначны, выбери любой из них.

Затем подготовь аналитический отчёт Word под названием Route_Analysis_Report.docx в рабочей директории. Документ должен содержать четыре раздела с заголовками. Заголовок первого раздела — "Executive Summary"; это короткий абзац с кратким изложением числа найденных маршрутов, даты и общего наблюдения по вариантам. Заголовок второго раздела — "Route Details"; в нём по одному абзацу на каждый доступный поезд с описанием его кода, пункта назначения, расписания, длительности и цен для обоих классов. Заголовок третьего раздела — "Recommendation"; в нём нужно указать единственный лучший поезд для деловой поездки с приоритетом скорости и обосновать выбор минимальной длительностью поездки. Заголовок четвёртого раздела — "Cost Comparison"; в нём нужно сравнить цены билетов по всем маршрутам и классам обслуживания.

Наконец, отправь письмо на адрес logistics@company.com с темой "Route Analysis Москва-СПб 2026-03-10". Тело письма должно содержать ключевые итоговые цифры: число маршрутов, код рекомендованного поезда, его время отправления и прибытия, а также цены эконом-класса по всем доступным поездам.
