Наш поставщик прислал свой свежий каталог товаров в виде PDF-файла Supplier_Catalog.pdf в рабочей директории. В этом каталоге перечислены товары, которые он может нам поставлять, вместе с рекомендуемыми розничными ценами. Нужно сверить этот каталог с текущим ассортиментом нашего интернет-магазина InSales и понять, какие товары у нас уже есть, а каких не хватает.

Сначала прочитайте PDF-каталог поставщика и извлеките все названия товаров и их цены. Затем запросите данные нашего магазина InSales и получите полный список товаров, которые мы сейчас продаём, включая их названия и текущие цены.

Создайте документ Word с названием Product_Comparison.docx в рабочей директории. В начале документа должен быть заголовок «Supplier Catalog vs Store Inventory Comparison».

Добавьте раздел с заголовком «Совпадающие товары», в котором перечислите все товары из каталога поставщика, которые уже есть в нашем магазине. Для каждого совпавшего товара добавьте один абзац, где указаны: название товара, цена поставщика из каталога и текущая цена в нашем магазине. Считайте товар совпавшим, если название товара из каталога поставщика встречается как подстрока в каком-либо названии товара в магазине (сравнение без учёта регистра).

Затем добавьте раздел с заголовком «Отсутствующие товары», в котором перечислите все товары из каталога поставщика, которых нет в нашем магазине. Для каждого отсутствующего товара добавьте один абзац, где указаны название товара и рекомендуемая цена поставщика.

В конце добавьте раздел с заголовком «Итоги» с абзацем, в котором указано общее число товаров в каталоге поставщика, сколько из них найдено в магазине и сколько отсутствует.

Важно: названия товаров берите дословно из каталога и из магазина — не переводите и не изменяйте их.

При работе с инструментом Word передавайте АБСОЛЮТНЫЙ путь к файлу Product_Comparison.docx внутри рабочей директории.
