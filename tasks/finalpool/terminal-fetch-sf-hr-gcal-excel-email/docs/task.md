Вы — HR-аналитик по компенсациям, которому поручено провести комплексное исследование по бенчмаркингу зарплат. Вам нужно сравнить внутренние данные о зарплатах с внешними рыночными бенчмарками, назначить совещания по пересмотру и сообщить о результатах HR-команде.

Начните с прочтения файла Compensation_Policy.pdf в вашей рабочей директории, в котором изложена философия компании в области компенсаций и критерии пересмотра зарплат.

Загрузите внешние данные о бенчмарках зарплат по адресу http://localhost:30405/api/benchmarks.json. В этом JSON-файле содержатся отраслевые бенчмарки зарплат, сгруппированные по отделам, включая медианную зарплату, 25-й процентиль, 75-й процентиль и рыночный тренд (growing — рост, stable — стабильность или declining — снижение) для каждого отдела.

Сделайте запрос к нашему хранилищу данных ClickHouse, чтобы получить внутреннюю статистику по зарплатам для каждого отдела. Данные находятся в таблице `sf_data."HR_ANALYTICS__PUBLIC__EMPLOYEES"`. Имена таблиц и столбцов записаны латиницей в ВЕРХНЕМ регистре и заключены в двойные кавычки; ключевые столбцы — "DEPARTMENT" (название отдела), "SALARY" (зарплата), "YEARS_EXPERIENCE" (стаж в годах) и "JOB_SATISFACTION" (оценка удовлетворённости работой). Для каждого из семи отделов (Инженерия, Финансы, Кадры, Операции, НИОКР, Продажи, Поддержка) рассчитайте количество сотрудников, среднюю зарплату, округлённую до двух знаков после запятой, минимальную зарплату, максимальную зарплату, средний стаж работы в годах и среднюю оценку удовлетворённости работой.

Напишите и запустите Python-скрипт под названием compensation_analysis.py в вашей рабочей директории. Скрипт должен сравнить среднюю внутреннюю зарплату каждого отдела с внешней медианой бенчмарка. Рассчитайте разрыв в процентах: (internal_avg минус benchmark_median) делить на benchmark_median умножить на 100. Отметьте отделы, у которых разрыв превышает плюс-минус 10 процентов, как требующие пересмотра. Скрипт также должен выявить отделы, где средняя удовлетворённость работой ниже 6.5, что может указывать на недовольство, связанное с компенсацией.

Назначьте совещания по пересмотру зарплат в общем календаре на неделю с 16 по 20 марта 2026 года. Создайте по одному совещанию для каждого из семи отделов. Заголовок каждого совещания должен состоять из названия отдела, за которым следует «Совещание по пересмотру зарплат» (например, «Инженерия Совещание по пересмотру зарплат»). Назначьте их как часовые совещания, по два в день, начиная в 10:00 и 14:00, при этом последний отдел приходится на утро пятницы. Включите в описание события среднюю зарплату отдела и процент разрыва по сравнению с бенчмарком.

Создайте книгу Excel под названием Compensation_Benchmark_Report.xlsx в вашей рабочей директории с тремя листами.

Первый лист должен называться Department_Analysis и содержать столбцы Department, Employee_Count, Avg_Salary, Min_Salary, Max_Salary, Avg_Experience и Avg_Satisfaction. Включите по одной строке для каждого из семи отделов.

Второй лист должен называться Salary_Benchmark и содержать столбцы Department, Internal_Avg, Benchmark_Median, Benchmark_25th, Benchmark_75th, Gap_Pct, Market_Trend и Needs_Review. Включите по одной строке для каждого отдела. Столбец Needs_Review должен показывать «Да», если разрыв превышает плюс-минус 10 процентов, иначе «Нет».

Третий лист должен называться Review_Summary и содержать столбцы Department, Review_Date, Review_Time, Key_Issue и Recommended_Action. Включите по одной строке для каждого отдела с датой и временем назначенного совещания, основной выявленной проблемой (разрыв в зарплате или проблема удовлетворённости) и рекомендуемым действием (повысить, сохранить или реструктурировать).

Отправьте письмо на адрес hr_team@company.com с темой «Compensation Benchmarking Study Complete». Тело письма должно резюмировать общие результаты: сколько отделов требуют пересмотра зарплат, у какого отдела самый большой разрыв с бенчмарком, средний разрыв по всем отделам, а также примечание о том, что совещания по пересмотру назначены на неделю с 16 марта. Также упомяните, что подробный отчёт Excel доступен.
