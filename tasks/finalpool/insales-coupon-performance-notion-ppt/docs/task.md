Мне нужен подробный отчёт об эффективности промокодов для нашего отдела маркетинга. Пожалуйста, выгрузи данные о промокодах из нашего магазина InSales и проанализируй эффективность каждого промокода.

Создай Excel-файл с именем Coupon_Performance.xlsx с листом Coupon Analysis, в котором будут колонки Coupon_Code, Discount_Type, Discount_Amount, Times_Used, Usage_Limit (укажи N/A, если лимита нет) и Usage_Rate_Pct, которая вычисляется как Times_Used, делённое на Usage_Limit и умноженное на сто, округлённое до двух знаков после запятой — укажи N/A, если лимита использования нет. Отсортируй по Times_Used по убыванию. После Excel создай презентацию PowerPoint с именем Coupon_Analysis_Presentation.pptx. Включи титульный слайд с заголовком Coupon Performance Analysis, по одному слайду на каждый промокод с указанием кода, типа скидки и числа использований, и финальный сводный слайд с общей статистикой.

Создай в Teamly страницу с заголовком «Coupon Strategy Overview 2026 / Обзор стратегии по промокодам 2026» с ключевыми выводами анализа промокодов, включая то, какие промокоды показывают наилучшие результаты, и рекомендации. Обязательно упомяни в тексте лидирующий промокод (с наибольшим числом использований).

Отправь письмо на адрес marketing@company.com с темой Coupon Campaign Performance Report Q1 2026, содержащее краткую сводку результатов по эффективности промокодов.
