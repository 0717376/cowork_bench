Здравствуйте, мне нужна ваша помощь с важной задачей для нашего отдела кадрового комплаенса. Мы проводим ежегодный аудит справедливости оплаты труда и нуждаемся в тщательном анализе распределения зарплат по всей нашей организации. В рабочей директории есть PDF-файл Equity_Policy.pdf, в котором изложена наша политика справедливости вознаграждения, включая определение уровней опыта (experience bands), пороговые значения разрыва в оплате и коэффициенты справедливости. Пожалуйста, сначала внимательно прочитайте его, чтобы понять методологию.

Когда вы разберётесь с политикой, выгрузите данные о сотрудниках из аналитического хранилища данных ClickHouse. Нам нужно рассмотреть зарплаты в разбивке по отделам (department), уровню образования (education level) и уровню опыта (experience band — эти уровни определены в документе с политикой). Для любых необходимых статистических вычислений используйте терминал.

Создайте файл Excel с названием Compensation_Equity.xlsx с тремя листами.

Первый лист должен называться "Salary Analysis" и показывать разбивку зарплат по каждой комбинации отдела, уровня образования и уровня опыта. Включите количество сотрудников (employee count), среднюю зарплату (average salary, округлённую до 2 знаков после запятой), медианную зарплату (median salary, округлённую до 2 знаков), минимальную зарплату, максимальную зарплату и стандартное отклонение зарплаты (salary standard deviation, округлённое до 2 знаков). Отсортируйте данные сначала по отделу, затем по уровню образования, затем по уровню опыта.

Второй лист должен называться "Equity Metrics" и сравнивать оплату труда между уровнями образования внутри каждой комбинации отдела и уровня опыта. Для каждой комбинации определите, какая группа по уровню образования является наиболее высокооплачиваемой (highest paid) и какая — наименее (lowest paid), вычислите процент разрыва в оплате (pay gap percentage, округлённый до 1 знака, рассчитываемый как разница между наибольшей и наименьшей средней зарплатой, делённая на наибольшую, умноженная на 100), коэффициент справедливости (equity ratio, округлённый до 3 знаков — это средняя минимальной группы, делённая на среднюю максимальной группы) и статус справедливости (equity status) на основе пороговых значений из политики. Отсортируйте по отделу, затем по уровню опыта.

Третий лист должен называться "Summary" и содержать пары "метка-значение" в двух столбцах. Включите: Total_Employees_Analyzed, Departments_Analyzed (только количество), Concerning_Gaps_Count, Monitor_Gaps_Count, Acceptable_Gaps_Count, Overall_Equity_Score (среднее всех коэффициентов справедливости, округлённое до 2 знаков), Highest_Gap_Dept (отдел с наибольшим единичным разрывом в оплате) и Highest_Gap_Band (уровень опыта, на котором существует наибольший единичный разрыв в оплате).

После анализа в Excel напишите документ Word с названием Equity_Report.docx, который служит резюме для руководства по итогам анализа. В нём должны быть разделы, охватывающие краткое резюме для руководства (executive summary), использованную методологию (methodology), ключевые выводы по отдельным отделам (department-by-department analysis highlights), основные выводы о том, где находятся наиболее тревожные разрывы (key findings), рекомендации по устранению этих разрывов (recommendations) и заявление о соответствии требованиям (compliance statement). При сохранении документа Word передавайте инструменту АБСОЛЮТНЫЙ путь к файлу Equity_Report.docx внутри рабочей директории (инструмент Word не сохраняет текущий рабочий каталог).

Наконец, создайте форму с названием "Compensation Fairness Survey" для сбора анонимных отзывов сотрудников о справедливости оплаты. В ней должно быть пять вопросов:
1. вопрос с вариантами ответа, спрашивающий, к какому отделу относится респондент (варианты — названия отделов организации);
2. текстовый вопрос, спрашивающий, сколько лет опыта у респондента;
3. вопрос с вариантами ответа, просящий оценить удовлетворённость своим текущим вознаграждением по шкале от 1 до 5 (варианты: 1, 2, 3, 4, 5);
4. вопрос с вариантами ответа, просящий оценить воспринимаемую справедливость своей оплаты по сравнению с коллегами по шкале от 1 до 5 (варианты: 1, 2, 3, 4, 5);
5. текстовый вопрос для любых дополнительных комментариев о вознаграждении.

Также в рабочей директории есть файл guide.md, в котором методология описана более подробно — пожалуйста, ознакомьтесь и с ним.
