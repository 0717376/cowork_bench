Нашей бухгалтерии нужен налоговый сводный отчёт, построенный на основе данных о заказах нашего интернет-магазина. Пожалуйста, получите из нашего магазина InSales все заказы, у которых статус равен либо completed, либо processing. По этим заказам составьте налоговую сводку с разбивкой по штату (billing state) каждого покупателя.

Создайте в рабочей директории файл Excel с именем Tax_Summary_Report.xlsx с двумя листами. Первый лист должен называться "By State" и содержать следующие колонки: State, Order Count, Total Sales и Total Tax. Перечислите каждый штат (billing state), встречающийся в подходящих заказах, отсортированный по Total Tax по убыванию, чтобы штаты с наибольшим налогом шли первыми. Все денежные значения округляйте до двух знаков после запятой. Второй лист должен называться "Overall" и содержать две колонки с названиями Metric и Value со следующими четырьмя строками: Total Orders с количеством всех подходящих заказов, Total Sales с суммой всех итогов заказов, Total Tax с суммой всех налогов, и Effective Tax Rate с общей долей налога как процентом от общей суммы продаж, округлённым до двух знаков после запятой.

Помимо файла Excel, создайте общую онлайн-таблицу (Google Sheet) с названием "Tax Summary for Accounting", чтобы бухгалтерия могла получить доступ к данным онлайн. Эта таблица должна содержать единственный лист с названием "Summary" с теми же данными Overall — колонками Metric и Value и теми же четырьмя строками, что и на листе Overall в файле Excel.

При работе с инструментом Excel передавайте АБСОЛЮТНЫЙ путь к файлу Tax_Summary_Report.xlsx внутри рабочей директории.
