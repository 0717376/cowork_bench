# Методология анализа пробелов в навыках

Модель оценки:
1. Определить ключевые требования к навыкам для каждого подразделения
2. Оценить покрытие текущими учебными программами
3. Сопоставить эффективность сотрудников с обучением
4. Рассчитать gap score (обратная величина покрытия * эффективность)

Расчёт Gap Score:
- 1-3: Хорошее покрытие, минимальные пробелы
- 4-6: Умеренные пробелы, требуется улучшение
- 7-10: Критические пробелы, необходимы немедленные действия

Классификация приоритетов:
- Critical: Эффективность ниже средней + низкое покрытие обучением
- Important: Либо эффективность ниже средней, либо низкое покрытие
- Nice-to-have: Выше среднего, но есть пространство для улучшения
