Мне нужно подготовить прогноз продаж на Q2 2026, объединив наши исторические данные о продажах с внешними рыночными прогнозами. По адресу http://localhost:30304/api/market_projections.json доступен API рыночной аналитики, который предоставляет региональные темпы роста и данные об объёме рынка. Пожалуйста, получите эти данные.

Затем выгрузите наши текущие данные о продажах из корпоративного хранилища данных ClickHouse с разбивкой по регионам, включая количество заказов и общую выручку.

С помощью терминала создайте и запустите Python-скрипт под названием forecast_builder.py в рабочей директории, который читает оба источника данных из JSON-файлов (создайте historical_sales.json и market_outlook.json), применяет темпы роста для расчёта прогнозируемой выручки и сохраняет результат в sales_forecast.json.

Создайте файл Excel под названием Sales_Forecast_Q2_2026.xlsx с тремя листами. Первый лист Regional_Forecast должен содержать столбцы Region, Current_Orders, Current_Revenue, Growth_Rate_Pct, Forecasted_Revenue (Current_Revenue умножить на 1 плюс Growth_Rate, делённый на 100; округлить до 2 знаков) и Revenue_Increase (Forecasted минус Current; округлить до 2 знаков). Отсортируйте по столбцу Region в алфавитном порядке.

Второй лист Forecast_Summary должен содержать два столбца Metric и Value: Total_Current_Revenue, Total_Forecasted_Revenue, Total_Revenue_Increase, Avg_Growth_Rate (простое среднее всех региональных темпов роста, округлить до 1 знака), Highest_Growth_Region, Lowest_Growth_Region, Total_Orders.

Третий лист Growth_Ranking должен ранжировать регионы по темпу роста по убыванию со столбцами Rank, Region, Growth_Rate_Pct, Current_Revenue и Market_Share_Pct (текущая выручка региона, делённая на общую текущую выручку, умноженная на 100; округлить до 1 знака).

Также создайте документ Word под названием Q2_Forecast_Report.docx с заголовком "Q2 2026 Sales Forecast Report" и разделами: Обзор рынка (опишите рыночные условия), Региональный анализ (обсудите перспективы роста каждого региона) и Стратегические рекомендации (не менее 3 рекомендаций). В каждом разделе должны упоминаться конкретные числа из анализа.

Наконец, отправьте письмо на адрес sales-team@company.com с темой "Q2 2026 Sales Forecast Ready", в котором обобщите общую прогнозируемую выручку и регион с наибольшим темпом роста.

Сохраните все выходные файлы в рабочей директории. ВАЖНО: при создании файлов .xlsx и .docx через соответствующие MCP-инструменты передавайте АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории, так как эти инструменты не сохраняют текущую рабочую директорию.
