Вы — координатор по логистике интернет-магазина на платформе InSales. Ваш руководитель попросил провести аудит расходов на доставку, чтобы определить, не занижает ли магазин стоимость доставки для клиентов по сравнению с фактическими тарифами перевозчиков. Тарифные таблицы перевозчиков опубликованы онлайн по адресу http://localhost:30201/index.html, где есть ссылки на страницы как с внутренними (domestic), так и с международными (international) тарифами.

Начните с изучения документа с правилами доставки (shipping guidelines) и CSV-файла с контрактами перевозчиков (carrier contracts) в вашем рабочем каталоге. Правила доставки описывают фиксированные ставки магазина за доставку и объясняют, как определить фактическую стоимость перевозчика для сравнения. Обратите внимание на допущение о весовом диапазоне и на то, какие тарифы перевозчиков использовать для каждого способа доставки.

Получите из вашего интернет-магазина все заказы, у которых стоимость доставки больше нуля. Для каждого заказа зафиксируйте идентификатор заказа (order ID), статус заказа, итоговую сумму заказа, сумму доставки, начисленную клиенту, и использованный способ доставки, например «Стандартная доставка» или «Экспресс-доставка».

Затем получите тарифные таблицы перевозчиков с сайта. Перейдите на страницу внутренних тарифов (domestic), чтобы найти тарифы FedEx, UPS и USPS для разных весовых диапазонов и уровней сервиса. Согласно правилам, используйте весовой диапазон 5–10 lbs для всех заказов. Для заказов со способом «Стандартная доставка» используйте тариф USPS standard в качестве фактической стоимости. Для заказов со способом «Экспресс-доставка» используйте тариф FedEx express в качестве фактической стоимости. Для любых заказов с другими способами доставки используйте тариф USPS standard как значение по умолчанию.

Рассчитайте разницу между фактической стоимостью перевозчика и суммой, начисленной клиенту, для каждого заказа. Заказ считается недозаряженным (undercharged), когда фактическая стоимость превышает начисленную сумму.

Создайте Excel-файл с именем Shipping_Audit.xlsx в вашем рабочем каталоге с тремя листами. Первый лист должен называться "Rate Comparison" со столбцами Order_ID, Status, Order_Total, Shipping_Charged, Shipping_Method, Carrier, Actual_Cost, Difference и Undercharged, где Undercharged принимает значение Yes или No. Включите каждый заказ, у которого есть стоимость доставки. Второй лист должен называться "Undercharged Orders" со столбцами Order_ID, Status, Order_Total, Shipping_Charged, Actual_Cost и Difference и должен содержать только заказы, по которым доставка была недозаряжена. Третий лист должен называться "Summary" с двумя столбцами Metric и Value, содержащими Total_Orders_Analyzed, Undercharged_Count, Total_Undercharged_Amount и Avg_Undercharge_Per_Order.

Отправьте письмо на адрес logistics@company.com с темой "Shipping Cost Audit Alert", в котором кратко укажите, сколько заказов оказались недозаряженными, общую сумму недозаряда и перечислите как минимум первые десять идентификаторов недозаряженных заказов. В письме рекомендуйте пересмотреть текущую структуру фиксированных ставок на доставку.

Когда вы выполните все задачи, просто ответьте без вызова инструментов, чтобы завершить работу.
