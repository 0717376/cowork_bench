Мне нужно подготовить презентацию с обзором производительности категорий товаров для нашего магазина. Пожалуйста, выгрузи данные о товарах из магазина InSales и проанализируй производительность по категориям товаров. Для каждой категории посчитай количество товаров, среднюю цену, суммарное количество проданных единиц по всем товарам этой категории и среднюю оценку покупателей.

Создай книгу Excel с именем Category_Data.xlsx в рабочей директории, с листом под названием "Category Performance", содержащим столбцы Category, Product_Count, Avg_Price, Total_Units_Sold и Avg_Rating. Отсортируй строки по названию категории в алфавитном порядке. Важно: используй именно эти английские названия файла, листа и столбцов без изменений.

Затем создай презентацию PowerPoint с именем Category_Review.pptx в рабочей директории. Первый слайд должен быть титульным со заголовком «Обзор производительности категорий товаров». После него создай по одному слайду на каждую категорию товаров, где заголовком слайда будет название категории, а в содержимом слайда — ключевые метрики (количество товаров, средняя цена, суммарное количество проданных единиц и средняя оценка). В конце добавь итоговый слайд с заголовком «Общая сводка», на котором выдели категорию с наибольшим количеством товаров, категорию с наивысшей средней оценкой и категорию с наибольшим суммарным количеством проданных единиц.

Названия категорий бери ровно так, как они приходят из магазина InSales (на русском языке), и используй их без изменений в Excel и в презентации.

Важно про пути: при создании Excel- и PowerPoint-файлов передавай АБСОЛЮТНЫЙ путь к Category_Data.xlsx и Category_Review.pptx внутри рабочей директории, так как инструменты работы с документами не сохраняют текущую рабочую директорию.
