Привет! Команде развития бизнеса нужно провести конкурентный анализ рынка. Мы хотим понять, как наш интернет-магазин (InSales) выглядит на фоне рынка в целом по каждой товарной категории.

В рабочей директории есть документ с описанием методологии конкурентной стратегии — Competitive_Strategy.pdf. В нём объясняется, как классифицировать ценовое позиционирование, рассчитывать долю рынка и выявлять возможности для роста. Пожалуйста, сначала прочитайте его, чтобы разобраться в методологии.

У нас работает портал рыночной аналитики по адресу http://localhost:30195/api/market_data.json — там лежат свежие отраслевые бенчмарки, включая средние цены, выручку рынка и темпы роста по каждой товарной категории. Заберите эти данные.

Затем выгрузите весь каталог товаров и историю заказов из нашего интернет-магазина (InSales). Нам нужно знать, сколько товаров у нас в каждой категории, какие у нас средние цены и сколько выручки мы генерируем по каждой категории по фактическим заказам.

Когда у вас будут оба набора данных, используйте терминал, чтобы сопоставить всё между собой и рассчитать конкурентные метрики, описанные в PDF. Определите наше ценовое позиционирование по каждой категории, сравнив нашу среднюю цену со средней по рынку. Рассчитайте процент нашей доли рынка в каждой категории. Выявите, какие категории представляют возможности для роста — по высокому темпу роста рынка в сочетании с нашей низкой текущей долей.

## Важно: сопоставление категорий (английский API → русский магазин)

Категории в нашем магазине (InSales) названы по-русски, а портал рыночной аналитики (market_data.json) использует английские названия категорий. Чтобы соединить два набора данных, используйте следующее соответствие. В итоговых таблицах используйте РУССКИЕ названия категорий — так, как они хранятся в магазине:

| Название в API (английское) | Название категории в магазине (русское) |
| --- | --- |
| Electronics | Электроника |
| Audio | Аудио |
| Cameras | Камеры |
| Headphones | Наушники |
| Home Appliances | Бытовая техника |
| Speakers | Колонки |
| TV & Home Theater | ТВ и домашний кинотеатр |
| Watches | Часы |

Поля JSON из API (`market_overview`, `categories`, `market_avg_price`, `market_revenue`, `growth_rate`, `total_market_size` и т. д.) оставьте на английском — это имена полей, а не данные.

## Excel-файл

Создайте Excel-файл с именем Competitive_Analysis.xlsx с тремя листами. Передавайте инструменту работы с Excel АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.

Первый лист Category_Comparison должен содержать столбцы Category, Own_Products, Own_Avg_Price, Market_Avg_Price, Price_Position, Own_Revenue, Market_Revenue, Market_Share_Pct и Market_Growth_Rate. Цены и выручку округлите до 2 знаков после запятой. Долю рынка — до 2 знаков.

Порядок строк: отсортируйте категории по алфавиту по их АНГЛИЙСКОМУ названию из API (в столбце Category при этом оставьте русское название магазина). Таким образом ожидаемый порядок строк следующий: Аудио, Камеры, Электроника, Наушники, Бытовая техника, Колонки, ТВ и домашний кинотеатр, Часы (это алфавитный порядок английских названий Audio, Cameras, Electronics, Headphones, Home Appliances, Speakers, TV & Home Theater, Watches).

Второй лист Strategic_Matrix должен содержать столбцы Category, Price_Position, Market_Share_Pct, Market_Growth_Rate, Growth_Opportunity, Strategic_Priority и Recommended_Action. Growth_Opportunity равно Yes, когда темп роста рынка выше 10 процентов И наша доля рынка ниже 5 процентов, иначе No. Strategic_Priority равно High, когда есть возможность для роста И у нас позиция Competitive или Price Leader; Medium, когда применим один из этих факторов; Low, когда не применим ни один. Добавьте рекомендуемое действие (Recommended_Action) для каждой категории. Используйте тот же порядок строк, что и на первом листе (по алфавиту английских названий категорий).

Значения столбцов Price_Position (Price Leader / Competitive / Premium), Growth_Opportunity (Yes / No) и Strategic_Priority (High / Medium / Low) оставьте на английском — именно эти значения и имена столбцов проверяются.

Третий лист Executive_Summary должен содержать два столбца Label и Value со следующими строками: Total_Own_Products, Total_Own_Revenue, Total_Market_Size, Overall_Market_Share_Pct (суммарная собственная выручка, делённая на общий размер рынка, умноженная на 100 и округлённая до 2 знаков), Categories_Price_Leader (количество), Categories_Premium (количество), High_Priority_Categories (количество), Growth_Opportunities_Count (количество).

Соберите презентацию PowerPoint с именем Strategy_Presentation.pptx минимум из 6 слайдов: титульный слайд, обзор рынка, ключевые моменты сравнения категорий, матрица стратегических приоритетов, возможности для роста и стратегические рекомендации. Передавайте инструменту работы с PowerPoint АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.

## Письма

Отправьте два письма. Первое — на ceo@company.com с темой "Competitive Market Analysis - Executive Summary", содержащее ключевые выводы из листа Executive_Summary, включая нашу общую долю рынка (overall market share), число категорий высокого приоритета (high priority) и количество возможностей для роста (growth opportunities). Второе письмо — на product_team@company.com с темой "Category Competitive Analysis - Detailed Findings", в котором по каждой категории указано её ценовое позиционирование (price position), доля рынка и стратегический приоритет, с акцентом на то, какие категории требуют немедленного внимания.
