# Руководство по порталу рыночной аналитики

## Эндпоинт API
Данные рыночной аналитики доступны по адресу:
```
http://localhost:30195/api/market_data.json
```

## Формат данных
API возвращает JSON-объект со следующими полями:
- `market_overview`: общий размер рынка, рост год к году и дата отчёта
- `categories`: массив данных по категориям, включая среднюю рыночную цену (`market_avg_price`), рыночную выручку (`market_revenue`) и темп роста (`growth_rate`)

Названия категорий в API указаны на английском. Сопоставление с русскими названиями категорий магазина приведено в task.md.

## Методология сопоставления
1. Получите рыночные данные с эндпоинта API
2. Запросите каталог товаров вашего магазина для подсчёта количества товаров и средних цен по категориям
3. Запросите данные о заказах, чтобы рассчитать фактическую выручку по категориям
4. Сравните собственные метрики с рыночными бенчмарками, используя методологию из Competitive_Strategy.pdf
