Проект медиа-анализа — Сравнение каналов

Цель: сравнить эффективность и контент-стратегию двух YouTube-каналов из видеобазы.

Каналы для сравнения:
- Fireship (технологический канал, короткие ролики)
- Veritasium (научный канал, длинные документальные ролики)

Необходимые результаты:
1. Облачная таблица с названием "Channel Comparison Analysis", содержащая:
   - Лист Comparison: одна строка на канал со всеми метриками
   - Лист Monthly_Fireship: помесячное количество видео для Fireship

2. Документ Word: Channel_Analysis_Report.docx с разделами:
   - Executive Summary
   - Channel Metrics Comparison
   - Key Findings
   - Recommendations

3. Письмо на адрес media@company.com с темой "YouTube Channel Comparison Report"

Метрики для вычисления по каждому каналу:
- Video_Count, Total_Views, Total_Likes
- Avg_Views (округлено до 0 знаков после запятой)
- Avg_Duration_Sec (округлено до 0 знаков после запятой)
- Avg_Like_Rate (like_count/view_count*100, округлено до 2 знаков после запятой)
- Top_Video_Title, Top_Video_Views
