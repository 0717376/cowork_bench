Вы — медиа-аналитик, которому поручено подготовить сравнительный анализ двух известных YouTube-каналов, данные о которых хранятся в видеобазе вашей команды. Эти два канала — Fireship (технологический канал) и Veritasium (научный канал).

Для каждого канала вычислите следующие метрики, используя данные из видеобазы: общее количество видео, суммарное число просмотров по всем видео, суммарное число лайков по всем видео, среднее число просмотров на одно видео (округлённое до целого), среднюю продолжительность видео в секундах (округлённую до целого), средний процент лайков, рассчитанный как суммарное число лайков, делённое на суммарное число просмотров и умноженное на 100 (округлённый до 2 знаков после запятой), а также название и число просмотров единственного самого просматриваемого видео на этом канале.

Создайте облачную таблицу с названием "Channel Comparison Analysis". Эта таблица должна содержать два листа.

Первый лист должен называться Comparison. В нём должна быть ровно одна строка заголовка, за которой следуют две строки данных — по одной на каждый канал. Столбцы должны идти в следующем порядке: Channel, Video_Count, Total_Views, Total_Likes, Avg_Views, Avg_Duration_Sec, Avg_Like_Rate, Top_Video_Title, Top_Video_Views.

Второй лист должен называться Monthly_Fireship. Он должен показывать данные о публикациях по месяцам только для канала Fireship, со столбцами Month (в формате YYYY-MM) и Video_Count, отсортированными по столбцу Month по возрастанию.

Затем создайте документ Word с названием Channel_Analysis_Report.docx в вашем рабочем каталоге. Документ должен содержать не менее четырёх разделов со следующими заголовками: Executive Summary, Channel Metrics Comparison, Key Findings и Recommendations. Раздел Executive Summary должен описывать цель отчёта. Раздел Channel Metrics Comparison должен включать вычисленную статистику по обоим каналам. Раздел Key Findings должен выделять заметные различия между каналами. Раздел Recommendations должен содержать стратегические рекомендации, основанные на данных.

Наконец, отправьте письмо на адрес media@company.com с темой "YouTube Channel Comparison Report" и приложите либо упомяните файл Channel_Analysis_Report.docx в тексте сообщения.
