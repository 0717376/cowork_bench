# Методология стресс-тестирования

## Расчёт месячной доходности (Monthly Return)
- Сгруппируйте ежедневные цены закрытия по календарным месяцам
- Используйте последнюю доступную цену закрытия за каждый месяц
- Месячная доходность = (Close_month_N - Close_month_N-1) / Close_month_N-1

## VaR портфеля (Value at Risk) при доверии 95%
1. Вычислите взвешенные месячные доходности портфеля: сумма (weight_i * return_i) за каждый месяц
2. VaR_95 = 5-й процентиль месячных доходностей портфеля * общая стоимость портфеля
3. Это отражает максимальный ожидаемый месячный убыток при доверии 95%

## Максимальная просадка (Maximum Drawdown)
1. Вычислите накопленную доходность портфеля: произведение (1 + monthly_return) по времени
2. Отслеживайте текущий максимум накопленной доходности (running maximum)
3. Просадка в каждой точке = (cumulative - running_max) / running_max
4. Максимальная просадка = минимум из всех значений просадки

## Коэффициент Шарпа (Sharpe Ratio, по каждой акции)
- Годовая доходность = mean(monthly returns) * 12
- Годовая волатильность = std(monthly returns) * sqrt(12)
- Для всех стандартных отклонений используйте выборочное стандартное отклонение (ddof=1, как pandas .std() по умолчанию)
- Sharpe = (annualized_return - 0.05) / annualized_volatility
- Используйте 5% в качестве годовой безрисковой ставки

## Применение стресс-теста
- Для каждого сценария примените доходность сценария к текущей распределённой стоимости каждой акции
- Scenario_Value = Current_Value * (1 + Scenario_Return)
- Scenario_PnL = Scenario_Value - Current_Value
- Итог по портфелю = сумма всех значений Scenario_Value по отдельным акциям
