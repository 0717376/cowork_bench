Вы — портфельный управляющий в российской инвестиционной компании среднего размера. Все суммы и оценки приводятся в рублях (RUB). Ваша компания отслеживает пять основных позиций (акции Московской биржи): SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME и MTSS.ME. Исследовательская служба опубликовала свежие прогнозы макроэкономических показателей, которые нужно учесть в анализе портфеля. Сначала прочитайте файл Investment_Strategy.pdf в вашем рабочем каталоге, чтобы понять подход компании к управлению портфелем с учётом макроэкономики.

API исследовательской службы доступен по адресу http://localhost:30211/api/macro_forecast.json и предоставляет прогнозы роста ВВП, уровня инфляции и процентной ставки по кварталам на 2026 год. Загрузите эти данные, чтобы понять макроэкономическую картину.

После получения макропрогноза запросите у финансового источника данных текущую информацию по всем пяти позициям: текущую цену, сектор (sector), рыночную капитализацию (marketCap), коэффициент цена/прибыль (trailingPE).

Создайте книгу Excel с именем Macro_Portfolio_Analysis.xlsx в вашем рабочем каталоге с тремя листами.

Первый лист должен называться "Macro Forecast" и содержать колонки: Quarter, GDP_Growth_Pct, Inflation_Rate_Pct, Interest_Rate_Pct. Заполните его четырьмя кварталами данных из API исследовательской службы (Q1 2026 по Q4 2026).

Второй лист должен называться "Stock Holdings" и содержать колонки: Symbol, Sector, Price, Market_Cap_B (рыночная капитализация в миллиардах, округлённая до 1 знака после запятой), PE_Ratio (округлённый до 1 знака после запятой). Перечислите все пять акций, отсортировав их по Symbol в алфавитном порядке. Используйте текущую цену (currentPrice), возвращаемую финансовым источником, в колонке Price.

Третий лист должен называться "Sector Sensitivity" и содержать колонки: Sector, Avg_GDP_Growth, Rate_Sensitivity, Recommended_Action. Для каждого уникального сектора среди пяти позиций вычислите средний годовой рост ВВП из макропрогноза (среднее по 4 кварталам, округлённое до 1 знака), присвойте метку Rate_Sensitivity на основе коэффициента P/E (trailingPE): "High", если средний P/E акций этого сектора выше 5.0, иначе "Low". Установите Recommended_Action равным "Overweight", если Rate_Sensitivity = High и средний рост ВВП выше 2.0 процента; "Underweight", если Rate_Sensitivity = Low и средний рост ВВП ниже 2.0 процента; и "Hold" во всех остальных случаях. Отсортируйте по Sector в алфавитном порядке.

Важно: коэффициент бета (beta) для российских эмитентов недоступен, поэтому чувствительность к ставке определяется по коэффициенту P/E (trailingPE), как описано выше, а не по бете.
