Нашей инвестиционной команде нужно подготовить еженедельный отчёт по анализу рынка. Аналитическая финансовая панель доступна по адресу http://localhost:30204/dashboard.html и показывает индексы эффективности секторов, недельную доходность, доходность с начала года (YTD), уровень волатильности и консенсус-рейтинги аналитиков для пяти секторов рынка MOEX: Энергетика, Финансы, Телекоммуникации, Потребительский сектор и Технологии.

Мы также отслеживаем пять отдельных акций, представляющих эти секторы: GAZP.ME для Энергетики, SBER.ME для Финансов, MTSS.ME для Телекоммуникаций, MGNT.ME для Потребительского сектора и TCSG.ME для Технологий. Нужно сопоставить эффективность на уровне секторов с данными по отдельным акциям.

Откройте рыночную панель в браузере и извлеките все данные по эффективности секторов, включая недельную доходность, значение индекса сектора, доходность с начала года, уровень волатильности, консенсус-рейтинг аналитиков и потенциал роста к целевой цене для каждого из пяти секторов.

Затем получите текущие данные по каждой из пяти акций (GAZP.ME, SBER.ME, MTSS.ME, MGNT.ME, TCSG.ME) из финансового сервиса через инструмент get_stock_info. Для каждой акции получите текущую цену (в рублях), сектор, рыночную капитализацию, коэффициент P/E и дивидендную доходность.

Создайте Google Sheet с названием "Weekly_Market_Analysis" с двумя листами. Первый лист "Sector Performance" должен содержать колонки Sector, Weekly_Return, Sector_Index, YTD_Return, Volatility, Consensus и Target_Upside, по одной строке на каждый из пяти секторов. Второй лист "Stock vs Sector" должен содержать колонки Ticker, Stock_Price, Sector, Market_Cap_B (рыночная капитализация в миллиардах рублей, округлённая до 1 знака после запятой), PE_Ratio, Dividend_Yield_Pct, Sector_Weekly_Return и Sector_YTD_Return. Сопоставьте каждую акцию с соответствующими данными её сектора из панели.

Далее создайте документ Word с именем "Weekly_Market_Report.docx" в рабочей директории в виде исполнительного резюме. Включите заголовок, обзорный абзац по рынку, раздел по каждому сектору с обсуждением эффективности индекса сектора и представительной акции, а также заключение, в котором указаны лучший и худший по эффективности секторы недели. ВАЖНО: при сохранении файла .docx передавайте инструменту word АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории агента (например, <workspace_dir>/Weekly_Market_Report.docx), так как инструмент word может не сохранять текущую рабочую директорию.

Наконец, отправьте письмо от analyst@investmentfirm.com на committee@investmentfirm.com с темой "Weekly Market Analysis Report". В теле письма выделите лучший по эффективности сектор и любые секторы с отрицательной недельной доходностью.

Сохраните все выходные файлы в рабочей директории.
