Я — администратор учебной части в Открытом университете, и мне нужно подготовить итоговые отчёты об успеваемости за весенний семестр 2014 года по всем курсам. Курсы весеннего семестра 2014 года определяются по кодам курсов, содержащим "2014B".

В рабочей директории есть файл grading_policy.pdf, в котором изложена наша официальная шкала оценивания и критерии особых отметок. Пожалуйста, сначала прочитайте его, чтобы разобраться в границах оценок, а также в правилах присвоения отличия и постановки на академический контроль.

Для каждого курса весеннего семестра 2014 года найдите курс в нашей системе управления обучением (LMS) и определите ведущего преподавателя. Ведущий преподаватель — это преподаватель (teacher), записанный на курс, чьё имя идёт первым по алфавиту. Если на курс не записан ни один преподаватель, используйте "N/A" для имени преподавателя и адреса электронной почты и не отправляйте письмо по этому курсу. Также определите количество студентов, у которых есть выставленные баллы (ненулевое значение current_score в их оценках по записи на курс), и вычислите средний балл по курсу на основе баллов этих студентов.

Присвойте каждому курсу буквенную оценку (letter grade) на основе его среднего балла по шкале из PDF. Также определите, заслуживает ли курс отметки "With Distinction" (с отличием) или подпадает под флаг "Academic Probation" (академический контроль) согласно критериям из PDF.

Создайте файл с именем semester_grade_report.xlsx в рабочей директории с тремя листами. ВАЖНО: при работе с excel MCP передавайте абсолютный путь к файлу semester_grade_report.xlsx внутри рабочей директории.

Первый лист должен называться "Course Grades" со следующими столбцами: Course_Code, Course_Name, Lead_Instructor, Instructor_Email, Students_Scored, Class_Average (округлённый до 2 знаков после запятой), Letter_Grade, Distinction (записывайте "Yes" или "No"), Probation (записывайте "Yes" или "No"). Отсортируйте строки по Course_Code в алфавитном порядке.

Второй лист должен называться "Grade Distribution" со столбцами: Letter_Grade (перечислите A, B, C, D, F именно в этом порядке), Course_Count (сколько курсов получили эту буквенную оценку), Courses (коды курсов через запятую, получившие эту оценку, отсортированные по алфавиту внутри каждой оценки; оставьте пустую строку, если ни один курс не получил эту оценку).

Третий лист должен называться "Summary" с двумя столбцами (Metric, Value), содержащими следующие строки: Total_Courses (количество курсов весеннего семестра 2014 года), Avg_Class_Average (среднее всех средних баллов по курсам, округлённое до 2 знаков после запятой), Highest_Average_Course (код курса с наивысшим средним баллом), Lowest_Average_Course (код курса с наименьшим средним баллом), Distinction_Count (сколько курсов получили отметку With Distinction), Probation_Count (сколько курсов помечены для Academic Probation).

Также создайте Google-таблицу с заголовком "Spring 2014 Grade Summary" с одним листом с именем "Grades", содержащим столбцы: Course_Code, Class_Average, Letter_Grade, Distinction, Probation для всех шести курсов, отсортированных по Course_Code.

Наконец, отправьте письмо с адреса registrar@openuniversity.ac.uk ведущему преподавателю каждого курса (для курсов, у которых есть преподаватель) с темой "End-of-Semester Grade Report: [Course Code]", где [Course Code] заменяется на фактический код курса. В теле письма должны быть указаны название курса, средний балл, буквенная оценка, а также то, получил ли курс отличие или поставлен на академический контроль.
