Проект «Карта научных образовательных ресурсов»

Цель: связать самые просматриваемые научные видео канала Veritasium с релевантными академическими статьями, чтобы создать кураторский обучающий ресурс.

Процесс:
1. Получить из базы видео топ-5 видео Veritasium по числу просмотров
2. Определить основную научную тему каждого видео на основе его названия
3. Найти в базе академических статей работы по каждой теме (до 3 статей на тему)
4. Создать Science_Resource_Map.xlsx с листами Videos и Papers
5. Создать страницу в корпоративной базе знаний teamly с заголовком «Science Video-Paper Resource Map»

Выходные файлы:
- Science_Resource_Map.xlsx
  - Лист Videos: Rank, Video_ID, Title, View_Count, Main_Topic, Paper_Count
  - Лист Papers: Topic, Paper_Title, Authors, Year, Citations
- Страница в корпоративной базе знаний teamly: «Science Video-Paper Resource Map»
