Вы — исследователь в сфере образования, создающий научную библиотеку ресурсов, которая связывает популярные научно-популярные видео с релевантными академическими статьями. Ваша задача — определить самые просматриваемые научные видео канала Veritasium и подобрать к каждому из них связанные академические работы.

Начните с запроса к базе видео, чтобы получить все видео канала Veritasium. Выберите 5 самых просматриваемых видео. Для каждого из 5 видео определите основную научную тему на основе названия видео. Например, у видео про погружение вглубь под микроскопом темой будет «microscopy» или «fractals», а у видео про инженерную проблему небоскрёба — «structural engineering».

Для каждой из 5 определённых тем выполните поиск в базе академических статей по релевантным работам. Соберите до 3 статей на каждую тему. Для каждой найденной статьи зафиксируйте название, авторов, год публикации и число цитирований.

Создайте Excel-файл с именем Science_Resource_Map.xlsx в вашем рабочем каталоге. Этот файл должен содержать два листа.

Первый лист должен называться Videos. В нём должна быть одна строка на каждое топовое видео со следующими столбцами в указанном порядке: Rank (от 1 до 5, ранжирование по числу просмотров по убыванию), Video_ID, Title, View_Count, Main_Topic и Paper_Count (число академических статей, которое вы нашли по этой теме). Строки должны быть отсортированы по Rank по возрастанию.

Второй лист должен называться Papers. В нём должна быть одна строка на каждую найденную статью по всем темам. Столбцы должны идти в таком порядке: Topic, Paper_Title, Authors, Year и Citations. Если по какой-то теме статей не найдено, эту тему можно опустить в данном листе.

После создания таблицы создайте страницу в корпоративной базе знаний teamly с заголовком ровно «Science Video-Paper Resource Map». Содержимое страницы должно быть написано связным текстом, перечисляющим 5 видео Veritasium с их темами и числом найденных релевантных статей для каждого. Текст должен быть оформлен как читаемые абзацы, а не как таблица или список со специальными символами.
