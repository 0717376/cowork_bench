Здравствуйте! Руководство HR-отдела попросило меня подготовить анализ оплаты труда на основе данных о сотрудниках, которые хранятся в нашем хранилище данных ClickHouse.

Мне нужно от вас две вещи.

Во-первых, запросите данные о сотрудниках и рассчитайте статистику по зарплатам с группировкой по отделам (DEPARTMENT). Для каждого отдела вычислите количество сотрудников, среднюю зарплату, минимальную зарплату, максимальную зарплату и медианную зарплату. Все значения зарплат округлите до 2 знаков после запятой. Затем сделайте более детальную разбивку по отделу и уровню образования (EDUCATION_LEVEL), вычислив для каждой комбинации количество сотрудников, среднюю, минимальную и максимальную зарплату.

Данные находятся в таблице `HR_ANALYTICS__PUBLIC__EMPLOYEES` (логическая база `HR_ANALYTICS`, схема `sf_data`).

Создайте PDF-файл с именем `Compensation_Report.pdf` в рабочей директории. В PDF должен быть заголовок "Compensation Analysis Report" и подзаголовок, упоминающий "HR Analytics Department" и дату 2026-03-06. Включите таблицу со сводкой по отделам со столбцами Department, Employees, Avg Salary, Min Salary, Max Salary и Median Salary. Затем включите вторую таблицу с разбивкой по отделу и уровню образования со столбцами Department, Education, Count, Avg Salary, Min Salary и Max Salary. Отсортируйте сводку по отделам по алфавиту по названию отдела. Разбивку по образованию отсортируйте по отделу по алфавиту, а затем по уровню образования по алфавиту внутри каждого отдела.

Также создайте файл Excel с именем `Compensation_Data.xlsx` в рабочей директории с двумя листами. Первый лист должен называться "Department Summary" со столбцами Department, Employees, Avg_Salary, Min_Salary, Max_Salary и Median_Salary. Второй лист должен называться "Education Breakdown" со столбцами Department, Education_Level, Count, Avg_Salary, Min_Salary и Max_Salary. Сортировка та же, что описана выше. Все значения зарплат округлите до 2 знаков после запятой.

Имена столбцов и листов оставьте на английском в точности так, как указано выше — они используются для проверки.

Наконец, создайте страницу в базе знаний Teamly с заголовком "Compensation Analysis 2026", в которой кратко изложите ключевые выводы, включая то, в каком отделе самая высокая средняя зарплата и какой уровень образования в целом имеет наибольшее число сотрудников. Названия отдела и уровня образования в тексте страницы указывайте на русском языке так, как они хранятся в базе данных.

Файлы `Compensation_Report.pdf` и `Compensation_Data.xlsx` записывайте по полному (абсолютному) пути внутри указанной рабочей директории.
