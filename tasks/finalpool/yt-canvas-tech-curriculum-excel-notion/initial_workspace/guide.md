# Памятка по задаче: Tech Course Video Resources

## Обзор
Сопоставить видео Fireship с YouTube релевантным университетским курсам и составить карту видеоресурсов для учебной программы.

## Шаг 1: Найти курсы
Запроси из системы управления обучением (Canvas) курсы, в названии которых встречается:
«вычисления» или «данных».

## Шаг 2: Подобрать видео
Для каждого курса подбери ровно 2 релевантных видео Fireship, сопоставляя ключевые слова:
- Курсы с «вычисления» (Computing): «code», «computing», «tech», «programming».
- Курсы с «данных» (Data): «data», «database», «code», «analytics».

Ключевые слова даны в порядке убывания приоритета: ранжируй видео по приоритету совпавшего слова, затем по View_Count по убыванию.

## Результат 1: Curriculum_Video_Map.xlsx

Лист: Course_Videos
Столбцы: Course_Name, Course_Code, Video_ID, Video_Title, View_Count, Duration_Min, Relevance_Match
Сортировка: Course_Name по возрастанию, затем View_Count по убыванию

Лист: Summary (ровно 3 строки)
Строка 1: Total_Courses_Mapped | [количество]
Строка 2: Total_Videos_Recommended | [количество]
Строка 3: Avg_Views_Recommended | [целое число]
