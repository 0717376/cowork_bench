Мне нужно параллельное сравнение всех наших курсов в Canvas. Выгрузи данные по каждому курсу: число зачисленных студентов, количество заданий, количество тестов и количество тем для обсуждений.

Создай Excel-файл с именем Canvas_Course_Comparison.xlsx с двумя листами. Лист "Course Comparison" должен содержать столбцы Course_Code, Course_Name, Students, Assignments, Quizzes и Discussions. Отсортируй строки по Course_Code.

Лист "Summary" должен содержать Total_Courses, Avg_Students_Per_Course (округлённое до ближайшего целого) и код самого популярного курса Most_Popular_Course.

Также создай Google Sheet с заголовком "Course Comparison Dashboard" и листом "Data", содержащим те же сравнительные данные.

Важно: при сохранении файла Canvas_Course_Comparison.xlsx через excel-инструмент указывай АБСОЛЮТНЫЙ путь к файлу внутри рабочего каталога агента (excel-инструмент не сохраняет текущий рабочий каталог).
