# course-enrollment-analytics-dashboard

Начальная рабочая директория для задачи об аналитике набора студентов и группе риска.

Файлы:
- `course_catalog.csv` — справочник курсов (коды и названия программ);
- `data.csv` — небольшой справочник студентов для консультантов;
- `config.json` — параметры задачи (адрес консультантов, пространство Teamly);
- `enrollment_template.xlsx` — пустой шаблон (можно не использовать).
