Учебному отделу университета нужен анализ сдачи работ по всем курсам в системе управления обучением (LMS) Canvas. Получите данные о сдачах по оцениваемым заданиям и агрегируйте их по курсам.

Создайте файл Excel под названием Canvas_Submissions.xlsx с двумя листами. На листе "Submission Stats" должны быть столбцы: Course_Code, Submissions (количество сдач), Avg_Score (средний балл, округлённый до 2 знаков после запятой), Late_Count (число сдач с опозданием) и Late_Pct (доля сдач с опозданием в процентах от общего числа, округлённая до 1 знака после запятой). Сдача считается опоздавшей, если время сдачи (submitted_at) позже срока сдачи (due_at). Отсортируйте строки по Course_Code.

На листе "Summary" должны быть столбцы Metric и Value со следующими метриками: Total_Submissions (общее число сдач), Overall_Avg_Score (средний балл, взвешенный по количеству сдач по каждому курсу), Total_Late (общее число опоздавших сдач) и Overall_Late_Pct (общая доля опоздавших сдач).

Сохраняйте без изменений на английском языке: имя файла Canvas_Submissions.xlsx, названия листов "Submission Stats" и "Summary", все названия столбцов и ключи метрик (Course_Code, Submissions, Avg_Score, Late_Count, Late_Pct, Metric, Value, Total_Submissions, Overall_Avg_Score, Total_Late, Overall_Late_Pct), а также коды курсов (например, AAA-2013J). При записи файла Excel передавайте инструменту АБСОЛЮТНЫЙ путь к Canvas_Submissions.xlsx внутри рабочей директории.

Наконец, создайте в корпоративной базе знаний Teamly (пространство учебного отдела) страницу под названием "Canvas Submission Analysis" с ключевыми выводами анализа. Название страницы "Canvas Submission Analysis" сохраняйте на английском языке без изменений.
