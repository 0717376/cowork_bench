Я организую корпоративный ужин для команды (тимбилдинг) и прошу помочь спланировать меню. Хочу посмотреть доступные рецепты для ужина и собрать меню из трёх блюд: закуска (appetizer), основное блюдо (main course) и десерт (dessert).

Используй кулинарную базу через инструменты kulinar (например, `mcp_kulinar_getAllRecipes`, `mcp_kulinar_getRecipesByCategory`, `mcp_kulinar_recommendMeals`, `mcp_kulinar_getRecipeById`), чтобы найти подходящий рецепт для каждого блюда. Бери реальные названия рецептов и реальные списки ингредиентов из базы — ничего не выдумывай.

Когда подберёшь рецепты, мне нужны два результата.

Во-первых, создай в рабочей директории презентацию PowerPoint с именем файла "Event_Menu_Presentation.pptx". В презентации должны быть:
- титульный слайд с текстом "Корпоративный ужин команды";
- по одному слайду на каждое блюдо (закуска, основное блюдо, десерт) с названием рецепта и списком его ингредиентов;
- финальный итоговый слайд, на котором указано общее число ингредиентов по всем трём блюдам.

Во-вторых, создай в рабочей директории файл Excel с именем "Menu_Budget.xlsx". В нём должно быть два листа.
- Лист "Menu Items" с колонками Course, Recipe_Name, Ingredients_Count, Difficulty. По одной строке на каждое блюдо. В колонке Course используй английские маркеры: appetizer, main course, dessert.
- Лист "Summary" с колонками Metric и Value и строками Total_Recipes, Total_Ingredients, Avg_Difficulty.

Для колонки Difficulty используй числовую шкалу: 1 — лёгкий рецепт, 2 — средний, 3 — сложный. База kulinar хранит сложность в диапазоне 1–4; считай сложность 1 как лёгкую (1), 2 как среднюю (2), а 3 и выше — как сложную (3). Если у рецепта сложность не указана, по умолчанию ставь 2.

Значения в листе "Summary" считай по фактическим данным листа "Menu Items":
- Total_Recipes — количество строк с блюдами;
- Total_Ingredients — сумма значений колонки Ingredients_Count;
- Avg_Difficulty — среднее значение колонки Difficulty.

То же общее число ингредиентов (Total_Ingredients) должно быть указано на итоговом слайде презентации.
