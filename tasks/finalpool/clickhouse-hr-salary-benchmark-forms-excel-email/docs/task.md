Вы — аналитик по компенсациям в отделе кадров. Ваша задача — проанализировать распределение зарплат по отделам и подготовить пакет для обзора компенсаций.

Запросите кадровую базу данных ClickHouse, чтобы рассчитать статистику по зарплатам для каждого отдела. Вам нужны: численность сотрудников, минимальная зарплата, максимальная зарплата, средняя зарплата и медианная зарплата по каждому отделу.

Создайте файл Excel с именем Salary_Analysis.xlsx в рабочей директории с двумя листами:

Лист с именем Department_Stats со столбцами: Department, Headcount, Min_Salary, Max_Salary, Avg_Salary, Median_Salary. Включите по одной строке на каждый отдел (всего 7 отделов). Отсортируйте строки по Avg_Salary по убыванию.

Лист с именем Summary с двумя столбцами: Metric и Value. Включите следующие строки: Total_Employees (общая численность сотрудников по всем отделам), Company_Avg_Salary (средневзвешенная зарплата по всем сотрудникам, округлённая до 2 знаков после запятой), Highest_Paid_Dept (название отдела с наивысшей средней зарплатой), Lowest_Paid_Dept (название отдела с наименьшей средней зарплатой). Названия отделов берите ровно в том виде, в каком они хранятся в базе данных.

Создайте опрос в Forms с заголовком Compensation Satisfaction Survey ровно из 5 вопросов. Первый вопрос спрашивает, насколько сотрудники удовлетворены своей текущей компенсацией, и предлагает варианты с множественным выбором от «Очень удовлетворён» до «Очень не удовлетворён». Второй спрашивает, является ли их оплата конкурентной по сравнению с отраслевыми стандартами, с вариантами «Да», «Нет» или «Не уверен». Третий спрашивает, какие льготы для них наиболее важны, в виде краткого текстового ответа. Четвёртый спрашивает, рассматривали бы они уход ради лучшей оплаты, с вариантами «Да», «Нет» или «Возможно». Пятый предлагает оставить любые дополнительные комментарии о компенсации в виде развёрнутого текстового ответа (абзац).

Отправьте письмо с адреса compensation@hr.example.com на адрес hr-leadership@company.example.com с темой Compensation Analysis Report - Action Required. В теле письма должны быть указаны средняя зарплата по компании, а также названы отделы с наивысшей и наименьшей оплатой.

Файл Compensation_Policy.pdf в вашей рабочей директории содержит руководящие принципы философии компенсаций для справки.
