Я планирую меню обедов на неделю для офиса и прошу помочь собрать рецепты, отфильтровать их и оформить аккуратный документ.

Начни с поиска рецептов в кулинарной базе минимум по четырём разным категориям. Просмотри доступные категории (например, супы, горячее, салаты, гарниры) и собери разнообразный набор вариантов: обязательно включи хотя бы один суп и хотя бы одно овощное блюдо. Используй инструменты kulinar (getAllRecipes, getRecipesByCategory, getRecipeById, recommendMeals, whatToEat). Для каждого найденного рецепта запиши его название (name), категорию (category), уровень сложности (difficulty) и список ингредиентов (ingredients).

Когда соберёшь достаточный пул рецептов, сохрани его в виде JSON-файла all_recipes.json в рабочей директории. Каждая запись должна содержать поля name, category, difficulty и ingredients. Затем запусти предоставленный скрипт filter_recipes.py, чтобы отфильтровать набор, оставив только рецепты со сложностью 3 или ниже (флаг --max-difficulty 3). Сохрани отфильтрованный результат в файл filtered_recipes.json в рабочей директории.

Перед окончательным выбором блюд запомни (в memory) следующие пищевые предпочтения для будущих ссылок: никаких блюд со сложностью выше 3; предпочитать разнообразие по категориям; за неделю должен быть хотя бы один суп и хотя бы одно овощное блюдо; не повторять одну и ту же категорию в соседние дни. Также запомни названия блюд, которые ты в итоге выберешь, чтобы можно было сослаться на них позже.

Из отфильтрованных результатов выбери пять блюд для меню рабочей недели — по одному на каждый день с понедельника по пятницу. Убедись, что они охватывают минимум четыре разные категории и соответствуют записанным пищевым предпочтениям.

Наконец, сформируй документ Word с именем Weekly_Menu.docx в рабочей директории. У документа должен быть заголовок «Меню обедов на неделю» (Weekly Lunch Menu). Затем для каждого дня недели с понедельника по пятницу добавь раздел с заголовком — названием дня (Понедельник, Вторник, Среда, Четверг, Пятница). Под каждым днём укажи название блюда, его категорию, уровень сложности и полный список ингредиентов этого блюда. В конце документа добавь раздел с заметками о пищевых предпочтениях, которыми ты руководствовался при выборе.

Важно: MCP-сервер Word не сохраняет текущую рабочую директорию, поэтому при создании Weekly_Menu.docx передавай АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.
