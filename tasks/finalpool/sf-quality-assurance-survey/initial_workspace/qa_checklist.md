# Чек-лист контроля качества

## Методология оценки

Каждый агент поддержки оценивается по трём основным измерениям:

1. Время отклика: сравнение среднего времени первого отклика с опубликованным целевым значением для каждого уровня приоритета.
2. Удовлетворённость клиентов: средняя оценка CSAT по всем обработанным тикетам по шкале от 1 до 5.
3. Доля решённых обращений: процент тикетов, успешно решённых агентом, от общего числа назначенных.

## Критерии оценки

- Агенты, выполняющие все три цели, получают рейтинг «Соответствует стандартам».
- Агенты, не выполняющие одно измерение, получают рейтинг «Требует улучшения» с конкретными мерами.
- Агенты, не выполняющие два и более измерений, помечаются статусом «Требуется наставничество».

## Периодичность проверок КК

Ежемесячные проверки проводятся командой контроля качества. Ежеквартальные сводки предоставляются руководству департамента.

## Источники данных

- Данные о тикетах из базы данных центра поддержки
- Целевые показатели качества с внутреннего портала стандартов
- Ростер агентов из системы управления командой
