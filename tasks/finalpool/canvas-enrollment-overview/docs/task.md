Мне нужен сводный отчёт по зачислению для всех курсов в нашей LMS Canvas. Пожалуйста, выгрузите данные о зачислении по каждому курсу и распределите их по типам ролей.

Создайте файл Excel с именем Canvas_Enrollment_Report.xlsx с двумя листами. На листе "Enrollment Data" должны быть столбцы Course_Code, Course_Name, Students (число студентов), Teachers (число преподавателей), TAs (число ассистентов) и Total_Enrolled — сумма всех трёх. Отсортируйте по Course_Code в алфавитном порядке.

На листе "Summary" со столбцами Metric и Value должны быть строки: Total_Courses, Total_Students по всем курсам, Total_Teachers, Largest_Course — код курса с наибольшим числом студентов, и Smallest_Course — код курса с наименьшим числом студентов.

После создания отчёта отправьте письмо на адрес registrar@openuniversity.ac.uk с темой "Canvas Enrollment Overview Report", в котором кратко изложите итоговые числа по зачислению.

Примечание: при работе с Excel-файлом через соответствующий инструмент указывайте абсолютный путь к Canvas_Enrollment_Report.xlsx внутри рабочей директории.
