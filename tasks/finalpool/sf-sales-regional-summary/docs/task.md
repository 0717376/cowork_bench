Мне нужна сводка региональных продаж по доставленным заказам из нашего хранилища данных ClickHouse. Пожалуйста, выгрузи данные и разбей их по регионам.

Создай файл Excel с именем Regional_Sales.xlsx в рабочей директории с двумя листами. На первом листе "Regional Breakdown" должны быть колонки Region, Orders (количество уникальных доставленных заказов), Customers (количество уникальных клиентов с доставленными заказами), Revenue (общая сумма, округлённая до 2 знаков после запятой), Avg_Order_Value (округлённое до 2 знаков после запятой) и Revenue_Share_Pct, показывающая долю каждого региона в общей выручке, округлённую до 1 знака после запятой. Отсортируй по колонке Region в алфавитном порядке.

Второй лист "Summary" с колонками Metric и Value должен содержать Total_Regions, Total_Revenue (округлённое до 2 знаков после запятой), Total_Orders, Total_Customers и Top_Region — название региона с наибольшей выручкой.

Также, пожалуйста, создай таблицу Google Sheet с заголовком "Regional Sales Dashboard" и листом "Overview", содержащим те же данные Regional Breakdown, чтобы команда могла получить к ним доступ онлайн.

При записи файла Regional_Sales.xlsx указывай абсолютный путь к нему внутри рабочей директории.
