Комиссия по аккредитации учебных программ нашего университета попросила меня оценить, соответствуют ли курсы университета стандартам аккредитации, опубликованным учёным советом. Требования к аккредитации размещены на странице стандартов по адресу http://localhost:30210/standards.html, где указано минимальное число заданий и тестов, необходимых для каждого курса, а также другие показатели качества.

Пожалуйста, начни с того, что зайди на эту страницу стандартов аккредитации и извлеки все перечисленные там критерии соответствия. На странице указано минимальное количество заданий, минимальное количество тестов и другие требования, которым должен удовлетворять каждый курс, чтобы считаться соответствующим (Compliant).

Затем обратись к системе управления обучением (LMS) и получи информацию обо всех курсах, имеющихся в системе. Для каждого курса мне нужно название курса, число заданий и число тестов. Всего в системе 22 курса по нескольким кафедрам и семестрам.

Используя стандарты с веб-страницы, оцени каждый курс по критериям аккредитации. Курс проходит проверку, если он удовлетворяет всем минимальным требованиям, и не проходит, если он не удовлетворяет хотя бы одному требованию.

Создай документ Word под названием Accreditation_Compliance_Report.docx в рабочей директории. Документ должен содержать заголовок «University Course Accreditation Compliance Report» (Отчёт о соответствии курсов университета стандартам аккредитации), вступление, объясняющее цель проверки и источник стандартов, а затем раздел для каждого курса с указанием названия курса, числа заданий, числа тестов и того, проходит ли он каждый критерий. В конце добавь итоговый раздел с общим числом соответствующих (compliant) курсов, несоответствующих (non-compliant) курсов и общим процентом соответствия. ВАЖНО: при сохранении документа через инструмент Word передавай АБСОЛЮТНЫЙ путь к файлу Accreditation_Compliance_Report.docx внутри рабочей директории (инструмент Word не сохраняет текущую директорию).

Затем создай в базе знаний Teamly страницу под названием «Course Compliance Tracker». На этой странице размести таблицу со следующими столбцами: Course_Name, Department, Assignment_Count, Quiz_Count, Assignments_Compliant, Quizzes_Compliant, Overall_Status (значения «Compliant» или «Non-Compliant») и Follow_Up_Date. Добавь по одной строке для каждого из 22 курсов. Для несоответствующих (non-compliant) курсов укажи в Follow_Up_Date дату 2026-04-15 (15 апреля 2026 года), а для соответствующих (compliant) курсов оставь Follow_Up_Date пустым.

Наконец, для каждой уникальной кафедры (department), у которой есть хотя бы один несоответствующий курс, отправь письмо с адреса accreditation@university.edu на адрес department-review@university.edu с темой «Accreditation Review: [Department Name] Courses» (где [Department Name] — название кафедры), перечислив несоответствующие курсы этой кафедры и то, каким стандартам они не соответствуют.
