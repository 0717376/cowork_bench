Я готовлюсь к заседанию кафедры, на котором нам нужно проанализировать результаты тестов (quizzes) по нашим курсам осеннего семестра 2014 года (Fall 2014). Мне нужно, чтобы вы извлекли данные о тестах из нашей системы управления обучением (LMS, Canvas) по всем курсам, которые читались в этом семестре, и обобщили результаты.

Для каждого курса осеннего семестра 2014 года (Fall 2014), в котором есть тесты, соберите названия тестов, количество студенческих сдач по каждому тесту, средний балл, минимальный балл и максимальный балл. Затем создайте файл Excel под названием `Quiz_Performance.xlsx` в рабочей директории. В книге должен быть лист с именем `Quiz Details` со столбцами `Course`, `Quiz_Title`, `Submissions`, `Avg_Score`, `Min_Score` и `Max_Score`. Значения `Avg_Score`, `Min_Score` и `Max_Score` должны быть округлены до двух знаков после запятой. Отсортируйте строки сначала по названию курса (`Course`) по алфавиту, затем по названию теста (`Quiz_Title`) по алфавиту внутри каждого курса.

Также добавьте второй лист с именем `Course Summary`, который агрегирует данные на уровне курса. В нём должны быть столбцы `Course`, `Total_Quizzes` (количество тестов в курсе), `Total_Submissions` (сумма сдач по всем тестам) и `Overall_Avg_Score` — средневзвешенный балл по всем сдачам этого курса, округлённый до двух знаков после запятой. Отсортируйте по алфавиту по названию курса (`Course`).

Наконец, создайте презентацию PowerPoint под названием `Quiz_Report.pptx` в рабочей директории. Первый слайд должен быть титульным с заголовком `Fall 2014 Quiz Performance Report` и подзаголовком `Faculty Meeting Summary`. Затем для каждого курса осеннего семестра 2014 года (Fall 2014), в котором есть тесты, создайте один слайд с названием курса в качестве заголовка и текстом в теле слайда, перечисляющим количество тестов, общее число сдач и общий средний балл для этого курса. Добавьте финальный слайд с заголовком `Key Takeaways` и текстом, указывающим, у какого курса самый высокий общий средний балл, а у какого — самый низкий.

Важные замечания:
- Имена файлов (`Quiz_Performance.xlsx`, `Quiz_Report.pptx`), имена листов (`Quiz Details`, `Course Summary`), заголовки столбцов и текст `Fall 2014 Quiz Performance Report`, `Faculty Meeting Summary`, `Key Takeaways` оставьте ровно на английском, как указано выше (по ним выполняется проверка). Названия курсов сохраняйте в точности так, как они приходят из Canvas, включая суффикс семестра (например, `(Осень 2014)`), не изменяя и не переводя их.
- При вызове инструментов Excel и PowerPoint указывайте АБСОЛЮТНЫЙ путь к файлам `.xlsx` и `.pptx` внутри рабочей директории (эти инструменты не сохраняют текущую рабочую директорию).
