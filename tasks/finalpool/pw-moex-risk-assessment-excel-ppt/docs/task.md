Помоги мне с оценкой рисков по нашему портфелю российских акций (MOEX). В нашем портфеле шесть бумаг: Сбербанк (SBER.ME), Газпром (GAZP.ME), ЛУКОЙЛ (LKOH.ME), Т-Банк (TCSG.ME), Магнит (MGNT.ME) и МТС (MTSS.ME).

Доступны внешние отраслевые ориентиры (бенчмарки). Зайди по адресу http://localhost:30317 и извлеки нужные метрики из таблицы. В таблице есть колонки Sector, Benchmark_PE, Benchmark_Yield_Pct и Market_Outlook — сохрани их английские названия как есть, по ним мы сопоставляем сектора.

Затем получи актуальные рыночные данные по акциям нашего портфеля через инструменты MOEX-финансов (get_stock_info, get_stock_price_by_date и т. п.). Все цены — в рублях (RUB).

С помощью терминала создай и запусти Python-скрипт под названием yf_risk_processor.py в рабочем каталоге. Скрипт должен читать собранные тобой данные из JSON-файлов, которые ты создашь, выполнять анализ и выводить результат в yf_risk_results.json.

Правила расчёта (применяй строго):
- Current_Price — это текущая цена бумаги (поле currentPrice из get_stock_info), в рублях.
- Target_Price = Current_Price * 1.10 (целевой рост 10%).
- Upside = Target_Price - Current_Price (абсолютная величина в рублях).
- Sector каждой бумаги бери из данных MOEX (поле sector) и сопоставляй с таблицей бенчмарков с панели.

Создай Excel-файл под названием Risk_Assessment_Report.xlsx с тремя листами.
1. Лист Data_Analysis с основными данными сравнения. Колонки: Symbol, Name, Sector, Current_Price, Target_Price, Upside. По одной строке на каждую бумагу портфеля (минимум 5 строк). Отсортируй строки по алфавиту по колонке Symbol (первичное измерение).
2. Лист Metrics с двумя колонками Metric и Value, обобщающий ключевые показатели. Включи как минимум строки: Total_Stocks (число бумаг = количеству строк в Data_Analysis), Avg_Upside (среднее значение колонки Upside) и Best_Opportunity (Symbol с максимальным Upside).
3. Лист Recommendations со списком приоритетных действий по итогам анализа. Колонки: Priority, Action, Symbol. Минимум 2 строки. Хотя бы одна рекомендация должна ссылаться на Symbol, который реально присутствует в Data_Analysis, и опираться на сопоставление сектора бумаги с отраслевым бенчмарком (Market_Outlook) с панели.

Создай презентацию PowerPoint под названием Risk_Assessment_Presentation.pptx минимум из 4 слайдов, освещающих ключевые выводы.

Важно: при работе с MCP-инструментами Excel и PowerPoint передавай АБСОЛЮТНЫЙ путь к файлам .xlsx и .pptx внутри рабочего каталога (эти инструменты не сохраняют текущий каталог).
