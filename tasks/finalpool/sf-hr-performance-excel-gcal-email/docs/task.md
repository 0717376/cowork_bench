Вы аналитик по эффективности персонала (HR), готовящий данные для ежегодного совета по оценке результативности сотрудников. Сделайте запрос к базе данных кадровой аналитики в ClickHouse, чтобы проанализировать распределение результативности по отделам.

Для каждого отдела определите лучших сотрудников (top performers — сотрудники с наивысшей оценкой результативности 5) и отстающих сотрудников (underperformers — сотрудники с оценкой результативности 1 или 2). Для каждой группы внутри каждого отдела вычислите количество сотрудников, среднюю зарплату и средний стаж работы (в годах).

Создайте файл Excel с именем Performance_Review_Summary.xlsx в рабочей директории с тремя листами:

Лист с именем Top_Performers со столбцами: Department, Count_Rating_5, Avg_Salary_Top, Avg_Experience_Top. Включите одну строку на каждый отдел (всего 7 строк), отсортированных по алфавиту по столбцу Department. Названия отделов берите ровно в том виде, в каком они хранятся в базе данных (они на русском языке) — не переводите их.

Лист с именем Underperformers со столбцами: Department, Count_Low_Rating, Avg_Salary_Low, Avg_Experience_Low. Включите одну строку на каждый отдел (всего 7 строк), отсортированных по алфавиту по столбцу Department.

Лист с именем Summary с двумя столбцами: Metric и Value. Включите строки для Total_Top_Performers (общее количество всех лучших сотрудников по всем отделам), Total_Underperformers (общее количество всех отстающих сотрудников по всем отделам) и Overall_Avg_Rating (средняя оценка результативности по всем сотрудникам, округлённая до 2 знаков после запятой).

Запланируйте событие в Google Calendar с названием Annual Performance Review Board Meeting ровно через 21 день после времени запуска. Событие должно длиться 2 часа и содержать описание, ссылающееся на данные оценки результативности.

Отправьте письмо с адреса hr@company.example.com на адрес executives@company.example.com с темой Annual Performance Review Summary - Action Required. В тексте письма укажите общее число лучших и отстающих сотрудников, а также отметьте общую среднюю оценку результативности.

Файл Performance_Review_Criteria.pdf в вашей рабочей директории содержит критерии оценки для справки.
