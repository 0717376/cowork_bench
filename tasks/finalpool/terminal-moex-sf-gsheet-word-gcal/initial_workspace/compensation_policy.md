# Политика компенсаций отдела продаж — структура бонусов за Q4

## Обзор
Квартальный бонус сотрудников отдела продаж (Продажи) рассчитывается на основе совокупной выручки, сгенерированной в каждом регионе продаж. Каждый сотрудник отдела продаж закрепляется за одним из пяти регионов клиентов. Совокупная выручка региона определяет, какой бонусный тир (tier) применяется ко всем сотрудникам, закреплённым за этим регионом.

## Закрепление за регионами
Сотрудники отдела продаж закрепляются за регионами по принципу round-robin (циклически). Отсортируйте всех сотрудников отдела продаж по имени в алфавитном порядке, затем циклически распределите их по пяти регионам, отсортированным в алфавитном порядке.

Пять регионов клиентов, отсортированные по алфавиту: Азиатско-Тихоокеанский регион, Ближний Восток, Европа, Латинская Америка, Северная Америка.

Например, первый по алфавиту сотрудник попадает в Азиатско-Тихоокеанский регион, второй — в Ближний Восток, третий — в Европу, четвёртый — в Латинскую Америку, пятый — в Северную Америку, шестой снова в Азиатско-Тихоокеанский регион и так далее.

## Расчёт бонуса
Бонус каждого сотрудника вычисляется как:

    Бонус = Оклад сотрудника x Процент бонуса (из тира)

Процент бонуса определяется по совокупной выручке региона по таблице бонусных тиров (bonus_tiers.csv).

## Рыночная корректировка
Перед финализацией бонусов комитет по компенсациям применяет коэффициент рыночной корректировки на основе годового изменения (year-over-year) котировок акций Сбербанка (тикер SBER.ME) — рыночного ориентира:

- Если SBER.ME вырос более чем на 10% год к году: коэффициент = 0.9 (снизить бонусы)
- Если SBER.ME упал более чем на 10% год к году: коэффициент = 1.1 (повысить бонусы)
- В остальных случаях: коэффициент = 1.0 (без изменений)

Скорректированный бонус = Текущий бонус x Коэффициент рыночной корректировки

## Бюджетный лимит
Если совокупный пул скорректированных бонусов превышает утверждённый бюджетный лимит (budget cap), все бонусы пропорционально масштабируются вниз так, чтобы итог в точности равнялся бюджетному лимиту.

## Правила валидации
- Индивидуальный скорректированный бонус не может превышать 20% оклада сотрудника
- Совокупный пул бонусов не должен превышать бюджетный лимит
