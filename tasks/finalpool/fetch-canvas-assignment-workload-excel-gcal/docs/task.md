Мне нужна помощь с анализом учебной нагрузки в Canvas. Есть внешние эталонные (бенчмарк) данные, которые нужно получить по адресу http://localhost:30312/api/data.json — извлеки оттуда релевантные метрики.

Затем зайди в нашу систему управления обучением (LMS) и собери данные по курсам и студентам.

С помощью терминала создай и запусти Python-скрипт course_workload_processor.py в рабочей директории. Скрипт должен читать собранные данные из создаваемых тобой JSON-файлов, выполнять анализ и выводить результат в course_workload_results.json.

Создай Excel-файл Assignment_Workload_Report.xlsx с тремя листами. Первый лист Data_Analysis должен содержать основные сравнительные данные по курсам. Второй лист Metrics должен резюмировать ключевые метрики. Третий лист Recommendations должен перечислять конкретные действия.

Лист Data_Analysis должен использовать курс как основное измерение и включать следующие колонки (именно с этими английскими заголовками): Course (название курса), Code (код курса), Enrollment (число записавшихся студентов), Avg_Score (средний балл) и Pass_Rate (доля сдавших). Отсортируй строки по колонке Course в алфавитном порядке.

Лист Metrics должен иметь две колонки Metric и Value. В колонке Metric используй именно такие английские метки: Total_Courses (общее число курсов = число строк на листе Data_Analysis), Total_Enrollment (сумма колонки Enrollment), Avg_Score (средний балл по курсам) и Avg_Pass_Rate (средняя доля сдавших).

Лист Recommendations должен иметь колонки Priority, Action и Course и перечислять приоритетные действия на основе анализа: в первую очередь — курсы с наибольшим отставанием от внешних эталонов (бенчмарков).
