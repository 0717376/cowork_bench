Вы — финансовый аналитик-исследователь, готовящий исследовательский отчёт FinTech, который объединяет анализ реальных рыночных данных с академической литературой об эффективности рынка. Ваша цель — проанализировать динамику акций, изучить релевантные научные статьи, провести статистические тесты и распространить результаты среди исследовательского комитета.

Начните с чтения файла Research_Brief.pdf в вашей рабочей директории, в котором изложены исследовательские вопросы об эффективности рынка и статистическая методология, которую нужно применить. Также прочитайте файл analysis_config.json, в котором указаны акции для анализа и уровень значимости для проверки гипотез.

Из сервиса финансовых данных получите исторические данные о ценах для SBER.ME (Сбербанк) и LKOH.ME (Лукойл). Для каждой акции получите всю доступную историю цен, включая ежедневные цены закрытия. Также получите базовую информацию об акции, включая название компании и отраслевую классификацию (сектор).

Выполните поиск в базе академической литературы по статьям, связанным с эффективностью рынка и алгоритмической торговлей. Ищите исследования о гипотезе эффективного рынка (Efficient Market Hypothesis), предсказуемости доходности акций и количественных торговых стратегиях. Соберите не менее четырёх релевантных статей с их названиями, авторами, годами публикации и краткими описаниями их релевантности.

С помощью инструментов командной строки напишите и выполните Python-скрипт с именем market_analysis.py в вашей рабочей директории. Скрипт должен выполнять статистическую проверку гипотез по данным о ценах акций. Рассчитайте ежедневную доходность для каждой акции. Проведите тест на автокорреляцию (autocorrelation) доходности, чтобы проверить, предсказывают ли прошлые доходности будущие, что было бы свидетельством против слабой формы эффективности рынка. Также выполните тест на нормальность (normality) распределений доходности. Для каждого теста зафиксируйте значение тестовой статистики, p-value и решение о том, отвергается ли нулевая гипотеза при заданном уровне значимости. Уровень значимости, который нужно использовать, — 0.05. Резюмируйте выводы, указав, предоставляют ли данные свидетельства за или против эффективности рынка.

Создайте книгу Excel с именем FinTech_Research_Report.xlsx в вашей рабочей директории с четырьмя листами. Первый лист с именем Market_Data должен содержать столбцы: stock symbol, company name, sector, average closing price, total return percentage за весь период и annualized volatility. Включите одну строку для SBER.ME и одну для LKOH.ME. Второй лист с именем Academic_Papers должен содержать столбцы: paper title, authors, publication year и relevance description. Включите не менее четырёх статей. Третий лист с именем Statistical_Tests должен содержать столбцы: test name, test statistic value, p-value и conclusion о том, отвергается ли нулевая гипотеза. Включите строки для теста на автокорреляцию и теста на нормальность для каждой акции. Четвёртый лист с именем Research_Summary должен содержать столбцы: finding description, supporting evidence и practical implication. Включите не менее трёх ключевых выводов, синтезирующих анализ рыночных данных и обзор академической литературы.

Создайте документ Word с именем FinTech_Research_Report.docx в вашей рабочей директории.
