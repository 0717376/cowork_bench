Мне нужен отчёт по эффективности категорий для нашего интернет-магазина (InSales). Пожалуйста, выгрузите все данные о товарах и категориях из магазина и создайте Excel-отчёт с именем "Category_Performance.xlsx" в рабочей директории. Передавайте абсолютный путь к файлу .xlsx внутри рабочей директории.

Первый лист должен называться "Category Summary" со следующими столбцами: Category, Product_Count, Total_Units_Sold, Total_Revenue (округлено до 2 знаков после запятой) и Avg_Price (округлено до 2 знаков после запятой). Значение Total_Units_Sold должно браться из поля total_sales каждого товара, а Total_Revenue вычисляется умножением цены товара (price) на его total_sales и суммированием внутри каждой категории. Отсортируйте строки по Total_Revenue от наибольшего к наименьшему.

Добавьте второй лист с названием "Top Products", в котором перечислены топ-3 товара по total_sales из каждой категории. Столбцы должны быть: Category, Product_Name, Price, Total_Sales и Stock_Quantity. Отсортируйте данные сначала по Category, затем по Total_Sales по убыванию внутри каждой категории.

Когда отчёт будет готов, отправьте письмо на адрес sales-team@shop.com с темой "Category Performance Report". В теле письма укажите, какая категория имеет наибольшую выручку, а какая — наименьшую (и, возможно, требует внимания).
