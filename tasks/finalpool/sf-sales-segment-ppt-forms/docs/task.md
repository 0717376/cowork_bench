Директору по продажам нужна презентация о результатах клиентских сегментов и форма обратной связи, чтобы команда продаж могла высказаться по стратегическим приоритетам. Подключитесь к хранилищу данных ClickHouse (логическая база SALES_DW) и получите выручку и количество заказов с группировкой по клиентскому сегменту (SEGMENT).

Создайте презентацию PowerPoint с именем Segment_Performance.pptx в рабочей директории, содержащую не менее четырёх слайдов. На первом слайде должен быть заголовок "Customer Segment Performance Report". Второй слайд должен иметь заголовок "Revenue by Segment" и представлять количество заказов и суммарную выручку по каждому клиентскому сегменту. Третий слайд должен иметь заголовок "Key Insights" и показывать, какой сегмент лидирует по выручке, у какого больше всего заказов и как сегменты сопоставляются между собой. Четвёртый слайд должен иметь заголовок "Strategy Recommendations" и содержать не менее трёх конкретных рекомендаций для команды продаж на основе данных по сегментам.

Важно: при работе с инструментом pptx передавайте АБСОЛЮТНЫЙ путь к файлу Segment_Performance.pptx внутри рабочей директории агента (инструмент pptx не сохраняет текущую рабочую директорию).

После создания презентации создайте форму с названием "Sales Strategy Feedback", чтобы команда продаж могла поделиться мнением о приоритетах сегментов. Форма должна содержать не менее трёх вопросов: один — о том, какой клиентский сегмент должен получить наибольшие инвестиции в продажи (с четырьмя названиями сегментов в качестве вариантов ответа), один — о предпочтительном подходе к распределению ресурсов (с вариантами выбора) и один открытый вопрос с просьбой оставить дополнительные комментарии или предложения.

Наконец, отправьте письмо на адрес sales_director@company.com с темой, ссылающейся на отчёт о результатах сегментов, и кратким текстом, который резюмирует сегмент-лидер по выручке и суммарные значения выручки, а также упоминает, что форма обратной связи для команды создана.
