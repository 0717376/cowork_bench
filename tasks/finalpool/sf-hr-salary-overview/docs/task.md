Мне нужен подробный отчёт по анализу заработной платы для нашего HR-отдела. В рабочей директории есть PDF-файл Salary_Benchmarks.pdf, в котором содержатся годовые ориентиры по зарплате (benchmarks), утверждённые руководством для каждого подразделения. Сначала прочитай его.

Затем подключись к корпоративному хранилищу данных ClickHouse и выгрузи текущие данные по зарплатам сотрудников. Мне нужна разбивка по подразделениям с указанием того, сколько сотрудников в каждом подразделении, какие у них средняя, минимальная и максимальная зарплаты, и как они соотносятся с ориентирами.

Создай Excel-файл HR_Salary_Report.xlsx в рабочей директории с двумя листами. Первый лист должен называться "Department Analysis" со столбцами Department, Employee_Count, Avg_Salary (округлённая до 2 знаков после запятой), Min_Salary, Max_Salary, Benchmark из PDF, Variance (это Avg_Salary минус Benchmark, округлённая до 2 знаков) и Variance_Pct (это Variance, делённая на Benchmark и умноженная на 100, округлённая до 1 знака после запятой). Отсортируй по алфавиту по столбцу Department.

Второй лист должен называться "Summary" с двумя столбцами Metric и Value. Включи Total_Employees (всего сотрудников по всем подразделениям), Overall_Avg_Salary (взвешенное среднее по всем сотрудникам, округлённое до 2 знаков), Overall_Benchmark (простое среднее всех ориентиров подразделений), Departments_Above_Benchmark (сколько подразделений имеют положительную Variance) и Departments_Below_Benchmark (для остальных).

При записи Excel-файла указывай абсолютный путь к HR_Salary_Report.xlsx внутри рабочей директории.
