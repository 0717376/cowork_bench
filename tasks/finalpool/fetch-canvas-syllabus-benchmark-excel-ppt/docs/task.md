Управление по учебной работе университета проводит ежегодный обзор учебных курсов и хочет сравнить метрики курсов нашего университета с национальными бенчмарками. Сервис бенчмаркинга предоставляет национальные средние значения через REST API эндпоинт по адресу http://localhost:30203/api/benchmarks.json, который возвращает данные, включая среднюю по стране численность студентов (enrollment), количество заданий (assignments) и количество тестов (quizzes) для семи предметных областей (course discipline areas).

Наш университет предлагает 22 курса через систему управления обучением (LMS), распределённые по семи различным типам курсов (course types). Нам нужно понять, как наши курсы соотносятся с национальными стандартами по численности студентов, количеству заданий и количеству тестов.

Начните с того, что получите данные национальных бенчмарков из API эндпоинта. Затем извлеките полный список курсов из системы управления обучением университета (LMS). Для каждого курса зафиксируйте название курса, общую численность студентов (enrollment), а также подсчитайте количество заданий (assignments) и тестов (quizzes), связанных с ним. Сгруппируйте курсы по их типу курса (course type) — это часть названия курса до семестра и года в скобках, например «Основы финансов» из «Основы финансов (Осень 2013)» — и вычислите среднюю численность студентов, среднее количество заданий и среднее количество тестов для каждого типа курса.

Создайте Excel-файл с именем "Course_Benchmark_Analysis.xlsx" в рабочей директории (workspace) с тремя листами.

Первый лист с названием "National Benchmarks" должен содержать столбцы Course_Type, Discipline, National_Avg_Enrollment, National_Avg_Assignments и National_Avg_Quizzes, заполненные данными из API бенчмарков для всех семи типов курсов.

Второй лист с названием "Our Courses" должен перечислять все 22 отдельных курса со столбцами Course_Name, Course_Type, Enrollment, Assignment_Count и Quiz_Count.

Третий лист с названием "Comparison" должен содержать столбцы Course_Type, Our_Avg_Enrollment, National_Avg_Enrollment, Enrollment_Diff, Our_Avg_Assignments, National_Avg_Assignments, Assignment_Diff, Our_Avg_Quizzes, National_Avg_Quizzes и Quiz_Diff. Столбцы с разницей (Diff) вычисляются как наше среднее значение минус национальное среднее. Должно быть 7 строк — по одной на каждый тип курса.

Затем создайте презентацию PowerPoint с именем "Academic_Benchmark_Presentation.pptx" в рабочей директории (workspace). Презентация должна иметь титульный слайд с заголовком "Course Benchmark Analysis" и подзаголовком, упоминающим учебный год. Включите по одному слайду на каждый тип курса (7 слайдов), на которых название типа курса выступает заголовком, а также приведено краткое сравнение метрик численности студентов (enrollment) и заданий (assignments) с национальными бенчмарками. Добавьте финальный итоговый слайд, на котором указано, какие типы курсов превышают национальные бенчмарки по численности студентов, а какие — ниже их.

Наконец, отправьте письмо от academic-affairs@university.edu на адрес dean@university.edu с темой "Annual Course Benchmark Report". Тело письма должно быть на русском языке и резюмировать, какие типы курсов находятся выше национальных бенчмарков по численности студентов, а какие — ниже.

Сохраните все выходные файлы в рабочую директорию (workspace).

Важно: инструменты для работы с Excel (.xlsx) и PowerPoint (.pptx) могут не сохранять текущую рабочую директорию, поэтому при их вызове всегда передавайте АБСОЛЮТНЫЙ путь к файлу .xlsx/.pptx внутри рабочей директории агента (workspace).
