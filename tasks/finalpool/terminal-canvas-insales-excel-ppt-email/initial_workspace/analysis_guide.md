# Руководство по анализу поведения студентов при покупке электроники

## Методология

Этот анализ исследует связь между академической успеваемостью на курсах «Прикладная аналитика» и поведением при покупке электроники в университетском книжном магазине.

### Источники данных
- Записи о зачислении и работах студентов по курсам 1 и 2 (секции «Прикладная аналитика»)
- Данные о заказах и товарах магазина с фокусом на категориях «Электроника» и «Камеры»
- Сопоставление студентов и покупателей через реестр программы лояльности

### Шаги анализа
1. Рассчитать среднюю оценку за работы по каждому студенту для курсов «Прикладная аналитика»
2. Агрегировать покупки электроники и камер по каждому сопоставленному покупателю
3. Связать академические записи и записи о покупках с помощью реестра лояльности
4. Вычислить корреляцию Пирсона между средними оценками и суммарными тратами на электронику
5. Сформировать практические рекомендации на основе направления корреляции

### Требования к результатам
- Excel-книга с четырьмя листами анализа
- Презентация PowerPoint из пяти слайдов
- Письма с отчётами для руководства магазина и учебной части
