Рекомендации по группам продуктов на один приём пищи

Каждый приём пищи должен по возможности включать следующее число порций по группам продуктов:

Группа продуктовПорций на приём пищиПримеры
Злаки2-3 порцииРис, лапша, хлеб (1 порция = 1/2 чашки в готовом виде)
Белок1-2 порцииМясо, рыба, тофу, яйца (1 порция = 85 г)
Овощи2-3 порцииЛистовая зелень, корнеплоды, грибы (1 порция = 80 г)
Фрукты1-2 порцииСвежие или сушёные фрукты (1 порция = 80 г)

Руководство по велнес-оценке

При оценке рецептов по велнес-критериям учитывайте:

  • Больше овощей = выше балл
  • Меньше сахара = выше балл
  • Сбалансированные группы продуктов = выше балл
  • Цельные ингредиенты вместо переработанных = выше балл
  • Подходящий размер порций = выше балл
