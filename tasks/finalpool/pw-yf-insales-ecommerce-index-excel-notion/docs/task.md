Мне нужна помощь с анализом yf wc ecommerce. Есть внешние эталонные (бенчмарк) данные: посетите http://localhost:30328 и извлеките оттуда нужные метрики.

Затем проверьте наш интернет-магазин InSales, чтобы получить актуальные данные о товарах и заказах.

С помощью терминала создайте и запустите Python-скрипт с именем yf_wc_ecommerce_processor.py в рабочей директории. Скрипт должен читать собранные данные из создаваемых вами JSON-файлов, выполнять анализ и сохранять результат в файл yf_wc_ecommerce_results.json.

Создайте Excel-файл с именем Wc_Ecommerce_Index_Report.xlsx с тремя листами. Первый лист Data_Analysis должен содержать основные сравнительные данные с соответствующими столбцами. Второй лист Metrics должен суммировать ключевые метрики. Третий лист Recommendations должен перечислять конкретные рекомендуемые действия.

Лист Data_Analysis должен включать столбцы Category (категория товаров), Product_Count (количество товаров), Our_Avg_Price (наша средняя цена), Total_Sales (суммарные продажи), Market_Avg_Price (рыночная средняя цена из внешнего бенчмарка) и Price_Gap_Pct (отклонение нашей цены от рыночной в процентах). Price_Gap_Pct рассчитывается как (Our_Avg_Price - Market_Avg_Price) / Market_Avg_Price * 100. Отсортируйте данные по столбцу Category в алфавитном порядке.

Обратите внимание: названия категорий во внешнем бенчмарке (http://localhost:30328) приведены на английском языке, а в нашем магазине InSales — на русском. Сопоставьте их по смыслу (например, Electronics — Электроника, Audio — Аудио, Cameras — Камеры, Home Appliances — Бытовая техника, "TV & Home Theater" — "ТВ и домашний кинотеатр", Watches — Часы). В итоговом отчёте используйте названия категорий так, как они хранятся в магазине InSales.

Лист Metrics должен содержать два столбца Metric и Value, суммирующие общие количества, средние значения и ключевые статистики (например, Total_Categories, Total_Products, Avg_Price_Gap).

Лист Recommendations должен перечислять приоритетные действия на основе анализа отклонений; используйте столбцы Priority, Action и Category.

Создайте в Teamly страницу с заголовком "Yf Wc Ecommerce Dashboard" с кратким резюме анализа (упомяните категории и величину ценового отклонения).

Все выходные файлы (Wc_Ecommerce_Index_Report.xlsx, yf_wc_ecommerce_processor.py, yf_wc_ecommerce_results.json и любые промежуточные JSON-файлы) сохраняйте в рабочей директории по абсолютному пути.
