Я формирую базу знаний нашего университета и хочу собрать структурированный каталог курсов на основе данных из системы управления обучением (Canvas). Найди, пожалуйста, все курсы осеннего семестра 2014 года (Fall 2014) — это те, у которых код курса заканчивается на 2014J.

По каждому курсу мне нужны: название курса, код курса, дата начала и дата окончания, общее число зачисленных студентов (считаем только зачисления студентов, преподавателей не учитываем), количество заданий и список преподавателей (учителей, зачисленных на курс) с их именами и адресами электронной почты.

Создай в нашей базе знаний (пространство Teamly) отдельную страницу wiki для каждого курса. Заголовок страницы должен быть в формате "[Course Code] - [Course Name]". Содержимое страницы должно включать в виде текста следующую информацию: код курса, семестр (Fall 2014), даты начала и окончания в формате YYYY-MM-DD, число зачисленных студентов, количество заданий и список всех преподавателей с их именами и email-адресами. Создавай страницы в алфавитном порядке по коду курса.

Также создай файл Excel под именем course_catalog.xlsx в рабочем каталоге с двумя листами. Первый лист должен называться "Courses" со столбцами Course_Code, Course_Name, Start_Date (формат YYYY-MM-DD), End_Date (формат YYYY-MM-DD), Student_Count, Assignment_Count и Instructor_Names (список имён преподавателей через запятую, отсортированный по алфавиту). Строки отсортируй по Course_Code по алфавиту. Второй лист должен называться "Summary" с двухколоночной разметкой, где в столбце A находятся метки, а в столбце B — значения. Строки должны быть: Total_Courses, Total_Students (сумма числа студентов по всем курсам), Total_Assignments (сумма количества заданий по всем курсам), Largest_Course (код курса с наибольшим числом студентов), Smallest_Course (код курса с наименьшим числом студентов) и Most_Assignments (код курса с наибольшим количеством заданий).

Важно: при сохранении course_catalog.xlsx через инструмент Excel укажи АБСОЛЮТНЫЙ путь к файлу внутри рабочего каталога, иначе файл может сохраниться не туда.
