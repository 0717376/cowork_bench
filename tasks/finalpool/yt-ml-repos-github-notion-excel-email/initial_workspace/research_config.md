Руководство по каталогизации исследований

Назначение
Это руководство объясняет, как каталогизировать ML-ресурсы исследований с канала ML Tech Reviews.
У канала есть тщательно отобранный плейлист из 7 видео, каждое из которых посвящено конкретному open-source ML-проекту.

Метаданные для видео
Для каждого видео фиксируйте: уникальный идентификатор видео, название (title), основную ML-тему или технику,
URL репозитория, найденный в описании видео, дату публикации и число просмотров.

Метаданные для статей
Для каждой статьи, связанной с темой видео, фиксируйте: идентификатор статьи, название, авторов
(в виде строки, разделённой запятыми), дату публикации, тему, к которой она относится,
и идентификатор видео, с которым она связана.

Связывание видео и статей
Сопоставляйте статьи с видео по теме. Используйте файл topic_mapping.json для списка ожидаемых тем.
У каждой темы должно быть хотя бы одно видео и желательно одна или несколько связанных статей.

Структура базы знаний
Создайте единую страницу-хаб со вступлением, а затем в теле страницы — структурированный каталог всех элементов
(видео и статей) с полями Name, Type (Video или Paper), Topic, URL и Status.

Лист Summary
Лист Summary должен агрегировать по каждой теме: сколько видео, сколько статей
и сколько различных URL репозиториев было найдено.
