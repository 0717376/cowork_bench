Мне нужно создать комплексную базу знаний для нашей исследовательской группы, изучающей большие языковые модели. Начни с поиска недавних научных статей по таким темам, как "large language models", "prompt engineering" и "in-context learning". Найди не менее 5 релевантных статей.

Затем используй репозиторий статей, чтобы скачать и прочитать полное содержание наиболее релевантных статей. Проанализируй их исходный код LaTeX, когда он доступен, чтобы извлечь подробные разделы с описанием методологии.

Используй терминал, чтобы создать и запустить Python-скрипт research_synthesizer.py в рабочей директории. Скрипт должен читать papers_metadata.json и paper_contents.json (сначала создай оба файла), извлекать ключевые методы, строить карту цитирований, рассчитывать оценки релевантности и выводить research_synthesis.json.

Создай файл Excel под названием Research_Knowledge_Base.xlsx с тремя листами. Первый лист Paper_Catalog должен содержать колонки Paper_ID, Title, Authors, Year, Category и Citation_Count, отсортированные по Citation_Count по убыванию. Второй лист Method_Comparison должен содержать колонки Method_Name, Paper_Source, Key_Innovation, Benchmark_Result и Applicability ("High", "Medium", "Low"). Третий лист Research_Gaps должен содержать колонки Gap_Area, Current_State, Opportunity и Priority ("Critical", "Important", "Nice-to-have"), с не менее чем 4 выявленными пробелами.

Важно: при работе с Excel-инструментом всегда указывай АБСОЛЮТНЫЙ путь к файлу Research_Knowledge_Base.xlsx внутри рабочей директории, чтобы файл создавался именно в ней.

Создай в Teamly страницу с заголовком "LLM Research Hub" и заголовком-секцией "Large Language Model Research Dashboard". Добавь абзацы, охватывающие обзор исследовательского ландшафта, краткое описание ключевых статей, сравнение методологий и выявленные исследовательские пробелы с рекомендациями для будущей работы.
