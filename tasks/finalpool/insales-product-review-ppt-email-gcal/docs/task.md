Вы — менеджер по продуктовой аналитике. Ваша задача — проанализировать отзывы покупателей о товарах по всем категориям в интернет-магазине InSales и подготовить презентацию для продуктовой команды.

Сначала получите все отзывы о товарах из магазина InSales. Для каждой категории товаров — Аудио (audio), Камеры (cameras), Электроника (electronics), Наушники (headphones), Колонки (speakers), ТВ и домашний кинотеатр (TV and Home Theater) — вычислите средний рейтинг и общее количество отзывов. Также определите топ-5 товаров с лучшими отзывами и топ-5 товаров с худшими отзывами по среднему рейтингу (для списка худших учитывайте только товары, у которых не меньше 2 отзывов).

Названия категорий в магазине заданы на русском языке (Аудио, Камеры, Электроника, Наушники, Колонки, ТВ и домашний кинотеатр) — используйте те названия, которые реально вернёт магазин.

Затем создайте файл презентации PowerPoint с именем Product_Review_Report.pptx в рабочей директории. Презентация должна содержать не менее 5 слайдов со следующей структурой: слайд 1 — титульный слайд с заголовком "Product Quality Review - Q1". Слайд 2 — "Category Rating Overview" (Обзор рейтингов по категориям) со средним рейтингом для каждой категории товаров. Слайд 3 — "Top 5 Best Reviewed Products" (Топ-5 товаров с лучшими отзывами) со списком товаров с наивысшим средним рейтингом. Слайд 4 — "Top 5 Worst Reviewed Products" (Топ-5 товаров с худшими отзывами) со списком товаров с наименьшим средним рейтингом. Слайд 5 — "Recommendations" (Рекомендации) минимум с двумя конкретными рекомендациями на основе данных отзывов.

ВАЖНО: при создании/сохранении .pptx через инструмент презентаций указывайте АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории (инструмент презентаций не сохраняет текущую рабочую директорию).

Далее создайте событие в Google Calendar с названием "Product Quality Review Meeting" на дату через 7 дней от сегодняшнего дня, продолжительностью один час. Обязательно добавьте описание встречи.

Наконец, отправьте сводку по презентации на адрес product-team@store.example.com с адреса analytics@store.example.com. Тема письма должна быть "Product Quality Review Report". В теле письма кратко изложите ключевые выводы, включая то, у какой категории самый высокий и самый низкий средний рейтинг.

Дополнительный контекст по методике анализа отзывов смотрите в файле Review_Analysis_Guidelines.pdf в вашей рабочей директории.
