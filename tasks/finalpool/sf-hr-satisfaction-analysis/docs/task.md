Мне нужен анализ удовлетворённости сотрудников по отделам из нашего корпоративного хранилища данных HR в ClickHouse. Включи как оценки удовлетворённости работой и баланса между работой и личной жизнью, так и оценки эффективности.

Создай файл Excel с именем HR_Satisfaction_Report.xlsx с двумя листами. Лист «Satisfaction Analysis» должен содержать столбцы Department, Avg_Satisfaction, Avg_Work_Life_Balance, Avg_Rating (все округлены до 2 знаков после запятой) и Employees (количество сотрудников). Отсортируй по Avg_Satisfaction по убыванию.

Лист «Summary» должен содержать Overall_Satisfaction и Overall_WLB как взвешенные средние, а также названия Happiest_Dept и Least_Happy_Dept.

Также создай документ Word с именем Satisfaction_Summary.docx с текстовым описанием тенденций удовлетворённости сотрудников. При сохранении документа Word передавай инструменту АБСОЛЮТНЫЙ путь к .docx внутри рабочей директории агента (Word MCP не сохраняет текущий рабочий каталог).
