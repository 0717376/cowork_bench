Ты — контент-стратег в сфере технологий, отвечающий за отслеживание обучения твоей команды через популярный канал по программированию и технологиям под названием Fireship. Твоя задача — проанализировать историю публикаций канала и метрики эффективности, хранящиеся в видеобазе команды, и подготовить структурированный отчёт.

Начни с запроса к видеобазе, чтобы получить все видео, опубликованные каналом Fireship. Для каждого видео доступны дата публикации, число просмотров (view count) и число лайков (like count).

Рассчитай помесячную разбивку активности канала. Для каждого календарного месяца, в котором было опубликовано хотя бы одно видео, вычисли количество опубликованных видео, среднее число просмотров (округлённое до целого), среднее число лайков (округлённое до целого) и суммарное число просмотров за этот месяц.

Создай в своём рабочем каталоге файл-таблицу с именем Fireship_Monthly_Stats.xlsx. Этот файл должен содержать два листа.

Первый лист должен называться Monthly_Stats. В нём должна быть одна строка на каждый месяц со следующими колонками в указанном порядке: Month (в формате YYYY-MM), Video_Count, Avg_Views, Avg_Likes и Total_Views. Отсортируй строки по колонке Month в порядке возрастания (хронологически).

Второй лист должен называться Summary. В нём должно быть ровно две колонки с именами Label и Value и следующие строки в указанном порядке: Total_Videos (общее число видео по всем месяцам), Total_Months_Active (сколько различных месяцев имели хотя бы одно видео), Peak_Month (строка месяца в формате YYYY-MM с наибольшим Video_Count), Peak_Month_Videos (число видео в этом пиковом месяце), Best_Avg_Views_Month (строка месяца в формате YYYY-MM с наибольшим Avg_Views) и Best_Avg_Views (среднее число просмотров для этого лучшего месяца как округлённое целое).

После создания таблицы создай в командной базе знаний Teamly страницу с заголовком ровно «Fireship Channel Analysis 2024-2025». Содержимое страницы должно представлять собой текстовый абзац, кратко резюмирующий ключевые выводы: общее число видео, число активных месяцев, пиковый месяц публикаций и месяц с лучшим средним числом просмотров.

Наконец, отправь письмо на адрес analytics@company.com с темой «Fireship Channel Analysis Complete». В теле письма кратко резюмируй основные выводы своего анализа.
