# Шаблон документа рецензирования конференции

## Структура

Документ рецензирования должен следовать этой структуре. Названия разделов можно оформлять на русском языке (рекомендуемые русские варианты указаны в скобках).

### Overview (Обзор)
Краткий абзац, описывающий процесс рецензирования, количество рассмотренных статей и общие темы.

### Per-Paper Review (Покритериальный обзор)
Для каждой статьи укажите:
- Название статьи в качестве заголовка подраздела
- Абзац "Сильные стороны" (Strengths), отмечающий, что в статье сделано хорошо
- Абзац "Слабые стороны" (Weaknesses), отмечающий области для улучшения
- Разбивку оценок: каждый критерий и его балл
- Итоговый балл (Total_Score)

### Comparative Analysis (Сравнительный анализ)
Обсуждение, сравнивающее методологические подходы статей, с указанием пересечений, взаимодополняющих техник и относительных преимуществ.

### Recommendations (Рекомендации)
Итоговый список каждой статьи с её рекомендацией (Accept/Revise/Reject) и обоснованием в одном предложении.
