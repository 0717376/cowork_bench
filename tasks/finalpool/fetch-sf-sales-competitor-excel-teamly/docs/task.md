Мне нужна помощь с анализом конкурентов по продажам (sf competitor analysis). Доступны внешние эталонные данные (external benchmark) по объёму рынка в разрезе регионов — их нужно получить по адресу http://localhost:30335/api/data.json и извлечь релевантные метрики (поле `market_size_m` для каждого региона `region`).

Затем выгрузите наши внутренние данные из корпоративного хранилища данных ClickHouse (база SALES_DW, схема sf_data) для сравнения. По таблице заказов `SALES_DW__PUBLIC__ORDERS`, соединённой с клиентами `SALES_DW__PUBLIC__CUSTOMERS` по `CUSTOMER_ID`, посчитайте в разрезе региона (`REGION`): количество заказов (Order_Count) и суммарную выручку (Revenue = сумма `TOTAL_AMOUNT`).

С помощью терминала создайте и запустите в рабочей директории Python-скрипт `sf_competitor_processor.py`, который читает собранные данные из создаваемых вами JSON-файлов, выполняет анализ и формирует на выходе `sf_competitor_results.json`.

Создайте файл Excel с именем `Sales_Competitor_Report.xlsx` с тремя листами.

Лист `Data_Analysis` должен содержать основную таблицу сравнения со следующими колонками:
- `Region` — регион (основное измерение);
- `Order_Count` — количество заказов из ClickHouse;
- `Revenue` — выручка из ClickHouse;
- `Market_Size_M` — внешний эталон объёма рынка для региона, полученный из JSON;
- `Market_Penetration_Pct` — проникновение на рынок, вычисляется как `Revenue / Market_Size_M * 100`.

Отсортируйте строки `Data_Analysis` по алфавиту по основному измерению (`Region`).

Лист `Metrics` должен иметь две колонки `Metric` и `Value` и суммировать ключевые показатели. Включите как минимум строки: `Total_Regions` (число регионов), `Total_Revenue` (сумма Revenue по всем регионам), `Total_Orders` (сумма Order_Count) и `Avg_Market_Penetration` (среднее значение Market_Penetration_Pct).

Лист `Recommendations` должен перечислять приоритетные действия на основе анализа разрывов и иметь колонки `Priority`, `Action`, `Region`. В первую очередь сосредоточьтесь на регионах с наименьшим проникновением на рынок (наибольшими разрывами).

Создайте в Teamly страницу с заголовком "Sf Competitor Dashboard" с кратким резюме анализа. В тексте упомяните хотя бы один регион и сделанный вывод о разрыве/проникновении.

Важно: при работе с Excel и при запуске скрипта указывайте АБСОЛЮТНЫЕ пути к файлам внутри рабочей директории агента.
