# Руководство по анализу — Sf Competitor

## Методология
Этот анализ сопоставляет внутренние метрики с внешними эталонами (external benchmarks).
Внутренние метрики (Order_Count, Revenue) берутся из хранилища ClickHouse (база SALES_DW).
Внешний эталон объёма рынка (Market_Size_M) берётся из benchmark API.
Проникновение на рынок: Market_Penetration_Pct = Revenue / Market_Size_M * 100.
Чем выше проникновение, тем сильнее наша позиция в регионе.

## Приоритетные действия
В первую очередь занимайтесь регионами с наименьшим проникновением (наибольшим разрывом).
Регионы в пределах 5% от целевого показателя считаются «в норме».

## Отчётность
Во всех отчётах указывайте источники данных, методологию и уровни доверия.
