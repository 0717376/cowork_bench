Я руководитель научно-исследовательской лаборатории и готовлю годовой отчёт о цитируемости публикаций. Мне нужно, чтобы вы собрали данные о научных статьях, запустили скрипт анализа и свели результаты в Google-таблицу.

Во-первых, найдите в академической базе данных статьи исследователей, работающих в области искусственного интеллекта и машинного обучения. Просмотрите научную базу (scholarly), чтобы найти все доступные статьи. Для каждой найденной статьи соберите название (title), авторов (authors), количество цитирований (citation_count) и год (year).

Во-вторых, определите всех уникальных авторов по всем статьям и вычислите для каждого автора суммарное число цитирований, число статей и среднее число цитирований на статью. Для этого сохраните собранные данные о статьях в JSON-файл с именем `papers.json` в вашей рабочей директории. Каждая запись должна содержать ключи `"title"`, `"authors"` (список объектов с ключом `"name"`), `"citation_count"` (целое число) и `"year"` (целое число).

В-третьих, в вашей рабочей директории уже находится скрипт `citation_analysis.py`. Запустите его, передав файл `papers.json` в качестве входных данных, чтобы вычислить статистику по авторам. Скрипт читает JSON-файл и выводит статистику авторов в формате JSON в stdout. Сохраните этот вывод для дальнейшего использования в таблице. При запуске скрипта и при работе с файлом `papers.json` указывайте абсолютный путь к файлу внутри вашей рабочей директории, чтобы избежать проблем с текущим рабочим каталогом.

В-четвёртых, создайте Google-таблицу с названием `Citation Impact Analysis`, содержащую два листа. Первый лист должен называться `Author Rankings` со столбцами `Author Name`, `Total Citations`, `Paper Count` и `Avg Citations Per Paper`. Заполните его статистикой авторов, отсортированной по суммарному числу цитирований по убыванию. Второй лист должен называться `Paper Details` со столбцами `Paper Title`, `Authors`, `Citation Count` и `Year`. Заполните его всеми статьями, отсортированными по числу цитирований по убыванию. Для столбца `Authors` на листе `Paper Details` объедините имена авторов через запятую с пробелом.

Убедитесь, что все данные точны и правильно отсортированы.
