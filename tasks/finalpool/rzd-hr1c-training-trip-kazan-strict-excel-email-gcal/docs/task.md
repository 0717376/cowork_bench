Вы — HR-менеджер российской компании, организующий командировку группы сотрудников на корпоративный тренинг в Казани 17 марта 2026 года. В рабочей директории лежат файл конфигурации `training_config.json` и гайд `hr_query_guide.md`. У вас есть доступ к HR-системе 1С:ЗУП, билетной системе РЖД, табличному редактору, почте и общему календарю.

# 1. Подбор участников

Выгрузите из HR-системы данные о сотрудниках. Возьмите сотрудников из отделов «Продажи» и «Маркетинг» со стажем не менее трёх лет. Размер группы — ровно пять человек, отбирать **наиболее опытных** (`ORDER BY YEARS_EXPERIENCE DESC, EMPLOYEE_ID ASC LIMIT 5`). По каждому участнику зафиксируйте: табельный номер, ФИО, отдел, стаж, годовой оклад, корпоративный email.

# 2. Поезда

Тренинг проходит в Казани, начало в 13:00 по местному времени. Команда едет утром и возвращается тем же вечером.

- **Туда** — Москва-Казанская → Казань-Пассажирская, утренний поезд серии «Стриж» с прибытием **до начала тренинга** (т.е. до 13:00).
- **Обратно** — Казань-Пассажирская → Москва-Казанская, вечерний поезд с прибытием в Москву **в тот же день** (не на следующий день).

Получите маршрут (промежуточные остановки) и цену билета «Эконом» для обоих рейсов.

# 3. Excel-отчёт

Создайте `Training_Travel_Report.xlsx` с тремя листами.

**Лист `Employees`** — колонки `Employee_ID`, `Name`, `Department`, `Years_Experience`, `Salary`, `Email`, `Training_Eligible`. По одной строке на каждого из пяти участников, `Training_Eligible` = «Да».

**Лист `Travel_Plan`** — колонки `Employee_Name`, `Train_No`, `Direction`, `Departure_Time`, `Arrival_Time`, `Seat_Class`, `Ticket_Price_RUB`. По две строки на сотрудника (туда и обратно) — итого 10 строк. `Direction` принимает значения `Туда` и `Обратно`. Время отправления и прибытия — в формате `HH:MM`.

**Лист `Budget_Summary`** — колонки `Item`, `Count`, `Unit_Price_RUB`, `Total_RUB`. Ровно три строки:
- `Билеты туда` — 5 билетов по цене эконома,
- `Билеты обратно` — 5 билетов по цене эконома,
- `Итого` — общая сумма по двум направлениям.

# 4. Календарь

Заведите два события на 17.03.2026:
- **«Корпоративный тренинг — выезд в Казань»** — начало за 30 минут до отправления утреннего поезда, окончание = время прибытия в Казань.
- **«Корпоративный тренинг — возврат в Москву»** — начало = время отправления вечернего поезда из Казани, окончание = время прибытия в Москву.

# 5. Письма

Отправьте сводное письмо на `training@hr.company.ru`. В тексте должны явно присутствовать: номера обоих поездов, дата поездки (17.03.2026 или «17 марта»), число участников (5) и итоговая сумма бюджета в рублях.

Дополнительно отправьте персональное письмо одному из пяти участников на его корпоративный адрес (из колонки `EMAIL`) — с подтверждением его деталей по поездке.
