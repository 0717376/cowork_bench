Нашей команде поддержки нужен комплексный обзор эффективности работы на основе данных по обращениям из нашего хранилища данных ClickHouse. Пожалуйста, выполните запросы к базе данных центра поддержки (SUPPORT_CENTER), чтобы собрать статистику по обращениям в разбивке по уровню приоритета и по типу проблемы. Для каждого уровня приоритета вычислите общее количество обращений, среднее время ответа в часах, округлённое до двух знаков после запятой, и средний балл удовлетворённости клиентов, округлённый до двух знаков после запятой. Также проверьте, какой процент обращений уложился в целевые сроки ответа по SLA, сравнив время ответа по каждому обращению с целевым временем ответа из политики SLA для соответствующего уровня приоритета. Кроме того, получите полный список агентов поддержки с указанием их команд и уровней квалификации.

Когда все данные собраны, создайте Excel-файл с именем Support_Analysis.xlsx в рабочей директории. При создании файла используйте абсолютный путь к Support_Analysis.xlsx внутри рабочей директории. Файл должен содержать четыре листа. Первый лист должен называться "Agent Roster" и содержать список всех агентов, отсортированный по имени, со столбцами Agent_Name, Team и Skill_Level. Второй лист должен называться "Priority Analysis" со столбцами Priority, Ticket_Count, Avg_Response_Hours, Avg_Satisfaction и SLA_Compliance_Pct, где процент соответствия отражает долю обращений, время ответа по которым уложилось в целевой срок SLA для данного приоритета. Третий лист должен называться "Issue Type Analysis" со столбцами Issue_Type, Ticket_Count, Avg_Response_Hours и Avg_Satisfaction, отсортированный по количеству обращений по убыванию. Четвёртый лист должен называться "Summary" с двумя столбцами Metric и Value, содержащий следующие строки: Total_Agents с количеством агентов, Total_Tickets с общим количеством обращений, Overall_Avg_Response_Hours с общим средним временем ответа, округлённым до двух знаков после запятой, и Overall_Avg_Satisfaction с общей средней удовлетворённостью, округлённой до двух знаков после запятой.

После создания Excel-файла также создайте общую таблицу, содержащую только сводные данные, чтобы другие заинтересованные лица могли её просмотреть. Таблица должна иметь заголовок "Support Performance Summary" и содержать единственный лист с именем "Summary" с теми же столбцами Metric и Value и теми же четырьмя строками, что и лист Summary в Excel-файле.
