# План презентации для совета директоров по D&I

## Рекомендуемая структура слайдов

### Слайд 1: Титульный слайд
- Заголовок: Отчёт о разнообразии и инклюзивности для совета директоров
- Подзаголовок: Анализ демографии персонала
- Дата

### Слайд 2: Краткая сводка для руководства (Executive Summary)
- Общая численность персонала
- Ключевые демографические показатели
- Общее соответствие отраслевым эталонам

### Слайд 3: Распределение по образованию
- Сравнение внутренних данных с отраслевым эталоном
- Выделение разрывов, превышающих 3 процентных пункта
- Выводы по отдельным отделам

### Слайд 4: Распределение по возрасту
- Разбивка по возрастным группам в сравнении с эталонами
- Выявление недопредставленных возрастных групп
- Последствия для трендов

### Слайд 5: Дополнительная демография
- Распределение по семейному положению
- Показатели баланса между работой и личной жизнью
- Метрики удовлетворённости работой

### Слайд 6: Рекомендации
- Приоритетные пункты действий
- Измеримые цели
- График следующего пересмотра

## Замечания по оформлению
- Используйте чёткий язык, основанный на данных
- Приводите как проценты, так и абсолютные числа там, где это уместно
- Выделяйте области, где разрывы превышают эталонные пороги
