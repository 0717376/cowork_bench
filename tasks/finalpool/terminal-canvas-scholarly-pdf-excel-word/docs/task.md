Учебно-методическая комиссия хочет оценить, насколько хорошо задания курса соответствуют актуальным научным исследованиям в области прикладной аналитики и алгоритмов. В рабочей директории находится PDF-файл Curriculum_Review_Guidelines.pdf, в котором описаны критерии проверки и методология оценивания. Также есть файл review_template.md, описывающий ожидаемую структуру итогового отчёта. Сначала прочитайте оба файла.

Подключитесь к системе управления обучением (LMS) и получите информацию о курсах и сведения о заданиях для двух потоков курса «Applied Analytics and Algorithms» (идентификаторы курсов 1 и 2). Для каждого задания зафиксируйте название задания, количество баллов и срок сдачи.

Выполните поиск в базе научной литературы по статьям, относящимся к прикладной аналитике, алгоритмам и образованию в области data science. Найдите не менее четырёх релевантных статей, которые могут быть полезны для совершенствования учебной программы. Для каждой статьи запишите название, авторов, год публикации и оценку релевантности от 1 до 10 — насколько тема статьи близка к заданиям курса.

Напишите и запустите Python-скрипт alignment_scorer.py в рабочей директории. Скрипт должен читать два создаваемых вами JSON-файла: assignments_data.json (содержит сведения о заданиях из обоих курсов) и papers_data.json (содержит информацию о научных статьях). Для каждого задания скрипт должен находить наиболее релевантную статью и вычислять оценку соответствия (alignment score) от 0 до 100 на основе пересечения ключевых слов между названием задания и названием/аннотацией статьи. Скрипт должен формировать на выходе файл alignment_matrix.json с результатами.

Создайте Excel-файл Curriculum_Research_Alignment.xlsx в рабочей директории с тремя листами.

Первый лист должен называться Course_Assignments со столбцами Course_ID, Course_Name, Assignment_Name, Points и Due_Date. Включите все двенадцать заданий из обоих курсов, отсортированные сначала по Course_ID, затем по Points по возрастанию. Сроки сдачи форматируйте как строки YYYY-MM-DD, либо оставляйте пустыми, если срок не задан.

Второй лист должен называться Related_Papers со столбцами Title, Authors (строка с именами авторов через запятую), Year и Relevance_Score (целое число от 1 до 10). Включите не менее четырёх статей, отсортированных по Relevance_Score по убыванию.

Третий лист должен называться Alignment_Matrix со столбцами Assignment_Name, Matched_Paper (название наиболее подходящей статьи) и Alignment_Score (от 0 до 100). Включите все двенадцать заданий, отсортированных по Alignment_Score по убыванию.

Создайте Word-документ Curriculum_Review_Report.docx в рабочей директории. Отчёт должен содержать заголовок, раздел с краткой сводкой (executive summary), где обсуждается общее соответствие учебной программы текущим исследованиям, раздел с описанием каждого курса и его заданий, раздел с обзором найденных релевантных статей, раздел с результатами анализа соответствия и заключение с рекомендациями по улучшению учебной программы. Отчёт можно писать на русском языке.

Важно: инструменты для работы с Excel и Word могут терять текущую рабочую директорию, поэтому при их вызове передавайте абсолютный путь к файлам .xlsx и .docx внутри рабочей директории.
