Руководство отдела кадров просит провести анализ оплаты труда по всем подразделениям компании. Пожалуйста, подключитесь к хранилищу данных ClickHouse и выгрузите записи о сотрудниках из базы данных кадров HR_ANALYTICS, затем рассчитайте ключевые показатели по зарплатам для каждого подразделения.

Создайте файл Excel с именем Compensation_Report.xlsx в рабочей директории с двумя листами. Первый лист должен называться "Department Compensation" и содержать столбцы Department, Headcount, Avg_Salary (средняя зарплата, округлённая до двух знаков после запятой), Min_Salary (округлённая до двух знаков после запятой), Max_Salary (округлённая до двух знаков после запятой) и Total_Payroll (сумма всех зарплат, округлённая до двух знаков после запятой). Отсортируйте строки по названию подразделения в алфавитном порядке.

Второй лист должен называться "Summary" со столбцами Metric и Value. Включите следующие записи: Total_Employees (общая численность по всем подразделениям), Total_Payroll (общий фонд оплаты труда по всем подразделениям, округлённый до двух знаков после запятой), Company_Avg_Salary (средняя зарплата по всей компании, округлённая до двух знаков после запятой), Highest_Avg_Department (подразделение с наибольшей средней зарплатой) и Lowest_Avg_Department (подразделение с наименьшей средней зарплатой).

Также создайте таблицу Google Sheet с заголовком "Workforce Compensation Dashboard" и листом "Overview", содержащим те же данные Department Compensation, чтобы отдел кадров мог совместно работать с ними онлайн.

Примечание: при работе с файлом Excel передавайте абсолютный путь к Compensation_Report.xlsx внутри рабочей директории.
