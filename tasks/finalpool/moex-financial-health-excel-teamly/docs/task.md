Вы — финансовый аналитик, который проводит оценку финансового здоровья компаний, торгующихся на Московской бирже (MOEX). В вашей рабочей директории нет заранее подготовленных файлов для этой задачи.

Подключитесь к источнику данных MOEX Finance и получите данные финансовой отчётности по отслеживаемым акциям. Для каждой компании вам нужны данные отчёта о прибылях и убытках (общая выручка, чистая прибыль) и данные баланса (совокупные активы, совокупные обязательства, собственный капитал акционеров) за последний годовой период.

Создайте Excel-файл с именем Financial_Health_Report.xlsx в вашей рабочей директории с двумя листами. Первый лист должен называться "Income Statement" со столбцами: Symbol, Total_Revenue, Net_Income, Profit_Margin_Pct (это Net_Income, делённое на Total_Revenue и умноженное на 100, округлённое до 1 знака после запятой). Включите строки для каждой из пяти акций: OZON.ME, PHOR.ME, ROSN.ME, VTBR.ME и YNDX.ME. Отсортируйте строки по столбцу Symbol в алфавитном порядке.

Второй лист должен называться "Balance Sheet" со столбцами: Symbol, Total_Assets, Total_Liabilities, Equity. Заполните его соответствующими данными баланса для тех же пяти акций, отсортированными по Symbol в алфавитном порядке.

Создайте в Teamly страницу с заголовком "Financial Health Dashboard", которая обобщает ключевые выводы вашего анализа. На странице должно быть указано, у каких компаний самая высокая выручка, лучшая рентабельность по чистой прибыли и самый сильный баланс — на основе полученных вами данных. Названия компаний пишите так, как они приводятся в источнике (например, Ozon Holdings, Rosneft, Yandex), либо используйте их тикеры.
