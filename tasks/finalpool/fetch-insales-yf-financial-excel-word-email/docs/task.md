Помоги мне с финансовым анализом нашего магазина. Доступны внешние рыночные (бенчмарк) данные — нужно получить их по адресу http://localhost:30333/api/data.json и извлечь релевантные метрики (средние рыночные цены по категориям).

Затем посмотри данные нашего магазина (InSales) — текущие товары и заказы.

Через терминал создай и запусти Python-скрипт wc_yf_finance_processor.py в рабочей директории. Скрипт читает собранные данные из созданных тобой JSON-файлов, выполняет анализ и сохраняет результат в wc_yf_finance_results.json.

Создай Excel-файл Yf_Financial_Report.xlsx с тремя листами. Первый лист Data_Analysis содержит основные сравнительные данные с релевантными колонками. Второй лист Metrics суммирует ключевые метрики. Третий лист Recommendations перечисляет приоритетные действия.

Лист Data_Analysis должен содержать колонки: Category (категория товара), Product_Count (число товаров), Our_Avg_Price (наша средняя розничная цена по данным магазина), Total_Sales (суммарные продажи), Market_Avg_Price (средняя рыночная цена из внешнего бенчмарка data.json) и Price_Gap_Pct — процентное отклонение нашей цены от рыночной, вычисленное как round((Our_Avg_Price - Market_Avg_Price) / Market_Avg_Price * 100, 1). Отсортируй строки по алфавиту по колонке Category. Лист Metrics должен иметь две колонки Metric и Value, суммирующие общие количества, средние значения и ключевую статистику (например Total_Categories, Total_Products, Avg_Price_Gap). Лист Recommendations должен перечислять приоритетные действия по итогам анализа разрывов с колонками Priority, Action, Category.

Также создай Word-документ Yf_Financial_Analysis.docx с разделами: краткое резюме (executive summary), ключевые выводы (key findings) и рекомендации (recommendations). При обращении к Word MCP передавай АБСОЛЮТНЫЙ путь к .docx-файлу внутри рабочей директории — у Word MCP теряется текущая рабочая директория.
