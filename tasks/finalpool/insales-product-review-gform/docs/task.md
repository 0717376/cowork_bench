Наша команда контроля качества интернет-магазина хочет собрать отзывы клиентов о товарах с самым низким рейтингом. Пожалуйста, изучите наш каталог товаров в InSales и определите пять товаров с самым низким средним рейтингом, учитывая только товары, у которых есть хотя бы какие-то оценки (рейтинг больше нуля). Для товаров с одинаковым рейтингом отдавайте предпочтение тем, у кого выше общее число продаж (total_sales), так как они отражают более широкий клиентский опыт.

Создайте опрос для сбора отзывов с названием "Отзыв о качестве продукции", чтобы собрать структурированную обратную связь от клиентов об этих товарах. У опроса должно быть описание, поясняющее, что мы собираем отзывы для улучшения качества продукции. Для каждого из пяти товаров с самым низким рейтингом добавьте вопрос с просьбой оценить удовлетворённость этим конкретным товаром по шкале от 1 до 5. Используйте название товара в каждом вопросе, чтобы респонденты понимали, о каком товаре идёт речь. Каждый вопрос должен быть обязательным.

Затем найдите, какие клиенты покупали любой из этих пяти низкорейтинговых товаров, изучив историю заказов и сопоставив идентификаторы товаров в позициях заказов (line items). Отправьте письмо каждому из этих клиентов с просьбой принять участие в опросе о качестве продукции. Используйте тему письма "We Value Your Feedback - Product Quality Survey" для каждого письма. В тексте письма должно упоминаться имя клиента и поясняться, что он получает этот опрос, потому что приобрёл один из товаров, по которым проводится проверка, а также содержаться просьба поделиться своим опытом.
