Мне нужен отчёт по всем возвращённым заказам из нашего магазина (InSales). Найдите каждый заказ со статусом "refunded" и соберите для него нужные данные: идентификатор заказа (Order_ID), имя клиента из платёжной (billing) информации, сумму возврата и дату создания заказа.

Создайте файл Excel с именем Refund_Report.xlsx в рабочей директории. Первый лист должен называться "Refund Details" со столбцами Order_ID, Customer_Name (имя и фамилия из billing, объединённые вместе: billing first_name и last_name), Total (сумма возврата, округлённая до 2 знаков после запятой) и Date (в формате YYYY-MM-DD). Отсортируйте строки по Order_ID по возрастанию.

Второй лист должен называться "Summary" с двумя столбцами Metric и Value и содержать три строки: Total_Refunds (количество возвращённых заказов), Total_Amount (сумма всех возвратов, округлённая до 2 знаков после запятой) и Avg_Refund (средняя сумма возврата, округлённая до 2 знаков после запятой).

Когда отчёт будет готов, отправьте сводное письмо на адрес manager@store.example.com от имени analytics@store.example.com с темой "Refund Analysis Report". В теле письма укажите общее количество возвращённых заказов, общую сумму возвратов и среднюю величину возврата.
