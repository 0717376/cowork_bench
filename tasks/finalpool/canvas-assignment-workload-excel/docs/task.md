Учебному отделу (академическому деканату) требуется анализ нагрузки студента по заданиям во всех курсах системы управления обучением (LMS). Запросите все курсы и их задания, подсчитайте количество отправленных работ (submissions) по каждому заданию и определите сроки сдачи заданий.

Создайте в рабочей директории файл Excel с именем Workload_Report.xlsx с двумя листами. На первом листе "Course Workload" должны быть столбцы: Course_Name; Total_Assignments — количество различных заданий в каждом курсе; Avg_Submissions — среднее число отправленных работ на одно задание в этом курсе, округлённое до 1 десятичного знака; Latest_Due_Date — самый поздний срок сдачи в формате YYYY-MM-DD. Отсортируйте по Total_Assignments по убыванию. На втором листе "Busiest Periods" перечислите все месяцы, в которых есть сроки сдачи заданий, со столбцами Month в формате YYYY-MM и Assignment_Count — сколько заданий нужно сдать в этом месяце. Отсортируйте по Assignment_Count по убыванию.

Для трёх самых загруженных месяцев создайте событие в календаре с названием "Study Session - [месяц]", запланированное на 15-е число этого месяца в 14:00 продолжительностью два часа. В описании события укажите, сколько заданий нужно сдать в этом месяце.

Также отправьте письмо на адрес academic.advisor@university.example.com с темой "Assignment Workload Analysis", в котором кратко изложите: общее число проанализированных курсов, общее число заданий по всем курсам и три самых загруженных месяца с количеством заданий в каждом.
