Инвестиционному комитету требуется ежеквартальный обзор результатов портфеля акций компании. Портфель состоит из пяти бумаг: Сбербанк (SBER.ME), Газпром (GAZP.ME), ЛУКОЙЛ (LKOH.ME), Т-Технологии (TCSG.ME) и Магнит (MGNT.ME). Перед началом работы изучите рекомендации по портфелю и параметры риска в рабочей директории (файлы Portfolio_Holdings.pdf и risk_parameters.json). Все цены и расчёты приведены в российских рублях.

Получите текущую информацию по акциям и исторические данные о ценах для всех пяти позиций из сервиса финансовых данных. Выполните скрипты финансового анализа, которые вычисляют совокупную доходность в процентах для каждой бумаги (на основе самой ранней и самой последней цены закрытия в доступной истории), годовую волатильность (стандартное отклонение дневных доходностей, умноженное на квадратный корень из 252) и упрощённый коэффициент Шарпа с использованием безрисковой ставки, указанной в файле параметров риска.

Создайте файл Excel с именем Quarterly_Portfolio_Review.xlsx с тремя листами. Первый лист «Performance_Summary» должен содержать столбцы Symbol, Name, Sector, Current_Price, Total_Return_Pct (округлено до 1 знака после запятой) и Annualized_Volatility_Pct (округлено до 1 знака после запятой), отсортированные по Symbol. Второй лист «Risk_Metrics» должен содержать столбцы Symbol, Sharpe_Ratio (округлено до 2 знаков после запятой), Max_Drawdown_Pct (округлено до 1 знака после запятой, рассчитанное как наибольшее снижение от пика до впадины), отсортированные по Symbol. Третий лист «Portfolio_Summary» должен содержать столбцы Metric и Value со строками: Total_Holdings (число 5), Best_Performer (символ с наибольшей совокупной доходностью), Worst_Performer (символ с наименьшей совокупной доходностью), Highest_Volatility (символ), Lowest_Volatility (символ), Avg_Total_Return_Pct (среднее всех совокупных доходностей, округлённое до 1 знака после запятой).

Напишите документ Word с именем Portfolio_Review_Report.docx, содержащий краткое резюме об общих результатах портфеля, раздел с анализом результатов отдельных бумаг и определением лучшей и худшей позиции, раздел анализа рисков с обсуждением волатильности и коэффициентов Шарпа, а также заключительные рекомендации по ребалансировке портфеля. Отчёт должен быть составлен на русском языке. При сохранении файла через инструмент Word передавайте АБСОЛЮТНЫЙ путь к .docx внутри рабочей директории агента.

Создайте онлайн-опрос с названием «Опрос мнения инвесторов, 1 кв. 2025» (Investor Feedback Survey) минимум с четырьмя вопросами: вопрос с рейтинговой оценкой общей удовлетворённости портфелем по шкале от 1 до 5, вопрос с выбором одного варианта о предпочтительной толерантности к риску, вопрос с кратким текстовым ответом для дополнительных комментариев и вопрос о предпочтительной частоте коммуникаций.

В завершение отправьте ежеквартальный обзор на адрес investment_committee@company.com с темой «Q1 2025 Portfolio Performance Review», включив в тело письма краткую сводку ключевых показателей результативности.
