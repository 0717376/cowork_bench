Мне нужна помощь с анализом набора студентов на курсы (canvas enrollment analysis) для нашей кафедры. Есть внешние эталонные (бенчмарк) данные аналитического института: зайди на http://localhost:30313 и извлеки оттуда нужные метрики.

Затем обратись к нашей системе управления обучением (Canvas LMS) за данными по курсам и студентам.

С помощью терминала создай и запусти Python-скрипт `course_enrollment_processor.py` в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые ты создашь, выполнять анализ и выводить результат в `course_enrollment_results.json`. Итоговый JSON должен быть получен именно работой этого скрипта (а не записан вручную).

Создай Excel-файл `Enrollment_Analysis_Report.xlsx` с тремя листами.

Первый лист `Data_Analysis` — основная таблица сравнения. Используй ровно следующие заголовки столбцов (на английском, без изменений): `Course`, `Code`, `Enrollment`, `Avg_Score`, `Pass_Rate`. Здесь `Course` — это основное измерение (название курса), `Code` — код курса, `Enrollment` — число набранных студентов (наша внутренняя метрика), `Avg_Score` — средний балл, `Pass_Rate` — доля сдавших в процентах (шкала 0–100); студент считается сдавшим, если его `current_score` >= 60. Отсортируй строки по алфавиту по основному измерению (столбец `Course`).

Второй лист `Metrics` — сводка ключевых метрик с ровно двумя столбцами: `Metric` и `Value`. Включи как минимум суммарные и средние показатели: `Total_Courses` (число курсов), `Total_Enrollment` (сумма по столбцу `Enrollment`), `Avg_Score` (средний балл) и `Avg_Pass_Rate` (средняя доля сдавших, в процентах 0–100).

Третий лист `Recommendations` — список приоритетных действий по результатам анализа разрывов (gap) с эталоном. Используй ровно следующие заголовки столбцов: `Priority`, `Action`, `Course`. В первую очередь сосредоточься на курсах с наибольшим отрицательным разрывом относительно эталона (см. методику ниже).

Методика анализа разрывов: разрыв (gap) = внутреннее значение − внешний бенчмарк. Положительный разрыв означает, что мы превышаем эталон, отрицательный — что отстаём. Сначала разбирай области с наибольшими отрицательными разрывами. Подробности — в файле `analysis_guide.md` в рабочей директории.

Также создай документ Word `Enrollment_Analysis_Analysis.docx` с разделами: краткое резюме (executive summary), ключевые выводы (key findings) и рекомендации (recommendations). ВАЖНО: при сохранении документа Word передавай инструменту АБСОЛЮТНЫЙ путь к файлу `Enrollment_Analysis_Analysis.docx` внутри рабочей директории агента (инструмент Word не сохраняет текущую директорию, и без абсолютного пути файл окажется не в той папке).

Наконец, создай Google-таблицу с названием "Canvas Enrollment Tracker" с ключевыми данными анализа.

Названия файлов, листов, столбцов, название Google-таблицы и имена `.py`/`.json` оставляй на английском в точности как указано выше.
