Мне нужен отчёт по статусам заказов для обзора операций нашего магазина InSales. Посмотрите все заказы в нашей системе и сгруппируйте их по текущему статусу.

Создайте Excel-файл с именем Order_Status_Report.xlsx с двумя листами. Первый лист с именем Status Breakdown должен содержать колонки Order_Status, Order_Count, Total_Value (округлить до 2 знаков после запятой) и Avg_Order_Value (округлить до 2 знаков после запятой), вычисленную как Total_Value, делённое на Order_Count. Отсортируйте по Order_Count по убыванию. Второй лист с именем Summary должен содержать колонки Metric и Value со следующими строками: Total_Orders, Total_Revenue (округлить до 2 знаков после запятой), Completion_Rate_Pct как процент завершённых заказов от всех заказов (округлить до 2 знаков после запятой), Most_Common_Status как статус с наибольшим числом заказов и Refund_Count как количество заказов со статусом refunded.

Создайте документ Word с именем Operations_Summary.docx с заголовком Order Status Summary и таблицей, показывающей полную разбивку по статусам с числом заказов и суммами выручки.
