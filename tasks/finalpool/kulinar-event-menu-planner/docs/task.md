Я организатор мероприятий и готовлю годовой корпоративный ужин. Мне нужно составить подробный план меню с учётом диетических ограничений, бюджетных рамок и нескольких блюд.

Начни с получения деталей мероприятия по адресу http://localhost:30234/api/event_details.json. В этом файле указаны название мероприятия, дата, количество гостей, бюджет на человека, разбивка по диетическим ограничениям и требуемые блюда (курсы). Внимательно запиши все детали — они задают рамки планирования меню.

Затем прочитай файл Dietary_Guidelines.pdf в рабочей директории. В этом документе описано, как учитывать различные диетические ограничения при планировании меню, включая стратегии замены продуктов и предотвращение перекрёстного загрязнения. Обрати внимание на ключевые рекомендации по вегетарианскому, безглютеновому питанию и аллергии на орехи.

Далее используй кулинарную базу через инструменты kulinar (например, `mcp_kulinar_getAllRecipes`, `mcp_kulinar_getRecipesByCategory`, `mcp_kulinar_getRecipeById`, `mcp_kulinar_recommendMeals`), чтобы подобрать подходящие рецепты для каждого требуемого блюда. Тебе нужны рецепты для закуски (appetizer — категория `закуска`), основного блюда (main — категория `горячее`) и десерта (dessert — категория `десерт`). Для каждого блюда выбери минимум два варианта рецептов: один обычный и один, который подходит под диетические ограничения (вегетарианский и безглютеновый). Учитывай стоимость и возможность масштабирования рецепта на 50 гостей. Бери реальные названия рецептов из базы kulinar (на русском языке) — ничего не выдумывай.

Также просмотри файл vendor_contacts.json в рабочей директории, чтобы найти адреса электронной почты координатора кейтеринга и поставщика.

Создай в рабочей директории книгу Excel с именем Event_Menu.xlsx с четырьмя листами. Первый лист назови "Menu Plan" с колонками Course, Recipe_Name, Serves, Cost_Per_Person и Dietary_Tags. Названия рецептов указывай на русском (из базы kulinar). В колонке Course используй английские маркеры: appetizer, main, dessert. Колонка Dietary_Tags должна указывать, какие ограничения учитывает каждый рецепт — используй английские теги Vegetarian, Gluten-Free, Nut-Free. Включи минимум два варианта на каждое блюдо. Второй лист назови "Ingredient List" с колонками Ingredient, Quantity_For_50, Unit и Estimated_Cost. Перечисли все ингредиенты для меню, рассчитанные на 50 гостей. Третий лист назови "Cost Summary" с колонками Item и Amount; включи строки Total_Food_Cost, Cost_Per_Person, Budget_Per_Person, Budget_Variance (бюджет минус фактическая стоимость) и Service_Fee_Estimate (15% от стоимости продуктов). Четвёртый лист назови "Dietary Accommodations" с колонками Restriction, Guest_Count, Menu_Options и Notes. Для каждого диетического ограничения из деталей мероприятия укажи количество затронутых гостей, какие блюда они могут есть, и любые особые примечания по приготовлению.

После создания файла Excel отправь письмо координатору кейтеринга на адрес catering@vendor.com с темой "Menu Plan for Annual Company Dinner". Письмо должно быть отправлено с адреса events@company.com и содержать краткое описание плана меню, итоговую оценку стоимости, количество гостей и необходимые диетические адаптации. Текст письма должен быть профессиональным и включать дату мероприятия.
