Я организую научно-просветительскую программу для школьников и хочу выстроить структурированный учебный план вокруг качественных научных видео. Для программы нужно запланировать сессии просмотра в нашем общем календаре, форму обратной связи для учащихся, расписание сессий, сохранённое в файл электронной таблицы, и письмо-уведомление участникам.

Из базы данных видео получите все видео с канала Veritasium, опубликованные в 2025 году. Из этого набора выберите топ-5 видео по числу просмотров.

Запланируйте сессию просмотра и обсуждения в общем календаре для каждого из пяти выбранных видео на следующие строго идущие подряд среды: 1 апреля 2026; 8 апреля 2026; 15 апреля 2026; 22 апреля 2026 и 29 апреля 2026. Каждая сессия должна проходить с 14:00 до 15:30. Название каждого события должно строго соответствовать шаблону: "Science Enrichment: " и затем название видео, усечённое максимум до 50 символов (если название длиннее 50 символов, обрежьте его на 50-м символе). В описании события должны быть указаны как число просмотров видео в виде целого числа без форматирования, так и длительность видео в секундах в виде целого числа без форматирования.

Создайте онлайн-форму опроса с названием "Science Enrichment Program Feedback", содержащую ровно четыре вопроса в указанном порядке. Первый вопрос должен быть вопросом с выбором одного варианта (multiple-choice) с названием "Which video topic interested you most?" и вариантами ответа из названий пяти выбранных видео (используйте для каждого варианта полное название). Второй вопрос должен быть вопросом с выбором одного варианта (multiple-choice) с названием "How would you rate the difficulty level?" и ровно тремя вариантами: Too Easy, Just Right, Too Hard. Третий вопрос должен быть вопросом с выбором одного варианта (multiple-choice) с названием "How often should we hold these sessions?" и ровно тремя вариантами: Weekly, Bi-weekly, Monthly. Четвёртый вопрос должен быть текстовым вопросом (со свободным ответом, не с выбором варианта) с названием "What other science topics would you like to explore?".

Создайте файл Excel с именем Viewing_Schedule.xlsx в вашей рабочей директории с единственным листом по имени Sessions. На листе должны быть следующие шесть колонок в указанном порядке: Session_Num (целые числа от 1 до 5), Date (дата сессии в формате YYYY-MM-DD), Video_ID (идентификатор видео), Title (полное название видео), View_Count (число просмотров в виде целого числа), Duration_Min (длительность видео в минутах, округлённая до одного знака после запятой). Каждая из пяти сессий занимает одну строку; строки отсортированы по Session_Num по возрастанию.

Отправьте письмо на адрес students@school.edu с темой "Science Enrichment Sessions Scheduled". В теле письма должны быть перечислены все пять дат просмотра и соответствующее каждой дате название видео.
