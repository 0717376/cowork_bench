Сначала найди все курсы осеннего семестра 2013 года. Это курсы, код которых оканчивается на "2013J". Для каждого из этих курсов найди задание (assignment), имя которого начинается с "Final Exam".

Для каждого курса, у которого есть задание Final Exam, собери следующую информацию: название курса, код курса, имя задания Final Exam вместе с количеством баллов (points possible) и сроком сдачи (due date), а также имя и email преподавателя. Преподаватель — это учитель (teacher), записанный на курс. Если у курса несколько преподавателей, используй первого по алфавиту по имени.

Для каждого Final Exam, у которого задан срок сдачи, создай учебные сессии в Google-календаре. Первая учебная сессия должна быть за два дня до даты экзамена, с 15:00 до 17:00 в часовом поясе America/New_York, с названием "Study: [Course Code] - Session 1", например "Study: DDD-2013J - Session 1". Вторая учебная сессия должна быть за один день до даты экзамена, с 10:00 до 12:00 в том же часовом поясе, с названием "Study: [Course Code] - Session 2". В описание каждой учебной сессии установи "Prepare for [Exam Assignment Name] in [Course Name]".

Создай Excel-файл с именем exam_review_plan.xlsx в рабочей директории, с единственным листом, названным "Exam Plan". Столбцы должны быть: Course_Name, Course_Code, Instructor_Name, Instructor_Email, Exam_Name, Points_Possible, Exam_Date (в формате YYYY-MM-DD, либо "TBD", если срок сдачи не задан), Study_Session_1 (дата и время первой учебной сессии в формате "YYYY-MM-DD HH:MM", либо "N/A", если дата экзамена отсутствует) и Study_Session_2 (тот же формат, что и Study_Session_1). Отсортируй строки по Course_Code по алфавиту.
