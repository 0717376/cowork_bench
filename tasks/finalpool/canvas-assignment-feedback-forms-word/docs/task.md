Вы — аналитик учебного курса «Основы финансов» (Foundations of Finance), который читается осенью 2013 года. В системе управления обучением этому курсу присвоен course ID 16.

Задача состоит из трёх частей.

Часть 1. Получите все задания курса 16 и соберите статистику по сдачам для каждого из них. Для каждого задания зафиксируйте: название задания, общее число сдач, средний балл (округлённый до 2 знаков после запятой), число поздних сдач и долю поздних сдач (число поздних сдач, делённое на общее число сдач, умноженное на 100, округлённое до 2 знаков после запятой).

Часть 2. Создайте опрос в сервисе форм с названием "Foundations of Finance - Assignment Feedback Survey" для сбора обратной связи студентов о заданиях. Опрос должен содержать ровно 5 вопросов, охватывающих следующие темы: общая нагрузка по заданиям, ясность формулировок заданий, время, требуемое на выполнение заданий, доступность поддержки со стороны преподавателя и общая удовлетворённость заданиями. Подберите подходящий тип вопроса для каждой темы. Формулировки вопросов можно писать на русском языке.

Часть 3. Создайте документ Word с именем Assignment_Analysis.docx в рабочей директории. В начале документа должен быть заголовок "Foundations of Finance (Fall 2013) - Assignment Analysis". Документ должен содержать таблицу со следующими колонками, строго в этом порядке: Assignment_Name, Total_Submissions, Avg_Score, Late_Submissions, Late_Rate(%). Отсортируйте строки по алфавиту по колонке Assignment_Name. Включите все задания, найденные в курсе.

Важно: при сохранении документа через MCP-сервер word передавайте ПОЛНЫЙ абсолютный путь к файлу Assignment_Analysis.docx внутри рабочей директории (значение workspace_dir из системного промпта + имя файла), иначе документ может быть записан вне рабочей директории.

После создания документа Word отправьте итоговое письмо на адрес instructor@financeou.example.com от имени analytics@university.example.com с темой "Assignment Analysis Report - Foundations of Finance (Fall 2013)". В теле письма должны быть указаны: общее число проанализированных заданий, задание с наивысшим средним баллом и задание с наибольшим числом поздних сдач. Также включите в тело письма ссылку на созданный опрос обратной связи.

В начальной рабочей директории находится файл Assignment_Guidelines.pdf — справочный документ с критериями оценивания, используемыми в этом курсе.
