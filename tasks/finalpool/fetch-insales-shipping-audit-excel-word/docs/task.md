Мне нужна помощь с аудитом доставки и ценообразования по нашему магазину InSales. Есть внешние эталонные (бенчмарк) данные: получите их по адресу http://localhost:30308/api/data.json и извлеките нужные показатели — это среднерыночные цены по категориям (`Market_Avg_Price`).

Затем посмотрите наш магазин InSales и соберите актуальные данные по товарам и заказам: для каждой категории товаров нужны количество товаров, наша средняя цена и суммарные продажи.

С помощью терминала создайте и запустите Python-скрипт `wc_shipping_processor.py` в рабочей директории. Скрипт читает собранные данные из создаваемых вами JSON-файлов, выполняет анализ и сохраняет результат в `wc_shipping_results.json`.

Создайте Excel-файл `Shipping_Audit_Report.xlsx` с тремя листами.

Лист `Data_Analysis` должен содержать основную сравнительную таблицу строго со следующими колонками (имена колонок оставьте на английском):
- `Category` — категория товара (используйте названия категорий из нашего магазина InSales, на русском);
- `Product_Count` — количество товаров в категории;
- `Our_Avg_Price` — наша средняя цена по категории;
- `Total_Sales` — суммарные продажи по категории;
- `Market_Avg_Price` — среднерыночная цена из эталонных данных;
- `Price_Gap_Pct` — процентное отклонение нашей цены от рыночной, рассчитанное как (Our_Avg_Price − Market_Avg_Price) / Market_Avg_Price × 100, округлите до одного знака.

Отсортируйте строки `Data_Analysis` по алфавиту по колонке `Category`.

Лист `Metrics` должен иметь две колонки `Metric` и `Value` и суммировать ключевые показатели: как минимум `Total_Categories` (число категорий, равное числу строк в `Data_Analysis`), `Total_Products` (сумма `Product_Count` по всем категориям) и `Avg_Price_Gap` (средний `Price_Gap_Pct`).

Лист `Recommendations` должен содержать приоритетные действия по итогам анализа разрывов цен с колонками `Priority`, `Action` и `Category`.

Также создайте документ Word `Shipping_Audit_Analysis.docx` с разделами: краткое резюме (executive summary / обзор), ключевые выводы и рекомендации. При работе с инструментом Word передавайте АБСОЛЮТНЫЙ путь к файлу `Shipping_Audit_Analysis.docx` внутри рабочей директории.
