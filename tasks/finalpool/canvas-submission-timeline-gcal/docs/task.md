Ты координатор учебного офиса университета. Тебе поручили проанализировать просроченные сдачи работ по курсам осеннего семестра 2013 года (Fall 2013) и создать в общем календаре напоминания о сроках сдачи заданий. Тебе доступны: система управления обучением (Canvas), общий календарь (Google Calendar), текстовый редактор Word и файловая система.

Сначала найди в Canvas все курсы осеннего семестра 2013 года. У этих курсов код заканчивается на «2013J». Всего должно быть шесть курсов: AAA-2013J, BBB-2013J, DDD-2013J, EEE-2013J, FFF-2013J и GGG-2013J.

По каждому курсу Fall 2013 проанализируй сдачи работ (submissions) и определи, сколько из них сданы с опозданием. Сдача считается просроченной, если её поле «late» равно true. Для каждого курса посчитай: общее число сдач, число просроченных сдач и процент просрочки (долю просроченных сдач от общего числа).

Создай в своей рабочей директории Word-документ с именем `Late_Submission_Report.docx`. Важно: при вызове word-инструмента передавай **абсолютный путь** до файла (рабочая директория указана в твоём системном промпте, шаблон вида `<workspace_dir>/Late_Submission_Report.docx`), иначе документ может оказаться в посторонней папке. В самом верху документа размести заголовок «Fall 2013 Late Submission Analysis». Затем включи отдельный раздел для каждого из шести курсов Fall 2013. В каждом разделе укажи код курса, название курса, общее число сдач, число просроченных сдач и процент просрочки. После разделов по курсам добавь итоговый сводный раздел (Итоги / Overall) с суммарными показателями по всем шести курсам.

Далее создай события в общем календаре (Google Calendar) для каждого задания курсов Fall 2013, у которого задан срок сдачи. Заголовок (summary) каждого события должен быть в формате «[Course_Code]: [Assignment_Name] Due» (например, «AAA-2013J: TMA 1752 Due»). Назначай каждое событие на дату срока сдачи соответствующего задания. Создавай события только для тех заданий, у которых установлен срок сдачи.
