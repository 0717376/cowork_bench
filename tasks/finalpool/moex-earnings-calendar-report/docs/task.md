Я веду портфель из пяти российских акций и хочу подготовить полную финансовую сводку для нашего комитета по инвестициям. Я отслеживаю следующие компании: Газпром, ЛУКОЙЛ, Магнит, МТС и Сбербанк. Используйте стандартные тикеры MOEX (формат XXXX.ME), чтобы найти информацию по ним.

Для каждой из этих пяти акций получите актуальную информацию о компании, включая краткое наименование компании (shortName), сектор, рыночную капитализацию, коэффициент trailing PE, коэффициент forward PE, дивидендную доходность, trailing EPS, forward EPS, общую выручку, темп роста выручки, маржу прибыли (profit margins) и рентабельность собственного капитала (return on equity).

Создайте файл Excel с именем Stock_Financial_Summary.xlsx в рабочей директории с двумя листами. Первый лист должен называться "Stock Overview" и содержать столбцы Ticker, Company_Name, Sector, Market_Cap, Trailing_PE, Forward_PE, Dividend_Yield и Trailing_EPS. Отсортируйте строки по алфавиту по столбцу Ticker. Для Market_Cap используйте необработанные числовые значения, а Dividend_Yield выражайте в виде десятичной дроби (например, 0.02 для 2 процентов). Если для какой-либо акции дивидендная доходность недоступна, оставьте соответствующую ячейку пустой.

Второй лист должен называться "Profitability Metrics" и содержать столбцы Ticker, Total_Revenue, Revenue_Growth, Profit_Margins, Return_On_Equity и Forward_EPS. Также отсортируйте строки по алфавиту по столбцу Ticker. Для Total_Revenue используйте необработанные числовые значения, а проценты выражайте в виде десятичных дробей.

Также создайте таблицу в нашем общем рабочем пространстве с названием "Portfolio Watch List" и листом "Summary". На этом листе разместите те же данные, что и на листе Stock Overview, чтобы команда могла работать с ними совместно. Столбцы должны точно соответствовать листу Stock Overview из файла Excel.
