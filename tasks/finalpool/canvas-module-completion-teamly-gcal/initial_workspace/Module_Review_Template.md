Шаблон обзора модулей (Module Review Template)

Creative Computing and Culture — Fall 2014

Назначение:
Этот шаблон помогает преподавательской команде проводить еженедельные сессии обзора модулей.
Каждая субботняя встреча должна охватывать перечисленные ниже пункты повестки.

Повестка:
1. Обзор статуса выполнения модулей (15 минут)
2. Обсуждение прогресса студентов и типичных трудностей (20 минут)
3. Задачи на предстоящую неделю (15 минут)
4. Планирование содержания следующего модуля (10 минут)

Чек-лист готовности модуля:
- Все элементы модуля доступны студентам
- Критерии оценивания заданий опубликованы и понятны
- Темы для обсуждения обновлены
- Видеоматериалы воспроизводятся корректно
- Вопросы тестов проверены на ясность формулировок

Определения статусов:
Not Started — модуль ещё не открыт для студентов
In Progress — модуль активен, студенты работают с ним
Completed — модуль завершён, оценки выставлены

Примечания:
После каждой встречи обновляйте трекер модулей в командном рабочем пространстве.
Фиксируйте любые проблемы или отзывы студентов для рассмотрения преподавателем.
