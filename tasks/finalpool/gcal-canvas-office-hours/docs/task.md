Я — ассистент преподавателя (TA) по курсу `Applied Analytics and Algorithms`, и мне нужно организовать консультации (office hours) на ближайшую неделю с 9 по 13 марта 2026 года. Пожалуйста, помогите мне со следующим.

Во-первых, найдите курс `Applied Analytics and Algorithms` в системе управления обучением (LMS). Найдите этот курс и получите список зачисленных студентов, собрав их имена и адреса электронной почты.

Во-вторых, проверьте форму с названием `Office Hours Booking`, где студенты уже отправили свои предпочтительные слоты для консультаций. В форме для каждого студента указаны: имя, email, предпочтительная дата (в диапазоне с 9 по 13 марта 2026 года), предпочтительный временной слот и тема, которую студент хочет обсудить.

В-третьих, создайте файл Excel с именем `Office_Hours_Schedule.xlsx` в рабочей директории с двумя листами.
- Первый лист должен называться `Bookings` и содержать по одной строке на каждый ответ из формы со столбцами `Student_Name`, `Student_Email`, `Preferred_Date`, `Preferred_Time_Slot` и `Topic`. Строки должны быть отсортированы сначала по `Preferred_Date`, затем по `Preferred_Time_Slot`.
- Второй лист должен называться `Summary` и содержать по одной строке на каждую дату, на которую есть записи, со столбцами `Date`, `Total_Bookings` и `Time_Slots`, где `Time_Slots` — это перечисленные через запятую различные временные слоты, забронированные на эту дату. Строки должны быть отсортированы по `Date`.

Важно: при создании файла `Office_Hours_Schedule.xlsx` укажите ПОЛНЫЙ абсолютный путь к нему внутри указанной выше рабочей директории, чтобы файл точно был сохранён в рабочей директории.

В-четвёртых, для каждой уникальной комбинации даты и временного слота из ответов формы создайте событие в календаре на соответствующую дату в соответствующий временной слот. Каждое событие должно длиться 30 минут. Название каждого события — `Office Hours: ` и далее одна из тем для этого слота. Если несколько записей приходятся на одни и те же дату и временной слот, выберите для названия любую из тем. Используйте часовой пояс `America/New_York`. В описании события перечислите имена всех студентов, забронировавших этот слот.

В-пятых, отправьте письмо-подтверждение каждому студенту, оставившему запись. Отправляйте письма от `ta@university.edu` с темой `Office Hours Confirmation`. В тексте письма укажите имя студента, его забронированную дату, временной слот и тему, а также краткое подтверждающее сообщение.
