Мне нужна помощь с анализом научных трендов (scholarly trend analysis). Наша российская исследовательская группа сопоставляет статьи из научной базы с внешними эталонными показателями. Внешние эталонные данные доступны на аналитической панели: зайдите на http://localhost:30324 и извлеките соответствующие метрики (таблица с колонками Research_Area, Publication_Count_2025, Avg_Citations, Trending_Score).

Источники данных:
1. Научная база (scholarly): получите статьи из arxiv (в базе есть 4 статьи с идентификаторами 2301.01234, 2302.05678, 2303.09012, 2304.03456 — их Paper_ID, Title и тематическая область Area).
2. Аналитическая панель http://localhost:30324: внешние эталонные показатели по областям исследований.

Сопоставьте каждую статью с её областью исследований (Research_Area) на панели и возьмите оттуда метрики: Citations = Avg_Citations этой области, Relevance_Score = Trending_Score этой области.

Используйте терминал, чтобы создать и запустить Python-скрипт под названием scholarly_trend_processor.py в рабочей директории. Скрипт должен читать собранные данные из создаваемых вами JSON-файлов, выполнять анализ и сохранять результат в scholarly_trend_results.json.

Создайте Excel-файл под названием Trend_Analysis_Report.xlsx с тремя листами.

Лист Data_Analysis должен содержать основную сопоставительную таблицу со следующими колонками (в первой строке, ровно эти заголовки на английском):
- Paper_ID — идентификатор статьи arxiv
- Title — название статьи
- Area — тематическая область
- Citations — число цитирований (из Avg_Citations соответствующей области на панели)
- Relevance_Score — оценка релевантности (из Trending_Score соответствующей области)
Отсортируйте строки по алфавиту по колонке Area. Лист должен содержать не менее 4 строк данных (по одной на каждую из 4 статей).

Лист Metrics должен иметь две колонки Metric и Value и содержать как минимум следующие строки (значения вычисляются из вашего же листа Data_Analysis):
- Total_Papers — общее число статей (количество строк данных в Data_Analysis)
- Avg_Citations — среднее значение по колонке Citations (округлённое)
- Top_Area — область с наивысшим Relevance_Score
- Avg_Relevance — среднее значение по колонке Relevance_Score (округлённое)

Лист Recommendations должен содержать колонки Priority, Action, Area и перечислять приоритетные действия (не менее 2 строк). Каждая строка должна ссылаться на область (Area), реально присутствующую в листе Data_Analysis.

Создайте презентацию PowerPoint под названием Trend_Analysis_Presentation.pptx минимум из 4 слайдов, освещающих ключевые выводы. В тексте слайдов упомяните хотя бы одну область исследований или название статьи из отчёта.

ВАЖНО: при сохранении Trend_Analysis_Report.xlsx и Trend_Analysis_Presentation.pptx через соответствующие MCP-инструменты передавайте АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории (excel/pptx MCP не сохраняет текущую рабочую директорию).
