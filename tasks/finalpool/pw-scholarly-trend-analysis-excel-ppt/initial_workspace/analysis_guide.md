# Руководство по анализу — Научные тренды (Scholarly Trend)

## Методология
Этот анализ сопоставляет статьи из научной базы (scholarly) с внешними
эталонными показателями (benchmarks) с аналитической панели по адресу
http://localhost:30324.
Для каждой статьи значения Citations и Relevance_Score берутся из
соответствующей области исследований (Research_Area) на панели:
Citations = Avg_Citations этой области, Relevance_Score = Trending_Score.

## Приоритетные действия
В первую очередь уделяйте внимание областям с наивысшей релевантностью
(Relevance_Score) и наибольшим числом цитирований (Citations).
Сформулируйте приоритетные действия (Recommendations) по этим областям.

## Отчётность
Во все отчёты включайте источники данных, методологию и уровни
достоверности.
