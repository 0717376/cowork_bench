# Руководство по планированию поездки на конференцию

## Обзор
Это руководство задаёт требования к формату вывода при планировании поездки на конференцию для 3 участников, едущих в Великий Новгород. Все требования должны строго совпадать с описанием задачи.

## Формат вывода Excel (файл Conference_Travel_Plan.xlsx)

### Лист: Outbound
Столбцы: Attendee, From_City, Train_Code, Depart_Time, Arrive_Time, Duration, Seat_Class, Price_RUB
По одной строке на каждого участника (поездка туда, 12.03.2026).

### Лист: Return
Столбцы: Attendee, From_City, Train_Code, Depart_Time, Arrive_Time, Duration, Seat_Class, Price_RUB
По одной строке на каждого участника (обратная поездка, 15.03.2026).

### Лист: Summary
Столбцы: Label и Value. Строки по порядку:
- Total_Attendees (3)
- Outbound_Date (2026-03-12)
- Return_Date (2026-03-15)
- Total_Travel_Cost_RUB (сумма всех шести цен билетов)
- Avg_Cost_Per_Person (итог / 3, округлить до 2 знаков)

## Требования к событиям календаря
- Создать одно событие на каждую поездку каждого участника (туда и обратно) — всего 6 событий.
- Формат заголовка поездки туда: «Conference Travel: From_City to Великий Новгород».
- Формат заголовка обратной поездки: «Conference Return: Великий Новгород to From_City».
- Время начала события — время отправления поезда, время окончания — время прибытия, в правильные даты.
- Описание: имя участника, код поезда и класс места.

## Требования к письмам
- Отправить отдельное персонализированное письмо каждому из 3 участников на его адрес.
- Тема: «План поездки на конференцию».
- Текст письма обращается к участнику по имени и содержит коды поездов, время отправления и прибытия, цены билетов туда и обратно.

## Класс места
- Для всех участников использовать класс «Эконом» как самый экономичный вариант.
