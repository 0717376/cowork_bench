Я ассистент кафедры по курсу Creative Computing & Culture (осенний семестр 2014 года, код курса CCC-2014J). Мне нужно подготовить методичку по заданиям для студентов. Пожалуйста, найди все задания этого курса в нашей системе управления обучением (Canvas).

Создай документ Word с именем Assignment_Guide.docx в рабочей директории. Когда вызываешь инструмент Word, указывай ПОЛНЫЙ АБСОЛЮТНЫЙ путь к файлу Assignment_Guide.docx внутри рабочей директории (иначе файл сохранится не туда). В начале документа должен быть заголовок "Assignment Guide - Creative Computing & Culture (Fall 2014)", под ним строка с кодом курса CCC-2014J. Затем добавь строку с текстом "This document lists all assignments for the course with due dates and point values."

Далее включи таблицу из четырёх столбцов:
- Assignment Name — название задания;
- Due Date — срок сдачи в формате YYYY-MM-DD, либо "No due date", если срок не задан;
- Points — числовое значение баллов;
- Status — напиши "Past Due", если у задания есть срок сдачи, либо "No due date", если срок не задан.

Отсортируй задания по сроку сдачи (Due Date) по возрастанию, разместив задания без срока сдачи в конце. Среди заданий с одинаковым сроком сдачи или без срока сортируй по идентификатору задания (assignment ID) по возрастанию.

После таблицы вставь пустую строку, затем строку "Total Assignments: N", где N — общее количество заданий, и ещё одну строку "Total Points: X", где X — сумма всех баллов, округлённая до 1 знака после запятой.

Также создай страницу в корпоративной базе знаний teamly с заголовком "CCC-2014J Assignment Overview", содержащую краткий обзор заданий курса. В содержании страницы должны быть упомянуты название курса, количество заданий и общая сумма доступных баллов. Страницу можно разместить в одном из существующих пространств.
