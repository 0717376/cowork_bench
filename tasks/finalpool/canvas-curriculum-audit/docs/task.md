Канцелярия проректора Российского государственного университета имени Н. И. Лобачевского запросила комплексный аудит соответствия учебных программ требованиям перед предстоящей аккредитационной проверкой. Стандарты аккредитации опубликованы на портале по адресу http://localhost:30217, где перечислены минимальные структурные требования, которым должен соответствовать каждый курс. Откройте этот портал, чтобы получить актуальные стандарты аккредитации, а также перейдите по ссылке на страницу стандартов качества содержания для дополнительного контекста о целевых показателях сложности оценивания.

После получения стандартов проведите аудит каждого курса в системе управления обучением (LMS): подсчитайте количество заданий (assignments), тестов (quizzes) и модулей (modules), а также проверьте, опубликован ли учебный план (syllabus). Сравните эти значения с пороговыми значениями аккредитации, полученными с портала, чтобы определить статус соответствия.

Ознакомьтесь с файлом контрольного списка аккредитации (accreditation_checklist.md) в рабочей директории для понимания методики оценивания и с файлом каталога курсов (course_catalog.txt) для справочной информации о курсах университета.

Создайте файл Excel под названием Curriculum_Audit.xlsx с двумя листами. Первый лист с именем "Compliance Matrix" должен содержать колонки: Course, Assignments_Count, Quizzes_Count, Modules_Count, Has_Syllabus (Yes или No), Compliant_YN (Yes, если все стандарты соблюдены, иначе No) и Issues (список конкретных недочётов через точку с запятой или "None", если курс соответствует требованиям). Включите все 22 курса, отсортированные по алфавиту по названию курса. Второй лист с именем "Summary" должен содержать строки для Total_Courses, Compliant_Courses, Non_Compliant_Courses и Compliance_Rate (в процентах).

Также создайте документ Word под названием Audit_Report.docx с повествовательным отчётом о соответствии. Документ должен включать резюме (executive summary) с описанием общего уровня соответствия и ключевых выводов, раздел с описанием наиболее распространённых областей несоответствия и рекомендации по приведению несоответствующих курсов в соответствие требованиям перед аккредитационной проверкой.

Важно: инструменты Excel и Word могут не видеть рабочую директорию по умолчанию. При создании Curriculum_Audit.xlsx и Audit_Report.docx указывайте полный абсолютный путь к этим файлам внутри рабочей директории агента.
