Портал стандартов аккредитации университета

Эти стандарты должны соблюдаться всеми курсами для прохождения аккредитации.

Требования к структуре курса

Требование Минимальный порог Описание
Задания (Assignments) 8 Каждый курс должен содержать не менее 8 оцениваемых заданий
Тесты (Quizzes) 3 Каждый курс должен включать не менее 3 тестов для формирующего оценивания
Модули (Modules) 4 Содержание курса должно быть организовано не менее чем в 4 модуля
Учебный план (Syllabus) Обязательно Действующий учебный план должен быть опубликован и доступен студентам

Подробные метрики качества содержания см. на странице Стандарты качества содержания.
