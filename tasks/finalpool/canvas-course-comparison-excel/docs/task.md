Я готовлю отчёт для отдела по учебной работе, сравнивающий успеваемость студентов по нашим курсам Fall 2013. Пожалуйста, найдите в нашей системе управления обучением все курсы, в названии которых есть «Fall 2013», и соберите по каждому из них соответствующую статистику.

Для каждого курса Fall 2013 мне нужны: название курса (Course_Name), код курса (Course_Code), количество зачисленных студентов (только зачисления со статусом студента), средний итоговый балл по оценкам студентов, округлённый до 1 знака после запятой (Avg_Final_Score), общее число заданий в курсе (Assignment_Count) и общее число тестов (Quiz_Count).

Создайте файл Excel с именем «Course_Comparison_Fall2013.xlsx» в рабочей директории. В нём должен быть лист с именем «Course Stats» со столбцами Course_Name, Course_Code, Student_Count, Avg_Final_Score, Assignment_Count и Quiz_Count, где строки отсортированы по алфавиту по Course_Name. Затем добавьте второй лист с именем «Summary» с двумя столбцами, Metric и Value, содержащий следующие строки: Total_Courses (число найденных курсов Fall 2013), Total_Students (сумма всех зачисленных студентов по этим курсам), Highest_Avg_Score_Course (название курса с наивысшим средним итоговым баллом) и Lowest_Avg_Score_Course (название курса с наименьшим средним итоговым баллом).

Важно: названия столбцов, имена листов, имя файла, поисковый запрос «Fall 2013», а также названия и коды курсов берутся из системы управления обучением как есть — не переводите и не изменяйте их.

Когда файл будет готов, отправьте письмо на адрес academic-affairs@university.edu с темой «Fall 2013 Course Performance Summary». Тему письма и адрес получателя оставьте на английском без изменений, а текст письма (тело) напишите на русском: дайте краткий обзор — сколько всего курсов проанализировано, сколько всего студентов зачислено, а также какой курс показал лучший и какой худший результат по среднему итоговому баллу.
