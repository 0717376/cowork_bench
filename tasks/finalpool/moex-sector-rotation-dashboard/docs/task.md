Я инвестиционный аналитик, и мне нужно, чтобы вы помогли построить дашборд по секторальной ротации в виде Excel-отчёта по акциям Московской биржи (МосБиржа). Я хочу сравнить пять акций, представляющих разные секторы, за период с 25 февраля 2026 года по 26 мая 2026 года: SBER.ME для сектора Financial Services, GAZP.ME для сектора Energy, LKOH.ME для сектора Energy, MGNT.ME для сектора Consumer Defensive и MTSS.ME для сектора Communication Services.

Сначала через MOEX Finance найдите для каждой акции цену закрытия на 2026-02-25 (базовая дата периода) и на 2026-05-26 (последний доступный торговый день), затем рассчитайте процентную доходность за период для каждой акции. Также возьмите из MOEX Finance последний годовой (annual) отчёт о прибылях и убытках (income_stmt) для каждой акции, чтобы получить выручку и чистую прибыль за последний отчётный год.

Далее зайдите на внутренний исследовательский портал по адресу http://localhost:30145, чтобы получить целевые секторальные эталонные доходности (benchmark) и консенсус-рейтинги аналитиков по каждой акции. На портале есть страница со всей этой информацией.

Наконец, скомпонуйте всё в Excel-файл с именем sector_rotation_report.xlsx, сохранённый в рабочей директории (workspace). Файл должен содержать три листа.

Первый лист должен называться "Performance" со следующими столбцами в указанном порядке: Symbol, Sector, Price_1Y_Ago, Current_Price, Return_Pct, Benchmark_Return_Pct, Alpha, Analyst_Rating, Target_Price, Upside_Pct. В столбец Price_1Y_Ago записывается базовая цена на 2026-02-25, а в Current_Price — цена на 2026-05-26. Return_Pct — это доходность за период в процентах, округлённая до двух знаков после запятой. Alpha — это Return_Pct минус Benchmark_Return_Pct, округлённое до двух знаков после запятой. Upside_Pct — это процентная разница между целевой ценой аналитиков (Target_Price) и текущей ценой (Current_Price), округлённая до двух знаков после запятой. Строки должны идти в порядке: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME, MTSS.ME.

Второй лист должен называться "Financials" со следующими столбцами: Symbol, Revenue_Latest_Q, Net_Income_Latest_Q, Profit_Margin_Pct. Выручку (Revenue_Latest_Q) и чистую прибыль (Net_Income_Latest_Q) нужно выразить в миллионах (разделить исходное число на один миллион) и округлить до двух знаков после запятой; берите значения из последнего годового отчёта о прибылях и убытках. Profit_Margin_Pct — это чистая прибыль, делённая на выручку и умноженная на 100, округлённая до двух знаков после запятой. Тот же порядок строк: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME, MTSS.ME.

Третий лист должен называться "Summary" и использовать простую раскладку «ключ-значение», где в столбце A находится метка, а в столбце B — значение. Строки должны быть: Best_Performer (тикер с наибольшим Return_Pct), Worst_Performer (тикер с наименьшим Return_Pct), Avg_Alpha (среднее значение Alpha по всем пяти акциям, округлённое до двух знаков после запятой), Stocks_Above_Benchmark (количество акций, у которых Alpha больше нуля), Stocks_Below_Benchmark (количество акций, у которых Alpha меньше нуля).
