Создайте Excel-файл с именем WC_Shipping_Report.xlsx с двумя листами. На листе "Shipping Zones" должны быть колонки Zone (название зоны) и Shipping_Methods (количество методов доставки). Отсортируйте строки по названию зоны (Zone).

На листе "Summary" должны быть метрики Total_Zones и Total_Methods.

Отправьте письмо на адрес logistics@shop.com с темой "Shipping Zone Configuration Review", перечислив зоны.
