Вы — инвестиционный ассистент, помогающий частному инвестору отслеживать предстоящие дивидендные выплаты по его портфелю акций Московской биржи (MOEX). Все суммы указаны в рублях (RUB).

Инвестор держит позиции в нескольких акциях и хочет понять, какие из них платят дивиденды, а затем структурировать информацию о дивидендах для удобного отслеживания.

Сначала получите информацию по следующим тикерам: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME, MTSS.ME. Для каждой акции определите, платит ли она дивиденды.

Считайте акцию дивидендной, если у неё значение поля `lastDividendValue` больше нуля И присутствует дата `exDividendDate` (экс-дивидендная дата). Включайте в результат только такие акции.

Важно: данные MOEX используют следующие имена полей (отличаются от обычных), читайте именно их:
- `shortName` — название компании;
- `lastDividendValue` — размер последней дивидендной выплаты на акцию (используйте как Dividend_Rate);
- `fiveYearAvgDividendYield` — средняя дивидендная доходность за 5 лет в процентах (используйте как Dividend_Yield);
- `exDividendDate` — экс-дивидендная дата в формате Unix-эпохи (целое число секунд). Преобразуйте её в дату в формате YYYY-MM-DD (UTC) для поля Ex_Dividend_Date;
- `payoutRatio` — коэффициент выплат (Payout_Ratio).

Для каждой дивидендной акции соберите: тикер (ticker symbol), название компании (company / short name), размер дивиденда (Dividend_Rate из `lastDividendValue`), дивидендную доходность в процентах (Dividend_Yield из `fiveYearAvgDividendYield`), экс-дивидендную дату (Ex_Dividend_Date из `exDividendDate`, преобразованную в YYYY-MM-DD) и коэффициент выплат (Payout_Ratio из `payoutRatio`).

Создайте Excel-файл с именем "Dividend_Tracker.xlsx" в вашей рабочей директории с двумя листами. ВАЖНО: при работе с Excel-инструментом передавайте АБСОЛЮТНЫЙ путь к файлу .xlsx внутри рабочей директории.

Первый лист должен называться "Dividend Stocks" и содержать колонки: Ticker, Company, Dividend_Rate, Dividend_Yield, Ex_Dividend_Date, Payout_Ratio. Включите по одной строке на каждую дивидендную акцию, отсортированные по тикеру в алфавитном порядке.

Второй лист должен называться "Summary" и содержать следующие строки (ключ в первой колонке, значение во второй):
- Total_Dividend_Stocks — количество акций, платящих дивиденды;
- Avg_Yield — средняя дивидендная доходность по дивидендным акциям (среднее значение Dividend_Yield);
- Highest_Yield_Ticker — тикер с наибольшей дивидендной доходностью.

Далее создайте в Google Calendar по одному событию на каждую дивидендную акцию, чтобы напомнить инвестору об экс-дивидендной дате. Каждое событие должно иметь заголовок (summary) "Ex-Dividend: [TICKER]" (заменив [TICKER] на реальный тикер), а в описании (description) должен упоминаться размер дивиденда. Каждое событие должно быть событием на весь день (all-day) с датой, равной экс-дивидендной дате этой акции.

Наконец, создайте Google-таблицу (Google Sheet) с названием "Dividend Watch List", содержащую лист "Overview" с теми же колонками и данными, что и лист "Dividend Stocks" в Excel-файле.
