# Структура оценки эффективности KPI

Уровни достижения:
- Met: >= 100% от цели
- Near: 90-99% от цели
- Missed: < 90% от цели

Приоритет для анализа:
1. KPI по выручке (финансовое влияние)
2. KPI по удовлетворённости клиентов (влияние на удержание)
3. KPI операционной эффективности (влияние на издержки)
4. KPI по сотрудникам (влияние на персонал)

Требования к отчётности:
- Сравнить цель с фактом по каждому KPI
- Рассчитать процент достижения
- Выявить первопричины отставаний
- Предложить корректирующие действия
