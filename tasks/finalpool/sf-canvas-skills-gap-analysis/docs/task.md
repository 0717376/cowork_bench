Вы — директор по обучению и развитию (L&D) крупной организации. Ваша цель — выявить пробелы в навыках по подразделениям, сопоставив данные о результативности сотрудников с доступными учебными курсами, а затем донести адресные рекомендации до руководителей подразделений.

Начните с посещения внутреннего портала компетенций компании по адресу http://localhost:30237, чтобы изучить требуемые навыки для каждого подразделения вместе с ожидаемыми уровнями владения. На портале две страницы: главная страница, где показаны требуемые навыки по подразделениям, и страница обучения по адресу http://localhost:30237/training.html, которая сопоставляет конкретные учебные курсы каждой области навыков.

Затем выполните запрос к корпоративному хранилищу данных ClickHouse, чтобы получить полную картину персонала каждого подразделения. Вам нужны названия подразделений, число сотрудников, средние оценки результативности и распределение по уровням образования (сколько сотрудников имеют каждый уровень образования, например «Среднее образование», «Диплом», «Бакалавр», «Магистр» или «Кандидат наук»). Учитывайте все семь подразделений в системе.

Затем проверьте систему управления обучением (LMS), чтобы узнать, какие курсы доступны в данный момент. Для каждого курса отметьте число зачислений (всего студентов), количество заданий и наличие тестов (квизов). Это поможет определить, насколько проработаны учебные предложения.

Используя методологию оценки навыков, описанную в файле с заметками в вашей рабочей директории, рассчитайте оценку пробела (gap score) для каждого подразделения на основе указанных там порога и формулы. Оценка пробела вычисляется вычитанием средней оценки результативности подразделения из порога, с минимумом 0. Более высокие оценки пробела указывают на более срочную потребность в обучении.

Создайте файл Excel под названием Skills_Gap.xlsx в вашей рабочей директории с тремя листами. Первый лист должен называться "Department Skills" со столбцами Department, Required_Skills (список через запятую с портала), Employee_Count, Avg_Performance, Education_Distribution (текстовая сводка) и Gap_Score. Включите все семь подразделений.

Второй лист должен называться "Training Mapping" со столбцами Skill_Area, Mapped_Course (со страницы обучения портала), Total_Students, Assignment_Count и Has_Quizzes. Этот лист сопоставляет каждый требуемый навык с ближайшим подходящим курсом в системе обучения.

Третий лист должен называться "Priority Actions" со столбцами Department, Gap_Score, Priority (High, если оценка пробела выше 0 даже немного; Low, если оценка пробела равна 0), Recommended_Training и Contact_Email (из файла с руководителями подразделений).

Наконец, отправьте письмо каждому руководителю подразделения, у которого оценка пробела ненулевая. В каждом письме тема (subject) должна включать название подразделения и фразу "Training Recommendations", а тело письма должно резюмировать выявленные пробелы в навыках и предлагать релевантные курсы. Используйте адреса электронной почты из файла с руководителями подразделений в вашей рабочей директории.

Важно: при работе с файлом Skills_Gap.xlsx через инструмент Excel передавайте АБСОЛЮТНЫЙ путь к файлу внутри вашей рабочей директории.
