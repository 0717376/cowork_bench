Я директор по продажам, и мне нужно построить аналитическую панель показателей продаж на основе наших исторических данных о заказах из корпоративного хранилища данных ClickHouse. Запроси базу продаж, чтобы получить помесячные данные о выручке: сгруппируй все заказы по месяцам. Для каждого месяца мне нужны общее количество заказов и итоговая сумма выручки, округлённая до двух знаков после запятой. Также получи данные о продажах в разрезе по регионам клиентов, показав суммарное количество заказов и суммарную выручку по каждому региону.

Создай в нашей корпоративной базе знаний Teamly страницу с заголовком "Sales Performance Dashboard", которая будет служить главной панелью. На этой странице размести раздел "Monthly Revenue" со следующими полями: Month (значение месяца), Order Count (количество заказов) и Revenue (выручка). Добавь по одной записи для каждого месяца из исторических данных, используя для значения месяца формат YYYY-MM. Включай только месяцы с полными данными, то есть исключи любой месяц, данные которого выглядят неполными по сравнению с остальными месяцами (последний незавершённый месяц нужно отбросить).

Также добавь на странице панели второй раздел "Regional Performance" с полями Region (регион), Order Count (количество заказов) и Revenue (выручка). Добавь по одной записи на каждый регион.

Наконец, создай резервный Excel-файл с именем Sales_Dashboard_Backup.xlsx в рабочей директории. В нём должно быть два листа. Первый лист с именем "Monthly Revenue" должен содержать столбцы Month, Order Count и Revenue с теми же помесячными данными, что и в базе знаний. Второй лист с именем "Regional Performance" должен содержать столбцы Region, Order Count и Revenue, соответствующие данным по регионам.

Важно: при записи Excel-файла передавай инструменту АБСОЛЮТНЫЙ путь к файлу Sales_Dashboard_Backup.xlsx внутри рабочей директории агента (Excel-инструмент может терять текущую рабочую директорию).
