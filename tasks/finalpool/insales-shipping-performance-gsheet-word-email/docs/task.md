Вы — аналитик логистики, отвечающий за ежемесячную отчётность по эффективности доставки. Проанализируйте данные по доставке из магазина InSales и подготовьте дашборд и отчёт по эффективности.

Сначала выгрузите заказы из магазина InSales (включите заказы в статусах completed, processing, on-hold и pending). Проанализируйте метод доставки каждого заказа. Сгруппируйте заказы по методу доставки (стандартная доставка, экспресс-доставка, а также заказы без платы за доставку) и для каждой группы посчитайте общее количество заказов, суммарную выручку (в рублях, ₽) и средний чек.

Названия методов доставки приходят из магазина уже на русском языке — используйте их как метки групп ровно в том виде, в каком они приходят: "Стандартная доставка" (тариф flat-rate), "Экспресс-доставка" (экспресс) и "Бесплатная доставка" (заказы без строки доставки).

Затем создайте Google-таблицу с названием "Shipping Performance Dashboard" со следующими колонками: Zone_Name, Order_Count, Total_Revenue, Avg_Order_Value. Заполните её — одна строка на каждую группу метода доставки. Используйте "Стандартная доставка" для заказов со стандартным тарифом flat-rate, "Экспресс-доставка" для экспресс-доставки и "Бесплатная доставка" для заказов без строки доставки. Денежные значения округляйте до 2 знаков после запятой.

Далее создайте документ Word с именем Shipping_Performance_Report.docx в рабочей директории. Документ должен включать: заголовок "Shipping Performance Report", абзац с краткой сводкой (executive summary), описывающий общую эффективность доставки, таблицу с теми же данными по зонам доставки (Zone_Name, Order_Count, Total_Revenue, Avg_Order_Value) и как минимум ещё один абзац с анализом или рекомендациями. Всего в документе должно быть не менее 3 абзацев. При создании документа через Word-инструмент укажите АБСОЛЮТНЫЙ путь к файлу Shipping_Performance_Report.docx внутри рабочей директории.

Наконец, отправьте письмо на адрес logistics@store.example.com от operations@store.example.com с темой "Monthly Shipping Performance Report". В теле письма кратко изложите ключевые метрики по доставке.

Для контекста по политикам доставки обратитесь к файлу Shipping_Strategy.pdf в вашей рабочей директории.
