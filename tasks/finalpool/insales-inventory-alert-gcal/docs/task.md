Пожалуйста, выгрузите полный каталог товаров из нашего магазина и определите все товары, которых сейчас нет в наличии, то есть тех, у которых остаток на складе равен нулю.

Для каждого отсутствующего товара создайте напоминание в календаре, чтобы мы не забыли сделать повторный заказ. Category: [Category Name]. This product is currently out of stock and needs to be reordered." Используйте первое название категории, указанное для товара. После создания событий в календаре отправьте одно итоговое письмо с адреса purchasing@ourstore.com на адрес warehouse@ourstore.com с темой "Out of Stock Alert - March 2026". В теле письма каждый отсутствующий товар должен быть на отдельной строке в формате "ID [Product ID]: [Product Name] (SKU: [SKU])", отсортированный по идентификатору товара по возрастанию. Внизу письма добавьте строку "Total out-of-stock products: [N]", где N — это количество таких товаров.

Также сохраните в рабочем каталоге простой текстовый файл с именем out_of_stock_report.txt. Каждая строка должна содержать "[Product ID] - [Product Name] - [SKU] - [Category]" для отсутствующих товаров, где поля разделены пробелом, дефисом и пробелом, отсортированные по идентификатору товара по возрастанию.
