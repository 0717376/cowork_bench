Для нашей исследовательской группы мне нужно собрать базу знаний по недавним статьям о дообучении (fine-tuning) и выравнивании (alignment) больших языковых моделей. Обратитесь к нашему репозиторию научных статей и прочитайте полный текст тех статей, которые относятся к темам дообучения LLM, выравнивания и улучшений архитектуры моделей. По каждой релевантной статье внимательно проработайте её разделы, чтобы понять методологию и ключевые результаты. Пропускайте статьи на нерелевантные темы — например, по робототехнике или физическим системам (такие статьи в базу знаний включать не нужно).

Создайте в нашей корпоративной базе знаний Teamly страницу с заголовком LLM Fine-Tuning Knowledge Base. На странице для каждой релевантной статьи разместите отдельный блок, где укажите название статьи, основной предложенный в ней метод и главные выводы. Для размещения страницы используйте подходящее пространство в Teamly — посмотрите список существующих пространств и при необходимости создайте новое.

Также создайте документ Word с именем LLM_Paper_Synthesis.docx в рабочей директории. Документ должен начинаться с заголовка LLM Fine-Tuning and Alignment Survey, за которым следует вводный абзац. Затем добавьте отдельный раздел по каждой релевантной статье, где приведите название статьи, имена авторов, описание метода и ключевые результаты, извлечённые при чтении разделов статьи.

Наконец, создайте Google Sheet с названием LLM Paper Registry со столбцами ArXiv_ID, Title, Authors, Published_Date, Key_Contribution и Method_Category, по одной строке на каждую релевантную статью.

Примечание: названия статей, arxiv ID, имена авторов, заголовок страницы LLM Fine-Tuning Knowledge Base, имя файла LLM_Paper_Synthesis.docx, заголовок документа LLM Fine-Tuning and Alignment Survey, название таблицы LLM Paper Registry и названия столбцов (ArXiv_ID, Title, Authors, Published_Date, Key_Contribution, Method_Category) оставляйте на английском — это технические идентификаторы. Текст заметок, описаний и выводов пишите на русском.

Важно: при создании файла LLM_Paper_Synthesis.docx передавайте инструменту Word полный абсолютный путь к файлу внутри рабочей директории (например, <рабочая директория>/LLM_Paper_Synthesis.docx), иначе документ может быть записан не в ту папку.
