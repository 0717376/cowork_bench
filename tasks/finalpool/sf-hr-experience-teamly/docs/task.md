Я руководитель HR-аналитики и мне нужно подготовить отчёт о стаже сотрудников для нашего директора по персоналу (CHRO). У нас есть хранилище данных ClickHouse с записями о сотрудниках по всем отделам, и мне нужно понять распределение стажа (лет опыта) по организации.

Запросите данные о сотрудниках из системы HR-аналитики ClickHouse, чтобы проанализировать колонку с годами опыта в разбивке по отделам. Для каждого отдела рассчитайте средний стаж (округлённый до 2 знаков после запятой), минимальный стаж, максимальный стаж и общее число сотрудников.

Создайте файл Excel с именем Tenure_Analysis.xlsx в рабочей директории с двумя листами. Первый лист должен называться "Department Tenure" с колонками Department, Avg_Tenure, Min_Tenure, Max_Tenure и Employee_Count. Отсортируйте строки по Department в алфавитном порядке. Второй лист должен называться "Summary" с двумя колонками, Metric и Value, и содержать следующие строки: Total_Employees (общая численность персонала), Department_Count (число отделов), Highest_Avg_Tenure_Dept (название отдела с наибольшим средним стажем) и Overall_Avg_Tenure (средний стаж по всем сотрудникам, округлённый до 2 знаков после запятой).

Создайте страницу в Teamly с заголовком "Employee Tenure Dashboard", которая описывает каждый отдел. Для каждого отдела добавьте заголовок с названием отдела, за которым следует абзац с числом сотрудников, средним стажем и диапазоном стажа (от минимума до максимума). Завершите итоговым абзацем, указывающим, в каком отделе в среднем наиболее опытный персонал.

Отправьте письмо с адреса hr-analytics@company.com на адрес chro@company.com с темой "Employee Tenure Analysis Report" и текстом, кратко излагающим общие выводы, включая общее число сотрудников, число проанализированных отделов и отдел с наибольшим средним стажем.
