Я работаю в HR-отделе нашей российской компании, и нам нужно провести ежеквартальный анализ зарплатных диапазонов по всем подразделениям. Данные о сотрудниках хранятся в нашем корпоративном облачном хранилище данных ClickHouse и включают информацию о зарплате (в рублях) и подразделении примерно по 50 000 сотрудников в 7 подразделениях.

Выгрузи данные о зарплатах сотрудников из хранилища данных ClickHouse и рассчитай для каждого подразделения следующие показатели: минимальную зарплату, 25-й перцентиль зарплаты, медианную зарплату, 75-й перцентиль зарплаты, максимальную зарплату, среднюю зарплату и общую численность сотрудников.

Создай в рабочей директории Excel-файл с именем "Salary_Band_Analysis.xlsx" с двумя листами. Первый лист с названием "Department Bands" должен содержать столбцы Department, Min_Salary, P25_Salary, Median_Salary, P75_Salary, Max_Salary, Avg_Salary и Employee_Count. Второй лист с названием "Summary" должен содержать столбцы Metric и Value со следующими строками: Total_Employees (должно быть 50000), Highest_Avg_Dept (название подразделения с самой высокой средней зарплатой), Lowest_Avg_Dept (название подразделения с самой низкой средней зарплатой) и Overall_Avg_Salary.

Далее запланируй встречу по пересмотру зарплат для каждого из 7 подразделений — по одной в день с 10 по 16 марта 2026 года. Каждая встреча должна проходить с 10:00 до 11:00 по часовому поясу America/New_York. Название события в календаре должно быть "Salary Review: [Название подразделения]", где Название подразделения — это фактическое название подразделения.

В завершение отправь письмо с адреса hr-analytics@company.com на адрес hr-director@company.com с темой "Quarterly Salary Band Analysis". В теле письма кратко изложи результаты анализа, упомянув подразделение с самой высокой и подразделение с самой низкой средней зарплатой.
