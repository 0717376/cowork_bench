Мне нужна помощь со сверкой записей о продажах из нашего хранилища данных с заказами из нашего интернет-магазина для аудита по итогам квартала. В корпоративном хранилище данных (ClickHouse) есть детальные записи о заказах с датами, клиентскими сегментами, регионами и суммами. У нашего интернет-магазина (InSales) тоже есть собственные записи о заказах с данными клиентов, товарными позициями и итоговыми суммами.

Нужно выгрузить данные из обеих систем и сопоставить их. Через terminal создай и запусти Python-скрипт reconciler.py в рабочей директории, который читает оба набора данных (сначала сохрани их как dw_orders.json и store_orders.json), сравнивает общую выручку, средние суммы заказа и выявляет любые значимые расхождения между двумя системами. Результаты анализа выведи в reconciliation_results.json.

Собери всё в файл Excel под названием Order_Reconciliation_Report.xlsx с четырьмя листами. Первый лист DW_Summary должен содержать столбцы Region, Total_Orders, Total_Revenue (округли до 2 знаков) и Avg_Order_Value (округли до 2 знаков), отсортированные по Region по алфавиту. Второй лист Store_Summary должен содержать столбцы Status, Order_Count и Total_Amount (округли до 2 знаков), отсортированные по Status по алфавиту. Третий лист Cross_Reference должен содержать столбцы Metric, DW_Value, Store_Value и Difference со строками для Total_Revenue, Total_Orders и Avg_Order_Value (все округлены до 2 знаков). Четвёртый лист Audit_Notes должен содержать столбцы Finding и Detail с не менее чем 3 наблюдениями по результатам сравнения данных.

Также создай документ Word под названием Reconciliation_Audit.docx с заголовком "Q1 Order Reconciliation Audit Report", абзацем-резюме (executive summary) и разделами "Data Warehouse Analysis", "Online Store Analysis" и "Reconciliation Findings" с ключевыми метриками и наблюдениями.

Отправь письмо на адрес finance@company.com с темой "Q1 Order Reconciliation Complete", в котором кратко изложены ключевые выводы аудита.

Важно: значения столбца Region берутся из хранилища данных как есть — записывай их в DW_Summary ровно в том виде, в котором они хранятся, не переводя и не изменяя.

Важно: инструменты Excel и Word могут не сохранить рабочую директорию между вызовами, поэтому при создании Order_Reconciliation_Report.xlsx и Reconciliation_Audit.docx указывай полный абсолютный путь к файлам внутри рабочей директории.
