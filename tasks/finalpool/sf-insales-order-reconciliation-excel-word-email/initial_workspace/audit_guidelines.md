# Рекомендации по квартальному аудиту

При сверке данных о заказах между несколькими системами обращайте внимание на:
- Сравнение общей выручки между системами
- Расхождения в количестве заказов
- Различия в средней сумме заказа
- Распределение заказов по регионам
- Аномалии в распределении по статусам

Помечайте любое отклонение между системами, превышающее 5%, как требующее проверки.
Сообщайте обо всех находках с подтверждающими данными.
