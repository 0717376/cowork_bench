Руководство по анализу риска оттока

Это руководство объясняет, как выполнить оценку риска оттока сотрудников по методике подсчёта баллов, определённой в PDF-документе Retention_Policy.pdf.

Расчёт роста зарплаты

Для каждого сотрудника, у которого есть записи об изменении зарплаты, вычислите процентное изменение по каждому событию как (NEW_SALARY - OLD_SALARY) / OLD_SALARY * 100. Затем вычислите среднее всех процентных изменений для этого сотрудника. Сотрудники без записей в истории зарплат не получают вклада в балл по фактору роста зарплаты (считается как 0 баллов).

Расчёт балла риска

Каждый сотрудник оценивается по четырём факторам, описанным в регламенте удержания. Факторы суммируются. После вычисления итогового балла отнесите сотрудника к уровню риска: Low (0-2), Medium (3-4) или High (5+).

Медианная зарплата подразделения

Для фактора опыта (Experience Factor) вам нужна медианная зарплата каждого подразделения. Рассчитайте её по всем сотрудникам подразделения, чтобы определить, получает ли опытный сотрудник зарплату ниже медианы.

Формат вывода

Excel-файл должен содержать три листа:
- Risk Assessment: сгруппированные по Department и Risk_Level агрегированные метрики
- Department Summary: одна строка на подразделение с количеством по уровням риска и бюджетом
- Top Risk Factors: единственный наиболее значимый фактор для каждого подразделения

Отчёт Word должен содержать описательные разделы с результатами анализа и рекомендуемыми мерами.
