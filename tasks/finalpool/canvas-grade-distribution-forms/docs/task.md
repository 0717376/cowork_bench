Я работаю в учебном офисе деканата, и нам нужно проанализировать распределение оценок по курсам весеннего семестра 2014 года (Spring 2014), а также собрать обратную связь студентов через опрос.

Сначала, пожалуйста, найдите все курсы весеннего семестра 2014 года в нашей системе управления обучением (LMS). В системе они называются на русском — ищите курсы со словами «Весна 2014» в названии (код курса оканчивается на «-2014B»). Для каждого курса нужно проанализировать сданные работы по заданиям и вычислить статистику оценок: средний балл, медианный балл, долю успешной сдачи (процент работ с баллом >= 60) и общее число студентов, сдавших работы.

Создайте файл Excel с именем "Grade_Distribution_Report.xlsx" в рабочей директории с двумя листами. Первый лист «Course Grades» должен содержать столбцы Course_Code, Course_Name, Avg_Grade, Median_Grade, Pass_Rate и Total_Students. Включите по одной строке на каждый курс Spring 2014. Второй лист «Summary» должен содержать столбцы Metric и Value со строками Total_Courses (число курсов Spring 2014), Overall_Avg_Grade (средний балл по всем курсам), Highest_Avg_Course (название курса с наивысшим средним баллом) и Lowest_Avg_Course (название курса с наименьшим средним баллом).

Во-вторых, создайте форму-опрос с заголовком "Spring 2014 Course Feedback Survey" для сбора мнений студентов. Форма должна содержать не менее 4 вопросов: общая удовлетворённость курсом (вопрос с выбором варианта, варианты ответа Excellent, Good, Fair, Poor), оценка качества преподавания (вопрос с выбором варианта, варианты от 1 до 5), оценка учебной нагрузки (вопрос с выбором варианта, варианты Too Light, Manageable, Heavy, Overwhelming) и открытый вопрос с предложениями и пожеланиями (текстовый вопрос, необязательный).

Наконец, отправьте письмо с адреса registrar@university.edu на адрес students@university.edu с темой "Spring 2014 Course Feedback Survey", в котором сообщите студентам об опросе и побудите их принять в нём участие.
