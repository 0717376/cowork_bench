Академический координатор университета хочет отслеживать динамику результатов тестов (quizzes) по всем курсам, чтобы выявлять студентов, которым нужна дополнительная поддержка. Проанализируйте данные о попытках прохождения тестов по всем курсам в Canvas и вычислите средний балл по каждому тесту в каждом курсе.

Создайте Google-таблицу под названием "Quiz Performance Tracker" с одним листом с названием "Quiz Scores", содержащим следующие колонки: Course_Name, Quiz_Title, Total_Attempts, Avg_Score (средний балл, округлённый до 1 знака после запятой), Max_Score и Pass_Rate (процент попыток, в которых балл составляет не менее 60 процентов от максимально возможного балла, округлённый до 1 знака после запятой). Включайте только те тесты, по которым есть хотя бы одна попытка с ненулевым (non-null) баллом. Отсортируйте строки сначала по Course_Name, затем по Quiz_Title. Названия курсов и тестов берутся из данных Canvas как есть (на английском) — не переводите их. Ожидается, что в таблице окажется порядка 76 тестов (по одной строке на каждый тест, прошедший фильтрацию).

Также создайте документ Word под названием Quiz_Performance_Summary.docx, который содержит: раздел-резюме с описанием общих тенденций результатов тестов, таблицу с лучшими тестами по среднему баллу и раздел с рекомендациями о том, каким курсам нужна дополнительная поддержка студентов, исходя из значений Pass_Rate. Раздел-резюме и рекомендации пишите на русском языке.

ВАЖНО: при сохранении документа Word через соответствующий инструмент укажите АБСОЛЮТНЫЙ путь к файлу Quiz_Performance_Summary.docx внутри вашей рабочей директории (инструмент Word не сохраняет текущую рабочую директорию между вызовами).

Отправьте результаты на адрес academic_coordinator@university.edu с темой письма "Quiz Performance Analysis" и включите в тело письма краткое резюме ключевых выводов на русском языке.
