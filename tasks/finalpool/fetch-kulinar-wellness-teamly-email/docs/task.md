Мне нужна помощь с анализом кулинарного оздоровительного отчёта (cook wellness analysis).

Сначала получите внешние эталонные данные по адресу http://localhost:30321/api/data.json и извлеките рекомендации по питанию из поля `nutrition_guidelines`. Каждая запись содержит нутриент (`nutrient`) и его суточную норму: Protein = 50 (`daily_g`), Fiber = 25 (`daily_g`), Vitamin_C = 90 (`daily_mg`). Сохраните разобранные нормы в JSON-файл в рабочей директории.

Затем найдите рецепты в нашей кулинарной базе рецептов (kulinar) — возьмите минимум 5 реальных рецептов из базы. Для каждого рецепта определите его категорию, а также пищевую ценность (калории и белок).

С помощью терминала создайте и запустите Python-скрипт `cook_wellness_processor.py` в рабочей директории, который читает собранные данные из созданных вами JSON-файлов, выполняет анализ и сохраняет результат в файл `cook_wellness_results.json`. Скрипт должен сопоставить белок каждого рецепта с суточной нормой Protein (50 г) и определить, удовлетворяет ли рецепт норме.

Создайте Excel-файл `Wellness_Report.xlsx` с тремя листами.

1. Лист `Data_Analysis` — основная таблица сравнения. Столбцы (именно с такими английскими заголовками):
   - `Recipe` — название рецепта (основное измерение);
   - `Category` — категория рецепта;
   - `Calories` — калорийность;
   - `Protein_g` — содержание белка в граммах;
   - `Meets_Guidelines` — соответствует ли рецепт норме Protein (значение Protein_g >= 50 => Yes/Да, иначе No/Нет).
   Отсортируйте строки по алфавиту по столбцу `Recipe`. Включите минимум 5 рецептов.

2. Лист `Metrics` — сводка ключевых метрик со столбцами `Metric` и `Value`. Включите минимум 4 строки: общее количество рецептов, среднюю калорийность, средний белок и количество рецептов, удовлетворяющих норме (Meets_Guidelines = Yes).

3. Лист `Recommendations` — список приоритетных действий со столбцами `Priority` и `Action`. Включите минимум 2 строки с рекомендациями на основе анализа разрыва между фактическими значениями и нормами питания.

Отправьте письмо на адрес team-lead@company.com с темой ровно `Analysis Report Complete`, кратко изложив в теле письма ключевые выводы (сколько рецептов проанализировано, сколько удовлетворяют норме по белку).

Создайте в Teamly (наш корпоративный аналог Confluence) страницу с заголовком `Cook Wellness Dashboard` с кратким текстовым резюме анализа (упомяните рецепты, нормы питания и итоговые выводы).
