Мне нужен анализ лояльности клиентов в разрезе сегмента и региона. Возьми данные из хранилища данных продаж (ClickHouse) и покажи закономерности пожизненной ценности клиентов (lifetime value) по различным комбинациям «сегмент — регион».

Создай файл Excel с именем Customer_Loyalty_Report.xlsx с двумя листами. На листе «Customer Loyalty» должны быть колонки: Segment_Region (в формате «Сегмент - Регион»), Customers (количество клиентов), Avg_LTV (средняя пожизненная ценность, округлённая до 2 знаков) и Total_Orders. Отсортируй по сегменту по алфавиту, а внутри каждого сегмента — по Avg_LTV по убыванию.

На листе «Summary» должны быть Total_Combinations, Highest_LTV_Group (комбинация «сегмент — регион» с наибольшим средним LTV) и Overall_Avg_LTV как средневзвешенное по количеству клиентов значение.
