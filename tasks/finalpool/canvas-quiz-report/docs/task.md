Мне нужен анализ успеваемости по тестам (quizzes) нашего университета в Canvas. В рабочей директории есть файл Quiz_Guidelines.pdf с целевыми показателями оценивания. Пожалуйста, ознакомьтесь с этими методическими указаниями.

Затем выгрузите из Canvas данные по всем тестам, у которых более 100 попыток сдачи (submissions). Для каждого теста мне нужны: код курса (course code), название теста (quiz title), максимальный балл (points possible), ограничение по времени (time limit), число попыток (submissions), средний балл (average score), а также Pass_Rate_Pct, который рассчитывается как средний балл, делённый на максимальный балл и умноженный на 100, округлённый до 1 знака после запятой.

Создайте Excel-файл с именем Canvas_Quiz_Report.xlsx с двумя листами. Лист "Quiz Performance" должен содержать колонки Course_Code, Quiz_Title, Points_Possible, Time_Limit, Submissions, Avg_Score (округлить до 2 знаков после запятой) и Pass_Rate_Pct. Отсортируйте строки по Course_Code, затем по Avg_Score по убыванию.

Лист "Summary" должен содержать Total_Quizzes, Avg_Pass_Rate по всем тестам (округлить до 1 знака после запятой), Hardest_Quiz — название теста с самым низким значением pass rate, и Easiest_Quiz — название теста с самым высоким значением pass rate.

Имена листов, колонок, файла, а также коды курсов и названия тестов из Canvas оставьте как есть на английском.
