# Методология прогнозирования продаж

## Подход

Региональный прогноз продаж использует методологию скользящей средней, скорректированную на экономические показатели.

## Формула

Месячный прогноз = Скользящая средняя за 3 месяца x (1 + Темп роста ВВП / 100)

Где:
- Скользящая средняя за 3 месяца = средняя выручка за последние три месяца (октябрь, ноябрь, декабрь 2025 года)
- Темп роста ВВП = годовой темп роста ВВП в процентах для региона

## Квартальный прогноз

Квартальный прогноз = Месячный прогноз x 3

Это предполагает стабильную помесячную динамику в течение квартала.

## Источники данных

- Исторические данные о продажах: корпоративное хранилище данных ClickHouse
- Экономические показатели: внешний API экономических данных (рост ВВП, потребительская уверенность, инфляция)

## Регионы

Прогнозы строятся для всех пяти регионов продаж:
- Северная Америка
- Европа
- Азиатско-Тихоокеанский регион
- Латинская Америка
- Ближний Восток

## Ограничения

- Не учитывает сезонные закономерности сверх того, что отражает скользящая средняя
- Предполагает, что экономические условия остаются стабильными
- Не учитывает запланированные запуски продуктов или промоакции
- Простая линейная корректировка может не отражать сложные экономические эффекты
