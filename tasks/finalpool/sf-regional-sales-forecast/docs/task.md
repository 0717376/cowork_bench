Финансовому отделу нужен региональный прогноз продаж на I квартал 2026 года, который объединяет историческую динамику продаж с внешними экономическими показателями. Данные по экономическим показателям доступны в виде JSON-файла по адресу http://localhost:30212/api/economic_indicators.json — они содержат темпы роста ВВП, индексы потребительской уверенности и уровни инфляции для каждого из наших пяти регионов продаж. Вам нужно получить эти данные и объединить их с историческими показателями продаж из хранилища данных ClickHouse, чтобы построить прогноз.

Начните с прочтения документа с методологией прогнозирования (forecast_methodology.md) в рабочей директории, в котором описана конкретная формула, а также файла с допущениями (assumptions.txt), где перечислены ключевые предпосылки, лежащие в основе прогноза. Затем получите экономические показатели из API и запросите в хранилище данных ClickHouse ежемесячную выручку по продажам в разрезе регионов за последние три месяца 2025 года, а именно за октябрь, ноябрь и декабрь.

Методология прогнозирования следующая. Для каждого региона рассчитайте скользящую среднюю выручку за 3 месяца, используя данные за октябрь–декабрь 2025 года. Затем примените экономическую корректировку по формуле: Скорректированный месячный прогноз равен скользящей средней за 3 месяца, умноженной на величину «единица плюс темп роста ВВП, делённый на 100». Квартальный прогноз для каждого региона затем равен утроенному скорректированному месячному прогнозу.

Создайте в рабочей директории файл Excel с именем Sales_Forecast.xlsx с тремя листами. Первый лист с именем Historical должен содержать колонки Region, Month в формате YYYY-MM, Orders как целое число заказов и Revenue, округлённую до двух знаков после запятой, с одной строкой для каждой комбинации регион-месяц за октябрь–декабрь 2025 года, отсортированными по региону, затем по месяцу. Второй лист с именем Forecast должен содержать колонки Region, Trailing_3M_Avg (округлённую до двух знаков), GDP_Growth_Pct, Economic_Adjustment (коэффициент, округлённый до четырёх знаков), Adjusted_Monthly_Forecast (округлённый до двух знаков) и Next_Quarter_Forecast (округлённый до двух знаков), с одной строкой на регион, отсортированными по алфавиту. Третий лист с именем Summary должен содержать колонки Metric и Value со следующими показателями: Total_Q4_2025_Revenue как сумма всей выручки за октябрь–декабрь, Total_Monthly_Average, Total_Q1_2026_Forecast как сумма всех региональных квартальных прогнозов, QoQ_Growth_Pct, показывающий процентное изменение от фактического значения IV квартала к прогнозу I квартала, округлённый до одного знака, Highest_Forecast_Region, Highest_Forecast_Amount, Lowest_Forecast_Region, Lowest_Forecast_Amount, Highest_GDP_Growth_Region и Regions_Count.

Создайте в рабочей директории документ Word с именем Forecast_Report.docx с разделами: краткое резюме (executive summary), описывающее общий прогноз; методология (methodology), объясняющая формулу и используемые источники данных; региональный анализ (regional analysis), обсуждающий прогноз по каждому региону с конкретными числами; ключевые допущения (key assumptions) с перечислением основных предпосылок; и рекомендации (recommendations), указывающие, на чём сосредоточить усилия по продажам с учётом прогноза и экономической ситуации. Текст разделов пишите на русском языке. При работе с инструментом Word передавайте АБСОЛЮТНЫЙ путь к файлу .docx внутри рабочей директории.

Наконец, создайте таблицу Google Sheets с заголовком Regional Sales Forecast и листом с именем Forecast Data, содержащим те же прогнозные данные, что и лист Forecast в файле Excel, чтобы финансовый отдел мог получить к ним доступ и совместно работать над ними онлайн.

Важно: используйте названия регионов ровно так, как они хранятся в ClickHouse и в API (на русском языке): Азиатско-Тихоокеанский регион, Европа, Латинская Америка, Ближний Восток, Северная Америка.
