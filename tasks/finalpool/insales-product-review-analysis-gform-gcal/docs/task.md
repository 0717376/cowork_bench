Мне нужно проанализировать рейтинги наших товаров и настроить процесс сбора обратной связи по товарам с низкими показателями. Пожалуйста, изучите все товары в нашем магазине InSales, у которых есть отзывы клиентов, и найдите пять товаров с самым низким средним рейтингом. Если средние рейтинги совпадают, отдавайте предпочтение товарам с большим числом отзывов.

Для этих пяти товаров с самым низким рейтингом создайте форму (Google Form) с названием Product Quality Improvement Survey и описанием, поясняющим, что мы собираем отзывы клиентов для улучшения качества продукции. Для каждого из пяти товаров добавьте обязательный вопрос с оценкой, в котором клиента просят оценить общую удовлетворённость данным товаром по шкале от одного до пяти; в тексте вопроса используйте название товара. Также добавьте в конце один необязательный текстовый вопрос с просьбой дать дополнительные предложения по улучшению.

Создайте Excel-файл с именем Product_Review_Analysis.xlsx с листом Review Summary, в котором есть столбцы Product_Name (название товара, обрезанное до шестидесяти символов), Avg_Rating (округлённый до двух знаков после запятой), Review_Count и Category (название первой категории каждого товара). Отсортируйте строки по Avg_Rating по возрастанию. Добавьте второй лист Stats со строками Metric и Value для метрик Total_Products_Reviewed (общее число всех товаров магазина, у которых есть отзывы, а не только пяти из отчёта), Lowest_Rated_Product (название товара, обрезанное до сорока символов) и Avg_Rating_Overall (средний рейтинг по всем товарам магазина с отзывами, округлённый до двух знаков после запятой).
