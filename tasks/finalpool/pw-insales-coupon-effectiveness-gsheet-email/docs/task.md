Мне нужна помощь с анализом купонов магазина. Есть внешние эталонные (бенчмарк) данные, которые тебе нужно получить: открой http://localhost:30309 и извлеки соответствующие метрики.

Затем проверь наш интернет-магазин InSales на наличие актуальных данных по товарам и заказам.

С помощью терминала создай и запусти Python-скрипт под названием wc_coupons_processor.py в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые ты создашь, выполнять анализ и выводить результат в wc_coupons_results.json.

Создай Excel-файл под названием Coupon_Effectiveness_Report.xlsx с тремя листами. Первый лист Data_Analysis должен содержать основные сравнительные данные с соответствующими столбцами. Второй лист Metrics должен суммировать ключевые метрики. Третий лист Recommendations должен перечислять конкретные действия.

Лист Data_Analysis должен включать столбцы для основного измерения (например, категория товара), наши внутренние значения метрик, внешние эталонные значения и разрыв (gap) или разницу между ними. Ключ объединения внутренних данных магазина с бенчмарком — это НАЗВАНИЕ категории товара. Отсортируй данные по алфавиту по основному измерению. Лист Metrics должен иметь два столбца Metric и Value, суммирующих общие количества, средние значения и ключевые статистики. Лист Recommendations должен перечислять приоритетные действия на основе анализа разрывов.
