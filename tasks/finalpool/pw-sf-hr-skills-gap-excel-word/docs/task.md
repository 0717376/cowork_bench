Мне нужна помощь в проведении анализа дефицита навыков (skills gap) для нашей компании. Отраслевой отчёт по навыкам, опубликованный TechTalent Research Institute, доступен по адресу http://localhost:30300 и содержит список самых востребованных навыков и зарплатных бенчмарков по ролям. Пожалуйста, зайдите на эту страницу и извлеките все данные о навыках и зарплатных бенчмарках.

Затем нужно выгрузить данные о наших сотрудниках из корпоративного хранилища данных ClickHouse (логическая база данных HR_ANALYTICS, схема sf_data). Я хочу понять, как распределена наша рабочая сила по подразделениям, как наши средние зарплаты выглядят по сравнению с отраслевыми бенчмарками и где у нас могут быть пробелы в навыках (skills gap) с учётом ролей в каждом подразделении.

Перед началом анализа, пожалуйста, с помощью терминала создайте Python-скрипт под названием skills_analysis.py в рабочей директории, который обрабатывает сырые данные. Скрипт должен читать JSON-файл raw_skills_data.json (который вы должны создать первым, со спарсенными с веб-страницы данными), очищать любые несоответствия и выводить обработанный файл processed_analysis.json. Запустите этот скрипт через терминал.

Соберите всё вместе в Excel-файле под названием Skills_Gap_Analysis.xlsx с тремя листами. Первый лист, Department_Overview, должен содержать колонки Department, Employee_Count, Avg_Salary, Avg_Experience, Avg_Performance, Industry_Benchmark_Salary (среднее всех отраслевых зарплат по сопоставимым ролям) и Salary_Gap (Avg_Salary минус Industry_Benchmark_Salary, округлённое до 2 знаков после запятой). Отсортируйте по Department в алфавитном порядке.

Второй лист, Skills_Matrix, должен перечислять каждое отраслевое требование к навыкам с колонками Role, Skill, Proficiency_Level, Demand_Score, Industry_Salary, Our_Avg_Salary (из соответствующего подразделения) и Gap_Pct (округлите (Our_Avg_Salary минус Industry_Salary), делённое на Industry_Salary, умноженное на 100, до 1 знака после запятой). Сопоставьте роли с подразделениями так: Data Engineer и Software Developer — к Engineering и R&D, HR Manager — к HR, Sales Executive — к Sales, Financial Analyst — к Finance, Operations Manager — к Operations, Support Specialist — к Support. У подразделения R&D нет собственной отраслевой роли на странице бенчмарков, поэтому оно использует тот же инженерный бенчмарк, что и Engineering (среднее отраслевых зарплат Data Engineer и Software Developer = 121500).

Третий лист, Summary, должен содержать две колонки Label и Value, начиная с ячейки A1, со строками: Total_Departments, Total_Skills_Tracked, Departments_Below_Benchmark (количество подразделений, где Salary_Gap отрицательный), Departments_Above_Benchmark, Largest_Gap_Department (подразделение с самым отрицательным Salary_Gap), Smallest_Gap_Department, Average_Salary_Gap (среднее значение Salary_Gap по всем подразделениям, округлённое до 2 знаков после запятой).

Также создайте документ Word под названием Skills_Gap_Report.docx с заголовком "Skills Gap Analysis Report", за которым следуют разделы: Executive Summary (один абзац-обзор), Department Analysis (опишите положение каждого подразделения относительно бенчмарков) и Recommendations (минимум 3 рекомендации на основе выводов). В каждом разделе должно быть минимум 2 предложения.

Важно: инструменты Excel и Word могут терять текущую рабочую директорию, поэтому при создании и сохранении файлов Skills_Gap_Analysis.xlsx и Skills_Gap_Report.docx передавайте АБСОЛЮТНЫЙ путь к ним внутри рабочей директории.
