Вы — финансовый аналитик инвестиционной компании, готовящей инвестиционный роадшоу по российскому рынку. Ваша задача: спланировать визиты в два города, провести анализ двух публичных эмитентов и подготовить материалы. Поездки осуществляются из Москвы в **Санкт-Петербург (2026-03-10)** и в **Великий Новгород (2026-03-12)** — два разных дня. Все даты везде записывайте в формате `YYYY-MM-DD` (ISO 8601), без вариаций.

## Шаг 1. Поездки (MCP `rzd`)

Получите коды станций для Москвы, Санкт-Петербурга и Великого Новгорода.

Найдите доступные поезда **Москва → Санкт-Петербург на 2026-03-10**. Из вариантов выберите Сапсан с номером **752А** (отправление 06:50, прибытие 10:50). Запросите тарифы по этому поезду, зафиксируйте цену билета в классе **Бизнес**.

Найдите доступные поезда **Москва → Великий Новгород на 2026-03-12**. Из вариантов выберите Ласточку с номером **818А** (отправление 13:50, прибытие 20:30). Запросите тарифы, зафиксируйте цену билета в классе **Бизнес**.

Для каждого направления зафиксируйте: номер поезда, время отправления, время прибытия, длительность в часах (десятичной дробью — например, `4.0` или `6.67`), класс, цену в рублях.

## Шаг 2. Финансовые данные (MCP `moex-finance`)

Основной эмитент — **`SBER.ME`** (ПАО Сбербанк). Эмитент-сравнение — **`GAZP.ME`** (ПАО Газпром). Тикеры с суффиксом `.ME` (Московская биржа), валюта — рубли. MCP `moex-finance` имеет тот же интерфейс tools, что и yahoo-finance (`get_stock_info`, `get_historical_stock_prices`, `get_financial_statement`, `get_recommendations` и т.д.), но данные — по российским эмитентам.

Для **SBER.ME**:
- получите общую информацию (`get_stock_info`),
- получите исторические цены за последние ~30 дней (`get_historical_stock_prices`),
- получите годовой отчёт о прибыли (`get_financial_statement`) — нужны Revenue и Net Income за два последних доступных финансовых года,
- получите рекомендации аналитиков (`get_recommendations`) — посчитайте сумму buy/hold/sell,
- получите последние новости (опционально, для контекста).

Для **GAZP.ME**:
- получите общую информацию и исторические цены за последние ~30 дней (для сравнительной динамики).

Посчитайте приблизительное изменение цены за 30 дней (`(last - first) / first * 100%`) для обоих эмитентов.

## Шаг 3. Excel-файл `Roadshow_Analysis.xlsx`

Создайте Excel-файл с тремя листами:

**`Travel_Plan`** — колонки: `City`, `Train_No`, `Departure`, `Arrival`, `Duration_Hours`, `Class`, `Ticket_Price_RUB`, `Meeting_Date`. Две строки — для СПб (752А) и для В.Новгорода (818А). Класс — `Бизнес`. Даты в ISO формате.

**`Stock_Summary`** — колонки: `Company`, `Ticker`, `Current_Price`, `Change_30D_Pct`, `Market_Cap_B`, `Revenue_B`, `EPS`, `Recommendation`. Две строки — для `SBER.ME` и `GAZP.ME`. `Market_Cap_B` и `Revenue_B` — в миллиардах рублей. `Recommendation` — общий консенсус (например, `Buy`, `Hold` и т.п.).

**`Financial_Highlights`** — колонки: `Metric`, `Year`, `Value_B`. Минимум 4 строки: Revenue и Net Income за два последних доступных финансовых года для `SBER.ME`. Значения — в миллиардах рублей.

## Шаг 4. Презентация `Investor_Roadshow.pptx`

Презентация должна содержать **ровно 5 слайдов**:

1. **Титульный.** Заголовок включает «Investor Roadshow 2026». Подзаголовок — упоминание двух городов (СПб и В.Новгород).
2. **Travel Overview.** Таблица или структурированный текст с двумя строками — для СПб и В.Новгорода: город, дата, поезд, отправление, прибытие, класс. На слайде должны быть **оба номера поездов** (`752А`, `818А`) и **оба города**.
3. **Financial Highlights.** Revenue, Net Income, EPS для `SBER.ME` за два года, плюс год-к-году рост в %.
4. **Stock Performance.** Сравнение `SBER.ME` и `GAZP.ME`: текущая цена в рублях, изменение за 30 дней в %. На слайде должны быть **оба тикера**.
5. **Analyst Consensus.** Сводка buy/hold/sell и общий рейтинг для `SBER.ME`.
