Учебному отделу нужна сводка по успеваемости (оценкам) во всех курсах. Выгрузите из нашей системы управления обучением (Canvas) все сданные работы с выставленными баллами и сгруппируйте их по курсам.

Создайте файл Excel с именем Canvas_Grade_Summary.xlsx с двумя листами. На листе "Grade Summary" должны быть столбцы: Course (название курса), Students_Submitted (число уникальных студентов, сдавших работы), а также Avg_Score, Max_Score и Min_Score — все округлённые до 2 знаков после запятой. Отсортируйте строки по Avg_Score по убыванию.

На листе "Summary" должны быть строки Total_Courses, Highest_Avg_Course и Overall_Avg_Score, округлённые до 2 знаков после запятой.

Также занесите сводные данные в Google-таблицу с названием "Grade Summary Report", чтобы их могла проверить академическая команда.
