Руководство HR хочет выявить подразделения, которым может угрожать риск оттока сотрудников из-за низкой удовлетворённости работой. Сделай запрос к аналитической базе данных HR в ClickHouse, чтобы рассчитать для каждого подразделения среднюю оценку удовлетворённости работой и средний показатель баланса между работой и личной жизнью, а также численность сотрудников и среднюю оценку эффективности. Отсортируй результаты по средней удовлетворённости работой по возрастанию, чтобы определить подразделения с самым низким моральным духом.

Создай в рабочей директории Excel-файл с именем Satisfaction_Report.xlsx с двумя листами. Первый лист должен называться "Department Scores" и содержать все семь подразделений со столбцами Department, Employee_Count, Avg_Satisfaction, Avg_Work_Life_Balance и Avg_Performance, при этом значения, где это применимо, округлены до двух знаков после запятой. Отсортируй строки по Avg_Satisfaction по возрастанию. Второй лист должен называться "Action Items" и содержать только три подразделения с самой низкой удовлетворённостью со столбцами Department, Avg_Satisfaction и Status, где статус для каждого должен быть установлен в "Needs Review".

Для каждого из трёх подразделений, определённых как требующие проверки, запланируй часовую встречу в календаре, название которой состоит из "HR Review - " и идущего следом названия подразделения. Эти встречи должны быть запланированы на 16 марта 2026 года, начиная в 9:00, 10:30 и 12:00 соответственно в часовом поясе America/New_York. Каждое описание события должно упоминать название подразделения и его среднюю оценку удовлетворённости работой.

После планирования встреч отправь письмо на адрес hr-director@company.example.com с темой "Department Satisfaction Review Summary". Тело письма должно перечислять все семь подразделений с их средними оценками удовлетворённости работой и выделять, какие три подразделения отмечены для проверки, вместе с запланированными датами встреч.
