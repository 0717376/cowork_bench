Вы — финансовый аналитик, отвечающий за подготовку ежемесячного отчёта по возвратам средств для компании.

Ваша задача — проанализировать все данные о возвратах из магазина InSales, систематизировать выводы в рабочем пространстве Teamly, подготовить подробный отчёт в Excel и отправить результаты финансовому директору по электронной почте.

Сначала получите все возвраты из магазина InSales. Для каждого возврата соберите: идентификатор возврата (refund ID), идентификатор заказа (order ID), сумму возврата, причину возврата и дату создания возврата.

Создайте файл Excel с именем «Refund_Analysis.xlsx» в вашем рабочем каталоге с двумя листами. Так как Excel-инструмент может терять текущий рабочий каталог, передавайте АБСОЛЮТНЫЙ путь к файлу внутри вашего рабочего каталога при каждом обращении.

Первый лист должен называться «Refund Details» и содержать колонки: Refund_ID, Order_ID, Amount, Reason, Date. Включите каждый найденный возврат, отсортировав строки по Refund_ID.

Второй лист должен называться «Summary» и содержать следующие строки с информацией: Total_Refunds (общее количество возвратов), Total_Refund_Amount (сумма всех возвратов), Avg_Refund_Amount (средняя сумма возврата), Refund_Rate (рассчитывается как число возвратов, делённое на общее число заказов, выраженное в процентах).

Затем создайте в Teamly страницу с заголовком «Refund Analysis Dashboard». На этой странице оформите структуру (таблицу/базу) под названием «Refund Trends», которая содержит сводные показатели по возвратам, включая общее количество возвратов, общую сумму возвратов и наиболее частую причину возврата.

Наконец, отправьте письмо с адреса finance@store.com на адрес cfo@company.com с темой «Monthly Refund Analysis Report». В теле письма приведите сводку ключевых выводов: общее количество возвратов, общую сумму возвратов, среднюю сумму возврата и наиболее частую причину возврата.
