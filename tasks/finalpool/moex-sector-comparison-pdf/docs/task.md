Мне нужно сравнить динамику акций из разных секторов с нашими внутренними целевыми показателями. В моём рабочем каталоге есть PDF-файл Sector_Targets.pdf, в котором указаны целевые значения доходности с начала года (YTD) для каждого сектора. Сначала прочитай этот файл, чтобы понять эталонные показатели.

Затем найди самые последние цены закрытия для следующих четырёх акций, представляющих разные секторы: GAZP.ME для сектора Energy, SBER.ME для сектора Financial Services, MTSS.ME для сектора Communication Services и MGNT.ME для сектора Consumer Defensive. Для каждой акции также рассчитай доходность с начала года (в процентах), сравнив последнюю цену закрытия с ценой закрытия в первый торговый день 2026 года. Округли YTD-доходность до 2 знаков после запятой.

Создай Excel-файл Sector_Analysis.xlsx в рабочем каталоге с двумя листами. Первый лист "Stock Prices" должен содержать колонки Symbol, Sector, Latest_Close (округлённую до 2 знаков) и YTD_Return_Pct (округлённую до 2 знаков). Второй лист "Sector Comparison" должен содержать колонки Sector, Stock, Target_Return (из PDF), Actual_Return (равную YTD-доходности) и Met_Target — значение Yes, если фактическая доходность достигает целевого значения или превышает его, и No в противном случае.

Важно: названия секторов в колонке Sector оставляй точно такими же, как в Sector_Targets.pdf (Energy, Financial Services, Communication Services, Consumer Defensive) — это англоязычные идентификаторы. Названия файла Sector_Analysis.xlsx, названия листов "Stock Prices" и "Sector Comparison", все заголовки колонок, тикеры (GAZP.ME, SBER.ME, MTSS.ME, MGNT.ME) и значения Yes/No также должны оставаться на английском.

Примечание: при работе с MCP-инструментами для Excel и PDF передавай ПОЛНЫЙ АБСОЛЮТНЫЙ путь к файлам Sector_Analysis.xlsx и Sector_Targets.pdf внутри рабочего каталога, поскольку эти инструменты не сохраняют текущий рабочий каталог.
