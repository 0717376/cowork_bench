Мне нужно проанализировать наши обращения в поддержку по типам проблем. Выгрузи данные по обращениям из центра поддержки (хранилище ClickHouse) и сгруппируй их по типу проблемы, чтобы понять, какие проблемы встречаются чаще всего и как меняется удовлетворённость клиентов.

Создай файл Excel с именем Support_Resolution_Report.xlsx с двумя листами. На листе "Issue Analysis" должны быть столбцы Issue_Type, количество Tickets, Avg_Response_Hrs (округлить до 2 знаков после запятой) и Avg_CSAT (округлить до 2 знаков после запятой). Отсортируй по Tickets по убыванию.

На листе "Summary" должны быть Total_Issue_Types, Total_Tickets, Most_Common_Issue (название типа проблемы) и Best_CSAT_Issue (тип проблемы с наивысшей удовлетворённостью).

Значения Issue_Type, Most_Common_Issue и Best_CSAT_Issue записывай ровно в том виде, в каком они хранятся в базе данных (на русском языке), не переводя их.

Также создай Google-таблицу с названием "Support Issue Dashboard" с листом "Issues", содержащим те же данные.
