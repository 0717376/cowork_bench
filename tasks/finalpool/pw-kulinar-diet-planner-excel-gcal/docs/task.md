Мне нужна помощь с анализом рациона питания (cook diet analysis). Доступны внешние эталонные данные (бенчмарки) по нутриентам: зайдите на http://localhost:30318 и извлеките оттуда нужные показатели (нормы по нутриентам и приоритеты).

Затем подберите рецепты блюд через нашу базу рецептов с помощью инструментов kulinar (например, `mcp_kulinar_getAllRecipes` / `mcp_kulinar_getRecipesByCategory` / `mcp_kulinar_recommendMeals`). Выберите минимум 5 блюд.

С помощью терминала создайте и запустите Python-скрипт `cook_diet_processor.py` в рабочей директории. Скрипт должен читать собранные данные из созданных вами JSON-файлов, выполнять анализ (сопоставление блюд с эталонными нормами, расчёт расхождений) и записывать результат в `cook_diet_results.json`.

Создайте Excel-файл `Diet_Planner_Report.xlsx` с тремя листами. При работе с Excel MCP передавайте АБСОЛЮТНЫЙ путь к файлу `Diet_Planner_Report.xlsx` внутри рабочей директории.

Требования к листам и к их колонкам (имена колонок указывайте ровно так, по-английски):

1. Лист `Data_Analysis` — основная таблица по блюдам, минимум 5 строк данных. Колонки (заголовки в первой строке):
   - `Recipe` — название блюда (берётся из базы рецептов kulinar);
   - `Category` — категория блюда;
   - `Calories` — калорийность порции (ккал);
   - `Protein_g` — содержание белка в граммах;
   - `Meets_Guidelines` — соответствует ли блюдо рекомендациям (`Yes`/`No`); считается соответствующим, если калорийность одной порции не превышает 800 ккал.
   Отсортируйте строки по алфавиту по колонке `Recipe`.

2. Лист `Metrics` — сводка ключевых метрик, минимум 4 строки данных. Ровно две колонки: `Metric` и `Value`. Включите как минимум: общее число блюд (`Total_Recipes`), число блюд, соответствующих рекомендациям (`Recipes_Meeting_Guidelines`), среднюю калорийность (`Avg_Calories`) и среднее содержание белка (`Avg_Protein`). Значения должны быть согласованы с данными листа `Data_Analysis`.

3. Лист `Recommendations` — список действий, минимум 2 строки данных. Ровно две колонки: `Priority` и `Action`. Перечислите приоритетные рекомендации на основе анализа расхождений с эталонными нормами.
