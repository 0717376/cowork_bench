Я академический координатор в Open University, и мне нужно составить сводное расписание заданий по всем курсам осеннего семестра 2014 года. Пожалуйста, найди в нашей системе управления обучением все курсы осеннего семестра 2014 года — это те, у которых код курса заканчивается на 2014J.

Для каждого курса получи все задания, у которых установлен срок сдачи (due date). Для каждого такого задания создай событие в календаре. Каждое событие должно иметь заголовок в формате "[Course Code] - [Assignment Name]" (например, "AAA-2014J - TMA 1758"), должно быть запланировано как событие на весь день (all-day) на дату сдачи задания, а описание должно быть таким: "Задание для [Полное название курса]. Срок сдачи: [YYYY-MM-DD].". Для всех событий используй часовой пояс America/New_York. Обрати внимание: срок сдачи (due date) в системе хранится в UTC, поэтому все даты сдачи [YYYY-MM-DD] — в событиях календаря, в письме и в файле — вычисляй, переводя due date в часовой пояс America/New_York.

После создания всех событий в календаре отправь одно сводное письмо от coordinator@openuniversity.ac.uk на адрес students@openuniversity.ac.uk с темой "Fall 2014 Assignment Schedule". В теле письма перечисли каждый курс в алфавитном порядке по коду курса, а под каждым курсом перечисли все его задания, отсортированные по сроку сдачи, указав название задания и срок сдачи в формате YYYY-MM-DD. Между курсами оставляй пустую строку для читаемости. В конце тела письма добавь строку "Total assignments with due dates: [N]", где N — общее количество заданий со сроком сдачи по всем курсам.

Также сохрани текстовый файл с именем assignment_schedule.txt в рабочей директории. Используй для него абсолютный путь внутри рабочей директории. Этот файл должен содержать каждое задание на отдельной строке в формате "[Course Code] - [Assignment Name] - [YYYY-MM-DD]", где поля разделены пробел-дефис-пробел, отсортированный сначала по коду курса в алфавитном порядке, а затем по сроку сдачи по возрастанию внутри каждого курса.
