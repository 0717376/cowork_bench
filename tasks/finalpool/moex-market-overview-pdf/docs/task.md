Мне нужен обзорный отчёт по рынку (Market Overview Report) Московской биржи, охватывающий пять акций: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME и MTSS.ME. Пожалуйста, выгрузи самые свежие доступные торговые данные и информацию о компаниях по этим пяти тикерам из финансового сервиса данных. Все цены — в рублях (RUB).

Создай в рабочей директории книгу Excel под названием Market_Overview_Report.xlsx с двумя листами.

Первый лист должен называться "Stock Data" и содержать столбцы Symbol, Company_Name, Date, Open, High, Low, Close, Volume, Market_Cap, PE_Ratio, 52W_High, 52W_Low и Sector. Каждая строка соответствует одной акции. Используй самый последний доступный торговый день. Для полей, где данные недоступны, оставь ячейку пустой. Отсортируй строки по алфавиту по столбцу Symbol.

Второй лист должен называться "Summary" и содержать два столбца: Metric и Value. Включи следующие строки: Report_Date (дата данных о ценах), Stocks_Covered (число 5), Highest_Close_Symbol (тикер с самой высокой ценой закрытия), Highest_Close_Price (это значение цены закрытия), Highest_Volume_Symbol (тикер с самым высоким объёмом торгов) и Highest_Volume (это значение объёма).

Затем создай PDF-отчёт под названием Market_Report.pdf в рабочей директории. В PDF вверху должен быть заголовок "Market Overview Report" (допускается русский вариант "Обзор рынка"), за ним — дата отчёта и количество охваченных акций. Ниже размести таблицу, в которой для каждой акции указаны символ (symbol), название компании, цена закрытия, объём, коэффициент PE и сектор. Отсортируй таблицу по алфавиту по символу.

Важно: при работе с инструментами Excel и PDF передавай АБСОЛЮТНЫЙ путь к файлам Market_Overview_Report.xlsx и Market_Report.pdf внутри рабочей директории (рабочая директория указана в системном промпте), поскольку эти инструменты могут не сохранять текущий каталог между вызовами.
