# Методические указания — Кейтеринг

## Методология
Этот анализ сопоставляет пищевую ценность блюд с дневными нормами по нутриентам.
Для каждого рецепта оценивается калорийность (Calories) и содержание белка (Protein_g).
Содержание белка сравнивается с дневной нормой по белку (Protein daily_g), полученной из внешнего источника нормативов питания.

## Приоритетные действия
В первую очередь обращайте внимание на блюда с низким содержанием белка относительно нормы.
Блюдо считается соответствующим норме (Meets_Guidelines = «Да»), если содержание белка не ниже дневной нормы.

## Отчётность
Указывайте источники данных, методологию оценки и ключевые показатели во всех отчётах.
