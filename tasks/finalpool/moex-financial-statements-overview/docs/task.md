Мне нужен обзор доступной финансовой отчётности по нашим отслеживаемым акциям на Московской бирже (MOEX). В рабочей директории лежит файл `Analysis_Criteria.pdf` с описанием типов отчётности, которые нас интересуют.

Отслеживаемые тикеры (суффикс `.ME` — Московская биржа, валюта — рубли):
`SBER.ME`, `GAZP.ME`, `LKOH.ME`, `TCSG.ME`, `MGNT.ME`, `MTSS.ME`.

Используй MCP `moex-finance` (тот же интерфейс, что и yahoo-finance: `get_financial_statement` и т.д.). Для каждого тикера запроси типы отчётности, перечисленные в `Analysis_Criteria.pdf` (`income_stmt`, `balance_sheet`, `cashflow`), и посчитай, за сколько периодов доступны данные по каждому сочетанию «тикер + тип отчётности». Включай в итог только те сочетания, по которым реально вернулись данные (где периодов нет — строку не добавляй).

Создай Excel-файл `YF_Financial_Statements.xlsx` с двумя листами.

Лист `Financial Overview` — колонки `Symbol_Type` (в формате `"SYMBOL - stmt_type"`, например `SBER.ME - income_stmt`) и `Periods` (количество доступных периодов). Отсортируй строки по `Symbol_Type` по алфавиту (A-Z).

Лист `Summary` — колонки `Metric` и `Value` со строками:
- `Total_Records` — общее число строк на листе `Financial Overview`;
- `Unique_Symbols` — число уникальных тикеров, попавших в обзор;
- `Statement_Types` — число различных типов отчётности (`stmt_type`), попавших в обзор.
