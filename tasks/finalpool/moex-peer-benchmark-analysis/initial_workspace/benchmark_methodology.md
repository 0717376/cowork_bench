# Методология бенчмаркинга по аналогам

## Расчёт потенциала роста (Upside Potential)

Потенциал роста акции рассчитывается так:

Upside_Pct = ((Target_Price - Current_Price) / Current_Price) * 100

Положительное значение означает, что акция торгуется ниже аналитического таргета (потенциально недооценена). Отрицательное значение означает, что акция торгуется выше таргета (потенциально переоценена).

## Относительная оценка по PE

Trailing PE каждой акции сравнивается с бенчмарком PE по её сектору:

- Если Trailing_PE > Sector_PE_Benchmark: акция торгуется с премией ("Premium") к аналогам.
- Если Trailing_PE <= Sector_PE_Benchmark: акция торгуется с дисконтом ("Discount") к аналогам.

## Величина премии/дисконта по PE

Величина премии или дисконта по PE рассчитывается так:

PE_Premium = Trailing_PE - Sector_PE_Benchmark

Положительное значение означает, что акция дороже относительно среднего по сектору.
