Вы — портфельный управляющий, который готовит сравнительный анализ по аналогам (peer benchmark) для инвестиционного комитета. В вашей рабочей директории лежат два справочных файла. Файл benchmark_methodology.md объясняет, как рассчитывать потенциал роста как процентную разницу между аналитическим таргетом и текущей рыночной ценой, и как сравнивать trailing PE акции с бенчмарком PE по сектору. Файл peer_list.json описывает состав группы аналогов по каждому сектору.

Начните с захода на портал аналитического консенсуса по адресу http://localhost:30223/index.html — там находится таблица консенсус-данных по пяти позициям портфеля: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME и MTSS.ME (тикеры Московской биржи, суффикс `.ME`, цены в рублях RUB). В таблице по каждой акции указаны таргет (target price), консенсус-рейтинг (Buy, Strong Buy, Hold и т.д.) и значение бенчмарка PE по сектору, используемое для относительной оценки. Внимательно считайте все данные с этой страницы.

Затем подключитесь к источнику биржевых данных через MCP `moex-finance` (интерфейс тот же, что и у yahoo-finance) и получите текущую рыночную цену (current price) и trailing PE по каждой из пяти акций. Эти фактические цены нужны для расчёта потенциала роста или снижения относительно аналитических таргетов.

Создайте Excel-книгу с именем Peer_Benchmark.xlsx в вашей рабочей директории с двумя листами. Первый лист назовите "Consensus Analysis" с колонками Symbol, Current_Price, Target_Price, Upside_Pct, Consensus, Trailing_PE, PE_Benchmark и PE_vs_Benchmark. Заполните его пятью акциями, отсортированными по символу (Symbol) в алфавитном порядке. Upside_Pct рассчитывается как (Target_Price минус Current_Price), делённое на Current_Price, умноженное на 100, округлённое до двух знаков после запятой. Колонка PE_vs_Benchmark должна содержать "Premium", если trailing PE акции превышает бенчмарк PE её сектора, либо "Discount", если он равен бенчмарку или ниже него.

Второй лист назовите "Summary" с колонками Metric и Value. Включите следующие строки: Avg_Upside_Pct — простое среднее процентов роста по всем пяти акциям, округлённое до двух знаков; Stocks_Above_Target — сколько акций имеют текущую цену выше таргета (то есть отрицательный потенциал роста); Stocks_Below_Target — сколько акций имеют текущую цену ниже таргета (положительный потенциал роста); Avg_PE_Premium — среднее по всем акциям значение (trailing PE минус бенчмарк PE), округлённое до двух знаков; Most_Undervalued — символ акции с наибольшим положительным потенциалом роста.

Также создайте Google-таблицу (Google Sheet) с названием "Peer Benchmark Data" с единственным листом "Benchmark", содержащим те же колонки и данные, что и лист Consensus Analysis в Excel-файле, чтобы инвестиционный комитет мог получить к ним доступ онлайн.

Если MCP для работы с Excel теряет текущую рабочую директорию, передавайте полный абсолютный путь к файлу Peer_Benchmark.xlsx внутри вашей рабочей директории.
