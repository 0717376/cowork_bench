Отдел кадров хочет понять взаимосвязь между уровнем образования и показателями эффективности сотрудников по всем отделам, чтобы определить, где инвестиции в обучение дадут наибольший эффект. В рабочей директории есть PDF-файл Training_Policy.pdf, который описывает распределение бюджета компании на обучение в зависимости от уровня образования. Сначала прочитайте его, чтобы понять контекст политики.

Затем подключитесь к хранилищу данных ClickHouse и сделайте запрос данных о сотрудниках. Получите количество сотрудников и средний показатель эффективности с группировкой по отделу и уровню образования, чтобы каждая комбинация «отдел — уровень образования» была представлена. Отсортируйте результаты в алфавитном порядке сначала по отделу, затем по уровню образования.

Важно: значения отделов и уровней образования в ClickHouse возвращаются на русском языке. В Excel записывайте эти значения ровно так, как их возвращает ClickHouse (то есть на русском), не переводите их обратно на английский.

Создайте в рабочей директории файл Excel с именем HR_Education_Analysis.xlsx с двумя листами.

Первый лист должен называться "Education Breakdown" и содержать четыре столбца: Department, Education_Level, Employee_Count и Avg_Performance (округлённый до 2 знаков после запятой). Каждая строка представляет одну уникальную комбинацию отдела и уровня образования, отсортированную в алфавитном порядке сначала по столбцу Department.

Второй лист должен называться "Department Summary" и содержать семь столбцов: Department (отсортированный по алфавиту), Total_Employees, High_School_Count, Bachelor_Count, Masters_Count, PhD_Count и Higher_Ed_Pct. Higher_Ed_Pct — это процент сотрудников с уровнем образования «Магистр» или «Кандидат наук» от общего числа сотрудников отдела, округлённый до 1 знака после запятой. Уровень образования «Диплом» должен учитываться в общем числе сотрудников (Total_Employees), но не входить ни в один из перечисленных выше столбцов по конкретным уровням образования.

Столбцы по уровням образования соответствуют следующим значениям из ClickHouse: High_School_Count — «Среднее образование», Bachelor_Count — «Бакалавр», Masters_Count — «Магистр», PhD_Count — «Кандидат наук».

После создания файла Excel создайте Google-форму под названием "Training Interest Survey" со следующими четырьмя вопросами: один — с запросом отдела респондента, один — с запросом текущего уровня образования, один вопрос с выбором вариантов о сферах интереса к обучению (с вариантами вроде «Технические навыки», «Лидерство», «Коммуникация», «Анализ данных») и один — о предпочитаемом формате обучения (с вариантами вроде «Очный семинар», «Онлайн в своём темпе», «Живой вебинар», «Обучение на рабочем месте»).

Наконец, отправьте письмо на адрес training@company.com с темой, кратко резюмирующей результаты анализа образования, и кратким текстом письма, в котором отражены ключевые выводы о распределении уровней образования по отделам.
