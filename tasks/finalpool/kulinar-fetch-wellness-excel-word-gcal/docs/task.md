Я разрабатываю корпоративную программу здорового питания (велнес-программу) для нашей компании. Сначала найдите рецепты в нашей кулинарной базе рецептов по разным категориям: блюда к завтраку (например, выпечка и десерты), блюда для обеда (например, горячее и супы) и полезные гарниры. Получите не менее 10 рецептов вместе с их ингредиентами и информацией о пищевой ценности.

По адресу http://localhost:30343/api/wellness_guidelines.json доступен API велнес-программы с диетическими нормами: целевая суточная калорийность, соотношение макронутриентов и рекомендации по группам продуктов. Пожалуйста, получите эти данные.

С помощью терминала создайте и запустите Python-скрипт wellness_planner.py в рабочей директории, который читает recipes.json и wellness_guidelines.json (сначала создайте оба файла), оценивает каждый рецепт по диетическим нормам, рассчитывает баллы соответствия по питательности, формирует недельный план питания и выводит результат в wellness_plan.json.

Создайте Excel-файл Corporate_Wellness_Plan.xlsx с четырьмя листами. Первый лист Recipe_Evaluation должен содержать столбцы Recipe_Name, Category, Difficulty, Ingredient_Count, Estimated_Calories и Wellness_Score (от 1 до 10, округлить до 1 знака после запятой), отсортированные по Wellness_Score по убыванию. Второй лист Weekly_Plan должен содержать столбцы Day (с понедельника по пятницу), Breakfast, Lunch и Snack с названиями рекомендованных рецептов. Третий лист Nutritional_Summary должен содержать столбцы Meal_Slot, Avg_Calories, Protein_Pct, Carb_Pct, Fat_Pct с расчётной разбивкой по макронутриентам. Четвёртый лист Program_Metrics должен содержать столбцы Metric и Value со значениями Total_Recipes_Evaluated, Recipes_Meeting_Guidelines, Weekly_Avg_Calories, Program_Compliance_Pct (округлить до 1 знака после запятой) и Recommended_Budget_Per_Person (округлить до 2 знаков после запятой, в рублях).

Создайте документ Word с названием Wellness_Program_Guide.docx с заголовком «Корпоративная велнес-программа питания» и разделами «Обзор программы», «Недельный план питания», «Соответствие нормам питания» и «Рекомендации по внедрению». Когда обращаетесь к инструменту Word, передавайте абсолютный путь к файлу .docx внутри рабочей директории.
