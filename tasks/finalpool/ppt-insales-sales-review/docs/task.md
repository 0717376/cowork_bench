Отделу продаж нужен полный пакет материалов для ежемесячного обзора продаж. Вы отвечаете за сбор данных о заказах из нашего интернет-магазина InSales, добавление рыночного контекста из финансового сервиса MOEX и доставку итоговых материалов по электронной почте.

Начните с получения всей информации о заказах из нашего интернет-магазина InSales. Рассчитайте общее количество заказов, число выполненных заказов (статус `completed`), суммарную выручку по всем заказам и средний чек (средняя стоимость заказа). Также определите топ-10 товаров по выручке, зафиксировав для каждого название товара, общее количество проданных единиц и суммарную полученную выручку.

Затем посмотрите динамику котировок акций Сбербанка на Московской бирже (тикер `SBER.ME`), чтобы добавить рыночный контекст. Получите самую последнюю цену закрытия и рассчитайте приблизительное процентное изменение за последний месяц, сравнив самую свежую цену закрытия с ценой закрытия примерно месячной давности.

Создайте книгу Excel с именем `Monthly_Sales_Review.xlsx` в рабочей директории с тремя листами. Первый лист назовите `Sales Summary`; он должен содержать два столбца — `Metric` и `Value` — и четыре строки данных: `Total_Orders`, `Completed_Orders`, `Total_Revenue` и `Avg_Order_Value`. Денежные значения округлите до двух знаков после запятой. Второй лист назовите `Top Products`; он должен содержать три столбца: `Product_Name`, `Units_Sold` и `Revenue`. Перечислите топ-10 товаров, отсортированных по выручке по убыванию. Третий лист назовите `Market Context`; он должен содержать три столбца: `Index`, `Recent_Close` и `Monthly_Change_Pct`. Включите одну строку для выбранного инструмента MOEX, используя тикер `SBER.ME` в качестве значения столбца `Index`. Процентное изменение за месяц округлите до двух знаков после запятой.

Создайте презентацию PowerPoint с именем `Sales_Review_Presentation.pptx` в рабочей директории. Она должна содержать не менее пяти слайдов. Первый слайд — титульный, с заголовком `Monthly Sales Review`. Второй слайд должен представлять обзор продаж: общее число заказов, число выполненных заказов, суммарную выручку и средний чек. Третий слайд должен выделять самые продаваемые товары. Четвёртый слайд должен представлять рыночный контекст с динамикой котировок MOEX (`SBER.ME`). Пятый слайд должен содержать рекомендации или ключевые выводы для отдела продаж.

Наконец, отправьте письмо на адрес `sales@company.com` с темой `Monthly Sales Review`. Тело письма должно кратко излагать ключевые результаты, включая общее число заказов, суммарную выручку, самый продаваемый товар по выручке и месячную динамику котировок MOEX (`SBER.ME`).

Примечание: при работе с инструментами Excel и PowerPoint передавайте АБСОЛЮТНЫЙ путь к файлам `.xlsx` и `.pptx` внутри рабочей директории.
