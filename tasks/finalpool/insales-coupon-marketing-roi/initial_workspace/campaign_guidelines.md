Рекомендации по оценке результативности кампаний

Минимальный порог ROI: 200%
Расчёт ROI: ((Выручка от заказов с купоном - Бюджет кампании) / Бюджет кампании) * 100

Целевая конверсия: 3% (заказы / клики * 100)

API данных о кампаниях: http://localhost:30204/api/campaigns.json

Этапы анализа:
1. Получить данные о кампаниях из API
2. Для каждой кампании с кодом купона найти этот купон в магазине
3. Использовать usage_count купона как число сгенерированных заказов
4. Рассчитать выручку, умножив usage_count на средний чек
5. Поскольку точную выручку по каждому купону из данных заказов отследить нельзя,
   использовать смоделированную выручку = usage_count * 50 (оценка среднего чека)
6. Рассчитать ROI и конверсию для каждой кампании
7. Анализировать только кампании, у которых есть действительный код купона
