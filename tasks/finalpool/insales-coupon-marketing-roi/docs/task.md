Вы — маркетинг-аналитик интернет-магазина на платформе InSales. Вам нужно оценить рентабельность (ROI) каждой маркетинговой кампании, связав данные о расходах на кампании с данными об использовании купонов магазина, а затем создать опрос для сбора обратной связи от команды маркетинга.

Начните с изучения таблицы маркетингового бюджета (marketing_budget.xlsx) и документа с рекомендациями по кампаниям (campaign_guidelines.md) в вашем рабочем каталоге. В рекомендациях описаны формула расчёта ROI, минимальный порог ROI в 200 процентов, целевая конверсия в 3 процента и методика оценки выручки.

Получите данные о результативности кампаний из API по адресу http://localhost:30204/api/campaigns.json, который возвращает список кампаний с указанием названия кампании, маркетингового канала, потраченного бюджета, числа показов, кликов и кода купона, связанного с каждой кампанией. Учтите, что у некоторых кампаний может не быть кода купона либо они могут ссылаться на коды, отсутствующие в вашем магазине.

Затем получите все данные о купонах из вашего магазина InSales. Для каждого купона запишите код, тип скидки, размер скидки и количество использований (usage count). Количество использований — это число заказов, в которых был применён данный купон.

Свяжите каждую кампанию с соответствующим купоном магазина, сопоставив код купона из данных кампании с кодом купона в магазине. Анализируйте только те кампании, у которых есть действительный код купона, существующий в магазине. Рассчитайте выручку для каждой кампании как количество использований купона, умноженное на 50 (средний чек оценивается в 50 ₽). Вычислите ROI в процентах как (выручка минус бюджет) делить на бюджет умножить на 100. Рассчитайте конверсию как количество использований делить на число кликов умножить на 100.

Создайте файл Excel с именем Campaign_ROI.xlsx в вашем рабочем каталоге с тремя листами. Первый лист с именем "Campaign Performance" должен содержать столбцы Campaign_Name, Channel, Coupon_Code, Budget, Usage_Count, Revenue, ROI_Pct, Conversion_Rate, Meets_ROI_Target и Meets_Conversion_Target. Установите Meets_ROI_Target в Yes, если ROI составляет 200 процентов или выше, и No в противном случае. Установите Meets_Conversion_Target в Yes, если конверсия составляет 3 процента или выше, и No в противном случае.

Второй лист с именем "Channel Summary" должен агрегировать данные кампаний по маркетинговым каналам со столбцами Channel, Total_Budget, Total_Revenue, Channel_ROI_Pct и Campaign_Count.

Третий лист с именем "Recommendations" должен содержать два столбца Category и Finding со строками: Campaigns Meeting ROI Target с указанием количества, Campaigns Below ROI Target с указанием количества, Best Channel by ROI с названием канала с наивысшим ROI и Worst Channel by ROI с названием канала с наименьшим ROI.

Создайте форму с заголовком "Campaign Effectiveness Feedback" для сбора мнений команды. Форма должна включать вопрос с множественным выбором о том, какой маркетинговый канал оказался наиболее эффективным, с вариантами, соответствующими каналам из данных кампаний; вопрос с коротким ответом о предложениях по улучшению ROI кампаний; и вопрос с множественным выбором о предпочтительной частоте кампаний с вариантами Monthly, Bi-weekly, Weekly и Quarterly.

Примечание: при вызове инструмента Excel для записи файла Campaign_ROI.xlsx передавайте АБСОЛЮТНЫЙ путь к этому файлу внутри вашего рабочего каталога.
