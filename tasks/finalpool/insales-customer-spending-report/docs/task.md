Мне нужен анализ расходов клиентов нашего магазина InSales. Выгрузите всех клиентов, оформивших хотя бы один заказ, и отранжируйте их по сумме потраченных средств.

Учитывайте только оплаченные/выполненные заказы — со статусом 'completed' или 'processing'; заказы в статусах on-hold, pending, cancelled, refunded, failed не считаются.

Создайте Excel-файл с именем WC_Customer_Report.xlsx с двумя листами. Лист "Customer Spending" должен содержать колонки Customer_ID, Name, Email, число заказов Orders, Total_Spent (округлить до 2 знаков после запятой) и Avg_Order_Value (округлить до 2 знаков после запятой). Отсортируйте по Total_Spent по убыванию.

Лист "Summary" должен содержать Total_Active_Customers, Total_Spent_All (округлить до 2 знаков после запятой), имя Top_Customer и Avg_Lifetime_Value (округлить до 2 знаков после запятой).

Также создайте Google-таблицу с названием "Customer Analysis" с листом "Spending", содержащим те же данные по клиентам.

Важно: при записи Excel-файла передавайте инструменту АБСОЛЮТНЫЙ путь к WC_Customer_Report.xlsx внутри рабочей директории агента (excel MCP не сохраняет текущую рабочую директорию).
