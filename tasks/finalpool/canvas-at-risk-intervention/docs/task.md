Учебной части университета нужно выявить студентов, которым грозит неуспеваемость по их курсам, и разработать план поддержки. По адресу http://localhost:30216/api/support_resources.json доступна страница с ресурсами поддержки, где перечислены доступные в университете службы академической помощи: репетиторство (tutoring), учебные группы (study groups) и психологическое консультирование (counseling). Сначала получите этот перечень ресурсов, чтобы понять, какие варианты поддержки существуют.

Затем выгрузите данные о записях студентов на курсы и их оценках по всем курсам. Студент считается «в зоне риска» (at risk), если его текущий балл (current score) ниже 50 по любому курсу. Студентов с баллом ниже 40 следует отнести к уровню риска «Critical», а студентов с баллом от 40 до 50 — к уровню «Warning». Для каждого студента из зоны риска определите наиболее подходящий ресурс поддержки исходя из предметной области курса. Студентам уровня «Critical» в дополнение к предметной поддержке всегда должно назначаться направление на консультирование (counseling referral).

Прочитайте файл контактов кураторов advisor_contacts.json в рабочем каталоге, чтобы определить правильных получателей письма с отчётом об интервенции. Также изучите документ с методическими указаниями intervention_guidelines.pdf, в котором изложена политика университета относительно порогов раннего оповещения и обязательных встреч.

Создайте файл Excel под названием At_Risk_Report.xlsx с тремя листами. Первый лист с именем «At Risk Students» должен содержать колонки Student_Name, Course, Current_Score, Risk_Level (либо Critical, либо Warning) и Recommended_Support. Отсортируйте строки по Current_Score от наименьшего к наибольшему. Второй лист с именем «Course Summary» должен содержать колонки Course, At_Risk_Count (число студентов в зоне риска на этом курсе) и Avg_Score (средний текущий балл по всем записанным на курс студентам, а не только тем, кто в зоне риска). Включите все 22 курса, отсортированные по алфавиту. Третий лист с именем «Action Plan» должен содержать колонки Priority, Action, Target_Group, Timeline и Responsible, минимум с пятью пунктами плана действий, охватывающими обязательные встречи для студентов уровня Critical, организацию репетиторства, создание учебных групп, направления на консультирование и мониторинг прогресса.

При работе с инструментами Excel/Word, теряющими текущий каталог, всегда передавайте абсолютный путь к файлу At_Risk_Report.xlsx внутри рабочего каталога.

После создания отчёта отправьте письмо на адрес academic-support@uni.edu с темой «At-Risk Student Intervention Report», в котором приведите итоговое число выявленных студентов в зоне риска, курсы с наибольшим числом таких студентов и рекомендуемые дальнейшие шаги.
