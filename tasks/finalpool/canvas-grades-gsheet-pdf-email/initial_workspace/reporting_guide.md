# Стандарты отчётности по оценкам

Стандартная шкала оценок:
- A: 90-100
- B: 80-89
- C: 70-79
- D: 60-69
- F: ниже 60

Успевающий: оценка C или выше (70+)
Порог академической задолженности: ниже 60

Отчёты должны включать:
- Распределение оценок по каждому курсу
- Доли успевающих и неуспевающих
- Сравнение между курсами
- Отметку курсов с долей успевающих ниже 70%
