Вы — академический консультант (научный руководитель) университета, помогающий студентам спланировать учебное расписание на предстоящий семестр. Вам нужно проанализировать данные о доступных курсах, создать структурированную базу знаний, запланировать учебные сессии и подготовить подробный отчёт по учебному плану.

Начните с чтения файла Study_Planning_Guide.pdf в вашей рабочей директории. В нём описана методология распределения учебного времени в зависимости от сложности курса и нагрузки по заданиям.

Обратитесь к системе управления обучением (LMS, Canvas) и получите информацию о пяти курсах с наибольшим числом записавшихся студентов (по показателю enrollment). Для каждого курса узнайте название курса (course name), число записавшихся (enrollment), а также количество заданий (assignments) и тестов (quizzes). Эти пять курсов имеют идентификаторы (ID) 7, 17, 4, 16 и 3 — именно они отобраны по наибольшему числу записавшихся.

Для каждого из этих пяти курсов также получите список заданий с их названиями и сроками сдачи. Кроме того, получите данные по тестам, включая названия тестов и количество баллов за них.

Затем напишите и запустите Python-скрипт под названием study_planner.py, используя инструменты командной строки (terminal) в вашей рабочей директории. Скрипт должен взять собранные данные по курсам и рассчитать рекомендуемое число учебных часов в неделю (weekly study hours) для каждого курса. Формула: базовые 3 часа на курс, плюс 0,5 часа за каждое задание (assignment) в курсе, плюс 1 час за каждый тест (quiz). Скрипт также должен присвоить каждому курсу уровень приоритета: High, если недельных часов больше 8; Medium, если от 5 до 8 часов включительно; и Low, если меньше 5 часов.

Создайте базу знаний в командной вики (Teamly). Заведите отдельное пространство (space) и в нём создайте страницу под названием «Student Study Planner». На этой странице разместите структурированную таблицу с колонками: Course Name, Enrollment, Assignments, Quizzes, Weekly Hours и Priority (значения Priority — одно из High, Medium, Low). Добавьте по одной строке на каждый из пяти курсов с рассчитанными значениями. В тексте страницы для каждого курса должны присутствовать его название и численное значение Weekly Hours.

Запланируйте учебные сессии в общем календаре (Google Calendar) на неделю с 9 по 13 марта 2026 года. Для каждого из пяти курсов создайте одно событие — учебную сессию. Курсы с приоритетом High получают сессии по 2 часа, Medium — по 1,5 часа, Low — по 1 часу. Распределите сессии по дням недели: начните в 9:00 утра в понедельник и планируйте по одной сессии в день (с понедельника по пятницу, без выходных). В названии (summary) каждого события должны присутствовать название курса и текст «Study Session».

Создайте в рабочей директории книгу Excel под названием Study_Plan_Report.xlsx с тремя листами. Когда вызываете инструмент Excel, передавайте АБСОЛЮТНЫЙ путь к файлу Study_Plan_Report.xlsx внутри рабочей директории агента.

Первый лист должен называться Course_Analysis и содержать колонки Course_Name, Enrollment, Assignments, Quizzes, Weekly_Hours и Priority. Включите по одной строке на каждый из пяти курсов.

Второй лист должен называться Weekly_Schedule и содержать колонки Day, Time, Course, Duration_Hours и Session_Type. Включите по одной строке на каждое созданное вами событие календаря, отражая дни с понедельника по пятницу и соответствующие данные об учебных сессиях.

Третий лист должен называться Priority_Matrix и содержать колонки Priority_Level, Course_Count, Total_Weekly_Hours и Avg_Hours_Per_Course. Включите по одной строке для каждого из уровней приоритета High, Medium и Low, обобщая курсы в каждой категории.
