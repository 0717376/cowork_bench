Мне нужна помощь в подготовке сверочного отчёта по продажам за Q1 2025. В рабочей директории есть PDF-файл Q1_2025_Regional_Targets.pdf, в котором указаны целевые показатели выручки по каждому региону, установленные руководством, — пожалуйста, сначала прочитайте его.

Затем выгрузите фактические показатели продаж за Q1 2025 из нашего корпоративного хранилища данных ClickHouse (логическая база данных SALES_DW). Q1 — это период с января по март 2025 года. Мне нужна фактическая выручка в разбивке по регионам, а также топ-3 самых продаваемых товаров по выручке внутри каждого региона.

Также есть внутренняя аналитическая панель по адресу http://localhost:30160 с отраслевыми бенчмарками — выгрузите их тоже, чтобы мы могли увидеть, как мы выглядим на их фоне.

Соберите всё вместе в Excel-файл Q1_Sales_Review.xlsx в рабочей директории. В нём должно быть три листа. Первый, Regional Performance, должен содержать по одной строке на регион, отсортированных по алфавиту, со столбцами Region, Target, Actual, Variance (это Actual минус Target), Variance_Pct (Variance делённое на Target, умноженное на 100, округлённое до 1 знака после запятой, как число, а не строка), Industry_Benchmark и vs_Benchmark (записать "Above", если Actual больше или равно бенчмарку, иначе "Below"). Второй лист, Top Products, перечисляет топ-3 товара по каждому региону, отсортированных по региону по алфавиту, затем по выручке по убыванию, со столбцами Region, Product_Name, Units_Sold и Revenue, округлённой до 2 знаков после запятой. Третий лист, Summary, — это простая таблица из двух столбцов, начинающаяся с ячейки A1, с подписями в столбце A и значениями в столбце B: Total_Target, Total_Actual (округлённое до 2 знаков после запятой), Overall_Variance, Overall_Variance_Pct (округлённое до 1 знака после запятой), Regions_Met_Target (количество регионов, где Actual >= Target), Regions_Missed_Target и Regions_Above_Benchmark.

Обратите внимание: названия регионов в хранилище данных, в PDF и на аналитической панели указаны на русском языке — используйте их как есть для сопоставления источников. Названия товаров (Product_Name) и все заголовки столбцов, имена листов, ключи на листе Summary и имя файла оставьте на английском без изменений.
