# Разработка программы благополучия сотрудников

Шаги для создания эффективных инициатив по благополучию:
1. Оценить текущий уровень удовлетворённости по отделам
2. Выявить болевые точки и области с низкими оценками
3. Разработать целевые мероприятия
4. Создать механизмы обратной связи (опросы)
5. Запланировать стартовое мероприятие и последующие обзоры

Шкала оценки удовлетворённости:
- Оценки выставляются по шкале от 1 до 5
- Ниже 3.0 требует немедленного внимания
- 3.0-3.5 требует мониторинга и улучшения
- Выше 3.5 приемлемо, но всё равно стоит улучшать
