Я профессор финансов и планирую серию семинаров по финансовой грамотности (Financial Literacy Workshop) для студентов двух моих курсов «Основы финансов» (Foundations of Finance, идентификаторы курсов 16 и 17 в системе управления обучением). Мне нужно объединить данные об успеваемости студентов с реальными рыночными данными, чтобы составить целевой план семинаров. Сначала прочитайте файлы workshop_template.xlsx и workshop_guidelines.md в вашей рабочей директории, чтобы узнать формат вывода и определения уровней (tiers), а также прочитайте contact_list.json, чтобы получить адреса электронной почты заинтересованных лиц.

Сначала выгрузите данные о сданных работах студентов из обоих курсов «Основы финансов». Для каждого студента рассчитайте его средний балл в процентах (балл, делённый на максимально возможное количество баллов, умноженное на 100, по всем оценённым работам). Затем распределите каждого студента по уровням успеваемости: «Needs Support» для среднего балла ниже 60, «Developing» для среднего балла от 60 до 74,99 и «Proficient» для среднего балла 75 и выше. Напишите Python-скрипт под названием categorize_students.py, который выполняет это распределение и выводит количество студентов по каждому уровню для каждого курса, и запустите его.

Затем выгрузите свежие данные о ценах акций GAZP.ME, MGNT.ME и MTSS.ME из финансового сервиса. Напишите Python-скрипт под названием find_market_events.py, который определяет 3 крупнейших дневных изменения цены (по абсолютному процентному изменению между ценой открытия и закрытия) за последние 30 торговых сессий (последние 30 строк дневных торговых данных по каждому тикеру, а не календарные 30 дней). Скрипт должен вывести файл под названием market_events.json, содержащий каждое событие с датой, тикером акции, процентным изменением, округлённым до 2 знаков после запятой, и кратким описанием, например «GAZP вырос на 1.45% на фоне сильной отчётности» или «GAZP упал на -1.45% из-за опасений в секторе». Запустите этот скрипт.

Теперь проверьте общий календарь на наличие существующих событий в двухнедельном окне, начиная с сегодняшнего дня. Я хочу запланировать три семинара, каждый продолжительностью 90 минут, в 14:00. Первый семинар «Intro to Markets» предназначен для уровня «Needs Support» и должен быть запланирован на первый доступный будний день (с понедельника по пятницу) в двухнедельном окне, в котором нет конфликтующих событий между 13:00 и 16:00. Второй семинар «Portfolio Basics» предназначен для уровня «Developing» и должен быть на следующий доступный будний день после первого. Третий семинар «Risk Management» предназначен для уровня «Proficient» и должен быть на следующий доступный будний день после второго. Создайте все три как события в календаре, добавив в описание, какой уровень успеваемости является основной аудиторией, и ожидаемое количество участников для этого уровня по обоим курсам вместе.

Создайте Excel-книгу под названием Financial_Literacy_Workshops.xlsx с тремя листами. Первый лист «Student_Tiers» должен иметь столбцы Course_ID, Course_Name, Needs_Support, Developing, Proficient и Total_Students, по одной строке на каждый курс, отсортированные по Course_ID. Второй лист «Market_Events» должен иметь столбцы Date, Symbol, Change_Pct и Event_Description, с тремя крупнейшими событиями, отсортированными по абсолютному изменению по убыванию. Третий лист «Workshop_Schedule» должен иметь столбцы Workshop_Date, Topic, Target_Tier и Expected_Attendance, где ожидаемое количество участников — это суммарное количество студентов на этом уровне из обоих курсов.

Отправьте письмо на адрес студентов-финансистов из списка контактов с темой «Financial Literacy Workshop Series Announcement», содержащее три даты семинаров, темы и краткое описание каждого занятия на основе выявленных рыночных событий. Затем отправьте отдельное письмо на адрес заведующего кафедрой из списка контактов с темой «Workshop Planning Summary», которое включает сводку по уровням успеваемости студентов по обоим курсам и запланированное расписание семинаров.

Наконец, напишите Python-скрипт под названием generate_outline.py, который читает market_events.json и создаёт файл workshop_materials.txt. План должен содержать раздел для каждого из трёх семинаров, включая релевантные примеры рыночных событий в качестве пунктов для обсуждения. Запустите скрипт, чтобы выходной файл оказался в вашей рабочей директории.
