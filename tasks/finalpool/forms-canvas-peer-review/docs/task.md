Вы помогаете преподавателю курса "Biochemistry & Bioinformatics" обработать результаты взаимного оценивания (peer review) недавнего группового проекта. Форма опроса с названием "Group Project Peer Review" уже была разослана и собрала оценки студентов друг о друге. Каждый ответ содержит имя оценивающего (reviewer), имя оцениваемого (reviewee) и оценки по трём критериям (Contribution, Communication и Quality of Work, каждый по шкале 1-5), а также необязательный текстовый комментарий.

Выполните, пожалуйста, следующие задачи:

Сначала запросите Canvas, чтобы найти курс "Biochemistry & Bioinformatics" и изучить список зачисленных студентов (class roster).

Затем получите все ответы из формы "Group Project Peer Review" через MCP forms (используйте list_forms, чтобы найти форму, и get_form_responses, чтобы получить ответы).

Затем создайте файл Excel с именем `Peer_Review_Analysis.xlsx` в рабочем каталоге, содержащий три листа:
   Лист "Raw Scores": перечислите каждый ответ взаимного оценивания как строку со столбцами Reviewer, Reviewee, Contribution, Communication, Quality, Average_Score (среднее трёх оценок данного ответа, округлённое до двух знаков после запятой).
   Лист "Individual Summary": для каждого уникального студента, которого оценивали, вычислите и перечислите: Student_Name, Avg_Contribution (среднее всех полученных оценок Contribution, округлённое до двух знаков), Avg_Communication (среднее всех полученных оценок Communication, округлённое до двух знаков), Avg_Quality (среднее всех полученных оценок Quality, округлённое до двух знаков), Overall_Avg (среднее трёх средних по категориям, округлённое до двух знаков), Review_Count (количество полученных оценок).
   Лист "Flagged": перечислите любого студента, чей Overall_Avg строго меньше 3.0. Столбцы: Student_Name, Overall_Avg, Flag_Reason. В поле Flag_Reason укажите, что общее среднее студента ниже порога 3.0.

Важно: MCP Excel теряет текущий рабочий каталог, поэтому при создании/записи файла передавайте АБСОЛЮТНЫЙ путь к `Peer_Review_Analysis.xlsx` внутри рабочего каталога агента.

Затем создайте Google-таблицу (Google Sheet) с названием "Peer Review Results" и заполните её данными Individual Summary (те же столбцы, что и на листе Individual Summary в файле Excel).

Затем создайте в Teamly страницу с названием "Peer Review Summary - Biochemistry Project", которая включает:
   обзор процесса взаимного оценивания (количество собранных оценок, количество оценённых студентов);
   список студентов, помеченных как flagged, с их баллами и причинами;
   рекомендации по дальнейшим действиям (follow-up) для flagged-студентов.
