Мне нужна помощь с анализом обзора публикаций arxiv. Доступны внешние бенчмарк-данные: получи их по адресу http://localhost:30325/api/data.json и извлеки нужные метрики (поле trending_topics, в каждом элементе — area и growth_pct).

Затем найди релевантные научные статьи по теме обзора.

С помощью терминала создай и запусти в рабочей директории Python-скрипт arxiv_survey_processor.py, который читает собранные данные из созданных тобой JSON-файлов, выполняет анализ и выводит результат в arxiv_survey_results.json.

Создай Excel-файл Survey_Report.xlsx с тремя листами. Передавай в инструмент Excel АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.

- Лист Data_Analysis должен содержать основную таблицу сопоставления по статьям со столбцами (заголовки именно на английском): `Paper_ID`, `Title`, `Area`, `Citations`, `Relevance_Score`. Поле `Area` должно соответствовать тематическим областям из бенчмарка (`LLMs`, `Computer Vision`, `RL`). Отсортируй строки по алфавиту по основному измерению (`Area`).
- Лист Metrics должен иметь два столбца `Metric` и `Value` и обобщать ключевые показатели: общее число статей, средние значения и ключевую статистику. Включи внешние бенчмарк-значения роста (`LLMs` — 45, `Computer Vision` — 12, `RL` — 8) и их среднее.
- Лист Recommendations должен перечислять приоритетные действия на основе анализа разрывов со столбцами `Priority`, `Action`, `Area`. Методология (см. analysis_guide.md): Gap = Internal Value − External Benchmark; в первую очередь приоритезируй области с наибольшим отрицательным разрывом.

Также создай документ Word под названием Survey_Analysis.docx с разделами: краткое резюме (executive summary), ключевые выводы и рекомендации. Передавай в инструмент Word АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.

Отправь письмо на адрес team-lead@company.com с темой "Analysis Report Complete", кратко излагающее ключевые выводы анализа (текст письма можно писать по-русски).

Создай в Teamly страницу с заголовком "Arxiv Survey Dashboard" с кратким резюме анализа.
