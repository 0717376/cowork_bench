Вы — координатор исследовательской группы, которому нужно задокументировать анализ трёх фундаментальных статей по языковым моделям. Ваша задача — проанализировать три конкретные статьи, доступные в базе статей, создать структурированные заметки в корпоративной базе знаний Teamly вашей команды и составить официальный отчёт в формате Word.

Три статьи для анализа: «Scaling Laws for Neural Language Models» (arXiv ID 2301.07041), «Training language models to follow instructions with human feedback» (arXiv ID 2203.11171) и «OPT: Open Pre-trained Transformer Language Models» (arXiv ID 2205.01068). Используйте инструменты arXiv и анализа LaTeX, чтобы получить полный текст и проанализировать структуру каждой статьи, её разделы и ключевой вклад.

Для каждой статьи создайте отдельную страницу в Teamly, где заголовком страницы будет название статьи с префиксом «Paper: » (название и префикс — на английском, как в оригинале). В теле страницы разместите структурированные заметки о главном вкладе статьи, методологии и ключевых выводах на основе полученного полного текста. Для размещения страниц используйте подходящее пространство в Teamly — посмотрите список существующих пространств и при необходимости создайте новое.

Создайте документ Word с именем Paper_Analysis_Report.docx в рабочей директории. Документ должен начинаться с заголовка «Comparative Paper Analysis Report» и содержать отдельный раздел по каждой из трёх статей. Заголовок каждого раздела должен чётко идентифицировать статью (используйте английское название статьи). Внутри раздела опишите название статьи, её ключевой вклад и методологию. После трёх отдельных разделов добавьте заключительный раздел с заголовком «Comparative Analysis», в котором проведите связи между тремя статьями и обсудите, как они соотносятся друг с другом по вкладу в исследования больших языковых моделей.

Отправьте письмо на адрес research_lead@university.edu с темой «Paper Analysis Report Ready», в котором кратко опишите выполненную работу и отметьте, что документ Word и страницы в Teamly готовы к проверке.

Примечание: названия статей, arxiv ID, имена авторов, префикс «Paper: », заголовки «Comparative Paper Analysis Report» и «Comparative Analysis», имя файла Paper_Analysis_Report.docx, адрес и тему письма оставляйте на английском — это технические идентификаторы.
