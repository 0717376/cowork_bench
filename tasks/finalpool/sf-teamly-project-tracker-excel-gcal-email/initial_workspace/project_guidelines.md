# Руководство по управлению портфелем проектов

Критерии отбора проектов:
- Стратегическое соответствие целям отдела
- Доступная ёмкость команды
- Ожидаемая отдача от инвестиций (ROI)
- Оценка рисков

Правила формирования команд:
- Руководитель проекта: лучший по результативности сотрудник профильного отдела
- Размер команды: 3-7 человек на проект
- Предпочтительны кросс-функциональные команды
- Не более 3 одновременных проектов на человека

Рекомендации по срокам:
- Стандартная длительность проекта: 3-6 месяцев
- Установочная встреча в течение 2 недель после утверждения
- Ежемесячные обзоры прогресса
- Ежеквартальный обзор портфеля проектов
