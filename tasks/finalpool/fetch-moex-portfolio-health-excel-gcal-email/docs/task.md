Мне нужна помощь с анализом инвестиционного портфеля. Есть внешние эталонные данные (benchmark), которые нужно получить по адресу http://localhost:30314/api/data.json — оттуда извлеки нужные метрики.

Затем подтяни актуальные рыночные данные по акциям нашего портфеля с Московской биржи (MOEX). Все цены в рублях (RUB).

Через терминал создай и запусти Python-скрипт под именем yf_portfolio_processor.py в рабочей директории. Скрипт должен прочитать собранные данные из JSON-файлов, которые ты создашь, выполнить анализ и записать результат в yf_portfolio_results.json.

Создай Excel-файл Portfolio_Health_Report.xlsx с тремя листами. Первый лист Data_Analysis должен содержать основную сравнительную таблицу с релевантными колонками. Второй лист Metrics — сводку ключевых метрик. Третий лист Recommendations — список действий, которые нужно предпринять.

Лист Data_Analysis должен включать колонки для основного измерения (Symbol — тикер), названия и сектора компании (Name, Sector), нашей текущей рыночной цены (Current_Price), внешней эталонной целевой цены (Target_Price) и разницы между ними — потенциала роста (Upside). Отсортируй данные по тикеру (Symbol) в алфавитном порядке. Лист Metrics должен иметь две колонки Metric и Value со сводными значениями: общее количество акций, средние величины и ключевые статистики. Лист Recommendations должен перечислять приоритетные действия на основе анализа Upside; используй колонки Priority, Action, Symbol.
