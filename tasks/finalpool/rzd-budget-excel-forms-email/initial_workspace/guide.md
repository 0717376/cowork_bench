# Руководство по бюджетному анализу: командировка Москва — Санкт-Петербург

## Обзор
Это руководство объясняет, как структурировать бюджетный анализ командировки восьми сотрудников из Москвы в Санкт-Петербург. Анализ охватывает три бюджетных сценария с разными профилями затрат.

## Бюджетные сценарии

### Сценарий Budget
- Класс обслуживания: «Эконом» (самый дешёвый доступный)
- Цель: минимизировать суммарную стоимость поездки
- Используйте этот сценарий, когда главное — контроль расходов

### Сценарий Standard
- Класс обслуживания: «Эконом+» на рекомендуемом утреннем поезде «Сапсан» (например, 754А, отправление 09:30)
- Цель: баланс комфорта и стоимости
- «Сапсан» — флагманский скоростной поезд Москва — Санкт-Петербург; класс «Эконом+» даёт больше места и более тихий вагон
- Используйте этот сценарий как рекомендацию по умолчанию

### Сценарий Premium
- Класс обслуживания: «Бизнес»
- Цель: максимальный комфорт для руководителей или сотрудников, работающих с клиентами
- Бизнес-класс предлагает кресла повышенной комфортности, приоритетную посадку и питание
- Используйте этот сценарий для руководства или при представлении компании клиентам

## Структура листов Excel

### Лист: Seat_Options
Колонки: Train_Code, Seat_Class, Price_Per_Ticket_RUB, Available_Seats

### Лист: Budget_Scenarios
Колонки: Scenario, Train_Code, Seat_Class, Price_Per_Person, Total_Cost_8_People

### Лист: Summary
Колонки: Label, Value

## Интерпретация разницы в бюджете
- Разница между сценариями Budget и Standard показывает доплату за комфорт в расчёте на человека
- Разница между сценариями Standard и Premium показывает стоимость апгрейда для руководства
- Полная экономия по поездке = (Premium_Total − Budget_Total) на всех 8 сотрудников
- В отчёте следует выделить рекомендуемый сценарий с обоснованием исходя из цели поездки
