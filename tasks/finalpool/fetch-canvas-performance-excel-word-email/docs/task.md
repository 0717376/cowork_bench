Мне нужна помощь с анализом успеваемости по курсам. Доступны внешние эталонные данные (benchmark): загрузите их с http://localhost:30310/api/data.json и извлеките нужные показатели (academic_benchmarks: pass_rate по уровням Introductory / Intermediate / Advanced).

Затем обратитесь к нашей системе управления обучением (Canvas) за данными по курсам и студентам. Названия и коды курсов читайте напрямую из Canvas — ничего не выдумывайте.

С помощью терминала создайте и запустите в рабочем каталоге Python-скрипт course_perf_processor.py, который читает собранные данные из созданных вами JSON-файлов, выполняет анализ и сохраняет результат в course_perf_results.json.

Создайте файл Excel под названием Performance_Report.xlsx с тремя листами.
- Лист Data_Analysis должен содержать основные данные по курсам со столбцами: Course (название курса), Code (код курса), Enrollment (число записавшихся), Avg_Score (средний балл), Pass_Rate (доля сдавших). Отсортируйте строки по алфавиту по столбцу Course.
- Лист Metrics должен суммировать ключевые показатели и содержать два столбца — Metric и Value: общее число курсов, суммарное число записавшихся (Total_Enrollment), а также средние значения Avg_Score и Avg_Pass_Rate по всем курсам.
- Лист Recommendations должен перечислять приоритетные действия со столбцами Priority, Action, Course. В первую очередь обращайте внимание на курсы с наименьшим Pass_Rate (наибольшее отставание от соответствующего внешнего эталона academic_benchmarks).

Также создайте документ Word под названием Performance_Analysis.docx с разделами: краткое резюме (executive summary), ключевые выводы (key findings) и рекомендации (recommendations).

Отправьте письмо руководителю команды на адрес team-lead@company.com с темой "Analysis Report Complete", в котором кратко изложите ключевые выводы.

Важно: инструменты Word и Excel могут терять текущий рабочий каталог, поэтому при создании и сохранении файлов .docx и .xlsx передавайте им АБСОЛЮТНЫЙ путь к файлам внутри рабочего каталога агента.
