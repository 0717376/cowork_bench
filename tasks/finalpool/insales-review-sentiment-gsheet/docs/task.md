Команде по работе с клиентами нужен анализ тональности отзывов о товарах из нашего интернет-магазина InSales. Мы хотим понять распределение оценок, выявить товары с наибольшим числом отзывов и найти товары с наивысшим рейтингом.

Выгрузите все отзывы о товарах вместе с названиями товаров из данных магазина InSales. Для топ-20 товаров с наибольшим числом отзывов (по количеству отзывов, а при равенстве — по алфавиту названия) рассчитайте количество отзывов, средний рейтинг (округлённый до 2 знаков после запятой), число отзывов с оценкой 5 звёзд и число отзывов с оценкой 1 звезда.

Создайте файл Excel с именем Review_Sentiment_Analysis.xlsx в рабочей директории с тремя листами.

Первый лист должен называться "Product Reviews" со столбцами Product_Name, Review_Count, Avg_Rating, Five_Star_Count и One_Star_Count. Включите только топ-20 товаров с наибольшим числом отзывов, отсортированных по Review_Count по убыванию, затем по Product_Name по возрастанию.

Второй лист должен называться "Rating Distribution" со столбцами Rating, Count и Percentage (округлённым до 1 знака после запятой). Включите по одной строке на каждое значение оценки (от 1 до 5), отсортированных по Rating по возрастанию.

Третий лист должен называться "Summary" с двумя столбцами, Metric и Value, содержащий следующие строки: Total_Reviews (общее число отзывов), Avg_Rating (общий средний рейтинг, округлённый до 2 знаков), Most_Reviewed_Product (название товара с наибольшим числом отзывов; при равенстве выберите первый по алфавиту) и Highest_Rated_Product (название товара с наивысшим средним рейтингом среди товаров с не менее чем 3 отзывами; при равенстве выберите первый по алфавиту).

Создайте документ Word с именем Review_Sentiment_Report.docx в рабочей директории с заголовком "Product Review Sentiment Analysis" и укажите дату 2026-03-06. Документ должен содержать раздел, описывающий общее распределение оценок, и раздел, выделяющий товары с наибольшим числом отзывов.

Создайте Google-таблицу с названием "Review Sentiment Dashboard" с листом "Rating Distribution", содержащим те же данные, что и лист Rating Distribution в Excel.

Примечание: при работе с инструментами Excel и Word указывайте ПОЛНЫЙ абсолютный путь к файлам .xlsx и .docx внутри рабочей директории, так как эти инструменты не сохраняют текущую рабочую директорию.
