# Методология анализа влияния обучения

Подход:
- Собрать показатели эффективности сотрудников по отделам
- Собрать данные о записях на курсы и их завершении
- Выявить корреляции между обучением и эффективностью
- Представить выводы с практическими рекомендациями

Ключевые метрики для отслеживания:
- Средняя оценка эффективности по отделам
- Распределение записей на курсы
- Доля завершения обучения в сравнении с ростом эффективности
