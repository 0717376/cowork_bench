Помоги, пожалуйста, с анализом нашего дивидендного портфеля акций Московской биржи (MOEX). Сначала загрузи внешние данные-бенчмарк по адресу http://localhost:30316/api/data.json — там по каждому тикеру указаны целевая цена аналитиков (`target_price`) и рейтинг (`analyst_rating`). Извлеки эти данные.

Затем получи текущие рыночные данные по каждой акции из нашего портфеля через MOEX (moex-finance): для каждого тикера из JSON возьми текущую цену, полное название компании и сектор.

Через терминал создай и запусти Python-скрипт с именем `yf_dividend_processor.py` в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые ты сформируешь, выполнять анализ и сохранять результат в `yf_dividend_results.json`.

Создай Excel-файл с именем `Dividend_Strategy_Report.xlsx` с тремя листами.

Лист 1 — `Data_Analysis` — с основными данными сравнения. Колонки (заголовки строго на английском, именно в таком виде):
- `Symbol` — тикер MOEX (например, SBER.ME)
- `Name` — полное название компании (из moex-finance)
- `Sector` — сектор (из moex-finance)
- `Current_Price` — текущая рыночная цена в рублях (из moex-finance)
- `Target_Price` — целевая цена из бенчмарка `data.json`
- `Upside` — потенциал роста в процентах: `(Target_Price - Current_Price) / Current_Price * 100`, округлённый до двух знаков

Отсортируй строки по алфавиту по колонке `Symbol`.

Лист 2 — `Metrics` — две колонки `Metric` и `Value`. Включи минимум три строки-метрики:
- `Total_Stocks` — общее число акций в анализе
- `Avg_Upside` — средний Upside по всем строкам листа Data_Analysis
- `Best_Opportunity` — тикер (Symbol) с максимальным Upside

Лист 3 — `Recommendations` — три колонки `Priority`, `Action`, `Symbol`. Перечисли приоритетные действия: `Priority` — порядковый номер (1, 2, ...) по убыванию Upside, `Action` — рекомендуемое действие из рейтинга аналитиков (Buy / Hold / Sell), `Symbol` — тикер. Минимум две строки.
