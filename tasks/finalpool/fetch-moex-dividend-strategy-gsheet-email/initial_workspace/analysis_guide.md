# Руководство по анализу — дивидендный портфель MOEX

## Методология
Этот анализ сравнивает текущую рыночную цену акций (Current_Price) с целевой
ценой аналитиков (Target_Price), полученной из внешнего источника-бенчмарка.
Upside (потенциал роста) = (Target_Price - Current_Price) / Current_Price * 100,
то есть процент потенциального роста до целевой цены.
Положительный Upside означает, что аналитики ожидают рост котировки.
Отрицательный Upside означает, что бумага торгуется выше целевой цены.

## Приоритетные действия
В первую очередь рассматривайте бумаги с наибольшим потенциалом роста (Upside).
Рекомендуемое действие (Action) опирается на рейтинг аналитиков: Buy / Hold / Sell.

## Отчётность
Включайте в отчёт источники данных, методологию и ключевые метрики.
Все цены указываются в рублях (RUB), портфель сформирован из бумаг Московской биржи (MOEX).
