Проректор хочет проанализировать баланс учебной нагрузки преподавателей по всем кафедрам. Начните с получения нормативов учебной нагрузки преподавателей по адресу http://localhost:30220/api/workload_standards.json, где заданы максимальная учебная нагрузка по числу курсов, пороговые значения числа студентов, оценки времени на проверку работ и критерии перегрузки.

Затем обратитесь к системе управления обучением, чтобы определить нагрузку каждого преподавателя. Для каждого преподавателя с преподавательской ролью (TeacherEnrollment) подсчитайте число различных курсов, которые он ведёт, общее число студентов по этим курсам и общее число заданий, за которые он отвечает. Рассчитайте оценочное общее число часов на проверку работ за семестр как общее число студентов, умноженное на число часов проверки на студента (grading_hours_per_student) из нормативов. Затем рассчитайте недельное число часов проверки, разделив семестровую сумму на число недель семестра (semester_weeks) из нормативов. Преподаватель считается перегруженным (overload), если он ведёт больше курсов, чем максимально допустимо, ИЛИ если его недельные часы проверки превышают максимальное недельное число часов (max_weekly_hours) из нормативов.

Изучите документ с политикой нагрузки (workload_policy.pdf) и файл справочника преподавателей (faculty_directory.csv) в рабочем каталоге, чтобы получить контекст о распределении по кафедрам и об институциональной политике.

Создайте файл Excel под названием Faculty_Workload.xlsx с двумя листами. Первый лист с названием "Instructor Load" должен содержать столбцы: Instructor, Courses_Count, Total_Students, Total_Assignments, Est_Grading_Hours (общее число часов проверки за семестр), Weekly_Grading_Hours и Overloaded_YN. Отсортируйте по алфавиту по имени преподавателя. Второй лист с названием "Department Summary" должен агрегировать данные по кафедре (на основе предметной области курса) со столбцами: Department, Instructor_Count, Course_Count и Total_Students.

Запишите данные о нагрузке в общую таблицу с заголовком "Faculty Workload Analysis", содержащую лист "Instructor Load" с теми же данными уровня преподавателей.

Также создайте презентацию под названием Workload_Review.pptx с кратким изложением ключевых выводов. Включите титульный слайд, слайд с общим числом преподавателей и числом перегруженных, слайд с разбивкой по кафедрам и слайд с рекомендациями о стратегиях перераспределения для кафедр с перегруженными преподавателями.

Примечание: при создании Faculty_Workload.xlsx и Workload_Review.pptx через соответствующие инструменты передавайте АБСОЛЮТНЫЙ путь к файлам внутри рабочего каталога агента (например, !!<<<<||||workspace_dir||||>>>>!!/Faculty_Workload.xlsx), поскольку эти инструменты не сохраняют текущий рабочий каталог.
