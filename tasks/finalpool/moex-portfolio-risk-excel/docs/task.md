Я портфельный управляющий российского инвестиционного фонда, и мне нужен отчёт по анализу рисков портфеля, состоящего из пяти акций Московской биржи (котировки в рублях): SBER.ME, GAZP.ME, LKOH.ME, TCSG.ME и MGNT.ME. Пожалуйста, загрузите историю цен закрытия по каждой из этих пяти акций из нашего источника финансовых данных (MOEX). Мне нужна вся доступная история цен по всем бумагам.

Создайте Excel-файл с именем Portfolio_Risk_Analysis.xlsx в рабочей директории с тремя листами. Передавайте Excel-инструменту АБСОЛЮТНЫЙ путь к файлу .xlsx внутри рабочей директории.

Первый лист должен называться "Price History" и содержать цены закрытия по всем пяти акциям: столбец Date, за которым следует по одному столбцу на каждый тикер с ценой закрытия за соответствующую дату. Включайте только те даты, по которым есть данные сразу по всем пяти акциям, и сортируйте по дате в порядке возрастания.

Второй лист должен называться "Risk Metrics" и содержать сводную таблицу со следующими столбцами: Ticker, Average Close Price, Standard Deviation, Minimum Close и Maximum Close. Все значения округляйте до двух знаков после запятой. Рассчитывайте эти статистики по всей истории цен.

Третий лист должен называться "Risk Assessment" и классифицировать каждую акцию по категории риска на основе коэффициента вариации, который равен стандартному отклонению, делённому на среднюю цену закрытия. Если коэффициент вариации меньше 0.10, акция классифицируется как "Low Risk"; если он находится в диапазоне от 0.10 до 0.20 включительно — как "Medium Risk"; если он больше 0.20 — как "High Risk". Этот лист должен содержать столбцы Ticker, Coefficient of Variation, округлённый до четырёх знаков после запятой, и Risk Category.
