Мне нужна помощь с анализом обучающих курсов в Canvas.

Сначала получи данные внутренних учебных курсов из Canvas: для каждого курса собери название (Course), код (Code), число записавшихся (Enrollment), средний балл (Avg_Score) и долю успешно прошедших (Pass_Rate).

Затем по адресу http://localhost:30327/api/data.json доступны внешние эталонные (benchmark) показатели отрасли — получи эти данные и извлеки нужные метрики, чтобы использовать их как отраслевой ориентир при выводах. Дополнительный контекст по подразделениям при необходимости можно взять из корпоративного хранилища данных ClickHouse (схема sf_data).

Через терминал создай и запусти Python-скрипт course_sf_training_processor.py в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые ты создашь, выполнять анализ и сохранять результат в course_sf_training_results.json. Указывай АБСОЛЮТНЫЕ пути к рабочей директории при создании и запуске скрипта, а также при чтении и записи всех JSON-файлов и итогового .xlsx, чтобы файлы гарантированно попадали в рабочую директорию агента.

Создай Excel-файл Sf_Training_Report.xlsx с тремя листами.

Лист Data_Analysis должен содержать основную таблицу по курсам. Включи столбцы:
- Course — название курса,
- Code — код курса,
- Enrollment — число записавшихся,
- Avg_Score — средний балл,
- Pass_Rate — доля успешно прошедших (в процентах).
Отсортируй строки по столбцу Course в алфавитном порядке.

Лист Metrics должен содержать два столбца, Metric и Value, и сводные показатели: Total_Courses (всего курсов), Total_Enrollment (суммарное число записавшихся), Avg_Score (средний балл по всем курсам) и Avg_Pass_Rate (средняя доля успешно прошедших). Значения должны быть арифметически согласованы со строками листа Data_Analysis.

Лист Recommendations должен содержать столбцы Priority, Action и Course и перечислять приоритетные действия на основе анализа (например, дополнительная поддержка для слабых курсов и обмен лучшими практиками для сильных).

Отправь письмо на адрес team-lead@company.com с темой "Analysis Report Complete", кратко резюмировав ключевые выводы анализа (укажи в тексте письма хотя бы один конкретный показатель: разрыв, средний балл, число курсов и т.п.).
