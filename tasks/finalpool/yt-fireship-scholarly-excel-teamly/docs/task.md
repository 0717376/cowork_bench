Наша исследовательская группа изучает связь между популярным техническим видеоконтентом и рецензируемыми научными работами по тем же темам. Мы хотим построить сопоставление между самыми просматриваемыми техническими видео в нашей базе данных и научными статьями, которые посвящены тем же темам.

Шаг первый: получите 8 самых просматриваемых видео с канала Fireship из нашей базы видео, отсортированных по числу просмотров по убыванию. Для каждого видео определите основную технологическую тему, которую оно охватывает, исходя из названия видео и его содержания. Отнесите каждое видео к одной из следующих тематических категорий: AI and Large Language Models, Linux and Systems, Software Engineering, Hardware and Computing или Security (используйте категорию, которая лучше всего соответствует тематике каждого видео).

Шаг второй: для каждой из восьми тем выполните поиск в нашей базе научных статей, чтобы найти релевантные исследовательские работы. Зафиксируйте название статьи, авторов (только имя первого автора), год публикации, число цитирований и краткий фрагмент аннотации (первые 200 символов).

Шаг третий: создайте Excel-файл с именем Tech_Research_Map.xlsx в вашем рабочем каталоге с двумя листами. Первый лист называется Video_Paper_Mapping со столбцами Video_Rank, Video_Title, Video_Views, Tech_Topic, Paper_Count, Top_Paper_Title, Top_Paper_Year, Top_Paper_Citations. Для столбцов Top_Paper_Title и связанных с ним используйте статью с наибольшим числом цитирований по данной теме. Второй лист называется All_Papers со столбцами Tech_Topic, Paper_Title, First_Author, Year, Citations, Abstract_Snippet. Строки в листе All_Papers должны быть отсортированы по Tech_Topic, а затем по Citations по убыванию внутри каждой темы.

Шаг четвёртый: создайте страницу в нашей корпоративной базе знаний teamly с заголовком Tech Content Research Bridge. Тело страницы должно начинаться с обзорного абзаца (две-три фразы), объясняющего цель этого упражнения по сопоставлению. Ниже обзора перечислите каждое из восьми видео с его рангом, названием и названием наиболее релевантной научной статьи, найденной по соответствующей теме.
