# Руководство: сопоставление видео Fireship с научными статьями

## Обзор
Эта задача сопоставляет 8 самых просматриваемых видео Fireship на YouTube с релевантными научными статьями, найденными через поиск по научной базе. Результат — рабочая книга Excel и страница в базе знаний teamly, суммирующая выводы.

## Формат вывода в Excel

### Лист: Video_Paper_Mapping
Этот лист даёт сводку «одна строка на видео», связывая каждое видео с наиболее подходящей научной статьёй.

Столбцы:
- Video_Rank: целое число 1–8 (1 = самое просматриваемое)
- Video_Title: полное название видео на YouTube
- Video_Views: число просмотров на момент анализа
- Tech_Topic: выведенная технологическая тематическая категория (например, AI and Large Language Models, Linux and Systems, Security)
- Paper_Count: число релевантных статей, найденных по данной теме
- Top_Paper_Title: название статьи с наибольшим числом цитирований
- Top_Paper_Year: год публикации топовой статьи
- Top_Paper_Citations: число цитирований топовой статьи

### Лист: All_Papers
Этот лист перечисляет все статьи, найденные по всем темам.

Столбцы:
- Tech_Topic: категория, под которой найдена эта статья
- Paper_Title: полное название статьи
- First_Author: имя первого автора
- Year: год публикации
- Citations: число цитирований
- Abstract_Snippet: первые 200 символов аннотации

## Структура страницы teamly
Создайте в базе знаний teamly страницу с заголовком "Tech Content Research Bridge", содержащую:
1. Обзорный абзац, описывающий цель анализа
2. Список, сопоставляющий каждое видео с его топовой статьёй
3. Раздел с наблюдениями, отмечающий любые интересные закономерности
