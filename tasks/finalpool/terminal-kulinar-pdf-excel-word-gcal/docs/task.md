Команда студенческой столовой организует пятидневный тематический гастрономический фестиваль русской кухни. В рабочей директории есть PDF-файл Dietary_Guidelines.pdf, в котором описаны требования к питанию и бюджетные ограничения мероприятия. Также есть JSON-файл event_config.json, в котором указаны количество гостей в день, бюджет на человека на приём пищи и обязательные категории блюд. Сначала прочитайте оба файла, чтобы понять параметры мероприятия.

Найдите в кулинарной базе `kulinar` блюда пяти категорий: основное блюдо/гарнир (staple food), мясное блюдо (meat dish), овощное блюдо/салат (vegetable dish), суп (soup) и десерт (dessert). Выберите ровно один рецепт из каждой категории. Учтите, что в базе `kulinar` русские категории называются иначе, поэтому используйте следующее соответствие категорий:
- гарнир -> staple (основное блюдо)
- горячее -> meat (мясное блюдо)
- салат -> vegetable (овощное блюдо)
- суп -> soup (суп)
- десерт -> dessert (десерт)

Для каждого выбранного рецепта зафиксируйте его название, категорию, уровень сложности, ориентировочное время приготовления и список ингредиентов. В колонке Category итоговых таблиц (см. ниже) используйте именно канонические английские ключи категорий (staple, meat, vegetable, soup, dessert), а не русские названия категорий из базы.

Напишите и запустите Python-скрипт budget_calculator.py в рабочей директории. Скрипт должен читать файл selected_recipes.json (который вы создаёте, перечислив пять выбранных рецептов с их деталями). Для каждого рецепта оцените стоимость одной порции: 15 рублей для категории staple, 25 рублей для meat, 10 рублей для vegetable, 12 рублей для soup и 8 рублей для dessert. Скрипт должен рассчитать общую стоимость на один день для 50 гостей (каждый гость получает одну порцию каждого блюда), общую стоимость за пять дней и проверить, укладывается ли итог в бюджет мероприятия 20000 рублей. Сохраните результаты в файл budget_analysis.json.

Создайте Excel-файл Campus_Dining_Plan.xlsx в рабочей директории с тремя листами. ВАЖНО: инструмент Excel может терять текущую рабочую директорию, поэтому при создании и сохранении передавайте АБСОЛЮТНЫЙ путь к Campus_Dining_Plan.xlsx внутри рабочей директории.

Первый лист должен называться Weekly_Menu с колонками Day (Monday — Friday), Category, Recipe_Name, Servings (50 для каждой строки) и Cost_Per_Serving (по ставкам выше). В каждый день входят одни и те же пять рецептов, что даёт 25 строк (5 дней × 5 категорий). Отсортируйте по Day, затем по Category в алфавитном порядке.

Второй лист должен называться Nutrition_Estimate с колонками Category, Recipe_Name, Estimated_Calories (400 для staple, 350 для meat, 150 для vegetable, 100 для soup, 200 для dessert), Estimated_Protein_g (15 для staple, 30 для meat, 5 для vegetable, 8 для soup, 3 для dessert) и Difficulty. Включите пять рецептов, отсортированных по Category в алфавитном порядке.

Третий лист должен называться Budget_Overview с колонками Category, Cost_Per_Serving, Daily_Servings (50), Daily_Cost, Five_Day_Cost и Pct_of_Budget (процент от 20000 рублей, округлённый до одного знака после запятой). Включите пять категорий, отсортированных по алфавиту, плюс итоговую строку (total) внизу. Рассчитайте общую дневную стоимость на основе 50 гостей и суммы стоимостей порций по всем пяти категориям, затем умножьте на пять дней, чтобы получить общую стоимость за пять дней.

Создайте Word-документ Meal_Plan_Report.docx в рабочей директории. ВАЖНО: инструмент Word также может терять текущую рабочую директорию, поэтому передавайте АБСОЛЮТНЫЙ путь к Meal_Plan_Report.docx внутри рабочей директории. В отчёте должны быть: заголовок о студенческом гастрономическом фестивале, раздел с параметрами мероприятия (50 гостей в день, 5 дней, бюджет 20000 рублей), раздел со списком каждого выбранного рецепта с его категорией и ключевыми ингредиентами, раздел бюджетного анализа с итоговыми затратами и подтверждением, что план укладывается в бюджет, и раздел с рекомендациями по приготовлению.

Наконец, создайте пять событий в календаре — по одному на каждый день с понедельника 9 марта по пятницу 13 марта 2026 года. Каждое событие должно иметь заголовок "Food Festival Prep - Day N" (где N — от 1 до 5; используйте именно эту английскую формулировку дословно), быть запланировано с 7:00 до 10:00 и содержать описание со списком пяти рецептов, которые готовят в этот день.
