Помоги мне с анализом цитируемости статей arxiv. Доступны внешние эталонные данные (benchmark), которые нужно загрузить с http://localhost:30322/api/data.json и извлечь из них релевантные метрики (рост популярности тематик, growth_pct по областям).

Затем найди релевантные научные статьи по теме в локальной базе arxiv/scholarly.

Через терминал создай и запусти Python-скрипт arxiv_citation_processor.py в рабочей директории, который читает собранные данные из создаваемых тобой JSON-файлов, выполняет анализ и сохраняет результат в arxiv_citation_results.json.

Создай Excel-файл Citation_Network_Report.xlsx ровно с тремя листами и точными названиями колонок (заголовки колонок и названия листов оставь на английском — они проверяются автоматически):

1. Лист `Data_Analysis` — основные данные по статьям. Колонки строго в таком виде:
   `Paper_ID`, `Title`, `Area`, `Citations`, `Relevance_Score`.
   Здесь `Citations` — это наша внутренняя метрика (число цитирований статьи): возьми её из научной базы цитирований (поиск по Google Scholar / scholarly), сопоставив статьи по названию. Данные о росте тематик growth_pct из data.json служат внешним эталоном (benchmark): разрыв (gap) между областью и эталоном выводится из их сопоставления. Включи все четыре релевантные статьи и исключи нерелевантную (шумовую) статью по квантовой физике. Отсортируй строки по алфавиту по колонке `Area`.

2. Лист `Metrics` — сводные ключевые метрики. Колонки: `Metric`, `Value`.
   Должен содержать как минимум: `Total_Papers` (общее число статей), `Avg_Citations` (среднее число цитирований), `Top_Area` (область с наибольшим ростом по внешнему эталону), `Avg_Relevance` (средняя релевантность).

3. Лист `Recommendations` — приоритетные действия по результатам анализа разрывов. Колонки: `Priority`, `Action`, `Area`.

Создай в Teamly страницу с заголовком "Arxiv Citation Dashboard" (заголовок оставь на английском) и кратким резюме анализа на русском языке: в тексте упомяни ключевые метрики и хотя бы одну область статей.

Важно: MCP-сервер excel не сохраняет текущую рабочую директорию, поэтому при создании и записи Citation_Network_Report.xlsx указывай АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории.
