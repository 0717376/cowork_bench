Привет, мне нужна твоя помощь с анализом клиентской лояльности для нашего интернет-магазина InSales. Мы хотим разбить клиентов на уровни лояльности в зависимости от того, сколько они потратили у нас в общей сложности.

Сначала выгрузи из InSales всех клиентов, которые оформили хотя бы один заказ. Для каждого такого клиента получи его ID, имя, email, количество заказов и общую потраченную сумму.

Затем распредели каждого клиента по уровню по следующим правилам. Уровень Gold — для клиентов, потративших в сумме $1,000 или больше. Уровень Silver — для клиентов, потративших от $300 до $999.99. Уровень Bronze — для тех, кто потратил меньше $300.

Создай документ Word под названием Customer_Loyalty.docx в рабочей директории. Документ должен начинаться с заголовка "Customer Loyalty Analysis Report" и содержать дату отчёта 2026-03-06, а также общее число активных клиентов (тех, у кого есть хотя бы один заказ).

Далее добавь раздел "Tier Summary", в котором для каждого уровня (Gold, Silver, Bronze) показано количество клиентов и их суммарная потраченная сумма, округлённая до 2 знаков после запятой. Также укажи суммарную выручку от всех активных клиентов.

После сводки добавь три раздела, по одному на каждый уровень: "Gold Tier Customers", "Silver Tier Customers" и "Bronze Tier Customers". В каждом разделе перечисли всех клиентов этого уровня, указав их имя, email, количество заказов, общую потраченную сумму (округлённую до 2 знаков) и среднюю стоимость заказа (округлённую до 2 знаков). Сортируй клиентов внутри каждого уровня по общей потраченной сумме по убыванию.

Важно: заголовки разделов в документе оставь точно на английском — "Customer Loyalty Analysis Report", "Tier Summary", "Gold Tier Customers", "Silver Tier Customers", "Bronze Tier Customers", — а сами имена клиентов используй так, как их возвращает InSales (на русском). Денежные пороги и суммы оставь в долларах ($) ровно так, как они приходят из InSales, ничего не конвертируя.

Также создай общую таблицу с названием "Customer Loyalty Dashboard" с листом "Tier Data", содержащим столбцы Customer_ID, Name, Email, Orders, Total_Spent, Avg_Order_Value и Tier. Включи всех активных клиентов, отсортированных по Total_Spent по убыванию.

Примечание: при создании файла Word передавай инструменту АБСОЛЮТНЫЙ путь к Customer_Loyalty.docx внутри указанной выше рабочей директории (Word MCP не сохраняет текущий рабочий каталог).
