Вы — HR-аналитик, которому поручено настроить процесс ежегодной оценки эффективности сотрудников. Ваш первый шаг — проанализировать текущее состояние оценок эффективности сотрудников по всем отделам, используя корпоративное хранилище данных компании.

Выполните запрос к базе данных ClickHouse, чтобы получить статистику по оценкам эффективности для каждого отдела. Используйте таблицу HR_ANALYTICS__PUBLIC__EMPLOYEES и сгруппируйте результаты по полю DEPARTMENT. Для каждого отдела вычислите общее количество сотрудников, среднюю оценку эффективности (PERFORMANCE_RATING), округлённую до двух знаков после запятой, количество сотрудников с низкой эффективностью (с оценкой ниже 3) и количество сотрудников с высокой эффективностью (с оценкой 4 и выше).

Создайте Excel-книгу с именем Performance_Review_Setup.xlsx в вашей рабочей директории с одним листом под названием "Performance Analysis". Включите столбцы Department, Employee_Count, Avg_Performance, Low_Performers_Count и High_Performers_Count. Добавьте по одной строке для каждого из семи отделов: Инженерия, Финансы, Кадры, Операции, НИОКР, Продажи и Поддержка (используйте названия отделов ровно так, как они хранятся в поле DEPARTMENT базы данных). Данные должны соответствовать фактическим значениям из запроса к ClickHouse.

Создайте форму под названием "Ежегодная оценка эффективности" со следующими шестью вопросами в указанном порядке: первый — текстовый вопрос «ID сотрудника»; второй — вопрос с выбором одного варианта «Отдел» с вариантами Инженерия, Финансы, Кадры, Операции, НИОКР, Продажи и Поддержка (ровно те значения отделов, что хранятся в базе); третий — вопрос с выбором одного варианта «Оценка эффективности» с вариантами 1-Требует улучшения, 2-Ниже ожиданий, 3-Соответствует ожиданиям, 4-Превосходит ожидания и 5-Выдающийся; четвёртый — текстовый вопрос «Ключевые достижения за год»; пятый — текстовый вопрос «Зоны для развития»; шестой — текстовый вопрос «Цели на следующий год».

Отправьте письмо на адрес hr_director@company.com с темой "Annual Performance Review Process Setup Complete", в котором кратко изложите текущее распределение эффективности по отделам и подтвердите, что форма оценки и Excel-отчёт подготовлены.
