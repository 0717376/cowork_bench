Учебному отделу университета нужна сводная картина по набору студентов на все курсы в системе управления обучением (LMS). Пожалуйста, получите список всех курсов, а затем соберите данные о наборе по каждому курсу, посчитав, сколько студентов, преподавателей и ассистентов (TA) зачислено на каждый из них. Тип зачисления определяется по роли в записи о зачислении: студенческие зачисления, преподавательские зачисления и зачисления ассистентов (TA) нужно считать раздельно.

После того как вы соберёте все данные о наборе, создайте в корпоративной базе знаний Teamly страницу-трекер под названием "Course Enrollment Tracker", которая будет служить централизованной доской учёта. На этой странице оформите таблицу, где каждый курс — отдельная строка. Для каждого курса укажите название курса, а также столбцы Student_Count, Teacher_Count, TA_Count и Total_Enrollment, где Total_Enrollment — это сумма студентов, преподавателей и ассистентов. Названия столбцов (Student_Count, Teacher_Count, TA_Count, Total_Enrollment) и название страницы "Course Enrollment Tracker" сохраняйте на английском языке без изменений.

Наконец, отправьте письмо на адрес admin@university.example.com с темой "Course Enrollment Summary Report". В теле письма укажите общее количество курсов, суммарный набор по всем курсам, а также краткий список пяти курсов с наибольшим набором (топ-5) с указанием названия каждого курса и его итогового числа зачислений (Total_Enrollment). Тему письма и адрес получателя оставьте на английском языке без изменений.
