# Соответствие внутренних должностей рыночным бенчмарк-категориям

В этом документе описано, как внутренние названия должностей соотносятся со стандартными рыночными бенчмарк-категориями.

## Отдел «Инженерия»
- DevOps-инженер -> рыночная категория: DevOps-инженер
- ML-инженер -> рыночная категория: ML-инженер
- Инженер-программист -> рыночная категория: Инженер-программист

## Отдел «Финансы»
- Аналитик -> рыночная категория: Аналитик
- Старший аналитик -> рыночная категория: Старший аналитик

## Отдел «Кадры»
- Специалист по кадрам -> рыночная категория: Специалист по кадрам
- Менеджер по кадрам -> рыночная категория: Менеджер по кадрам

## Отдел «Операции»
- Координатор операций -> рыночная категория: Координатор операций
- Руководитель операций -> рыночная категория: Руководитель операций

## Отдел «НИОКР»
- Исследователь -> рыночная категория: Исследователь
- Научный сотрудник -> рыночная категория: Научный сотрудник

## Отдел «Продажи»
- Менеджер по работе с клиентами -> рыночная категория: Менеджер по работе с клиентами
- Менеджер по продажам -> рыночная категория: Менеджер по продажам

## Отдел «Поддержка»
- Специалист по работе с клиентами -> рыночная категория: Специалист по работе с клиентами
- Инженер поддержки -> рыночная категория: Инженер поддержки

## Примечания
- Рыночные бенчмарки соответствуют 50-му перцентилю (медиане) отраслевого вознаграждения
- Бенчмарки обновляются ежегодно по результатам отраслевых обзоров зарплат
- Коэффициент оплаты (Pay Ratio) рассчитывается как: средний внутренний оклад / рыночный бенчмарк
