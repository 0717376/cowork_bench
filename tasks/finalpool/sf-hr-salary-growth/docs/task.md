Нам нужно проанализировать динамику роста зарплат по отделам. В рабочем каталоге есть файл Compensation_Policy.pdf, описывающий наши правила оплаты труда. Используйте данные истории изменений зарплат из хранилища данных ClickHouse, объединённые с данными о сотрудниках, чтобы понять, как распределялись повышения.

Создайте файл Excel с именем HR_Salary_Growth.xlsx с двумя листами. На листе "Salary Growth" должны быть колонки Department, Employees_With_Changes (количество уникальных сотрудников, у которых были изменения), счётчик Total_Changes, Avg_Increase, округлённое до 2 знаков, и Avg_Pct_Change, округлённое до 1 знака. Отсортируйте по Avg_Pct_Change по убыванию.

На листе "Summary" должны быть Total_Salary_Changes, Highest_Avg_Increase_Dept и Overall_Avg_Increase как взвешенное среднее, округлённое до 2 знаков.

Сохраните файл по абсолютному пути внутри рабочего каталога агента (передавайте в инструмент Excel полный абсолютный путь к HR_Salary_Growth.xlsx).
