# Планирование кейтеринга конференции «Импульс-2026»

Рабочая директория для задачи планирования кейтеринга и логистики корпоративной
конференции на 500 человек.

Файлы-источники:
- `Event_Brief.md` — бриф мероприятия (детали, бюджет, требования).
- `participant_list.xlsx` (лист **Participants**) — список 500 участников с колонкой **Dietary Notes**.
- `menu_templates.xlsx` — шаблоны меню по приёмам пищи.
- `config.json` — параметры мероприятия. `data.csv` — список поставщиков (вендоров).
