Вы — координатор кейтеринга и логистики корпоративной конференции «Импульс-2026» в Москве. Мероприятие рассчитано на **500 участников**, бюджет питания — **1500 ₽ на человека** (итого **750 000 ₽**). В рабочей директории находятся файлы-источники: `Event_Brief.md` (бриф мероприятия), `participant_list.xlsx` (лист **Participants** с колонкой **Dietary Notes**), `menu_templates.xlsx` (шаблоны меню) и `data.csv` (список поставщиков). Сначала внимательно изучите эти файлы.

Выполните задачу из пяти частей.

## Часть 1. Подбор меню из базы рецептов kulinar
Подберите в базе рецептов **kulinar** от **6 до 8** блюд для кейтеринга конференции. Используйте **реальные** блюда из базы (не выдумывайте названия). Меню должно быть сбалансированным (горячие блюда, гарниры, салаты/закуски, десерт) и покрывать пищевые ограничения участников: вегетарианские, веганские, халяльные и безглютеновые позиции. Для каждого блюда зафиксируйте название, категорию и предполагаемое число порций.

## Часть 2. Опрос пищевых предпочтений (forms)
Создайте в **forms** опрос с заголовком, содержащим слова «Пищевые предпочтения» (Dietary Preferences), чтобы собрать данные о питании участников. Опрос должен содержать **не менее 3 вопросов**, охватывающих: пищевые ограничения/предпочтения (вегетарианство, веганство, халяль, без глютена), наличие пищевых аллергий и поле для дополнительных комментариев или пожеланий. Подберите подходящий тип вопроса для каждого пункта.

## Часть 3. План кейтеринга в Excel — `Catering_Plan.xlsx`
Создайте в рабочей директории Excel-файл **`Catering_Plan.xlsx`** с тремя листами.

Лист **`Menu`** — колонки строго в этом порядке: `Recipe_Name`, `Category`, `Servings`, `Cost_Per_Person`. По одной строке на каждое выбранное блюдо.

Лист **`Dietary_Summary`** — колонки `Diet_Type` и `Count`. Сведите количество участников по каждому типу питания из колонки **Dietary Notes** файла `participant_list.xlsx`. Должны присутствовать все типы из списка участников, а числа — точно соответствовать данным.

Лист **`Summary`** — колонки `Metric` и `Value`, со строками: `Attendees` (500), `Budget_Per_Person` (1500), `Total_Budget` (750000).

## Часть 4. Предложение для руководства — `Catering_Proposal.docx`
Через **word** MCP создайте документ **`Catering_Proposal.docx`** в рабочей директории. ВАЖНО: word MCP не сохраняет рабочую директорию между вызовами — **передавайте абсолютный путь** к файлу внутри рабочей директории (например, `<workspace_dir>/Catering_Proposal.docx`). Документ должен содержать: рекомендованное меню (выбранные блюда), итоговую оценку стоимости (общий бюджет **750 000 ₽**, совпадает с листом `Summary`), а также раздел с резервным планом (контингенс) — запасные блюда и меры на случай изменения числа участников или пищевых ограничений. Используйте слова вроде «резерв», «запасн» или «контингенс».

## Часть 5. Рассылка и календарь
Отправьте письмо на адрес **events@company.ru** с кратким изложением плана кейтеринга: число участников (500), общий бюджет, перечень выбранных блюд. Затем создайте в общем календаре событие(я) с **дедлайнами для поставщиков (вендоров)** — в теме/описании укажите подтверждение поставщика (слова «вендор», «поставщик» или «дедлайн»).
