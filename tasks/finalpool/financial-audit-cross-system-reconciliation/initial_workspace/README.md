# financial-audit-cross-system-reconciliation

Исходная рабочая директория для задачи кросс-системной финансовой сверки.

Файлы:
- `authoritative_ledger.csv` — авторитетная выгрузка заказов магазина (InSales), колонки `Order ID`, `Date`, `Amount`, `Status`.
- `q1_manual_tracking.xlsx` — ручной реестр финансового отдела (лист `Manual Tracking`).
- `system_mapping_guide.pdf` — руководство по сопоставлению систем и правилам сверки.
