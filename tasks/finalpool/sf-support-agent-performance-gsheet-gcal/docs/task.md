Вы — руководитель службы поддержки. Проанализируйте индивидуальную эффективность агентов поддержки на основе базы данных тикетов центра поддержки в хранилище данных ClickHouse.

Сделайте запрос к таблице тикетов (TICKETS), чтобы рассчитать метрики по каждому агенту, используя колонку REPORTER как идентификатор агента. Для каждого уникального REPORTER вычислите общее количество обработанных тикетов, среднее время отклика в часах (RESPONSE_TIME), округлённое до 2 знаков после запятой, средний балл удовлетворённости клиентов (CSAT), округлённый до 2 знаков после запятой, и долю соблюдения SLA в процентах, округлённую до 2 знаков после запятой. Для расчёта соблюдения SLA тикет считается соответствующим требованиям, если время отклика укладывается в порог для его уровня приоритета (PRIORITY): тикеты с приоритетом High должны быть обработаны в течение 4 часов, Medium — в течение 8 часов, Low — в течение 24 часов.

Также сделайте запрос к таблице агентов (AGENTS), чтобы получить команду (TEAM) и уровень навыков (SKILL_LEVEL) для перечисленных там агентов. В данных тикетов пять значений REPORTER: Алиса, Борис, Карл, Эмилия и Иван.

Создайте Google-таблицу с названием Agent Performance Scorecard с двумя листами:

Лист с названием Scorecards со столбцами: Agent_Name, Total_Tickets, Avg_Response_Hrs, Avg_CSAT, SLA_Compliance_Rate. Включите по одной строке на каждого агента (всего 5 агентов, отсортированных по Total_Tickets по убыванию).

Лист с названием Rankings со столбцами: Rank_Type, Rank, Agent_Name, Value. Включите топ-3 агента по Avg_CSAT (Rank_Type = Top_CSAT) и топ-3 агента по Total_Tickets (Rank_Type = Top_Volume).

Запланируйте событие в Google Calendar с названием Agent Performance Review ровно через 10 дней после времени запуска. Событие должно длиться 1 час.

Отправьте письмо с адреса support-manager@company.example.com на адрес performance-review@company.example.com с темой Agent Performance Review - Monthly Scorecard. В теле письма кратко изложите данные о лучшем агенте по CSAT и об агенте с наибольшим объёмом тикетов.
