Отделу продуктовой стратегии нужен конкурентный анализ. Данные о продуктах конкурентов доступны через внутренний REST API по адресу http://localhost:30156/api/competitors.json. Этот эндпоинт возвращает список из 10 продуктов, для каждого указаны поля: name, price, rating и features. Кроме того, по трём ключевым продуктам доступны подробные карточки:

- http://localhost:30156/api/detail/alpha_pro.json
- http://localhost:30156/api/detail/gamma_ultra.json
- http://localhost:30156/api/detail/zeta_max.json

Каждая такая карточка содержит поля description, pros, cons и target_market.

Выполните следующие шаги.

1. Запросите эндпоинт http://localhost:30156/api/competitors.json и получите список всех 10 продуктов. Затем запросите три карточки с подробностями по продуктам Alpha Pro, Gamma Ultra и Zeta Max. Можно использовать python_execute (например, urllib/requests) или filesystem для обращения к API.

2. Создайте в рабочей директории Excel-файл с именем Competitor_Analysis.xlsx с двумя листами.

   Лист "Product Listing" — строка заголовка и ровно 10 строк данных (по одной на каждый продукт из competitors.json). Столбцы (именно в таком порядке): Product, Price, Rating, Features. В столбец Features поместите список особенностей продукта (например, через запятую). Названия продуктов, цены и рейтинги должны точно соответствовать данным API.

   Лист "Detailed Analysis" — строка заголовка и ровно 3 строки данных по продуктам Alpha Pro, Gamma Ultra и Zeta Max. Столбцы (именно в таком порядке): Product, Description, Pros, Cons, Target_Market. Значения возьмите из соответствующих карточек /api/detail/.

   В файле должно быть ровно два листа — Product Listing и Detailed Analysis, никаких лишних листов добавлять не нужно.

3. Запишите ключевые выводы анализа в файл памяти memory/memory.json в рабочей директории (в массив notes). Заметки должны быть содержательными: упомяните, что конкурентный анализ выполнен, что проанализировано 10 продуктов, и зафиксируйте наблюдения (например, продукт с наивысшим рейтингом — Gamma Ultra).

4. Создайте в Teamly страницу (заголовок должен содержать слово «конкурент» или Competitor) с кратким резюме конкурентного анализа. В тексте страницы обязательно отразите следующие факты: всего проанализировано 10 продуктов; средняя цена по 10 продуктам составляет примерно 289; продукт с наивысшим рейтингом — Gamma Ultra (рейтинг 4.8). Добавьте также короткий обзор ценового диапазона и рекомендации для продуктовой команды.

Названия продуктов, листов и столбцов (Product Listing, Detailed Analysis, Product, Price, Rating, Features, Description, Pros, Cons, Target_Market, Gamma Ultra, Alpha Pro, Zeta Max и т. д.), а также имя файла Competitor_Analysis.xlsx указывайте латиницей точно так, как в этом задании.
