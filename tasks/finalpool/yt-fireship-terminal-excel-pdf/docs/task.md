Наша инженерная команда хочет получить комплексный анализ эффективности популярного технологического канала из нашей базы видео. Канал публикует короткие ролики о программировании и разработке программного обеспечения. Ваша задача — обработать исходные данные о видео, рассчитать поквартальную статистику и подготовить готовые материалы для нашего ежеквартального отчёта.

Шаг первый: извлеките все видео из нашей базы данных для канала с названием Fireship. Экспортируйте исходные данные в файл fireship_raw.csv в вашей рабочей директории со следующими столбцами именно в таком порядке: video_id, title, published_date, duration_sec, view_count, like_count. Поле published_date должно быть в формате YYYY-MM-DD, а duration_sec — это длительность в секундах в виде целого числа.

Шаг второй: напишите и запустите скрипт анализа на Python с именем analysis.py в вашей рабочей директории. Скрипт должен читать fireship_raw.csv, группировать видео по календарным кварталам (например, 2024-Q1, 2024-Q2, 2025-Q1 и т. д.) и для каждого квартала вычислять: общее количество опубликованных видео (Video_Count), сумму всех просмотров (Total_Views) и средний уровень вовлечённости (engagement rate), определяемый как сумма всех лайков, делённая на сумму всех просмотров и умноженная на 100, округлённый до 3 знаков после запятой (Avg_Engagement_Rate). Запишите результаты в файл quarterly_stats.json в вашей рабочей директории. JSON-файл должен быть списком объектов, каждый из которых содержит ключи Quarter, Video_Count, Total_Views, Avg_Engagement_Rate, отсортированный хронологически по Quarter.

Шаг третий: создайте книгу Excel с именем Fireship_Analytics.xlsx в вашей рабочей директории с тремя листами. Первый лист называется Raw_Data и содержит все доступные видео Fireship со столбцами video_id, title, published_date, duration_sec, view_count, like_count. Второй лист называется Quarterly_Stats со столбцами Quarter, Video_Count, Total_Views, Avg_Engagement_Rate. Строки в Quarterly_Stats должны быть отсортированы хронологически. Третий лист называется Summary и должен содержать следующие пары «ключ — значение» в виде строк: Total_Videos (общее количество видео), Total_Views (сумма всех просмотров по всем видео), Best_Quarter (квартал с наибольшим Total_Views), Worst_Quarter (квартал с наименьшим Total_Views), Overall_Avg_Engagement (общий средний уровень вовлечённости по всем видео, округлённый до 3 знаков после запятой). Лист Summary должен иметь два столбца с именами Metric и Value.

Шаг четвёртый: напишите документ Word с именем Fireship_Analytics_Report.docx в вашей рабочей директории со следующими четырьмя разделами. Первый раздел озаглавлен Executive Summary и содержит абзац с кратким описанием размера канала, общего числа просмотров и общего среднего уровня вовлечённости. Второй раздел озаглавлен Quarterly Performance и содержит краткое описание того, в каком квартале было больше всего просмотров, а в каком — меньше всего. Третий раздел озаглавлен Key Observations и отмечает не менее двух интересных наблюдений по данным. Четвёртый раздел озаглавлен Appendix и сообщает, что данные получены из внутренней базы видео и охватывают все доступные опубликованные видео.
