Мне нужен отчёт о региональных продажах из нашего хранилища данных ClickHouse. Пожалуйста, выгрузите показатели продаж за всё время с разбивкой по регионам и создайте файл Excel под названием Regional_Sales_Report.xlsx. В файле должно быть два листа. Первый лист под названием Regional Performance должен содержать колонки Region, Order_Count, Total_Revenue (округлённое до 2 знаков после запятой), Avg_Order_Value (округлённое до 2 знаков после запятой — это общая выручка, делённая на количество заказов) и Revenue_Share_Pct (выручка каждого региона в процентах от общей выручки, округлённая до 2 знаков). Отсортируйте по общей выручке по убыванию. Второй лист под названием Summary должен содержать строки Metric и Value для Total_Revenue, Total_Orders, Top_Region (регион с наибольшей выручкой), Bottom_Region (регион с наименьшей выручкой) и Avg_Regional_Revenue (округлённое до 2 знаков).

После файла Excel создайте Google Sheet под названием Sales Regional Performance с той же разбивкой данных по регионам.

Затем отправьте отдельные письма каждому региональному менеджеру: europe.manager@company.com для региона Европа, apac.manager@company.com для Азиатско-Тихоокеанского региона, na.manager@company.com для региона Северная Америка, me.manager@company.com для региона Ближний Восток и latam.manager@company.com для региона Латинская Америка. В теме каждого письма должны быть указаны название региона и слова "sales performance", а в теле письма — показатели выручки и количества заказов этого региона.

Наконец, создайте документ Word под названием Regional_Summary.docx с заголовком Regional Sales Summary и таблицей, показывающей разбивку по регионам.

Примечание: названия регионов хранятся в хранилище данных на русском языке (например, Европа, Азиатско-Тихоокеанский регион, Северная Америка, Ближний Восток, Латинская Америка). Используйте их в том виде, в каком они записаны в хранилище, без перевода.

Примечание: инструмент Word может не сохранять рабочую директорию, поэтому при создании Regional_Summary.docx указывайте абсолютный путь к файлу внутри рабочей директории агента.
