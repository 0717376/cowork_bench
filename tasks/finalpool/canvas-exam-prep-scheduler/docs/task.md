Учебному координатору нужно организовать экзаменационные разборы (сессии повторения) по курсам, в которых студенты слабо справляются с тестами. Начните с того, что получите методические рекомендации по адресу http://localhost:30218/api/prep_guidelines.json — в них заданы пороговое значение (focus_threshold), длительность сессии и параметры расписания.

Затем проанализируйте данные об успеваемости по тестам из системы управления обучением (Canvas). Учитывайте только тесты, оцениваемые по 100-балльной шкале, — это нужно для корректного сравнения. Вычислите средний балл по каждому тесту и определите, какие тесты оказались ниже порога focus_threshold из рекомендаций. Определите, у каких курсов есть хотя бы один тест ниже порога — именно для них требуется сессия повторения.

Изучите документ с правилами составления расписания (scheduling_rules.md) и файл с информацией о помещениях (room_info.txt) в рабочей директории, чтобы понять доступные временные слоты и вместимость аудиторий.

Создайте файл Excel под названием Exam_Prep.xlsx с тремя листами. Первый лист с названием "Quiz Performance" должен содержать столбцы Course, Quiz, Avg_Score и Below_Threshold (значение Yes или No в зависимости от порога focus_threshold из рекомендаций; используйте именно английские Yes/No). Включайте только тесты по 100-балльной шкале, отсортированные сначала по названию курса, затем по названию теста. Второй лист с названием "Review Schedule" должен содержать по одной сессии повторения на каждый курс, которому требуется разбор, со столбцами Course, Topic (в формате "Quiz Review" с последующим названием предмета курса), Date (начиная с 2026-03-16, только будние дни), Time (используйте предпочтительный дневной слот из рекомендаций) и Room (перебирая доступные аудитории по кругу). Третий лист с названием "Summary" должен содержать строки Total_Quizzes_Analyzed, Below_Threshold_Quizzes, Courses_Needing_Review и Review_Sessions_Scheduled.

Для каждого курса, которому требуется сессия повторения, создайте событие в календаре с названием курса и фразой "Quiz Review Session" в заголовке, назначенное на дату и время из листа "Review Schedule". В описании события должен быть указан средний балл по тесту, который послужил поводом для разбора.

Наконец, отправьте письмо на адрес academic-coordinator@university.edu с темой "Exam Review Sessions Scheduled", в котором кратко изложите количество запланированных сессий повторения и какие курсы они охватывают.
