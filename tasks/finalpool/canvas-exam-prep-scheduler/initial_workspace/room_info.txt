Доступные аудитории для разборов

Аудитория 101 — Главный корпус, вместимость 30, есть аудио-видеооборудование
Аудитория 203 — Научный корпус, вместимость 25, только маркерная доска
Аудитория 305 — Библиотека, вместимость 35, проектор и экран
Аудитория 410 — Студенческий центр, вместимость 40, полный мультимедийный комплект
Аудитория 512 — Вычислительный центр, вместимость 20, есть компьютерные терминалы

Все аудитории доступны для бронирования с понедельника по субботу. Бронирование на воскресенье требует отдельного согласования с хозяйственной службой.
