Вы — академический куратор (advisor), готовящий итоговые обзоры за семестр для программы Global Governance and Geopolitics. Ваша цель — проанализировать данные об успеваемости студентов по двум секциям курса, выявить студентов, которым требуется академическое сопровождение, опубликовать аналитику и развернуть инфраструктуру для записи на консультации.

Начните с чтения файла Advising_Handbook.pdf в вашем рабочем каталоге, где описаны пороги GPA для разных категорий сопровождения и процедуры записи на консультации. Также прочитайте файл semester_config.json, в котором указаны курсы для анализа и параметры шкалы оценивания.

Из системы управления обучением (LMS) получите данные по двум курсам: курс Global Governance and Geopolitics за Fall 2013 с идентификатором 20 и тот же курс за Fall 2014 с идентификатором 21. Для каждого курса получите название курса, общее число зачисленных студентов, количество заданий (assignments) и количество тестов (quizzes). Также получите все студенческие отправки (submissions) по этим курсам, включая баллы по каждому заданию. Баллы из отправок нужны для расчёта распределения оценок.

С помощью инструментов командной строки напишите и выполните Python-скрипт под названием advising_analyzer.py в вашем рабочем каталоге. Скрипт должен вычислить для каждого курса следующую аналитику: средний балл отправок, медианный балл отправок и долю успешных (pass rate), определяемую как процент отправок с баллом 60 и выше из 100. Постройте распределение оценок, группируя баллы по диапазонам: ниже 60 — failing, от 60 до 69 — D range, от 70 до 79 — C range, от 80 до 89 — B range, от 90 до 100 — A range. Подсчитайте число отправок в каждом диапазоне и вычислите проценты. На основе общего анализа классифицируйте потребности в сопровождении: студентам в диапазоне failing нужна Urgent Intervention, в диапазоне D — Academic Support, в диапазоне C — Progress Monitoring, а в диапазонах B или A действия не требуются. Оцените число студентов, которым нужен каждый тип сопровождения, исходя из распределения баллов отправок.

Создайте книгу Excel под названием Academic_Advising_Report.xlsx в вашем рабочем каталоге с четырьмя листами. Первый лист с названием Course_Summary должен содержать столбцы: идентификатор курса, название курса, число зачисленных, средний балл, медианный балл и процент pass rate. Включите по одной строке на каждый курс. Второй лист с названием Grade_Distribution должен содержать столбцы: метка диапазона оценок, число отправок в этом диапазоне и процент от общего числа отправок. Включите по одной строке на каждый из пяти диапазонов оценок по обоим курсам суммарно. Третий лист с названием Advising_Needs должен содержать столбцы: категория сопровождения, оценочное число студентов и описание требуемого действия. Включите по одной строке на каждую категорию: Urgent Intervention, Academic Support, Progress Monitoring и No Action Required. Четвёртый лист с названием Appointment_Schedule должен содержать столбцы: дата, время и число доступных слотов для консультаций. Включите записи на пять будних дней недели с 16 по 20 марта 2026 года, по три слота в день в 9:00, 11:00 и 14:00.

Опубликуйте аналитику оценок в облачную таблицу. Создайте таблицу с заголовком "Academic Advising Analytics" минимум с одним листом, содержащим данные сводки по курсам, включая названия курсов, числа зачисленных, средние баллы и pass rate.

Создайте документ Word под названием Advising_Recommendations.docx в вашем рабочем каталоге. Документ должен включать заголовок, упоминающий академическое сопровождение для программы Global Governance, раздел-обзор (overview), обобщающий данные об успеваемости по обеим секциям курса, раздел с выявлением студентов группы риска на основе числа отправок в диапазонах failing и D, а также раздел с рекомендациями (recommendations) с конкретными действиями для каждой категории сопровождения. Текст документа пишите на русском языке, сохраняя английские названия категорий (Urgent Intervention, Academic Support, Progress Monitoring, No Action Required) и название программы Global Governance. Важно: при вызове инструмента Word передавайте АБСОЛЮТНЫЙ путь к файлу Advising_Recommendations.docx внутри рабочего каталога, так как инструмент не сохраняет текущий рабочий каталог.

В нашей базе знаний (пространство Teamly) создайте структуру отслеживания "Student Advising Tracker": заведите родительскую (parent) страницу с заголовком "Student Advising Tracker", а под ней — по одной дочерней (child) странице для каждой категории сопровождения. На каждой дочерней странице в теле текста укажите: категорию (Student Category), число студентов (Count), описание требуемого действия (Action Required) и приоритет (Priority) — High, Medium или Low. Создайте по одной дочерней странице для каждой из четырёх категорий: Urgent Intervention, Academic Support, Progress Monitoring и No Action Required.
