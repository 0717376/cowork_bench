Стратегия работы с отзывами бренда — 2026

Целевые показатели качества:
- Средний рейтинг товара должен быть 4.0 или выше в каждой категории
- Доля ответов на отзывы должна составлять не менее 80%
- Для товаров с рейтингом ниже бенчмарка конкурента нужны планы по улучшению

Дашборд с бенчмарками конкурентов доступен по адресу http://localhost:30203/index.html,
при этом для каждой категории товаров есть отдельная страница.

Требования к анализу:
- Сравнить наш средний рейтинг по каждой категории со средними значениями конкурентов
- Выявить товары, которые ниже бенчмарка конкурента для своей категории
- Создать страницу в базе знаний Teamly с итогами анализа
- Товары ниже бенчмарка должны быть отмечены конкретными действиями по улучшению
