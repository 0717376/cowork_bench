Проведите аудит ценообразования нашего магазина InSales. Просмотрите все товары и найдите каждую позицию, у которой одновременно установлены и обычная цена (regular price), и цена со скидкой (sale price), причём цена со скидкой больше нуля. Для каждого такого товара со скидкой вычислите процент скидки по формуле (regular_price - sale_price) / regular_price * 100 и округлите его до одного знака после запятой.

Создайте файл Excel с именем "Pricing_Audit.xlsx" в вашей рабочей директории. Первый лист должен называться "Discounted Products" и содержать следующие столбцы: Product_Name, Category, Regular_Price, Sale_Price, Discount_Pct и Stock_Quantity. Отсортируйте все строки по Discount_Pct по убыванию, чтобы самые сильно уценённые товары шли первыми.

Второй лист должен называться "Category Summary" и агрегировать данные по категориям товаров. Для каждой категории, в которой есть товары со скидкой, укажите название Category, количество Products_On_Sale, средний Avg_Discount_Pct (округлённый до одного знака после запятой) и максимальный Max_Discount_Pct (округлённый до одного знака после запятой). Отсортируйте этот лист по Avg_Discount_Pct по убыванию.

При сохранении файла Excel передавайте инструменту АБСОЛЮТНЫЙ путь к "Pricing_Audit.xlsx" внутри вашей рабочей директории.

Помимо отчёта Excel, создайте онлайн-форму обратной связи с заголовком "Pricing Review Feedback", чтобы наша команда могла оставлять свои оценки. Форма должна содержать четыре вопроса. Первый вопрос должен запрашивать имя рецензента в виде короткого текстового поля и быть обязательным. Второй вопрос должен предлагать рецензенту выбрать категорию товаров из списка категорий, в которых есть товары со скидкой, в виде вопроса с множественным выбором (варианты ответа), и тоже быть обязательным. Третий вопрос должен быть обязательным вопросом с множественным выбором, запрашивающим оценку ценообразования с вариантами "Too Aggressive", "Appropriate" и "Too Conservative". Четвёртый и последний вопрос должен быть необязательным текстовым полем для любых дополнительных комментариев.
