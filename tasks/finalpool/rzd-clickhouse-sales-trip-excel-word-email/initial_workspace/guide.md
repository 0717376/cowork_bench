# Руководство по планированию деловой поездки: Москва — Санкт-Петербург

## Обзор
Это руководство объясняет, как объединить данные о железнодорожных перевозках с данными о клиентах из хранилища, чтобы спланировать деловую поездку и подготовить требуемые результаты.

## Методология приоритизации клиентов
1. Запросите в хранилище данных (ClickHouse, логическая база SALES_DW) всех клиентов региона «Северная Америка».
2. Рассчитайте для каждого клиента суммарную сумму его заказов (сумма поля Total_Amount по всем его заказам).
3. Отсортируйте клиентов по суммарной сумме заказов по убыванию.
4. Выберите топ-5 клиентов для посещения.
5. Анкорные ориентиры устойчивости: адреса электронной почты клиентов (@example.com) и числовые суммы не локализуются и являются надёжными идентификаторами.

## Структура листов Excel (Sales_Trip_Plan.xlsx)

### Лист: Travel_Details
Столбцы:
- Train_Code: код выбранного поезда
- Depart_Time: время отправления (HH:MM)
- Arrive_Time: время прибытия (HH:MM)
- Duration_Min: продолжительность поездки в минутах
- First_Class_Price_RUB: цена места бизнес-класса (наивысший класс), в рублях
- Business_Class_Price_RUB: цена места эконом+ (следующий по уровню класс), в рублях

### Лист: Customer_Priority
Столбцы:
- Priority_Rank: от 1 до 5 (1 = клиент с наибольшей суммой)
- Customer_Name: имя клиента
- Customer_Email: адрес электронной почты
- Region: регион
- Order_Count: количество заказов
- Total_Amount: суммарная сумма заказов
- Lifetime_Value: пожизненная ценность клиента

### Лист: Summary
Столбцы: Metric, Value
Строки: Travel_Date, Train_Code, Departure_Time, Arrival_Time, Top_Customer (имя клиента ранга 1), Trip_Purpose.

## Разделы документа Word (Sales_Trip_Brief.docx)

### Раздел 1: Travel Logistics
Дата, маршрут, выбранный поезд, время отправления и прибытия, продолжительность.

### Раздел 2: Customer Visit Priority List
Список топ-5 клиентов с именами, количеством заказов и суммарными суммами.

### Раздел 3: Meeting Objectives
Два-три тезиса о целях встреч.

### Раздел 4: Follow-up Actions
Три действия после поездки.
