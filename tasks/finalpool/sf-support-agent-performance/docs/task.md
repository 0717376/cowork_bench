Мне нужно оценить работу наших агентов поддержки. Выгрузите агентов и связанные с ними обращения из хранилища данных поддержки в ClickHouse. Включайте только активных агентов (ACTIVE = true).

Создайте файл Excel под названием Support_Agent_Performance.xlsx с двумя листами. На листе "Agent Performance" должны быть столбцы Agent_Name, Team, Skill_Level, количество обработанных обращений Tickets_Handled и средняя удовлетворённость Avg_Satisfaction, округлённая до 2 знаков после запятой. Отсортируйте по Tickets_Handled по убыванию.

На листе "Summary" должны быть метрики Total_Active_Agents, Total_Tickets (суммарно по всем агентам), Top_Agent (агент с наибольшим числом обращений) и Avg_Satisfaction_Overall — взвешенное среднее, округлённое до 2 знаков после запятой. Используйте два столбца с заголовками Metric и Value.

Также создайте документ Word под названием Agent_Review.docx с кратким обзором ключевых показателей работы агентов и указанием агентов, которым требуется внимание. При сохранении документа Word передавайте абсолютный путь к файлу Agent_Review.docx внутри рабочей директории.

Сохраняйте названия файлов, имена листов, заголовки столбцов и ключи метрик на английском языке в точности так, как указано выше.
