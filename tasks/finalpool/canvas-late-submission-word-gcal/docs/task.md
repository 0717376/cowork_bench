Учебному отделу (деканату) нужен отчёт о соблюдении политики по поздним сдачам работ, а также требуется запланировать корректирующие (профилактические) встречи для курсов с самой высокой долей поздних сдач. Проанализируйте долю поздних сдач по всем курсам, чтобы выявить, где студенты сильнее всего испытывают трудности с дедлайнами.

Создайте документ Word с именем Late_Submission_Report.docx. Документ должен содержать заголовочную секцию, затем секцию с названием "Late Submission Statistics by Course", включающую таблицу со столбцами Course_Name, Total_Submissions, Late_Submissions и Late_Rate_Pct (округлите до 1 знака после запятой). Включите все курсы, по которым есть сдачи. Затем добавьте секцию с названием "Policy Recommendations" с письменными рекомендациями (на русском языке) по снижению высокой доли поздних сдач.

Определите 3 курса с наибольшей долей поздних сдач. Создайте в Google Calendar по одному событию с названием "Late Submission Review" для каждого из этих 3 курсов на дату 2026-04-01. Запланируйте их на 10:00, 11:00 и 14:00 соответственно, каждое продолжительностью один час, и включите название курса в описание события.

Отправьте отчёт письмом на адрес registrar@university.edu с темой "Late Submission Analysis Report" и кратким резюме (на русском языке) основных курсов-нарушителей по поздним сдачам в теле письма.

Примечание: при сохранении документа Word передавайте инструменту Word АБСОЛЮТНЫЙ путь к Late_Submission_Report.docx внутри рабочей директории агента (Word MCP не сохраняет текущую директорию).
