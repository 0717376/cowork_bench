Мне нужен отчёт для руководителей в разрезе отделов из нашего хранилища данных по персоналу в ClickHouse. По каждому отделу я хочу видеть общее число сотрудников, сколько из них высокорезультативных (рейтинг 4 и выше), сколько низкорезультативных (рейтинг 2 и ниже), среднюю зарплату, средний стаж и долю высокорезультативных сотрудников.

Создай файл Excel с именем HR_Manager_Report.xlsx с двумя листами. На листе "Manager Report" должны быть колонки Department, Total (общее число сотрудников), High_Performers, Low_Performers, Avg_Salary (округлённая до целого), Avg_Experience (округлённый до 1 знака после запятой) и High_Perf_Pct (доля высокорезультативных сотрудников в процентах от общего числа, округлённая до 1 знака после запятой). Отсортируй по Department.

На листе "Summary" должны быть строки Total_Employees, Total_High_Performers и Best_Dept (отдел с наибольшей долей высокорезультативных сотрудников).

Важно: при работе с Excel передавай АБСОЛЮТНЫЙ путь к файлу HR_Manager_Report.xlsx внутри рабочего каталога.

Создай страницу Teamly с заголовком "Отчёт о результативности по отделам", кратко описывающую полученные результаты.
