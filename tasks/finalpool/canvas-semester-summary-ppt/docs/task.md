Здравствуйте! Семестр Fall 2014 только что завершился, и мне нужно подготовить сводку для кафедры (отчёт для деканата). Не могли бы вы собрать обзор по всем курсам этого семестра?

Пожалуйста, найдите все курсы, которые шли в семестре Fall 2014, и соберите по каждому следующие данные: название курса, код курса, общее число студентов, количество зачислений (enrollments), число заданий и среднее количество возможных баллов по всем заданиям (округлённое до 1 знака после запятой).

Создайте в рабочей директории файл Excel с именем Semester_Summary.xlsx с двумя листами.

Первый лист должен называться "Course Overview" со столбцами: Course Name, Course Code, Total Students, Enrollments, Assignments, Avg Points Possible. По одной строке на каждый курс, отсортированными по алфавиту по названию курса (Course Name).

Второй лист должен называться "Summary" со столбцами Metric и Value и содержать следующие строки: Total Courses, Total Students (сумма total_students по всем курсам), Total Enrollments (сумма всех значений enrollments), Total Assignments (сумма всех значений по заданиям), Average Assignments per Course (округлённое до 1 знака после запятой).

Также создайте презентацию PowerPoint с именем Semester_Summary.pptx в рабочей директории. Она должна содержать:

Титульный слайд с заголовком "Fall 2014 Semester Summary".

Обзорный слайд с итоговой статистикой (total courses, total students, total enrollments, total assignments, average assignments per course).

По одному слайду на каждый курс: название курса в качестве заголовка и его детали (course code, total students, enrollments, assignments, average points possible).

Финальный слайд "Key Takeaways", определяющий самый крупный курс по числу студентов, самый малочисленный курс по числу студентов и курс с наибольшим числом заданий.

Важно: заголовки слайдов и метки, перечисленные в кавычках выше (например "Fall 2014 Semester Summary", "Course Overview", "Summary", "Key Takeaways", названия столбцов и метрик), оставьте на английском языке точно так, как указано.

Примечание: при сохранении .xlsx и .pptx через соответствующие инструменты передавайте АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории агента (инструменты pptx/excel могут не сохранять текущую рабочую директорию).
