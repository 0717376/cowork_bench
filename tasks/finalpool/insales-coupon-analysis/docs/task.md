Мне нужно проанализировать использование купонов в нашем магазине InSales. Получи все купоны и статистику их использования.

Создай Excel-файл с именем `WC_Coupon_Report.xlsx` с двумя листами (имена листов и колонок оставь на английском, как указано ниже).

Лист `Coupon Analysis` должен содержать колонки `Code`, `Discount_Type`, `Amount`, `Usage_Count`, `Usage_Limit` и `Utilization_Pct`, где `Utilization_Pct` — это число использований, делённое на лимит использований, умноженное на 100 и округлённое до 1 знака после запятой (0, если лимита нет). Отсортируй строки по `Usage_Count` по убыванию.

Лист `Summary` должен содержать `Total_Coupons`, `Total_Usage` (суммарное число использований по всем купонам), `Most_Used_Code` и `Avg_Utilization` (среднее значение `Utilization_Pct` по купонам, у которых задан лимит).

Создай документ Word с именем `Coupon_Strategy.docx` с рекомендациями по итогам анализа. При работе с инструментом Word передавай АБСОЛЮТНЫЙ путь к файлу `Coupon_Strategy.docx` внутри рабочей директории.
