# market-competitive-intelligence-report

Исходные данные для задачи конкурентной разведки.

Файл `competitors.csv` содержит выгрузку по пяти конкурентам на рынке
(название, город, сегмент, ежемесячная цена в рублях, число функций,
наличие бесплатного тарифа). Используйте эти данные вместо обхода сайтов.
