Мне нужен обзор отслеживаемых акций на Московской бирже (MOEX). Собери базовую информацию по всем акциям: название компании, сектор, рыночную капитализацию и коэффициент P/E.

Отслеживаемые тикеры (суффикс `.ME` — Московская биржа, валюта — рубли):
`SBER.ME`, `GAZP.ME`, `LKOH.ME`, `TCSG.ME`, `MGNT.ME`, `MTSS.ME`.

Используй MCP `moex-finance` (тот же интерфейс, что и yahoo-finance: вызывай `get_stock_info` для каждого тикера).

Создай Excel-файл `YF_Stock_Overview.xlsx` с двумя листами. Названия листов и колонок оставь на английском (как указано ниже).

Лист `Stock Overview` — колонки `Symbol`, `Name`, `Sector`, `Market_Cap`, `PE_Ratio`. Отсортируй строки по `Market_Cap` по убыванию. Включай только те акции, у которых указан сектор (`Sector` не пустой) — без сектора строку не добавляй.

Лист `Summary` — колонки `Metric` и `Value` со строками:
- `Total_Stocks` — количество акций, попавших в обзор;
- `Total_Market_Cap` — суммарная рыночная капитализация этих акций;
- `Largest_Company` — название компании с наибольшей рыночной капитализацией;
- `Largest_Market_Cap` — её рыночная капитализация.

Также создай Google-таблицу с названием `Stock Watch Dashboard` и листом `Stocks`, содержащим те же данные обзора (`Symbol`, `Name`, `Sector`, `Market_Cap`, `PE_Ratio`).
