Мне нужно разобраться в распределении заданий по нашим курсам в Canvas. Я сотрудник учебного отдела нашего университета и готовлю сводку для деканата. Пожалуйста, выгрузи данные о заданиях по всем курсам и проанализируй их количество и типы.

Создай файл Excel с именем Canvas_Assignment_Stats.xlsx с двумя листами. Лист "Assignment Stats" должен содержать колонки Course_Code, Total_Assignments, Avg_Points (округлить до 1 знака после запятой), TMA_Count (количество заданий типа Tutor Marked Assessment) и CMA_Count (количество заданий типа Computer Marked Assessment). Отсортируй строки по Course_Code в алфавитном порядке.

Лист "Summary" должен содержать Total_Assignments по всем курсам, Total_TMAs, Total_CMAs, а также Course_Most_Assignments — код курса с наибольшим количеством заданий.

Также создай документ Word с именем Assignment_Overview.docx с краткой сводкой по результатам анализа на русском языке.

Важно: при вызове инструментов excel и word передавай АБСОЛЮТНЫЙ путь к файлам Canvas_Assignment_Stats.xlsx и Assignment_Overview.docx внутри твоей рабочей директории (workspace).
