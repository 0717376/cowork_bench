Я хочу навести порядок по курсу «Data-Driven Design» за семестр Spring 2014. Пожалуйста, найдите этот курс и выгрузите все его задания, у которых есть срок сдачи (due date).

Создайте в рабочей директории файл Excel под названием `Assignment_Tracker.xlsx`. Первый лист должен называться `Assignments` и содержать следующие столбцы: `Assignment_Name`, `Due_Date` (в формате `YYYY-MM-DD`), `Points_Possible`, `Assignment_Group` и `Submission_Count` (сколько работ было сдано по каждому заданию). Отсортируйте строки по `Due_Date` от самой ранней даты к самой поздней.

Добавьте второй лист с названием `Summary` и двумя столбцами — `Metric` и `Value`. Включите в него следующие строки: `Total_Assignments` (количество заданий, у которых есть срок сдачи), `Avg_Points_Possible` (среднее число возможных баллов, округлённое до 1 знака после запятой), `Earliest_Due_Date` и `Latest_Due_Date` (обе даты в формате `YYYY-MM-DD`).

Наконец, создайте событие в календаре для каждого задания, у которого есть срок сдачи. Название каждого события задайте в виде `Due: [Assignment Name]`, подставив реальное название задания. Событие должно начинаться и заканчиваться в день срока сдачи, используя время из поля `due_at`. В описании события укажите число возможных баллов (`Points_Possible`) для этого задания.
