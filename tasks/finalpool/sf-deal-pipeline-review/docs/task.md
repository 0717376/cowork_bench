Вице-президенту по продажам нужен подробный обзор воронки продаж за 1 квартал 2025 года, сравнивающий фактические показатели продаж с целевыми. Квартальные целевые показатели воронки опубликованы на внутреннем дашборде продаж по адресу http://localhost:30209/, где есть сводная страница с итогами по регионам и отдельные страницы с детализацией по каждому региону с разбивкой по сегментам. Вам нужно посетить этот дашборд, чтобы собрать все целевые цифры, а затем сделать запрос в облачное хранилище данных ClickHouse за фактическими показателями за 1 квартал 2025 года, чтобы определить, какие регионы и сегменты выполняют свои цели.

Начните с просмотра шаблона повестки совещания и шаблона целевых показателей воронки (pipeline_targets.xlsx) в рабочей директории. Затем перейдите на внутренний дашборд, начав с главной страницы, и посетите страницу детализации каждого региона, чтобы собрать целевые показатели на 1 квартал 2025 года с разбивкой по сегментам для всех пяти регионов. Сделайте запрос в хранилище данных ClickHouse по всем заказам, размещённым в период с 1 января 2025 по 31 марта 2025 года, объединив их с информацией о клиентах, чтобы получить регион и сегмент для каждого заказа. Рассчитайте суммарную выручку для каждой комбинации регион-сегмент. Названия регионов и сегментов берите ровно так, как они хранятся в хранилище данных (на русском языке), и используйте их единообразно во всех артефактах.

Создайте файл Excel под названием Pipeline_Review.xlsx в рабочей директории с тремя листами. Первый лист с названием Regional Performance должен иметь столбцы Region, Q1_Target, Q1_Actual (фактическая выручка, округлённая до двух знаков после запятой), Gap (фактическое минус целевое), Gap_Pct (в процентах, округлённое до одного знака), и On_Track со значением Yes, если фактическое значение достигает или превышает целевое, и No в противном случае; строки отсортированы по региону. Второй лист с названием Segment Breakdown должен иметь столбцы Region, Segment, Target, Actual, Gap, Gap_Pct и On_Track, по одной строке для каждой комбинации регион-сегмент, отсортированные по региону, затем по сегменту. Третий лист с названием Summary должен иметь столбцы Metric и Value, перечисляющие Total_Q1_Target, Total_Q1_Actual, Total_Gap, Total_Gap_Pct, Regions_On_Track (количество), Regions_Behind (количество), Worst_Region (название региона с наибольшим отрицательным процентом отставания) и Best_Region (название региона с наименьшим отрицательным процентом отставания или лучшим положительным).

Для каждого региона, отстающего от своего целевого показателя более чем на 15 процентов, запланируйте срочное совещание по обзору воронки в Google Calendar. Заголовок совещания должен быть в формате: название региона, за которым следует "Q1 Pipeline Review Meeting" (маркер "Pipeline Review" обязательно сохраните на английском в заголовке). Совещание длительностью один час назначается утром на 9:00 UTC в последовательные рабочие дни, начиная с 10 марта 2026 года. Включите описание на русском языке, упоминающее процент отставания и необходимость корректирующих действий. Если отстают все регионы, запланируйте совещания для всех пяти на последовательные дни.

Примечание: при создании Pipeline_Review.xlsx передавайте инструменту Excel абсолютный путь к файлу внутри рабочей директории.
