# Повестка квартального совещания по обзору воронки продаж

## Структура совещания

1. Приветствие и цели (5 минут)
2. Обзор результатов по регионам (15 минут)
3. Детальный разбор по сегментам (20 минут)
4. Анализ первопричин для отстающих направлений (15 минут)
5. План действий и следующие шаги (10 минут)
6. Вопросы и ответы (10 минут)

## Ключевые вопросы для обсуждения

- Сравнить фактические показатели за 1 квартал 2025 года с целевыми показателями воронки
- Выявить регионы и сегменты, отстающие от целевых показателей
- Обсудить корректирующие действия для отстающих направлений
- При необходимости установить пересмотренные целевые показатели

## Участники

- Вице-президент по продажам (председатель)
- Региональные директора по продажам
- Руководители сегментов
- Финансовый бизнес-партнёр
- Операционный отдел продаж

## Подготовка к совещанию

- Изучить отчёт Pipeline_Review.xlsx
- Подготовить объяснения по любым существенным расхождениям
- Прийти с предложениями по корректирующим действиям
