Продуктовая команда интернет-магазина электроники хочет проанализировать качество клиентских отзывов по категориям товаров, чтобы расставить приоритеты в улучшении продукции. Получи данные об отзывах из InSales и свяжи их с категориями товаров, чтобы понять, как разные категории воспринимаются покупателями. Названия категорий бери непосредственно из InSales «как есть» (они на русском языке) — не переводи и не подменяй их.

Создай Excel-файл с именем Product_Review_Analysis.xlsx в своей рабочей директории с двумя листами. Первый лист назови "Category Analysis" со столбцами: Category, Product_Count, Review_Count, Avg_Rating (округление до 2 знаков после запятой), Five_Star_Count и Five_Star_Rate (доля отзывов с оценкой 5 звёзд в процентах, округление до 1 знака). Отсортируй по столбцу Category по алфавиту. Второй лист назови "Top Products" — перечисли 10 товаров с наивысшим рейтингом, у которых есть не менее 3 отзывов, со столбцами: Product_Name (обрезать до 50 символов при необходимости), Category, Review_Count и Avg_Rating (округление до 2 знаков). Отсортируй по Avg_Rating по убыванию, затем по Review_Count по убыванию.

Создай в корпоративной базе знаний Teamly (пространство PRODUCTS) страницу с заголовком "Product Review Insights". На странице размести таблицу со столбцами Category, Product_Count, Review_Count, Avg_Rating и Five_Star_Rate и заполни её — по одной строке на каждую категорию из листа Category Analysis (значения должны совпадать с Excel).

Отправь результаты анализа на адрес product_team@store.com с темой письма "Product Review Analysis Report". В тексте письма приведи краткую сводку: укажи, какие категории имеют наивысший и наинизший средний рейтинг.

Примечания:
- Для инструментов работы с Excel передавай ПОЛНЫЙ абсолютный путь к файлу Product_Review_Analysis.xlsx внутри рабочей директории.
- Названия файла, листов ("Category Analysis", "Top Products"), столбцов, заголовок страницы Teamly ("Product Review Insights"), адрес e-mail (product_team@store.com) и тему письма ("Product Review Analysis Report") оставляй точно на английском, как указано выше. Содержательную сводку в письме и текст на странице Teamly пиши на русском.
