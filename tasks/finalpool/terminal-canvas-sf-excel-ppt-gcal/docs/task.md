Вы аналитик по развитию персонала, которому поручено устранить разрыв между университетской учебной программой и потребностями отрасли в кадрах. Ваша цель — сопоставить то, чему обучают на курсах, с реальными профилями навыков и образования сотрудников компании, выявить пробелы и представить рекомендации консультативному совету (advisory board).

Начните с прочтения файла Skills_Framework.pdf в вашей рабочей директории, который описывает отраслевые категории навыков и эталонные показатели (benchmarks), используемые для оценки соответствия учебной программы. Также прочитайте файл gap_analysis_config.json, который определяет правила сопоставления структур академических курсов с областями профессиональных компетенций, а также пороговые значения для выявления значимых пробелов.

Из системы управления обучением (LMS) выгрузите подробную информацию о четырёх курсах. В частности, получите данные по курсам с идентификаторами от 1 до 4. По каждому курсу вам нужны название курса, общее число записавшихся студентов, количество заданий (assignments) и количество тестов (quizzes). Эти курсы охватывают темы прикладной аналитики (applied analytics) и биохимии (biochemistry) в течение двух академических лет.

Из корпоративного хранилища данных ClickHouse сделайте запрос к базе данных отдела кадров, чтобы получить профиль персонала. По каждому подразделению (department) получите численность сотрудников (headcount), среднюю заработную плату, среднюю оценку эффективности и средний стаж работы в годах. Также выгрузите распределение по уровням образования по всему персоналу, показывая, сколько сотрудников имеют каждый тип степени или квалификации.

С помощью инструментов командной строки напишите и выполните Python-скрипт под названием gap_analyzer.py в вашей рабочей директории. Скрипт должен вычислять анализ пробелов, сопоставляя интенсивность оценивания курсов с ожиданиями по профессиональным компетенциям. Для каждой из четырёх областей навыков, определённых в документе framework, рассчитайте балл учебной программы (curriculum score) на основе количества оценочных мероприятий в соответствующих курсах, нормированный к десятибалльной шкале. Балл отраслевого спроса (industry demand score) для каждой области навыков возьмите из раздела industry_benchmarks в gap_analysis_config.json. Используйте данные об эффективности и образовании персонала для обоснования рекомендаций. Балл пробела (gap score) — это абсолютная разница между баллом учебной программы и отраслевым спросом. Балл пробела выше 3.0 должен помечаться как приоритет Critical, от 1.5 до 3.0 — как приоритет High, ниже 1.5 — как приоритет Moderate.

Создайте книгу Excel под названием Skills_Gap_Analysis.xlsx в вашей рабочей директории с четырьмя листами. Первый лист должен называться Curriculum_Coverage и содержать столбцы для идентификатора курса, названия курса, числа записавшихся, количества заданий, количества тестов и общего числа оценочных мероприятий. Включите по одной строке на курс. Второй лист должен называться Workforce_Profile и содержать столбцы для названия подразделения, численности, описания среднего уровня образования и средней оценки эффективности. Включите по одной строке на подразделение. Третий лист должен называться Gap_Matrix и содержать столбцы для области навыков, балла учебной программы, балла отраслевого спроса, балла пробела и уровня приоритета. Включите по одной строке для каждой из четырёх областей навыков. Четвёртый лист должен называться Recommendations и содержать столбцы для приоритета, области пробела, рекомендуемого действия и срока реализации. Включите не менее четырёх практических рекомендаций на основе результатов анализа пробелов.

Создайте презентацию под названием Skills_Gap_Presentation.pptx в вашей рабочей директории не менее чем из пяти слайдов. Первый слайд должен быть титульным с названием презентации и датой. Включите слайды, охватывающие результаты анализа учебной программы, сводку профиля персонала, результаты матрицы пробелов и рекомендации. Каждый содержательный слайд должен иметь чёткий заголовок и содержательные маркированные пункты или сводки данных.

Наконец, запланируйте три встречи консультативного совета (advisory board) в общем календаре на неделю с 16 по 20 марта 2026 года. Первая встреча в понедельник в 10:00 продолжительностью два часа должна быть посвящена пересмотру учебной программы (тема: «Пересмотр учебной программы»). Вторая встреча в среду в 14:00 продолжительностью девяносто минут должна охватывать выводы по данным о персонале (тема: «Выводы по данным о персонале»). Третья встреча в пятницу в 09:00 продолжительностью два часа должна быть посвящена рекомендациям по анализу пробелов (тема: «Рекомендации по анализу пробелов»). Сводка (summary) каждого события должна включать «Advisory Board» и тему встречи.

Важно: инструменты Excel и PowerPoint могут не сохранять рабочую директорию, поэтому при создании Skills_Gap_Analysis.xlsx и Skills_Gap_Presentation.pptx указывайте АБСОЛЮТНЫЙ путь к файлам внутри рабочей директории.
