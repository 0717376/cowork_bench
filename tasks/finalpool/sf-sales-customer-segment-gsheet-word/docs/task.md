Вы аналитик по стратегии продаж. Ваша задача — проанализировать эффективность клиентских сегментов на основе данных из хранилища продаж и подготовить отчёт по сегментному анализу.

Сначала выполните запрос к хранилищу данных продаж ClickHouse, чтобы получить метрики по каждому клиентскому сегменту. Соедините таблицу CUSTOMERS с таблицей ORDERS по полю CUSTOMER_ID. Сгруппируйте по столбцу SEGMENT и рассчитайте для каждого сегмента: число уникальных клиентов, общее количество заказов и суммарную выручку (сумма TOTAL_AMOUNT). Логическая база данных — SALES_DW, схема — PUBLIC. В данных представлены четыре сегмента: «Частные клиенты», «Корпоративный», «Государственный» и «Малый и средний бизнес».

Затем создайте Google Sheet с названием "Customer Segment Analysis" со следующими столбцами: Segment, Customer_Count, Order_Count, Total_Revenue, Revenue_Pct. Столбец Revenue_Pct должен показывать долю каждого сегмента в общей выручке (округлённую до 2 знаков после запятой). Отсортируйте строки по Total_Revenue по убыванию. В таблице должно быть 4 строки с данными (по одной на сегмент) плюс строка заголовка.

После этого создайте документ Word под названием Customer_Segment_Report.docx в рабочей директории. Документ должен содержать: заголовок "Customer Segment Analysis Report", вводный абзац с описанием охвата анализа, таблицу с данными по сегментам (Segment, Customer_Count, Order_Count, Total_Revenue) и раздел с заголовком "Recommendations", за которым следуют как минимум две рекомендации, основанные на данных по сегментам. ВАЖНО: при сохранении документа через инструмент Word передавайте АБСОЛЮТНЫЙ путь к файлу Customer_Segment_Report.docx внутри рабочей директории.

Наконец, отправьте отчёт на адрес sales-strategy@company.example.com с адреса analytics@company.example.com с темой "Customer Segment Analysis Report". В теле письма укажите, какой сегмент генерирует наибольшую выручку и какой имеет наибольшее число клиентов.

Файл Segment_Strategy.pdf в вашей рабочей директории описывает каждый клиентский сегмент и стратегические цели.
