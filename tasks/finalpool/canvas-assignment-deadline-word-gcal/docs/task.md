Мне нужно помочь студентам курса Foundations of Finance (Fall 2013) не пропускать сроки сдачи заданий. Для этого подготовьте комплект материалов по управлению дедлайнами.

Найдите все задания этого курса в Canvas и создайте файл Excel с именем Assignment_Deadlines_FFF2013J.xlsx, содержащий два листа. Первый лист с названием All Assignments должен иметь столбцы Assignment_Name, Points_Possible и Due_Date (дата в формате YYYY-MM-DD), отсортированные по Due_Date по возрастанию, при этом задания без даты сдачи помещаются в конец. Второй лист с названием Summary должен иметь два столбца Metric и Value со строками: Total_Assignments — общее количество всех заданий, Total_Points_Possible — сумма баллов по всем заданиям, и Avg_Points_Per_Assignment — среднее количество баллов на задание, округлённое до 2 знаков после запятой.

Также создайте документ Word с именем Assignment_Schedule_FFF2013J.docx с заголовком Assignment Schedule for Foundations of Finance Fall 2013, после которого идёт таблица со списком каждого задания: его название, дата сдачи в формате YYYY-MM-DD и баллы (Points_Possible). Если у задания нет даты сдачи, в столбце с датой укажите TBD. Важно: при работе с MCP-сервером Word передавайте АБСОЛЮТНЫЙ путь к файлу .docx внутри рабочей директории (workspace).

Для каждого задания, у которого есть дата сдачи, создайте в календаре событие-напоминание, которое начинается за 7 дней до даты сдачи в 8:00 утра и длится 30 минут. Заголовок события должен быть Assignment Due: и далее название задания. Используйте часовой пояс America/New_York.

Отправьте письмо на адрес fff2013j.students@university.edu с темой Foundations of Finance Assignment Deadline Reminder. Текст письма напишите на русском языке: перечислите все задания с их датами сдачи и баллами и напомните студентам, чтобы они проверили свои календари на наличие запланированных напоминаний.
