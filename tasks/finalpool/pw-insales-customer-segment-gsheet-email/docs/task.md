Мне нужна помощь с сегментным анализом для нашего интернет-магазина. Есть внешние эталонные (benchmark) данные: зайди на http://localhost:30338 и извлеки оттуда нужные рыночные метрики по категориям товаров.

Затем проверь наш магазин InSales и собери актуальные данные по товарам и заказам.

С помощью терминала создай и запусти Python-скрипт wc_segment_processor.py в рабочем каталоге, который читает собранные данные из JSON-файлов (их создаёшь ты сам), выполняет анализ и сохраняет результат в wc_segment_results.json.

Создай Excel-файл Customer_Segment_Report.xlsx с тремя листами.

Лист Data_Analysis должен содержать основные сравнительные данные. Используй РОВНО следующие названия колонок (заголовки оставь на английском): Category, Product_Count, Our_Avg_Price, Total_Sales, Market_Avg_Price, Price_Gap_Pct. В колонке Category — название категории товаров (как в магазине InSales и на эталонной странице, например Аудио, Камеры, Электроника, Бытовая техника, ТВ и домашний кинотеатр, Часы). Our_Avg_Price — наша средняя цена по категории, Total_Sales — суммарные продажи, Market_Avg_Price — рыночная эталонная средняя цена из benchmark-страницы, Price_Gap_Pct — относительный разрыв цены в процентах: (Our_Avg_Price - Market_Avg_Price) / Market_Avg_Price * 100. Отсортируй строки по алфавиту по колонке Category.

Лист Metrics должен иметь две колонки Metric и Value, суммирующие ключевые показатели (общее число категорий Total_Categories, общее число товаров Total_Products, средний разрыв цены Avg_Price_Gap и т.п.).

Лист Recommendations должен перечислять приоритетные действия на основе анализа разрывов цен; колонки: Priority, Action, Category.
