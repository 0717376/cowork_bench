Мне нужна помощь с анализом результатов тестов (quiz) в Canvas.

Сначала зайди на внешний дашборд с эталонными (benchmark) данными по адресу http://localhost:30311 и извлеки оттуда нужные метрики. На странице приведена таблица с колонками Course_Level, National_Pass_Rate, Avg_Score и Student_Engagement_Index по уровням Introductory, Intermediate, Advanced и Graduate — это внешние отраслевые ориентиры.

Затем подключись к нашей системе управления обучением (Canvas LMS) и собери данные по курсам и студентам: список курсов, число записанных студентов и результаты тестов.

С помощью терминала создай и запусти в рабочем каталоге Python-скрипт course_quiz_processor.py, который читает собранные данные из созданных тобой JSON-файлов, выполняет анализ и сохраняет результат в course_quiz_results.json.

Создай Excel-файл Quiz_Benchmark_Report.xlsx ровно с тремя листами.

Лист Data_Analysis — основная таблица сравнения. Колонки именно такие (на английском, в этом порядке):
- Course — название курса;
- Code — код курса из Canvas;
- Enrollment — число записанных студентов;
- Avg_Score — наш внутренний средний балл по курсу;
- Pass_Rate — наша внутренняя доля сдавших (%).
Отсортируй строки по алфавиту по колонке Course. Значения Avg_Score и Pass_Rate должны быть честно посчитаны по данным из Canvas, а не выдуманы.

Лист Metrics — сводка ключевых метрик ровно с двумя колонками Metric и Value. Включи как минимум строки Total_Courses (число курсов), Total_Enrollment (суммарное число записанных), Avg_Score (средний балл) и Avg_Pass_Rate (средняя доля сдавших). Значения должны быть согласованы с листом Data_Analysis: Total_Courses равно числу строк, Total_Enrollment равно сумме Enrollment, Avg_Pass_Rate равно среднему Pass_Rate.

Лист Recommendations — список приоритетных действий по итогам анализа, с колонками Priority, Action и Course. В первую очередь (наивысший приоритет) — курсы с самым большим отставанием от внешнего ориентира (наименьшая доля сдавших). Сравнение веди по методике: Gap = внутреннее значение − внешний benchmark; в первую очередь занимайся самыми большими отрицательными разрывами.

Создай в Teamly страницу с заголовком "Canvas Quiz Dashboard" с кратким резюме анализа (на русском): главные выводы, сравнение с внешним ориентиром и рекомендации.
