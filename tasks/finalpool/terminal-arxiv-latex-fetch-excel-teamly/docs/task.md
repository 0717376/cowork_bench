Я готовлюсь к выступлению на «Международной конференции по методам машинного обучения». Расписание конференции доступно в виде JSON по адресу http://localhost:30412/api/conference.json. Сначала получите данные расписания конференции.

Затем мне нужно изучить научные статьи в нашем архиве LaTeX-статей. Найдите статьи, связанные с машинным обучением, а именно про законы масштабирования (scaling laws), instruction tuning и открытые языковые модели. Для каждой найденной статьи извлеките полное содержание, включая заголовки всех разделов, аннотации и ключевые выводы из исходного LaTeX.

Создайте и запустите Python-скрипт под названием conference_prep_builder.py в рабочей директории. Скрипт должен прочитать расписание конференции из conference_schedule.json и данные анализа статей из paper_analysis.json (оба файла вы создаёте из полученных данных), сопоставить, какие статьи относятся к каким сессиям конференции, извлечь количество слов по разделам и сформировать на выходе conference_prep_results.json.

Создайте Excel-файл под названием Conference_Prep_Tracker.xlsx с тремя листами. Первый лист Paper_Sections должен содержать столбцы Paper_ID, Paper_Title, Section_Title и Section_Word_Count (приблизительное количество слов в каждом разделе). Включите все разделы всех найденных релевантных статей. Отсортируйте по Paper_ID, затем по порядку разделов.

Второй лист Conference_Schedule должен содержать столбцы Session_ID, Session_Title, Date, Time, Room и Related_Papers (названия статей через запятую, которые соответствуют теме сессии). Данные берутся из JSON конференции.

Третий лист Presentation_Notes должен содержать столбцы Slide_Number (от 1 до не менее 8), Topic, Key_Points (резюме в 2-3 предложения) и Source_Paper. Постройте логичную последовательность презентации, начиная с вводной части, охватывая основные выводы каждой статьи и заканчивая заключением.

Наконец, создайте в базе знаний Teamly (в пространстве «Подготовка к конференции») страницу с заголовком «Conference Prep Notes» со структурированным содержанием. В тексте страницы укажите следующие сведения: Conference_Name (название конференции), Presentation_Date (дата выступления), Status (со значением «In Progress») и Paper_Count (число обработанных статей). Также добавьте описательные блоки содержания, кратко излагающие ключевой вклад каждой статьи и то, как он связан с темами конференции.
