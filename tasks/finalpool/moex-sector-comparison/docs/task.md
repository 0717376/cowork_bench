Мне нужно сравнить секторы наших отслеживаемых российских акций (MOEX). Получите информацию по каждой акции и сгруппируйте по секторам, чтобы понять отраслевую картину.

Наши отслеживаемые тикеры: SBER.ME, GAZP.ME, LKOH.ME, TCSG.ME, MGNT.ME, MTSS.ME. Капитализацию указывайте в миллиардах рублей.

Создайте файл Excel с именем YF_Sector_Comparison.xlsx с двумя листами. Названия листов, заголовки столбцов и имена файлов пишите ровно так, как указано ниже, латиницей (на английском).

Лист "Sector Comparison" должен содержать столбцы: Sector (название сектора), Stock_Count, Avg_Market_Cap_B (средняя рыночная капитализация в миллиардах, округлённая до 2 знаков), Avg_PE_Ratio (средний коэффициент P/E, округлённый до 2 знаков). Отсортируйте строки по столбцу Avg_Market_Cap_B по убыванию.

Если у какой-либо акции сектор не указан (пустое значение), отнесите её к сектору с названием "Unknown".

Лист "Summary" должен содержать столбцы Metric и Value со строками: Total_Sectors, Total_Stocks и Largest_Sector (сектор с наибольшей средней рыночной капитализацией).

Также создайте документ Word с именем Sector_Analysis.docx с содержательными наблюдениями на русском языке об отраслевом распределении и различиях в оценке между секторами. При сохранении документа Word передавайте АБСОЛЮТНЫЙ путь к файлу .docx внутри рабочей директории (Word MCP теряет текущую директорию).
