Мне нужна помощь с анализом рыночных настроений по нашему портфелю российских акций. Доступны внешние бенчмарк-данные: загрузите их по адресу http://localhost:30337/api/data.json и извлеките нужные показатели (массив market_indicators, где у каждого инструмента есть symbol, target_price и analyst_rating).

Затем получите текущие рыночные данные по акциям из нашего портфеля (тикеры MOEX из бенчмарк-файла: SBER.ME, GAZP.ME, LKOH.ME, TCSG.ME, MGNT.ME) через финансовый сервис.

С помощью терминала создайте и запустите Python-скрипт под названием yf_sentiment_processor.py в рабочей директории. Скрипт читает собранные данные из создаваемых вами JSON-файлов, выполняет анализ и сохраняет результат в yf_sentiment_results.json.

Создайте Excel-файл под названием Market_Sentiment_Report.xlsx с тремя листами. Первый лист Data_Analysis содержит основные сравнительные данные с релевантными колонками. Второй лист Metrics обобщает ключевые показатели. Третий лист Recommendations перечисляет конкретные действия.

Лист Data_Analysis должен включать колонки: Symbol (тикер, основное измерение), Name (название эмитента), Sector (сектор), Current_Price (текущая цена из финансового сервиса), Target_Price (целевая цена из бенчмарка) и Upside (разница между целевой и текущей ценой). Отсортируйте данные по алфавиту по колонке Symbol (по возрастанию). Лист Metrics должен иметь две колонки Metric и Value, обобщающие итоговые показатели: Total_Stocks (число акций), Avg_Upside (средний Upside) и Best_Opportunity (тикер с наибольшим Upside). Лист Recommendations должен перечислять приоритетные действия на основе анализа с колонками Priority, Action и Symbol.

Создайте в Teamly страницу с заголовком «Дашборд рыночных настроений» (Дашборд) с кратким изложением анализа; в тексте упомяните проанализированные тикеры MOEX.

Важно: при работе с Excel-сервисом передавайте АБСОЛЮТНЫЙ путь к файлу Market_Sentiment_Report.xlsx внутри рабочей директории.
