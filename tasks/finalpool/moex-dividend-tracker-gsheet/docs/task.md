Мне нужно собрать сводный трекер дивидендов и корпоративных действий для нашего инвестиционного портфеля по российским эмитентам. Все котировки рублёвые, тикеры — с суффиксом `.ME` (Московская биржа). Используйте MCP `moex-finance` (тот же интерфейс tools, что и у yahoo-finance: `get_stock_info` и т.д.). Посмотрите историю дивидендов и корпоративных действий по следующим пяти акциям: SBER.ME, GAZP.ME, LKOH.ME, MGNT.ME и TCSG.ME.

По каждой акции через `get_stock_info` соберите информацию о дивидендах (размер последнего дивиденда `lastDividendValue` в рублях, дивидендную доходность `trailingAnnualDividendYield`, годовую дивидендную ставку `trailingAnnualDividendRate`, а также дату закрытия реестра / экс-дивидендную дату `exDividendDate`), а также о дроблениях акций (split-фактор `lastSplitFactor` и дату `lastSplitDate`).

Учтите, что часть эмитентов в настоящий момент дивиденды не выплачивает — у такой акции поле `exDividendDate` отсутствует. Такие акции в сводку «кто платит дивиденды» включать не нужно.

Собрав все данные, создайте Google-таблицу с названием "Dividend Tracker" с единственным листом "Stock Actions", где перечислите каждое найденное действие. Колонки должны быть такими (названия колонок оставьте на английском): Date, Symbol, Action_Type (либо "Dividend", либо "Stock Split"), Value (для дивидендов — размер дивиденда `lastDividendValue`; для дроблений — split-фактор вида "1000:1"), Dividend_Yield (дивидендная доходность в процентах, либо 0 для дроблений), Trailing_Annual_Rate (годовая дивидендная ставка, либо 0 для дроблений). Строки отсортируйте сначала по Symbol, затем по Date.

Для строк типа "Dividend" в колонку Date запишите экс-дивидендную дату (`exDividendDate`) в формате YYYY-MM-DD. Для строк типа "Stock Split" — дату дробления (`lastSplitDate`) в формате YYYY-MM-DD.

После создания таблицы отправьте письмо на адрес investor@portfolio.example.com с темой "Dividend Action Summary" (тему оставьте на английском), в котором на русском языке кратко опишите, какие из акций сейчас платят дивиденды, их годовые дивидендные ставки и любые недавние корпоративные действия (дробления). Письмо должно быть отправлено с адреса portfolio-alerts@finance.example.com.
