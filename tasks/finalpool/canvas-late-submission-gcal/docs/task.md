Учебной части (academic affairs) нужен сводный отчёт о просроченных сдачах работ по всем курсам в системе управления обучением (LMS). Запросите все сдачи работ, отмеченные как просроченные (late), и сведите результаты воедино.

Создайте Excel-файл с именем Late_Submissions.xlsx в рабочей директории с двумя листами. На первом листе "By Course" должны быть колонки Course_Name, Late_Count (общее число просроченных сдач по курсу) и Unique_Students (число уникальных студентов, сдавших работу с опозданием по этому курсу). Отсортируйте строки по Late_Count в порядке убывания. На втором листе "Top Offenders" перечислите топ-10 комбинаций «студент-курс» с наибольшим числом просроченных сдач; колонки: Student_Name, Course и Late_Count, отсортированные по Late_Count в порядке убывания.

При записи Excel-файла передавайте АБСОЛЮТНЫЙ путь к Late_Submissions.xlsx внутри рабочей директории агента.

Для каждого курса, по которому есть просроченные сдачи, создайте событие в календаре с названием "Review Meeting - [course name]", запланированное на 13 марта 2026 года (March 13, 2026) на 10:00 утра продолжительностью один час. В описании события должны быть указаны число просроченных сдач и число уникальных затронутых студентов по этому курсу.

Наконец, отправьте письмо на адрес academic.affairs@university.example.com с темой "Late Submission Report", в котором кратко изложены результаты. В теле письма должны быть указаны: общее число курсов с просроченными сдачами, суммарное количество просроченных сдач по всем курсам, а также названия трёх курсов с наибольшим числом просроченных сдач.
