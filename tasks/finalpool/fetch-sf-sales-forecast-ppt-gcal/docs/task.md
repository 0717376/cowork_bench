Вице-президент по продажам нашей компании попросил меня подготовить презентацию прогноза продаж на 2 квартал 2026 года для предстоящего заседания совета директоров. Аналитическое агентство опубликовало свои отраслевые прогнозы роста на 2026 год, и эти данные доступны в виде JSON API по адресу http://localhost:30209/api/projections.json. Этот API предоставляет ожидаемые темпы роста по регионам для сектора электроники и потребительских товаров.

Начните с получения рыночных прогнозов роста из этого API. Данные включают прогнозируемые проценты роста для каждого из наших пяти регионов продаж.

Затем запросите хранилище данных продаж ClickHouse (логическая база данных SALES_DW, схема PUBLIC), чтобы получить наши исторические показатели за 1 квартал 2026 года. Мне нужны суммарная выручка и количество заказов в разбивке по регионам за январь, февраль и март. Также получите суммарную выручку 1 квартала по сегментам клиентов для каждого региона, чтобы понять структуру сегментов (segment mix).

Используя фактические данные 1 квартала как базу и применяя рыночные прогнозы роста, рассчитайте прогноз на 2 квартал 2026 года для каждого региона. Прогнозируемая выручка 2 квартала для каждого региона должна рассчитываться как суммарная выручка 1 квартала, умноженная на (1 + growth_rate/100), где growth_rate берётся из рыночного API прогнозов. Округлите все денежные значения до двух знаков после запятой.

Создайте файл Excel под названием Sales_Forecast_Data.xlsx в рабочей директории с тремя листами. Первый лист должен называться "Q1_Actuals" со столбцами Region, Month, Order_Count и Revenue, показывающими помесячную разбивку. Второй лист должен называться "Q2_Forecast" со столбцами Region, Q1_Revenue, Growth_Rate_Pct, Q2_Forecast_Revenue и Q2_Forecast_Orders. Третий лист должен называться "Segment_Mix" со столбцами Region, Segment, Q1_Revenue и Revenue_Share_Pct, показывающими долю каждого сегмента в общей региональной выручке в процентах.

Затем создайте презентацию PowerPoint под названием Q2_Sales_Forecast.pptx в рабочей директории. Презентация должна содержать титульный слайд с заголовком "Q2 2026 Sales Forecast" и сегодняшней датой. Добавьте слайд с обзором показателей 1 квартала по регионам с суммарной выручкой. Добавьте слайд с прогнозами роста из рыночного исследования. Добавьте слайд с прогнозом на 2 квартал по регионам. Добавьте финальный слайд с ключевыми выводами и регионом с наивысшим прогнозируемым ростом.

Наконец, запланируйте презентацию для совета директоров в календаре 28 марта 2026 года с 10:00 до 11:30 с заголовком "Board Meeting: Q2 Sales Forecast Presentation". В описании укажите суммарную прогнозируемую выручку компании за 2 квартал (по всем регионам) и регион с наивысшим прогнозируемым ростом.

Примечание: при создании файлов .xlsx и .pptx через соответствующие инструменты передавайте ПОЛНЫЙ абсолютный путь к файлу внутри рабочей директории (указанной выше), так как эти инструменты могут не сохранять текущую рабочую директорию.
