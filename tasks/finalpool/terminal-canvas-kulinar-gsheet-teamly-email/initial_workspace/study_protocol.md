# Исследование корреляции питания и успеваемости

## Study ID: SA-2026-NUT-001

## Главный исследователь (Principal Investigator)
Д-р Дарья Чен, отдел по работе со студентами

## Согласование IRB
IRB Protocol #2026-0142, одобрено 15 января 2026 года

## Цель исследования
Определить, связаны ли целевые вмешательства в питание с улучшением академической вовлечённости и успеваемости среди студентов бакалавриата, обучающихся на курсах «Экологическая экономика и этика».

## Методология

### Этап 1: Сбор данных
- Извлечь данные о зачислении и сдаче работ с курсов 13 и 14 («Экологическая экономика и этика», наборы осени 2013 и осени 2014 годов)
- Распределить студентов по уровням вовлечённости на основе паттернов опозданий при сдаче работ
- Определения уровней:
  - **High Engagement** (высокая вовлечённость): доля опозданий 0%
  - **Medium Engagement** (средняя вовлечённость): доля опозданий >0% и <=25%
  - **Low Engagement** (низкая вовлечённость): доля опозданий >25%

### Этап 2: Разработка вмешательства
- Разработать вмешательства по планам питания в зависимости от уровня вовлечённости
- Контрольная группа (High): без вмешательства в питание
- Частичное вмешательство (Medium): только рекомендации к завтраку
- Полное вмешательство (Low): полные планы питания на завтрак и обед
- Использовать корпоративную кулинарную базу рецептов для подбора здоровых блюд

### Этап 3: Пилотное внедрение
- Целевая дата старта: 1 апреля 2026 года
- Длительность: 12 недель
- Отслеживать изменения успеваемости по уровням

## Бюджетные ограничения
- См. campus_dining_budget.json по лимитам стоимости на одного студента
- Общий бюджет пилота: 50 000 ₽

## Конфиденциальность данных
- Все данные студентов должны быть обезличены в отчётах
- Использовать только идентификаторы студентов, без имён
- Агрегированная статистика — только для групп от 10 студентов и более

## Результаты (Deliverables)
1. Анализ вовлечённости студентов с разбивкой по уровням
2. Рекомендации по планам питания для каждого уровня
3. Облачная таблица для текущего учёта
4. Записи в базе знаний для управления программой
5. Уведомления заинтересованным сторонам по электронной почте
6. Итоговый отчёт
