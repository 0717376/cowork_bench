Я научный библиотекарь университета и провожу перекрёстный анализ (cross-reference) двух академических баз данных. Мне нужно сравнить статьи, доступные в базе Scholarly (а именно в коллекции arxiv_papers), со статьями из базы arxiv, чтобы найти пересекающиеся статьи и понять различия в покрытии.

Пожалуйста, создайте Excel-файл с именем "Citation_Cross_Reference.xlsx" в моей рабочей директории с тремя листами. ВАЖНО: при вызове инструментов excel передавайте АБСОЛЮТНЫЙ путь к файлу Citation_Cross_Reference.xlsx внутри рабочей директории.

Первый лист, "Scholarly_Papers", должен содержать все статьи из коллекции Scholarly arxiv_papers со столбцами: Paper_ID, Title, Authors, Published_Year, Journal_Ref.

Второй лист, "Arxiv_Papers", должен содержать все статьи из базы arxiv со столбцами: Paper_ID, Title, Authors, Published_Year, Categories.

Третий лист, "Overlap_Analysis", должен содержать статьи, которые присутствуют в обеих базах (сопоставление по их paper ID), со столбцами: Paper_ID, Title, In_Scholarly (Yes/No), In_Arxiv (Yes/No). Также включите статьи, уникальные для каждой из баз.

После создания таблицы, пожалуйста, отправьте письмо с кратким изложением результатов. Отправьте его от librarian@university.edu на research-lead@university.edu с темой "Cross-Reference Analysis: Scholarly vs Arxiv Paper Databases". Текст письма (на русском языке) должен указывать общее количество статей в каждой базе, число пересекающихся статей, а также выделять статьи, уникальные для одной из баз. Названия статей и имена авторов оставляйте в оригинале (на английском), не переводите их.
