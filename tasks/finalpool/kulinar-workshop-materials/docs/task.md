Наша компания в следующем месяце организует командный кулинарный воркшоп (тимбилдинг), и мне нужно, чтобы ты подготовил материалы. Мы хотим включить три русских блюда, которые относительно просто готовить, занимают не более 30 минут активной готовки и относятся к разным категориям, чтобы было разнообразие. Поищи в доступной кулинарной базе три подходящих блюда, удовлетворяющих этим критериям.

Используй кулинарную базу через инструменты kulinar (например, `mcp_kulinar_getAllRecipes`, `mcp_kulinar_getRecipesByCategory`, `mcp_kulinar_recommendMeals`, `mcp_kulinar_getRecipeById`), чтобы подобрать блюда. Бери реальные названия рецептов, реальные списки ингредиентов и реальные шаги приготовления из базы — ничего не выдумывай. Выбирай блюда лёгкого уровня сложности, со временем активной готовки не более 30 минут, и из разных категорий (например, салат, гарнир, десерт).

Когда подберёшь три рецепта, мне нужно, чтобы ты создал в рабочей директории три результата.

Во-первых, создай документ Word с именем файла Workshop_Handbook.docx. Он должен служить подробным пошаговым руководством по готовке для участников. Начни с заголовка верхнего уровня "Поваренная книга кулинарного воркшопа". Затем для каждого из трёх блюд добавь заголовок второго уровня с названием блюда, после него — абзац со списком всех необходимых ингредиентов с количествами, а затем ещё один абзац (или несколько абзацев) с пошаговым описанием приготовления по порядку. В конце документа добавь раздел "Советы и заметки" с общими кулинарными советами, которые помогут новичкам.

Во-вторых, создай презентацию PowerPoint с именем файла Workshop_Slides.pptx, которую инструктор будет использовать во время воркшопа. Первый слайд должен быть титульным с заголовком "Командный кулинарный воркшоп". Затем для каждого блюда создай два слайда: один слайд с названием блюда в качестве заголовка и списком ингредиентов в качестве содержимого, и второй слайд с заголовком из названия блюда и слова "Шаги" и шагами приготовления в виде маркированного списка. В конце добавь завершающий слайд с заголовком "Приятного аппетита!".

В-третьих, создай PDF-файл с именем файла Shopping_List.pdf в рабочей директории. Это должен быть сводный список покупок, объединяющий все ингредиенты из всех трёх блюд, аккуратно оформленный так, чтобы сотрудник нашего отдела закупок мог использовать его для похода в магазин. По возможности группируй похожие ингредиенты вместе и указывай суммарные необходимые количества. В верхней части документа должен быть чёткий заголовок "Список покупок для кулинарного воркшопа".

Совет по PDF: чтобы кириллица в PDF отображалась и извлекалась корректно, используй встроенный Unicode-шрифт с поддержкой кириллицы (например, зарегистрируй TTF-шрифт вроде Arial/DejaVuSans через reportlab); шрифт по умолчанию (Helvetica) кириллицу не поддерживает.

Важно: инструменты для работы с Word (.docx) и PowerPoint (.pptx) могут не сохранять текущую рабочую директорию. Поэтому при вызове этих инструментов передавай АБСОЛЮТНЫЙ путь к файлам .docx и .pptx внутри рабочей директории агента (например, <workspace_dir>/Workshop_Handbook.docx).
