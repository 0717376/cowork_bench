Мне нужна помощь с аналитическим обзором литературы по arxiv. Есть внешние данные-ориентир (benchmark), которые нужно получить с http://localhost:30323/api/data.json и извлечь оттуда релевантные показатели (тренды роста по областям).

Затем найди релевантные академические статьи по теме в локальном arxiv. В выборке присутствует шумовая статья не по теме (область quant-ph) — её нужно исключить из анализа, оставив только релевантные статьи по машинному обучению и ИИ.

Используя терминал, создай и запусти Python-скрипт arxiv_litreview_processor.py в рабочей директории. Скрипт читает собранные данные из JSON-файлов, которые ты создашь, выполняет анализ и сохраняет результат в arxiv_litreview_results.json.

Создай Excel-файл Lit_Review_Report.xlsx с тремя листами.

Первый лист Data_Analysis должен содержать основные данные по статьям со столбцами: Paper_ID, Title, Area, Citations, Relevance_Score. Включи только релевантные статьи (шумовую статью quant-ph не включай). Отсортируй строки по основному измерению (Area или Paper_ID) по возрастанию / в алфавитном порядке.

Второй лист Metrics должен суммировать ключевые показатели со столбцами Metric и Value: Total_Papers (общее число статей в Data_Analysis), Avg_Citations (среднее число цитирований по столбцу Citations), Top_Area (область с наибольшим числом цитирований) и Avg_Relevance (средний Relevance_Score). Значения должны быть согласованы с данными листа Data_Analysis.

Третий лист Recommendations должен перечислять приоритетные действия со столбцами Priority, Action, Area на основе анализа областей и данных-ориентира.

Также создай документ Word под названием Lit_Review_Analysis.docx с тремя разделами: краткое резюме (executive summary), ключевые выводы (key findings) и рекомендации (recommendations). Важно: при работе с инструментом Word передавай АБСОЛЮТНЫЙ путь к файлу .docx внутри рабочей директории агента.

Наконец, создай Google-таблицу под названием "Arxiv Litreview Tracker" с ключевыми данными — она должна содержать как минимум идентификаторы Paper_ID релевантных статей.
