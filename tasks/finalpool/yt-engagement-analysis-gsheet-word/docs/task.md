Наша команда исследования социальных сетей хочет получить полный анализ вовлечённости по всем доступным видео технического канала Fireship. Коэффициент вовлечённости (engagement rate) определяется как число лайков, делённое на число просмотров, и выражается в процентах.

Извлеките все доступные видео Fireship из базы данных видео. Для каждого видео вычислите коэффициент вовлечённости, разделив like_count на view_count и умножив на 100, округлив до трёх знаков после запятой. Также извлеките месяц публикации в формате YYYY-MM и преобразуйте длительность из секунд в целое число.

Создайте облачную таблицу под названием "Fireship Engagement Analysis". Таблица должна содержать два листа.

Первый лист должен называться Engagement_Data и содержать следующие столбцы в указанном порядке: Video_ID, Title, Published_Month, Duration_Sec, View_Count, Like_Count, Engagement_Rate_Pct. Каждая строка соответствует одному видео Fireship. Отсортируйте все строки по Engagement_Rate_Pct по убыванию (наибольшая вовлечённость сверху). Столбец Published_Month должен быть в формате YYYY-MM (например, 2024-07). Duration_Sec должен храниться как целое число. Engagement_Rate_Pct должен быть округлён до 3 знаков после запятой.

Второй лист должен называться Monthly_Summary и содержать следующие столбцы: Month, Video_Count, Avg_Engagement_Rate, Top_Video_Title. Каждая строка соответствует одному календарному месяцу. Month должен быть в формате YYYY-MM. Video_Count — это число видео Fireship, опубликованных в этом месяце. Avg_Engagement_Rate — это среднее значение Engagement_Rate_Pct по всем видео этого месяца, округлённое до 3 знаков после запятой. Top_Video_Title — это название видео с наивысшим коэффициентом вовлечённости в этом месяце. Отсортируйте строки по Month по возрастанию.

Подготовьте документ Word под названием Engagement_Analysis_Report.docx в вашем рабочем каталоге. Документ должен содержать следующие пять разделов с точными заголовками: Overview, Methodology, Key Findings, Monthly Trends и Conclusions. Раздел Key Findings должен называть и включать процент коэффициента вовлечённости для трёх наиболее вовлекающих видео Fireship. Раздел Methodology должен указывать, что коэффициент вовлечённости равен like_count, делённому на view_count и умноженному на 100.

Отправьте письмо на research@company.com с темой "Fireship YouTube Engagement Analysis". Тело письма должно суммировать общее число проанализированных видео и упоминать по имени хотя бы одно из наиболее вовлекающих видео.
