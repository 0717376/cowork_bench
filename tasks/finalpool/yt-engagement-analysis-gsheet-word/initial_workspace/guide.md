# Руководство по задаче: анализ вовлечённости Fireship

## Обзор
Проанализируйте коэффициенты вовлечённости по всем видео Fireship на YouTube и подготовьте исследовательский отчёт.

## Определение
Коэффициент вовлечённости (%) = (like_count / view_count) * 100, округлённый до 3 знаков после запятой

## Результаты
1. Облачная таблица с названием "Fireship Engagement Analysis" с двумя листами
2. Документ Word Engagement_Analysis_Report.docx
3. Письмо на research@company.com

## Лист 1: Engagement_Data
Столбцы: Video_ID, Title, Published_Month (YYYY-MM), Duration_Sec (целое число), View_Count, Like_Count, Engagement_Rate_Pct
Сортировка: по убыванию Engagement_Rate_Pct

## Лист 2: Monthly_Summary
Столбцы: Month (YYYY-MM), Video_Count, Avg_Engagement_Rate, Top_Video_Title
Сортировка: по возрастанию Month

## Разделы документа Word (требуются точные заголовки)
1. Overview
2. Methodology
3. Key Findings (включить топ-3 наиболее вовлекающих видео с коэффициентами)
4. Monthly Trends
5. Conclusions

## Письмо
- Кому: research@company.com
- Тема: "Fireship YouTube Engagement Analysis"
- Тело: общее число видео + хотя бы одно название наиболее вовлекающего видео
