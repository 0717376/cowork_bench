Вы — аналитик-исследователь, готовящий отчёт по сравнению акций из разных секторов. В вашей рабочей директории находятся файл Investment_Universe.md, описывающий пять акций и их секторы, и файл Analysis_Criteria.txt, перечисляющий финансовые метрики для сравнения.

Ваша задача — сравнить пять акций из разных секторов по ключевым финансовым метрикам. Выполните следующие шаги в точности.

Шаг 1: Получите актуальную информацию по акциям SBER.ME, GAZP.ME, MGNT.ME, MTSS.ME и TCSG.ME с помощью инструментов moex-finance. Соберите для каждой акции текущую цену, рыночную капитализацию, коэффициент P/E (trailing), 52-недельный максимум и 52-недельный минимум.

Шаг 2: Создайте файл Excel под названием Sector_Comparison.xlsx в вашей рабочей директории с двумя листами. Первый лист должен называться Metrics и содержать ровно пять строк (по одной на акцию) со следующими столбцами: Symbol, Company, Sector, Market_Cap_B (рыночная капитализация в миллиардах рублей, округлённая до двух знаков после запятой), Current_Price, PE_Ratio (trailing P/E, округлённый до двух знаков после запятой, или N/A, если значение недоступно), High_52w, Low_52w. Второй лист должен называться Sector_Summary и содержать ровно пять строк со столбцами: Sector, Symbol, Market_Cap_B, Price_Assessment (значение "Above_Avg", если текущая цена акции выше средней цены всех пяти акций, или "Below_Avg" в противном случае).

При создании файла Excel передавайте инструменту АБСОЛЮТНЫЙ путь к Sector_Comparison.xlsx внутри вашей рабочей директории.

Шаг 3: Создайте опрос в Google Forms с названием "Investment Preference Survey" и ровно четырьмя вопросами. Первый вопрос должен звучать "Какой сектор вы предпочитаете для долгосрочных инвестиций?" с вариантами ответа (множественный выбор): Financial Services, Energy, Consumer Defensive, Communication Services, Fintech. Второй вопрос должен звучать "Каков ваш инвестиционный горизонт?" с вариантами: Краткосрочный (менее 1 года), Среднесрочный (от 1 до 3 лет), Долгосрочный (более 3 лет). Третий вопрос должен звучать "Какова ваша терпимость к риску?" с вариантами: Низкая, Средняя, Высокая. Четвёртый вопрос должен звучать "Какая финансовая метрика для вас важнее всего?" с вариантами: Дивидендная доходность, P/E Ratio, Темп роста выручки, Рыночная капитализация.

Шаг 4: Отправьте письмо от research@fund.example.com на адрес investors@fund.example.com. Тема письма должна содержать "Sector Comparison". В теле письма на русском языке резюмируйте ключевые выводы: акцию с наибольшей рыночной капитализацией, акцию с наибольшим коэффициентом P/E, а также особенности 52-недельного диапазона как минимум для двух акций.

Выполните все четыре шага. Завершите работу, ответив без вызова каких-либо инструментов.
