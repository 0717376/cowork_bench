Я готовлю ежеквартальный обзор эффективности продуктов и мне нужно создать комплексный анализ, который объединяет наши фактические данные о продажах с информацией о рыночных трендах. Наш внутренний портал рыночной аналитики доступен по адресу http://localhost:30207 и содержит текущие темпы роста категорий и квартальные прогнозы.

Начните с чтения файла presentation_template.md в рабочей директории, который описывает структуру, ожидаемую для презентации обзора. Также изучите файл Q1_targets.xlsx, который содержит наши целевые показатели по выручке и количеству единиц для каждой категории продуктов в этом квартале. Затем прочитайте файл category_notes.txt с качественными заметками продуктовой команды по каждой категории и review_criteria.csv, где перечислены метрики, которые мы оцениваем для каждой категории.

Откройте дашборд рыночных трендов по адресу http://localhost:30207, чтобы собрать темпы роста год к году и данные об объёме рынка для каждой категории продуктов. Также откройте страницу квартальных прогнозов http://localhost:30207/trends.html, чтобы получить прогнозируемые темпы роста на оставшиеся кварталы 2026 года.

Далее выгрузите данные о товарах и заказах из нашего интернет-магазина InSales. Для каждой категории продуктов рассчитайте: общую выручку (сумму произведения quantity на price по всем позициям line_items во всех заказах), общее количество проданных единиц (сумму всех quantity), среднюю цену позиции (общая выручка, делённая на общее количество единиц, округлённая до 2 знаков после запятой) и средний пользовательский рейтинг отзывов для товаров этой категории (округлённый до 2 знаков после запятой). Категории товаров в нашем магазине: ТВ и домашний кинотеатр, Электроника, Аудио, Камеры, Часы и Бытовая техника.

Создайте Excel-книгу с именем Product_Review.xlsx в рабочей директории с тремя листами. Первый лист должен называться "Category Performance" со столбцами: Category, Revenue (округлённая до 2 знаков после запятой), Units_Sold, Avg_Price (округлённая до 2 знаков после запятой), Avg_Rating (среднее значение поля average_rating по всем товарам этой категории, округлённое до 2 знаков после запятой) и Revenue_Target (из файла Q1_targets.xlsx). Отсортируйте строки по Revenue по убыванию.

Второй лист должен называться "Market Comparison" со столбцами: Category, Store_Revenue, Market_Growth_Rate (процент роста год к году из рыночного дашборда), Market_Size_M (объём рынка в миллионах из дашборда) и Trend (метка тренда из дашборда, например Strong Growth, Moderate Growth, Slow Growth или Declining). Отсортируйте по Market_Growth_Rate по убыванию.

Третий лист должен называться "Opportunities" со столбцами: Category, Revenue, Market_Growth_Rate, Avg_Rating и Opportunity_Score. Opportunity_Score рассчитывается как Market_Growth_Rate, умноженный на Avg_Rating, округлённый до 2 знаков после запятой. Этот показатель помогает выявить категории, где высокий рост рынка совпадает с высоким качеством продукции. Отсортируйте по Opportunity_Score по убыванию.

Наконец, создайте презентацию PowerPoint с именем Q1_Product_Review.pptx в рабочей директории. Презентация должна содержать не менее 5 слайдов. Первый слайд — титульный со заголовком "Q1 2026 Product Performance Review" и датой 7 марта 2026 года в подзаголовке. Второй слайд — резюме для руководства с общей выручкой магазина по всем категориям, общим количеством проданных единиц и числом проанализированных категорий. Третий слайд — разбивка эффективности по категориям с выручкой и количеством единиц для каждой категории. Четвёртый слайд — рыночные тренды и то, как наши категории соотносятся с темпами роста рынка. Пятый слайд — возможности роста с выделением лучших категорий по показателю возможностей и рекомендациями по приоритетным направлениям.

Примечание: при создании файлов .xlsx и .pptx через соответствующие инструменты передавайте АБСОЛЮТНЫЙ путь к файлу внутри рабочей директории, так как эти инструменты не сохраняют текущую рабочую директорию.
