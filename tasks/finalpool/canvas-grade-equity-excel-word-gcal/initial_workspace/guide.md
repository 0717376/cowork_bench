# Руководство по анализу справедливости оценивания

## Сопоставление осенних курсов по годам

Осенние курсы определяются по суффиксу "J" в кодах курсов (например, AAA-2013J для осени 2013). Чтобы сравнить курсы год к году, сопоставляйте курсы с одинаковым базовым префиксом кода, различающиеся только частью с годом. Например, AAA-2013J и AAA-2014J — это один и тот же курс, прочитанный в двух подряд идущих осенних семестрах.

## Расчёт распределения оценок

Для каждого курса соберите все баллы по сданным студентами заданиям. Усредните баллы каждого студента, чтобы получить единый средний балл на студента. Затем рассчитайте статистику по курсу на основе этих средних значений по студентам.

Диапазоны оценок (grade bands):
- A: 90-100
- B: 70-89
- C: 50-69
- D: 30-49
- F: 0-29

## Метрики справедливости

Сравните средний балл за осень 2013 (Fall 2013) и осень 2014 (Fall 2014) для каждой сопоставленной пары курсов.

- Score Difference (разница баллов) = средний балл Fall 2014 минус средний балл Fall 2013
- Equity Status (статус справедливости) по модулю |Score Difference|:
  - Меньше 5 баллов: Acceptable
  - От 5 до 10 баллов: Concerning
  - Больше 10 баллов: Action Required

## Доля успешно сдавших (Pass Rate)

Pass Rate = процент студентов со средним баллом >= 50.
