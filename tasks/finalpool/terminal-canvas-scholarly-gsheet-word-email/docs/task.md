Я декан биологического факультета, и мне нужен комплексный анализ того, насколько хорошо научные исследования наших преподавателей согласуются с их преподавательской деятельностью. В вашей рабочей директории находится файл `faculty_roster.csv`, в котором перечислены наши преподаватели с указанием их кафедр (department), ID курсов, которые они ведут (course_id), их научных областей (research_area) и адреса электронной почты заведующего их кафедрой (dept_head_email). Также есть файл `integration_rubric.md`, описывающий методику подсчёта баллов, и файл `report_format.txt`, описывающий ожидаемую структуру итогового отчёта.

Начните с чтения реестра преподавателей (`faculty_roster.csv`), чтобы понять, кто и что преподаёт. Затем найдите каждый из перечисленных в реестре курсов в системе управления обучением (LMS), чтобы получить название курса и содержимое программы (syllabus). Напишите Python-скрипт, который извлекает тематические ключевые слова из HTML-содержимого программы каждого курса. Запустите скрипт и сохраните результат в файл `course_keywords.json` в рабочей директории.

Далее для каждого преподавателя найдите научные статьи, относящиеся к его научной области. Нужно найти статьи, соответствующие ключевым словам из заявленной специализации преподавателя. Напишите ещё один Python-скрипт, который вычисляет балл согласованности (alignment score) для каждого преподавателя, подсчитывая пересечения ключевых слов между тематическими ключевыми словами его курса и аннотациями (abstracts) найденных по его научной области статей. Балл согласованности — это процент ключевых слов курса, которые встречаются хотя бы в одной аннотации его научных статей. Запустите скрипт и сохраните результаты в файл `alignment_scores.json`.

Создайте облачную таблицу с названием "Research_Teaching_Alignment" с единственным листом "Alignment Matrix". Столбцы должны быть следующими: `Faculty Name`, `Department`, `Course Name`, `Research Area`, `Alignment Score (%)` и `Status`. В столбце `Status` должно быть указано "Aligned", если балл составляет 30% или выше, и "Review Needed", если ниже 30%.

Создайте документ Word с именем `Research_Teaching_Report.docx` в рабочей директории. Он должен следовать структуре, описанной в `report_format.txt`. Включите краткое резюме (Executive Summary / Сводка) с общим средним баллом согласованности и количеством преподавателей, нуждающихся в пересмотре. Затем — раздел поэтапного анализа по каждому преподавателю с указанием его имени, курса, балла согласованности и краткого примечания о том, согласован ли он (Aligned) или помечен для пересмотра (Review Needed). Затем — сводку по кафедрам со средним баллом согласованности для каждой кафедры. Наконец — раздел рекомендаций (Recommendations / Рекомендации), в котором поимённо указаны все преподаватели с баллом ниже 30% и предложено провести пересмотр учебной программы (curriculum review), чтобы лучше интегрировать их научные исследования. Важно: инструмент Word работает в собственной директории, поэтому при создании документа передавайте АБСОЛЮТНЫЙ путь к файлу `Research_Teaching_Report.docx` внутри рабочей директории.

Отправьте письмо на адрес dean@university.edu с темой "Research-Teaching Integration Summary", содержащее краткий обзор результатов: количество проанализированных преподавателей, средний балл согласованности и количество преподавателей, помеченных для пересмотра учебной программы.

Также отправьте отдельное письмо каждому уникальному заведующему кафедрой, указанному в реестре преподавателей. Каждое такое письмо должно иметь тему "Department Research-Teaching Alignment Update" и содержать название кафедры, имена и баллы согласованности преподавателей этой кафедры, а также информацию о том, помечен ли кто-либо для пересмотра. Кафедра Biochemistry адресуется на head_biochem@university.edu, кафедра Bioinformatics — на head_bioinfo@university.edu (не смешивайте преподавателей разных кафедр в одном письме).

Напишите Python-скрипт, который читает `alignment_scores.json` и вычисляет сводную статистику: общее количество преподавателей (`total_faculty`), средний балл согласованности (`average_alignment_score`), максимальный балл (`max_score`), минимальный балл (`min_score`) и количество преподавателей с баллом ниже 30%. Запустите его и сохраните результат в файл `summary_stats.json` в рабочей директории.

Наконец, создайте файл Excel с именем `Alignment_Summary.xlsx` в рабочей директории. На его листе должна быть строка заголовков и по одной строке данных на каждого преподавателя, повторяющая содержимое листа "Alignment Matrix": `Faculty Name`, `Department`, `Course Name`, `Research Area`, `Alignment Score (%)` и `Status`. При создании файла Excel также передавайте АБСОЛЮТНЫЙ путь к `Alignment_Summary.xlsx` внутри рабочей директории.
