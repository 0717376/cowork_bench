Формат отчёта об интеграции исследований и преподавания
=======================================================

Документ отчёта должен содержать следующие разделы по порядку:

1. Executive Summary (Сводка)
   - Общий средний балл согласованности по всем преподавателям
   - Общее число проанализированных преподавателей
   - Количество преподавателей, помеченных для пересмотра учебной программы (балл ниже 30%)

2. Per-Faculty Analysis (Анализ по преподавателям)
   - Для каждого преподавателя:
     - Имя преподавателя (Faculty name)
     - Кафедра (Department)
     - Преподаваемый курс (Course)
     - Научная область (Research area)
     - Балл согласованности (Alignment score, в процентах)
     - Статус (Status): "Aligned" или "Review Needed"

3. Department Summary (Сводка по кафедрам)
   - Для каждой кафедры:
     - Название кафедры
     - Средний балл согласованности
     - Число преподавателей на кафедре
     - Число помеченных для пересмотра

4. Recommendations (Рекомендации)
   - Поимённое указание преподавателей с баллом ниже 30%
   - Предложение по процессу пересмотра учебной программы (curriculum review)
   - Любые наблюдения на уровне кафедры
