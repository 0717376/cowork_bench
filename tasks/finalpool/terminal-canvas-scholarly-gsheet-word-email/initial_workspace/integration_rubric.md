# Рубрика подсчёта согласованности исследований и преподавания

## Расчёт балла согласованности (Alignment Score)

Балл согласованности показывает, насколько научные публикации преподавателя связаны с курсами, которые он ведёт.

1. Извлеките тематические ключевые слова из содержимого программы курса (минимум 3 символа, исключая распространённые стоп-слова и HTML-теги).
2. Для каждого преподавателя найдите его научные публикации на основе его научной области (research area).
3. Подсчитайте, сколько ключевых слов курса встречается хотя бы в одной аннотации статьи (сравнение без учёта регистра).
4. Вычислите балл согласованности как: (количество совпавших ключевых слов / общее число ключевых слов курса) * 100, округлённое до одного знака после запятой.

## Пороги (Thresholds)

- 30% и выше: "Aligned" (Согласовано) — исследования преподавателя достаточно интегрированы с преподаванием.
- Ниже 30%: "Review Needed" (Требуется пересмотр) — преподавателю следует обновить учебную программу, чтобы лучше отразить свои исследования, либо скорректировать научный фокус.

## Агрегация по кафедрам

Баллы на уровне кафедры — это среднее арифметическое баллов согласованности всех преподавателей данной кафедры, округлённое до одного знака после запятой.
