Мне нужна помощь с анализом ресурсов sf canvas. Есть внешние эталонные (benchmark) данные: загрузите их по адресу http://localhost:30329/api/data.json и извлеките нужные метрики. Это среднеотраслевые значения зарплат (`industry_avg`) по подразделениям (ключ `department`).

Затем выгрузите наши внутренние данные из корпоративного хранилища данных ClickHouse для сравнения. Внутренний показатель по каждому подразделению — это средняя зарплата (`SALARY`) сотрудников этого подразделения из таблицы `HR_ANALYTICS__PUBLIC__EMPLOYEES` (группировка по полю `DEPARTMENT`). Названия подразделений во внутреннем хранилище уже даны на русском языке (Инженерия, Финансы, Кадры, Операции, НИОКР, Продажи, Поддержка), а ключи во внешнем JSON — на английском (Engineering, Finance, HR, Operations, R&D, Sales, Support), поэтому сопоставьте их между собой.

С помощью терминала создайте и запустите Python-скрипт sf_canvas_resource_processor.py в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые вы создадите, выполнять анализ и выводить результат в sf_canvas_resource_results.json.

Создайте Excel-файл Canvas_Resource_Report.xlsx с тремя листами.

Первый лист Data_Analysis должен содержать основную сравнительную таблицу со следующими колонками:
- Department — название подразделения (основное измерение);
- Internal — наш внутренний показатель (средняя зарплата по подразделению);
- Benchmark — внешнее эталонное значение (`industry_avg` из data.json);
- Gap — разница: Gap = Internal - Benchmark (положительное значение — мы выше эталона, отрицательное — ниже).
Отсортируйте строки по алфавиту по колонке Department. Должна быть строка на каждое из 7 подразделений.

Второй лист Metrics должен содержать две колонки Metric и Value со сводными показателями (общее число подразделений, средние значения и ключевая статистика).

Третий лист Recommendations должен содержать колонки Priority, Action и Department и перечислять приоритетные действия на основе анализа разрывов: в первую очередь — подразделения с наибольшим отрицательным разрывом (Gap).
