Мне нужно проанализировать распределение заказов по статусам в нашем корпоративном хранилище данных ClickHouse (логическая база данных SALES_DW). В рабочей директории есть файл Order_KPIs.pdf с нашими целевыми показателями обработки заказов — используй его как справку.

Создай Excel-файл Sales_Order_Status.xlsx с двумя листами. Лист «Order Status» должен содержать колонки Status, Orders (количество заказов), Revenue (выручка в рублях, округлить до 2 знаков после запятой), Avg_Quantity (среднее количество, округлить до 1 знака), Avg_Discount (средняя скидка, округлить до 2 знаков) и Revenue_Pct (доля статуса в общей выручке, округлить до 1 знака). Отсортируй строки по Revenue по убыванию. Значения статусов бери как есть из базы данных, не переводи их.

Лист «Summary» должен содержать колонки Metric и Value со следующими метриками: Total_Orders, Total_Revenue, Delivered_Revenue (выручка только по доставленным заказам) и Cancel_Rate_Pct (доля отменённых заказов в процентах).

При записи .xlsx укажи абсолютный путь к файлу внутри рабочей директории.
