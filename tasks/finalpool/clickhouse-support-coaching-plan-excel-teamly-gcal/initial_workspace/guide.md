# Методология коучинга по эффективности агентов

## Расчёт соблюдения SLA

Соблюдение SLA измеряет, как часто агент отвечает на тикеты в пределах целевого времени ответа, заданного в таблице политик SLA. Целевое время ответа зависит от приоритета тикета:

- Critical: 1 час
- High: 4 часа
- Medium: 8 часов
- Low: 24 часа

Чтобы рассчитать соблюдение SLA для каждого агента:
1. Соедините (join) таблицу tickets с таблицей политик SLA по полю priority.
2. Для каждого тикета проверьте, что RESPONSE_TIME_HOURS меньше или равно RESPONSE_TARGET_HOURS из политик SLA.
3. Соблюдение SLA = (количество тикетов в пределах цели / общее количество тикетов) * 100, округлённое до 1 знака после запятой.

## Расчёт CSAT

Средний CSAT — это среднее арифметическое значения поля CUSTOMER_SATISFACTION (шкала 1-5) по всем тикетам, обработанным агентом, округлённое до 2 знаков после запятой.

## Идентификация агентов

Поле reporter в таблице tickets определяет, какой агент обработал тикет. Каждое уникальное значение reporter соответствует одному агенту.

## Назначение уровней эффективности (Performance Tiers)

Используйте определения уровней из PDF-документа Coaching Framework. Для того чтобы агент попал в определённый уровень, должны быть выполнены оба порога — и по соблюдению SLA, и по CSAT. Оценивайте уровни от высшего (Elite) к низшему (Needs Improvement) и присваивайте первый уровень, где выполнены оба критерия.

## Назначение зон развития (Focus Area)

Применяйте правила назначения зон развития из методологии в указанном порядке приоритета. Первое подходящее правило определяет основную зону развития (primary focus area). Второе подходящее правило определяет вторичную зону развития (secondary focus area). Если подходит меньше двух правил, заполните оставшиеся слоты значением Leadership Development.

Для проверки на высокий объём тикетов сравнивайте количество тикетов агента со средним количеством тикетов по всем агентам.
Для проверки на CSAT ниже среднего сравнивайте со средним CSAT по всем агентам.
