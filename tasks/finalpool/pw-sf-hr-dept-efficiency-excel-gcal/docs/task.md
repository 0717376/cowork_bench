Мне нужна помощь с анализом эффективности HR-подразделений. По адресу http://localhost:30334 опубликованы внешние отраслевые бенчмарки — зайди на эту страницу и извлеки все приведённые там метрики по подразделениям.

Затем выгрузи наши внутренние данные из корпоративного хранилища данных ClickHouse (логическая БД HR_ANALYTICS, таблицы HR_ANALYTICS__PUBLIC__DEPARTMENTS и HR_ANALYTICS__PUBLIC__EMPLOYEES) для сравнения. По каждому подразделению рассчитай численность сотрудников, среднюю зарплату, средний стаж и среднюю оценку производительности.

Через терминал создай и запусти Python-скрипт sf_efficiency_processor.py в рабочей директории. Скрипт должен читать собранные данные из JSON-файлов, которые ты создашь, выполнять анализ и сохранять результат в sf_efficiency_results.json (валидный JSON с разбором разрыва по каждому подразделению).

Создай Excel-файл Hr_Dept_Efficiency_Report.xlsx с тремя листами.

Лист Data_Analysis должен содержать основные данные сравнения со следующими колонками (заголовки колонок оставь на английском):
- Department — название подразделения;
- Employee_Count — численность сотрудников;
- Our_Avg_Salary — наша средняя зарплата (в рублях);
- Avg_Experience — средний стаж;
- Avg_Performance — средняя оценка производительности;
- Industry_Benchmark — отраслевой эталон зарплаты;
- Salary_Gap — разрыв, рассчитанный как round(Our_Avg_Salary - Industry_Benchmark, 2). Положительный разрыв означает, что мы превышаем эталон, отрицательный — отстаём.

Отсортируй строки листа Data_Analysis в алфавитном порядке по столбцу Department.

Лист Metrics должен иметь две колонки Metric и Value и обобщать ключевую статистику. Обязательно включи строки: Total_Departments (общее число подразделений), Total_Employees (суммарная численность), Avg_Salary_Gap (средний разрыв), Departments_Above (число подразделений с положительным Salary_Gap) и Departments_Below (число подразделений с отрицательным Salary_Gap).

Лист Recommendations должен перечислять приоритетные действия на основе анализа разрывов и содержать колонки Priority, Action, Department, Impact.
