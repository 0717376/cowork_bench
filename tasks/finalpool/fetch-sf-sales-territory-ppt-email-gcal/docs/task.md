Вы — региональный директор по продажам, готовящий обзор эффективности территорий для руководства компании. CRM-панель компании по адресу http://localhost:30215/api/territory_data.json предоставляет квоты по территориям, распределение торговых представителей и данные по воронке продаж для каждого из пяти регионов.

Сначала прочитайте файл Territory_Review_Guide.pdf в вашем рабочем каталоге, чтобы понять методологию отчётности и то, как рассчитываются выполнение квоты (quota attainment) и покрытие воронкой (pipeline coverage).

Затем получите данные о квотах и воронке продаж по территориям из CRM-панели через API. Затем выполните запрос к хранилищу данных ClickHouse (схема SALES_DW), чтобы получить фактические результаты по выручке. Вам нужны суммарная выручка и количество заказов с разбивкой по регионам (Азиатско-Тихоокеанский регион, Европа, Латинская Америка, Ближний Восток, Северная Америка) и по сегментам клиентов (Корпоративный, Малый и средний бизнес, Частные клиенты, Государственный).

ВАЖНО: названия регионов и сегментов записывайте в презентации ровно в том виде, в каком они возвращаются из базы данных ClickHouse и из CRM API — не переводите и не изменяйте эти строковые значения, чтобы ключи объединения данных (выручка из БД ↔ квота/воронка из CRM) совпадали.

Создайте презентацию PowerPoint с именем файла Territory_Scorecard.pptx в вашем рабочем каталоге. При работе с инструментом pptx передавайте АБСОЛЮТНЫЙ путь к файлу Territory_Scorecard.pptx внутри рабочего каталога. Презентация должна содержать следующие слайды.

Слайд 1 — титульный слайд с заголовком «Territory Performance Review - Q1 2026» и подзаголовком «Regional Sales Director Report».

Слайд 2 с заголовком «Revenue by Region» (Выручка по регионам) должен содержать сводку, показывающую для каждого региона суммарную выручку (сумма по всем сегментам), назначенную квоту из данных CRM и процент выполнения квоты (фактическая выручка делённая на квоту и умноженная на 100, округлённая до 1 знака после запятой).

Слайд 3 с заголовком «Segment Performance» (Эффективность сегментов) должен показывать для каждого сегмента суммарную выручку по всем регионам, долю этого сегмента в общей выручке компании (округлённую до 1 знака после запятой) и регион-лидер для данного сегмента.

Слайд 4 с заголовком «Pipeline Coverage Analysis» (Анализ покрытия воронкой) должен показывать для каждого региона значение воронки продаж из данных CRM, фактическую выручку и коэффициент покрытия воронкой (воронка делённая на выручку, округлённая до 2 знаков после запятой).

Слайд 5 с заголовком «Recommendations» (Рекомендации) должен включать маркированный список с указанием того, какие регионы перевыполнили квоту (выполнение выше 100 процентов), какие отстали, а также общий процент выполнения квоты по компании.
