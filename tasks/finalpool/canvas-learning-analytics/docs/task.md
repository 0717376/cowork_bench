Проректор по учебной работе просит подготовить комплексную панель учебной аналитики, сравнивающую успеваемость по курсам с институциональными контрольными показателями (бенчмарками). Начните с посещения портала аналитических бенчмарков по адресу http://localhost:30219, чтобы получить целевые значения ключевых показателей эффективности (KPI). Также перейдите по ссылке на страницу с подробными бенчмарками, чтобы узнать определения уровней успеваемости.

Затем сделайте запрос к системе управления обучением (LMS) и рассчитайте следующие метрики для каждого из 22 курсов. Коэффициент завершения (Completion Rate) — это процент проверенных работ от общего числа сданных работ. Средний балл (Average Grade) — это среднее текущее значение оценки по всем зачисленным студентам. Доля просроченных работ (Late Submission Rate) — это процент работ, помеченных как просроченные, от общего числа сданных работ. Количество обсуждений (Discussion Participation Count) — это число тем для обсуждения в курсе.

Изучите документ с методологией расчёта KPI (analytics framework) в рабочей директории, а также файл-шаблон панели (dashboard template) для понимания структуры сводной страницы.

Создайте файл Excel с именем Learning_Analytics.xlsx с тремя листами. Первый лист с именем "Course KPIs" должен содержать столбцы Course, Completion_Rate (процент), Avg_Grade (числовой балл), Late_Rate (процент), Discussion_Count и Meets_Benchmark (значение Yes только если коэффициент завершения не менее 75 процентов И средний балл не менее 70 И доля просроченных работ не более 15 процентов). Отсортируйте по названию курса в алфавитном порядке. Второй лист с именем "Trends" должен содержать столбцы Course, Category (Completion, Grade или Late Submissions) и Performance (Above/Below для завершения и оценки, Acceptable/High для просроченных работ). На каждый курс должно приходиться три строки — по одной на каждую категорию. Третий лист с именем "Summary" должен содержать строки Total_Courses, Meets_All_Benchmarks (количество курсов, удовлетворяющих всем целевым показателям), Below_Benchmark (количество курсов, не достигших хотя бы одного целевого показателя) и Benchmark_Rate (процент курсов, удовлетворяющих всем показателям).

В завершение создайте в системе заметок Teamly страницу с заголовком "Learning Analytics Dashboard" с разделами: краткая сводка общей результативности по бенчмаркам, перечень курсов, удовлетворяющих всем целевым показателям, выделение курсов, требующих внимания, и рекомендации по улучшению.
