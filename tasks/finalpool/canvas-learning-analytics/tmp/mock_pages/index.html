Портал бенчмарков учебной аналитики

Ключевые показатели эффективности для оценки качества курсов.

Целевые показатели вовлечённости

KPI Целевое значение Описание
Коэффициент завершения (Completion Rate) 75% Процент проверенных работ от общего числа сданных. Курсы ниже этого порога требуют вмешательства.
Средний балл (Average Grade) 70 Среднее текущее значение оценки по зачисленным студентам. Цель обеспечивает достаточные образовательные результаты.
Порог просроченных работ (Late Submission Threshold) 15% Максимально допустимый процент просроченных работ. Курсы выше этого значения требуют лучшего управления сроками.
Участие в обсуждениях (Discussion Participation) минимум 3 темы Минимальное число тем для обсуждения на курс, обеспечивающее вовлечённость студентов.

Курс удовлетворяет общему бенчмарку только если выполнены ВСЕ целевые показатели KPI.

Подробные целевые показатели по категориям см. на странице Подробные бенчмарки.
