Мне нужно проанализировать, как каждое подразделение исполняет выделенный бюджет. В рабочей директории лежит PDF-файл Budget_Targets.pdf, в котором указан утверждённый годовой бюджет по каждому подразделению. Сначала прочитай его.

Затем выгрузи фактические данные по сотрудникам из нашего хранилища данных HR-аналитики ClickHouse (логическая база данных HR_ANALYTICS). По каждому подразделению нужно получить плановую численность (Planned_Headcount) из таблицы подразделений, фактическое число сотрудников, среднюю зарплату и суммарные затраты на зарплату. Также рассчитай Budget_Utilization_Pct — суммарные затраты на зарплату в процентах от бюджета подразделения, округлённые до 1 знака после запятой.

Создай Excel-файл Department_Budget_Report.xlsx с двумя листами. Первый лист «Budget Analysis» должен содержать колонки Department, Budget, Planned_Headcount, Actual_Headcount, Avg_Salary (округлить до 2 знаков), Total_Salary_Cost (округлить до 2 знаков) и Budget_Utilization_Pct (округлить до 1 знака). Отсортируй по Department по алфавиту.

Лист «Summary» должен содержать колонки Metric и Value со следующими метриками: Total_Budget, Total_Salary_Cost, Avg_Budget_Utilization по всем подразделениям (округлить до 1 знака) и Over_Budget_Depts — число подразделений, у которых использование бюджета превышает 100 процентов.

При записи .xlsx укажи абсолютный путь к файлу внутри рабочей директории.
