Вы — координатор корпоративного велнеса, отвечающий за запуск программы обедов для сотрудников. Ваша цель: создать опрос пищевых предпочтений, подобрать блюда, соответствующие распространённым предпочтениям сотрудников, спланировать недельное меню, наполнить базу знаний рецептов и сообщить о программе персоналу.

Начните с прочтения файла Cafeteria_Budget.pdf в вашей рабочей директории, где описаны бюджет на одно блюдо и ожидаемое число сотрудников. Также изучите файл dietary_requirements.json, в котором перечислены распространённые пищевые ограничения и аллергены для коллектива, категории блюд и стоимость порции по каждой категории.

Создайте опрос пищевых предпочтений на платформе форм (forms). Опрос должен называться «Опрос предпочтений по программе обедов» (Employee Lunch Program Preferences) и включать следующие пять вопросов:
- вопрос с выбором одного варианта о предпочитаемой категории блюд с вариантами: горячее, салат, гарнир, суп, закуска;
- вопрос с выбором одного варианта об остроте с вариантами: Не острое, Средне, Остро;
- вопрос о пищевых ограничениях с вариантами: Вегетарианец, Без свинины, Без морепродуктов, Без глютена, Без ограничений;
- вопрос-шкала от 1 до 5 о том, насколько для сотрудника важно разнообразие в обеденном меню;
- текстовый вопрос для любых дополнительных пожеланий по еде или аллергий.

Примечание: платформа форм поддерживает вопросы с выбором одного варианта (multiple choice) и текстовые вопросы. Вопрос об ограничениях оформите как вопрос с выбором/перечислением вариантов, вопрос-шкалу 1–5 — как вопрос с выбором вариантов 1, 2, 3, 4, 5, а вопрос о дополнительных пожеланиях — как текстовый.

Обратитесь к базе рецептов kulinar, чтобы найти блюда разных категорий. Найдите не менее двух блюд из категории «горячее», двух из «салат», одного «гарнир», одного «суп» и одного «закуска». Для каждого полученного рецепта запишите название (recipe_name), категорию (category), уровень сложности (difficulty) и список ингредиентов.

Напишите Python-скрипт menu_planner.py в вашей рабочей директории и выполните его средствами командной строки. Скрипт должен выбрать пять блюд для недельного меню с понедельника по пятницу (Monday–Friday), балансируя категории (никакие два соседних дня не должны иметь одну и ту же категорию) и сохраняя сложность не выше 4 для удобства приготовления в столовой. Скрипт должен оценить стоимость порции по категориям из dietary_requirements.json (горячее — 8 руб., закуска — 7 руб., гарнир — 6 руб., салат — 5 руб., суп — 4 руб.) и рассчитать суммарную недельную стоимость из расчёта 50 порций в день.

Создайте Excel-книгу Meal_Program_Plan.xlsx в вашей рабочей директории с четырьмя листами. Передавайте инструменту Excel абсолютный путь к файлу внутри рабочей директории.

Первый лист назовите Survey_Questions со столбцами question_num, text и type. Включите по одной строке на каждый из пяти вопросов опроса.

Второй лист назовите Recipe_Selection со столбцами recipe_name, category и difficulty. Включите по одной строке на каждый рецепт, полученный из базы рецептов, всего не менее семи рецептов.

Третий лист назовите Weekly_Menu со столбцами day, lunch_recipe и estimated_cost. Включите пять строк, по одной на каждый будний день с понедельника по пятницу (Monday–Friday), с назначенным блюдом и оценочной стоимостью порции. Значения столбца day записывайте по-английски: Monday, Tuesday, Wednesday, Thursday, Friday.

Четвёртый лист назовите Program_Summary со столбцами metric и value. Включите строки: total_weekly_cost (сумма дневных стоимостей, умноженная на 50 порций), avg_daily_cost_per_person, survey_question_count (5), recipes_evaluated (всего полученных рецептов) и menu_days (5).

Создайте в системе командной вики (teamly) страницу «База знаний рецептов» (Recipe Knowledge Base). На странице для каждого блюда из недельного меню укажите его название (Recipe Name), категорию (Category), сложность (Difficulty) и пригодность (Suitable For — например: обед, ужин, быстрое блюдо). Добавьте по одной записи на каждое блюдо, включённое в недельное меню.

Отправьте письмо на адрес all_staff@company.com с темой «New Employee Lunch Program - Take Our Survey», описав новую программу обедов, упомянув, что недельное меню составлено с разнообразием блюд, и призвав сотрудников заполнить опрос пищевых предпочтений, чтобы помочь улучшить будущее меню. Тему письма оставьте на английском языке ровно как указано; текст письма пишите по-русски.
