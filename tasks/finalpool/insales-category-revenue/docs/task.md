Мне нужно проанализировать выручку по категориям товаров в нашем магазине InSales. В рабочей директории есть файл Category_Targets.pdf с целевыми показателями выручки по категориям — он приложен исключительно для справки.

Извлеки данные о товарах и категориях через инструменты InSales и рассчитай оценочную выручку для каждой категории как цену, умноженную на количество проданных единиц. Если товар относится к нескольким категориям, он учитывается в каждой категории, к которой он относится.

Создай Excel-файл с именем WC_Category_Revenue.xlsx с двумя листами. Лист "Category Revenue" должен содержать колонки: Category (название категории), Products (количество товаров), Avg_Price (средняя цена, округлённая до 2 знаков), Units_Sold (суммарное количество проданных единиц) и Est_Revenue (оценочная выручка, округлённая до 2 знаков). Отсортируй строки по Est_Revenue по убыванию.

Лист "Summary" должен содержать колонки Metric и Value со строками Total_Categories, Total_Est_Revenue и Top_Category (название категории-лидера).
